Du kan använda Azure Cloud Shell för att köra de flesta PowerShell-cmdlets och CLI-kommandon, i stället för att installera Azure PowerShell eller CLI lokalt. Azure Cloud Shell är ett kostnadsfritt interaktivt skal som har vanliga Azure-verktyg förinstallerade och är konfigurerade att användas med ditt konto.
Om du vill köra en kod som finns i den här artikeln på Azure Cloud Shell öppnar du en Cloud Shell-session, använder knappen **Kopiera** på ett kodblock för att kopiera koden och klistrar in den i Cloud Shell-sessionen med __Ctrl+Skift+V__ på Windows och Linux eller __Cmd+Skift+V__ på macOS. Inklissad text körs inte automatiskt, tryck på **Retur** för att köra kod.

Det finns flera olika sätt att starta Cloud Shell:

|  |   |
|-----------------------------------------------|---|
| Klicka på **Prova** i det övre högra hörnet av ett kodblock. | ![Cloud Shell i den här artikeln](./media/vpn-gateway-cloud-shell-powershell/cloud-shell-powershell-try-it.png) |
| Öppna Cloud Shell i din webbläsare. | [![https://shell.azure.com/powershell](./media/vpn-gateway-cloud-shell-powershell/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Klicka på knappen **Cloud Shell** längst upp till höger i Azure-portalen. | [![Cloud Shell i portalen](./media/vpn-gateway-cloud-shell-powershell/cloud-shell-menu.png)](https://portal.azure.com) |
