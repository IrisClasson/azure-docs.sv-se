## <a name="prerequisite-install-the-azure-cli"></a>Förutsättning: Installera Azure CLI

Att utföra stegen i den här artikeln [installera Azure CLI](/cli/azure/install-azure-cli) och [logga in på Azure](/cli/azure/authenticate-azure-cli). 

> [!NOTE]
> Om du inte har ett Azure-konto, behöver du en. Registrera dig för en [kostnadsfri utvärderingsversion här](../articles/active-directory/fundamentals/sign-up-organization.md).
> Dessutom att följa instruktionerna du måste ha antingen [jq](https://stedolan.github.io/jq/) eller några andra JSON parsning av verktyget eller ett bibliotek som är installerad.
