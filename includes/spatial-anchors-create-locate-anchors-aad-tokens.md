### <a name="azure-active-directory-authentication"></a>Azure Active Directory-autentisering

Azure Spatial Anchors gör det också möjligt för program att autentisera med användare Azure AD (Active Directory) token. Du kan till exempel använda Azure AD-token för att integrera med Azure Spatial Anchors. Om ett företag underhåller användare i Azure AD kan du ange en Azure AD-token för användare i Azure Spatial Anchors SDK. På så sätt kan du autentisera direkt till Azure Spatial Anchors-tjänsten för ett konto som är en del av samma Azure AD-klientorganisation.
