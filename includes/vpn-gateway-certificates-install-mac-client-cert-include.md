När du installerar ett klientcertifikat behöver du lösenordet som skapades när klientcertifikatet exporterades.

1. Leta reda på PFX-certifikatfilen och kopiera den till din Mac. Du kan hämta certifikatet till Mac på flera sätt, till exempel kan du skicka certifikatfilen via e-post.
2. När certifikatet har kopierats till Mac dubbelklickar du på certifikatet för att öppna rutan Lägg till **certifikat,** klicka på **Lägg** till för att påbörja installationen.

   ![Lägga till certifikat](./media/vpn-gateway-certificates-install-mac-client-cert-include/addcert.png)
3. Ange det lösenord som du skapade när klientcertifikatet exporterades. Lösenordet skyddar certifikatets privata nyckel. Klicka på **OK** för att slutföra installationen.

   ![lösenord](./media/vpn-gateway-certificates-install-mac-client-cert-include/password.png)
