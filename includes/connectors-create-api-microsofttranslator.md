## <a name="prerequisites"></a>Krav

* Ett [MicrosoftTranslator-konto](https://www.microsoft.com/translator)  

Innan du kan använda ditt MicrosoftTranslator-konto i en Logic-app måste du auktorisera Logic-appen för att ansluta till ditt MicrosoftTranslator-konto. Lyckligtvis kan du enkelt göra detta inifrån logikappen på Azure Portal.  

Här är stegen för att auktorisera din Logic-app för att ansluta till ditt MicrosoftTranslator-konto:  

1. Om du vill skapa en anslutning till MicrosoftTranslator väljer du **Visa Microsoft-hanterade API:er** i listrutan Visa Microsoft i listrutan och anger sedan *MicrosoftTranslator* i sökrutan. Välj den utlösare eller åtgärd som du vill använda:  
   ![Steget för att skapa Anslutning i MicrosoftTranslator](./media/connectors-create-api-microsofttranslator/microsofttranslator-1.png)  
2. Om du inte har skapat några anslutningar till MicrosoftTranslator tidigare uppmanas du att ange dina MicrosoftTranslator-autentiseringsuppgifter. Dessa autentiseringsuppgifter används för att auktorisera din Logic-app att ansluta till och komma åt data för ditt MicrosoftTranslator-konto:  
   ![Steget för att skapa Anslutning i MicrosoftTranslator](./media/connectors-create-api-microsofttranslator/microsofttranslator-2.png)  
3. Observera att anslutningen har skapats och du är nu fri att fortsätta med de andra stegen i logikappen:  
   ![Steget för att skapa Anslutning i MicrosoftTranslator](./media/connectors-create-api-microsofttranslator/microsofttranslator-3.png)
