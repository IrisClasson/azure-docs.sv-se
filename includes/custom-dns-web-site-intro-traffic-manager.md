När du använder en Microsoft Azure Traffic Manager kan belastningsutjämna trafik till din Azure-webbplats som webbplatsen kan sedan användas med den  **\*. trafficmanager.net** domännamn tilldelning av Azure. Du kan även associera ett anpassat domännamn, till exempel www.contoso.com, med din webbplats för att tillhandahålla ett mer beskrivande domännamn för dina användare.
