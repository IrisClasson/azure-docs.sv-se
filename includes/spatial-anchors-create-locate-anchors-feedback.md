## <a name="providing-feedback-to-the-user"></a>Ge feedback till användaren

Du kan skriva koden för att hantera sessionshändelsen som har uppdaterats. Göra det, kan du:

- Ge feedback till användaren som enheten flyttar och sessionen uppdaterar sin miljö kunskap.
- Fastställa vid vilken tidpunkt det har tillräckligt spåras rumsliga data för att skapa eller hitta spatial fästpunkter – vi Lär dig mer i ett senare skede.
