#### <a name="to-exit-maintenance-mode"></a>Så här avslutar du underhållsläge
1. Vid kommandotolken skriver du:
   
     `Exit-HcsMaintenanceMode`
2. Ett varningsmeddelande och ett bekräftelsemeddelande visas. Skriv **Y** för att avsluta underhållsläget.
   
    Båda styrenheterna startas om. När omstarten är klar visas ett annat meddelande som anger att enheten är i normalläge.
