> [!NOTE]
>
> Den här artikeln har uppdaterats för att använda den nya [Az för Azure PowerShell-modulen](/powershell/azure/new-azureps-module-az). AzureRM kommer att fortsätta att få kritiska bugfix-uppdateringar, men nya funktioner kommer enbart till Az-modulen.
> * Az-modulen finns redan på Cloud Shell med AzureRM-kompatibilitet aktiverad.
> * För att installera modulen Az lokalt, se [installera Azure PowerShell](/powershell/azure/install-az-ps).
> * Om du installerar Az-modulen lokalt kan du aktivera AzureRM-kompatibilitet genom att köra `Enable-AzureRmAlias`.
