> [!NOTE]
>
> Den här artikeln har nyligen uppdaterats till att använda kommandon från den nya Azure PowerShell Az-modulen. Den här modulen är den nya PowerShell-modulen för Azure, som är kompatibel med PowerShell-versionerna 5.x och 6. AzureRM kommer att fortsätta att få kritiska bugfix-uppdateringar, men nya funktioner kommer enbart till Az-modulen.
>
> * Läs mer om Az-modulen och fortsatt stöd för AzureRM-modulen i [Introduktion till den nya Azure PowerShell Az-modulen](/powershell/azure/new-azureps-module-az).
> * Az-modulen finns redan tillgänglig i Cloud Shell med AzureRM-kompatibilitet automatiskt aktiverat.
>   Anvisningar om hur du installerar Az-modulen finns i [Installera Azure PowerShell](/powershell/azure/install-az-ps).
> * Om du vill aktivera AzureRM-kompatibilitetskommandon så kör du `Enable-AzureRmAlias` efter installationen.
>   Mer information om kompatibilitet finns i [Migrera från Azure PowerShell AzureRM till Az](/powershell/azure/migrate-from-azurerm-to-az).
