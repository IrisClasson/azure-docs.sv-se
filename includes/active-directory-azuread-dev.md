> [!IMPORTANT]
> [Microsoft identity platform (v2.0)](/azure/active-directory/develop/v2-overview) är en utveckling av utvecklarplattformen för Azure Active Directory (Azure AD) (v1.0). Den hjälper utvecklare att bygga program som loggar in alla Microsoft-identiteter och hämtar tokens för att anropa Microsoft API:er som Microsoft Graph eller API:er som utvecklare har byggt.
> Det här innehållet är för den äldre Azure AD v1.0-slutpunkten. Vi rekommenderar att du använder v2.0-slutpunkten för nya projekt. Mer information finns i [Varför uppdatera till Microsoft identity platform (v2.0)?](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) samt Begränsningar för [Microsofts identitetsplattform](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations).
