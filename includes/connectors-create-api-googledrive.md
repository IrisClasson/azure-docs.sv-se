## <a name="prerequisites"></a>Krav

* Ett [GoogleDrive](https://www.google.com/drive/) -konto  

Innan du kan använda ditt GoogleDrive-konto i en Logic app måste du godkänna att Logic-appen ansluter till ditt GoogleDrive-konto. Lyckligt vis kan du enkelt göra det i din Logic-app på Azure-portalen.  

Här följer stegen för att godkänna din Logic app för att ansluta till ditt GoogleDrive-konto:  

1. För att skapa en anslutning till GoogleDrive, i Logic App Designer, väljer du **Visa Microsoft-hanterade API: er** i list rutan och anger sedan *GoogleDrive* i sökrutan. Välj den utlösare eller åtgärd som du vill använda:  
   steg för att skapa ![GoogleDrive-anslutning](./media/connectors-create-api-googledrive/googledrive-1.png)  
2. Om du inte har skapat några anslutningar till GoogleDrive tidigare uppmanas du att ange dina GoogleDrive-autentiseringsuppgifter. Dessa autentiseringsuppgifter används för att auktorisera din Logi Kap par för att ansluta till och komma åt ditt GoogleDrive-kontos data:  
   ![Steg för skapande av GoogleDrive-anslutning](./media/connectors-create-api-googledrive/googledrive-2.png)  
3. Ange din GoogleDrive-e-post adress:  
   ![Steg för skapande av GoogleDrive-anslutning](./media/connectors-create-api-googledrive/googledrive-3.png)  
4. Ange ditt GoogleDrive-lösenord för att auktorisera din Logic app:  
   ![Steg för skapande av GoogleDrive-anslutning](./media/connectors-create-api-googledrive/googledrive-4.png)
5. Tillåt anslutningen till GoogleDrive  
   ![Steg för skapande av GoogleDrive-anslutning](./media/connectors-create-api-googledrive/googledrive-5.png)  
6. Observera att anslutningen har skapats och att du nu är kostnads fri att fortsätta med de andra stegen i din Logic app:  
   ![Steg för skapande av GoogleDrive-anslutning](./media/connectors-create-api-googledrive/googledrive-6.png)
