1. I bladet inställningar för resurs, resursgrupp eller prenumeration som du vill låsa väljer **låser**.

      ![Välj Lås](./media/resource-manager-lock-resources/select-lock.png)
2. Om du vill lägga till ett lås, Välj **Lägg till**. Om du vill skapa ett lås på överordnad nivå väljer du överordnat. Den markerade resursen ärver låset från överordnat. Exempelvis kan du låsa resursgruppen du vill tillämpa ett lås på alla dess resurser.

      ![Lägg till Lås](./media/resource-manager-lock-resources/add-lock.png) 
3. Ge låset ett namn och på låsskärmen. Alternativt kan du lägga till uppgifter som beskriver låset.

      ![set-Lås](./media/resource-manager-lock-resources/set-lock.png) 
4. Välj ellipsen för att ta bort låset och **ta bort** bland de tillgängliga alternativen.

      ![Ta bort lås](./media/resource-manager-lock-resources/delete-lock.png)
