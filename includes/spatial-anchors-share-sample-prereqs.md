## <a name="prerequisites"></a>Krav

För att kunna följa den här självstudien måste du ha:

* Läs igenom [översikten Azure Spatial Anchors](../articles/spatial-anchors/overview.md).
* Avslutade en av de [5-minuters snabbstarter](../articles/spatial-anchors/index.yml).
* Grundläggande kunskaper om C# och Unity.
* Grundläggande kunskaper om <a href="https://developers.google.com/ar/discover/" target="_blank">ARCore</a> om du vill använda Android eller <a href="https://developer.apple.com/arkit/" target="_blank">ARKit</a> om du vill använda iOS.
* En Windows-dator där <a href="https://www.visualstudio.com/downloads/" target="_blank">Visual Studio 2017</a> eller senare installeras med **arbetsbelastningen för ASP.NET och webbutveckling.**
* [.NET Core 2.2 SDK](https://dotnet.microsoft.com/download).
* En eller flera enheter (iOS eller Android) där du kan distribuera och köra en app.
  * Om du använder Android måste du ha:
    * <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3</a> eller senare, <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2019.1 eller 2019.2</a>och <a href="https://git-scm.com/download/win" target="_blank">Git för Windows</a> som är installerat på din Windows-dator.
    * En <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">utvecklare-aktiverad</a> och <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">ARCore-kompatibel Android-enhet.</a>
  * Om du använder iOS måste du ha:
    * En macOS-dator där <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Xcode 10</a> eller senare, <a href="https://cocoapods.org" target="_blank">CocoaPods</a>och <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2019.1 eller 2019.2</a> är installerade.
    * En utvecklaraktiverad <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">ARKit-kompatibel iOS-enhet.</a>
    * Git installeras via Homebrew. Ange följande kommando på en enda `/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"` rad `brew install git`i terminalen: Kör sedan .
