> [!TIP]
> 
> Hantera Azure Blob-lagringsresurser med Azure Storage Explorer. 
> [Azure Storage Explorer](https://azure.microsoft.com/features/storage-explorer/) är en kostnadsfri, fristående app från Microsoft som gör att du kan [hantera Azure Blob-lagringsresurser](../articles/vs-azure-tools-storage-explorer-blobs.md). Med Hjälp av Azure Storage Explorer kan du visuellt skapa, läsa, uppdatera och ta bort blob-behållare och blobbar samt hantera åtkomst till dina blobbar-behållare och blobbar.
