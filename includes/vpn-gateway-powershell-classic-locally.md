När du arbetar med den klassiska distributionsmodellen kan du inte använda Azure Cloud Shell. I stället måste du installera den senaste versionen av Azure Service Management (SM) PowerShell-cmdlets lokalt på din dator. Dessa cmdlets skiljer sig från AzureRM- eller Az-cmdlets. Så här installerar du SM-cmdlets finns i [Installera servicehanterings-cmdlets](/powershell/azure/servicemanagement/install-azure-ps). Mer information om Azure PowerShell i allmänhet finns i [Azure PowerShell-dokumentationen](/powershell/azure/).
