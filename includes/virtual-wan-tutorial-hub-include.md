Ett nav innehåller gatewayen. När hubben har skapats debiteras du för hubben, även om du inte kopplar några platser. Det tar 30 minuter att skapa navet och gatewayen.

1. Leta reda på det virtuella WAN som du skapade. Klicka på **Hubbar**under arkitekturavsnittet Virtuellt WAN på sidan **Virtuell WAN.**
2. På sidan Hubbar klickar du på **+Ny hubb** för att öppna sidan **Skapa virtuellt nav.**
3. Fyll i följande fält på sidan **Skapa virtuellt nav:**

   * Location
   * Namn
   * Privat adressutrymme för hubb

Klicka på **Bekräfta** för att skapa hubben. Klicka på **Uppdatera** om du vill visa navet på **hubbarsidan.**
