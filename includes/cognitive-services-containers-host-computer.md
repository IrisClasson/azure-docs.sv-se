Den **värden** är en x64-baserad dator som kör docker-behållaren. Det kan vara en dator i din lokala enhet eller en docker som är värd för tjänsten i Azure, inklusive:

* [Azure Kubernetes Service](../articles/aks/index.yml)
* [Azure Container Instances](../articles/container-instances/index.yml)
* [Kubernetes](https://kubernetes.io/) kluster som distribueras till [Azure Stack](/azure-stack/operator). Mer information finns i [distribuera Kubernetes i Azure Stack](/azure-stack/user/azure-stack-solution-template-kubernetes-deploy).
