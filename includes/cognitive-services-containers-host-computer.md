* [Azure Kubernetes Service](../articles/aks/index.yml).
* [Azure Container Instances](../articles/container-instances/index.yml).
* En [Kubernetes](https://kubernetes.io/) kluster som distribueras till [Azure Stack](/azure-stack/operator). Mer information finns i [distribuera Kubernetes i Azure Stack](/azure-stack/user/azure-stack-solution-template-kubernetes-deploy).
