#### <a name="to-install-regular-updates-via-the-azure-classic-portal"></a>Installera regelbundna uppdateringar via den klassiska Azure-portalen
1. På den **enheter** väljer du den enhet som du vill installera uppdateringar.
2. Gå till **enheter** > **Underhåll** och bläddra ned till **programuppdateringar**.
3. Om du vill söka efter uppdateringar, klickar du på **Kontrollera uppdateringar** längst ned på sidan.
4. Du ser ett meddelande om programuppdateringar är tillgängliga. Klicka på **installera uppdateringar** att börja uppdatera enheten.
   
    Du kommer att meddelas när uppdateringen har installerats.
