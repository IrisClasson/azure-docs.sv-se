Utför följande steg i Azure Portal för att visa en enhets händelse.

1. Gå till Data Box Edge/Data Box Gateway-resursen i Azure Portal och gå sedan till **övervakning > enhets händelser**.
2. Välj en händelse och Visa aviserings informationen. Åtgärda varnings villkoret genom att vidta lämpliga åtgärder.

    ![Välj händelse och Visa information](media/data-box-edge-gateway-view-device-events/view-device-events.png)
