## <a name="supported-ssh-key-formats"></a>SSH-nyckelformat som stöds

Azure stöder för närvarande SSH protocol 2 (SSH-2) RSA offentlig-privata nyckelpar med en minsta längd på 2048 bitar. Andra viktiga format som ED25519 och ECDSA stöds inte.
