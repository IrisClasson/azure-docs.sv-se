1. I Cloud Shell skapar du en fil som heter `rg.yml`.

    ```bash
    code rg.yml
    ```

1. Klistra in följande kod i redigeringsprogrammet:

   ```yaml
   ---
   - hosts: localhost
     connection: local
     tasks:
       - name: Create resource group
         azure_rm_resourcegroup:
           name: ansible-rg
           location: eastus
         register: rg
       - debug:
           var: rg
   ```

1. Spara filen och avsluta redigeringsprogrammet.

1. Kör spelboken med `ansible-playbook` kommandot:

   ```bash
   ansible-playbook rg.yml
   ```
