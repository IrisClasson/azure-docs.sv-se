Inställningen `Eula` anger att du har accepterat licensen för behållaren. Du måste ange ett värde för den här konfigurationsinställningen och värdet måste anges till `accept`.

|Krävs| Namn | Datatyp | Beskrivning |
|--|------|-----------|-------------|
|Ja| `Eula` | String | Licensacceptans<br><br>Exempel:<br>`Eula=accept` |

Cognitive Services-behållare licensieras enligt [ditt avtal](https://go.microsoft.com/fwlink/?linkid=2018657) som styr din användning av Azure. Om du inte har ett befintligt avtal som reglerar din användning av Azure godkänner du att ditt avtal om användning av Azure är [Microsoft Online-prenumerationsavtalet](https://go.microsoft.com/fwlink/?linkid=2018755), som innehåller [villkoren för onlinetjänster](https://go.microsoft.com/fwlink/?linkid=2018760). För förhandsgranskningar godkänner du även de [kompletterande användningsvillkoren för Microsoft Azure Previews](https://go.microsoft.com/fwlink/?linkid=2018815). Genom att använda behållaren godkänner du dessa villkor.
