> [!TIP]
> Azure Storage stöder auktorisera åtkomst till Blob- och Queue storage-resurser med Azure Active Directory (AD Azure). Auktorisering av användare eller program med hjälp av en OAuth 2.0-token som returneras av Azure AD tillhandahåller överlägsen säkerhet och användarvänlighet över delad nyckel auktorisering och signaturer för delad åtkomst (SAS). Med Azure AD finns det inget behov att lagra kontoåtkomstnyckel med din kod och risk potentiella säkerhetsproblem.
>
> Microsoft rekommenderar att du använder Azure AD med dina Azure Storage-program när det är möjligt. Mer information finns i [auktorisera åtkomst till Azure-blobbar och köer med hjälp av Azure Active Directory](../articles/storage/common/storage-auth-aad.md).
