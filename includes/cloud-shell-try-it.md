## <a name="use-azure-cloud-shell"></a>Använda Azure Cloud Shell

Azure är värd för Azure Cloud Shell, en interaktiv skalmiljö som du kan använda via din webbläsare. Du kan använda antingen Bash eller PowerShell med Cloud Shell för att arbeta med Azure-tjänster. Du kan använda kommandona för förinstallerade i Cloud Shell för att köra koden i den här artikeln utan att behöva installera något på din lokala miljö.

Så här startar du Azure Cloud Shell:

| Alternativ | Exempel/länk |
|-----------------------------------------------|---|
| Välj **Prova** i det övre högra hörnet av ett kodblock. Om du väljer **Prova** kopieras inte koden automatiskt till Cloud Shell. | ![Exempel på Prova för Azure Cloud Shell](./media/cloud-shell-try-it/hdi-azure-cli-try-it.png) |
| Gå [https://shell.azure.com](https://shell.azure.com)till eller välj knappen **Starta Cloud Shell** för att öppna Cloud Shell i din webbläsare. | [![Starta Cloud Shell i ett nytt fönster](media/cloud-shell-try-it/hdi-launch-cloud-shell.png)](https://shell.azure.com) |
| Välj knappen **Cloud Shell** i menyraden längst upp till höger i [Azure-portalen](https://portal.azure.com). | ![Cloud Shell-knappen i Azure Portal](./media/cloud-shell-try-it/hdi-cloud-shell-menu.png) |

Så här kör du koden i den här artikeln i Azure Cloud Shell:

1. Starta Cloud Shell.

1. Markera knappen **Kopiera** på ett kodblock för att kopiera koden.

1. Klistra in koden i Cloud Shell-sessionen genom att välja **Ctrl**+**Shift**+**V** på Windows och Linux eller genom att välja **Cmd**+**Shift**+**V** på macOS.

1. Välj **Ange** om du vill köra koden.
