## <a name="use-azure-cloud-shell"></a>Använda Azure Cloud Shell

Azure är värd för Azure Cloud Shell, ett interaktivt gränssnitt-miljö som du kan använda via webbläsaren. Cloudshell kan du använda antingen `bash` eller `PowerShell` att arbeta med Azure-tjänster. Du kan använda förinstallerade Cloud Shell-kommandon för att köra koden i den här artikeln utan att behöva installera något på din lokala miljö.

Att starta Azure Cloud Shell:

| Alternativ | Exempel/länk |
|-----------------------------------------------|---|
| Välj **Prova** i det övre högra hörnet av ett kodblock. Att välja **prova** inte automatiskt att kopiera koden till Cloud Shell. | ![Exempel på Testa för Azure Cloudshell](./media/cloud-shell-try-it/cli-try-it.png) |
| Gå till [ https://shell.azure.com ](https://shell.azure.com) eller Välj den **starta Cloud Shell** knappen för att öppna Cloud Shell i webbläsaren. | <a href="https://shell.azure.com" title="Starta Azure Cloudshell"><img name="launch-cloud-shell" src="https://docs.microsoft.com/azure/includes/media/cloud-shell-try-it/launchcloudshell.png" /></a> |
| Välj den **Cloud Shell** knapp i verktygsfältet längst upp till höger i den [Azure-portalen](https://portal.azure.com). | ![Cloud Shell-knappen i Azure Portal](./media/cloud-shell-try-it/cloud-shell-menu.png) |

Att köra koden i den här artikeln i Azure Cloud Shell:

1. Starta Cloudshell.
1. Välj den **kopia** knappen i ett kodblock att kopiera koden. 
1. Klistra in koden i Cloud Shell-sessionen med **Ctrl**+**SKIFT**+**V** på Windows och Linux eller **Cmd** + **SKIFT**+**V** på macOS. 
1. Tryck på **RETUR** att köra koden.
