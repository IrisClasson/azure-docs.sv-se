### <a name="prerequisites"></a>Nödvändiga komponenter
* En [Facebook](https://www.facebook.com/) konto 

Innan du kan använda ditt Facebook-konto i en logikapp, måste du godkänna logikappen som ska ansluta till Facebook-konto. Som tur är kan göra du det enkelt från i logikappen på Azure Portal. 

Här följer stegen för att auktorisera din logikapp för att ansluta till Facebook-konto:

1. Om du vill skapa en anslutning till Facebook, i Logic appdesigner väljer **visa Microsoft hanterade API: er** i nedrullningsbara listan anger *Facebook* i sökrutan. Välj utlösaren eller åtgärden som du kommer att tycka att använda:  
   ![Facebook-steg 1](./media/connectors-create-api-facebook/facebook-1.png)
2. Om du inte skapat några anslutningar till Facebook innan du kan hämta uppmanas du att ange dina autentiseringsuppgifter för Facebook. Dessa autentiseringsuppgifter används för att auktorisera din logikapp för att ansluta till och komma åt data i ditt Facebook-konto:  
   ![Facebook-steg 2](./media/connectors-create-api-facebook/facebook-2.png)
3. Ange ditt Facebook-användarnamn och lösenord för att auktorisera din logikapp:  
   ![Facebook-steg 3](./media/connectors-create-api-facebook/facebook-3.png)   
4. Lägg märke till anslutningen har skapats och du kan nu fortsätta med andra steg i logikappen:  
   ![Facebook-steg 4](./media/connectors-create-api-facebook/facebook-4.png)
