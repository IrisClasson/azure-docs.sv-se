## <a name="prerequisites"></a>Krav

* Ett [Facebook](https://www.facebook.com/) -konto 

Innan du kan använda ditt Facebook-konto i en Logic app måste du godkänna att Logic-appen ansluter till ditt Facebook-konto. Lyckligt vis kan du enkelt göra det i din Logic-app på Azure-portalen. 

Här följer stegen för att godkänna din Logic app för att ansluta till ditt Facebook-konto:

1. Om du vill skapa en anslutning till Facebook, i Logic App Designer, väljer du **Visa Microsoft Managed API: er** i list rutan och anger sedan *Facebook* i sökrutan. Välj den utlösare eller åtgärd som du vill använda:  
   ![Facebook steg 1](./media/connectors-create-api-facebook/facebook-1.png)
2. Om du inte har skapat några anslutningar till Facebook tidigare uppmanas du att ange dina Facebook-autentiseringsuppgifter. Dessa autentiseringsuppgifter används för att auktorisera din Logi Kap par för att ansluta till och komma åt Facebook-kontots data:  
   ![Facebook, steg 2](./media/connectors-create-api-facebook/facebook-2.png)
3. Ange ditt Facebook-användarnamn och lösen ord för att auktorisera din Logic app:  
   ![Facebook, steg 3](./media/connectors-create-api-facebook/facebook-3.png)   
4. Observera att anslutningen har skapats och att du nu är kostnads fri att fortsätta med de andra stegen i din Logic app:  
   ![Facebook, steg 4](./media/connectors-create-api-facebook/facebook-4.png)
