## <a name="setupdevenv"></a>Konfigurera utvecklingsmiljön
Starta genom att ställa in din utvecklingsmiljö med Visual Studio och Azure SDK.

* Visual Studio 2017 och senare innehåller Azure SDK. Om du använder Visual Studio 2017 eller senare, krävs inga ytterligare inställningar för utvecklingsmiljön.
* För Visual Studio 2015 klickar du på länken nedan för att installera [Azure SDK för Visual Studio 2015](https://go.microsoft.com/fwlink/?linkid=518003).
* För Visual Studio 2013 klickar du på länken nedan för att installera [Azure SDK för Visual Studio 2013](https://go.microsoft.com/fwlink/?LinkID=324322).

* Om du inte har Visual Studio installerat kan du installera [Visual Studio 2019](https://www.visualstudio.com/) med Azure SDK.

> [!NOTE]
> Beroende på hur många av SDK-beroendena du redan har på datorn kan det ta lång tid att installera SDK:n, från några minuter till drygt en halvtimma.
