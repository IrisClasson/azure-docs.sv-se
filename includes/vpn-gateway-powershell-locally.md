Du kan också installera och köra Azure PowerShell-cmdlets lokalt på datorn. PowerShell-cmdletar uppdateras ofta. Om du inte har installerat den senaste versionen kan de värden som anges i instruktionerna Miss lyckas. Ta reda på vilka versioner av Azure PowerShell som är installerade på datorn genom att använda `Get-Module -ListAvailable Az`-cmdleten. Information om hur du installerar eller uppdaterar finns i [installera Azure PowerShell-modulen](/powershell/azure/install-az-ps).
