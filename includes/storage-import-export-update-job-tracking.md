Efter diskarna, gå tillbaka till den **Import/Export** sidan på Azure portal. 

> [!IMPORTANT] 
> Om Spårningsnumret inte uppdateras inom 2 veckor för att skapa jobbet, jobbet upphör att gälla. 

Utför följande steg för att uppdatera Spårningsnumret.
 
1. Välj och klicka på jobbet.
2. Klicka på **uppdatera jobbstatus och spårningsinformation när enheter levereras**. 
3. Markera kryssrutan med **Markera som levererat**.
4. Ange den **operatör** och **spårningsnummer**.
5. Spåra jobbförloppet på instrumentpanelen i portalen. Gå till en beskrivning av varje jobbstatus [visa jobbets status](../articles/storage/common/storage-import-export-view-drive-status.md).
