## <a name="planning-and-implementation"></a>Planering och implementering
Rubrik: SAP NetWeaver på Azure Virtual Machines – planera och Implementeringsguide

Sammanfattning: Starta med det här dokumentet om du funderar på att köra SAP NetWeaver i virtuella Azure-datorer. Den här planerings- och implementeringsguiden hjälper dig att utvärdera om ett befintligt eller planerat SAP NetWeaver-baserat system kan distribueras till en miljö för virtuella Azure-datorer. Den omfattar flera SAP NetWeaver distributionsscenarier och innehåller SAP-konfigurationer som är specifika för Azure. Pappret listar och beskriver all nödvändig konfigurationsinformation som du behöver på SAP/Azure-sidan för att köra ett SAP-hybridlandskap. Åtgärder som du kan vidta för att garantera hög tillgänglighet för SAP NetWeaver-baserade system på IaaS omfattas också.

## <a name="sap-dbms-on-azure"></a>SAP DBMS på Azure
Rubrik: SAP DBMS i Azure Deployment Guide

Sammanfattning: Det här dokumentet beskriver planerings- och implementeringsöverväganden för DBMS system som ska köras tillsammans med SAP. I den första delen listas och presenteras allmänna överväganden. Följande delar av papperet avser distributioner av olika DBMS i Azure som stöds av SAP. Olika DBMS som visas är SQL Server, SAP ASE, Oracle, SAP MaxDB och IBM DB2 för Linux, Unix- och Windows. I de specifika delarna diskuteras de överväganden du behöver beakta när du kör SAP system på Azure tillsammans med de olika DBMS. Ämnen som vilka metoder för säkerhetskopiering och hög tillgänglighet som stöds av olika DBMS på Azure presenteras för användning med SAP-program.

## <a name="sap-netweaver-on-azure"></a>SAP NetWeaver på Azure
Rubrik: SAP NetWeaver - bygga en Azure baserad Haveriberedskapslösning

Sammanfattning: det här dokumentet innehåller stegvisa anvisningar för att skapa en Azure-baserad lösning för Haveriberedskap för SAP NetWeaver. Lösningen beskrivs förutsätter att SAP-landskap körs virtualiserade lokalt baserat på Hyper-V. I den första delen av dokumentet Azure Site Recovery (ASR) introduceras tjänster i dess komponenter. Den andra delen av dokumentet beskriver informationen för för SAP NetWeaver-baserade landskap. Möjligheterna att använda ASR med instanser av programmet SAP NetWeaver och SAP Central Services beskrivs och beskrivs. Fokus i den andra delen utnyttjar ASR för SAP Central Services som är skyddade med windows Server Failover Cluster konfigurationer.
