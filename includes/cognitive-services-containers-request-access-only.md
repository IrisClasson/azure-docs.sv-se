Formuläret begär information om dig, ditt företag och det användarscenario som du ska använda behållaren för. När du har skickat formuläret granskar Azure Cognitive Services-teamet det för att säkerställa att du uppfyller kriterierna för åtkomst till det privata behållarregistret.

> [!IMPORTANT]
> Du måste använda en e-postadress som är associerad med antingen ett MSA-konto (Microsoft Account) eller Azure Active Directory (Azure AD) i formuläret .

Om din begäran godkänns får du ett e-postmeddelande med instruktioner som beskriver hur du hämtar dina autentiseringsuppgifter och får tillgång till det privata behållarregistret.
