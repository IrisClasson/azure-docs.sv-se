> Den här funktionen är en förhandsversion. Om du vill använda den måste du installera [Event Grid-tillägget](/cli/azure/azure-cli-extensions-list) för [Azure CLI](/cli/azure/install-azure-cli). Du kan installera det med `az extension add --name eventgrid`. Om du tidigare har installerat tillägget kan du uppdatera den med `az extension update -n eventgrid`. Om du använder REST API måste du använda `api-version=2018-09-15-preview`.
