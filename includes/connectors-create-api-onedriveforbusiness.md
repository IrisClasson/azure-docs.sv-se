### <a name="prerequisites"></a>Nödvändiga komponenter
* En [OneDrive](http://OneDrive.com) konto 

Innan du kan använda ditt OneDrive för företag-konto i en logikapp, måste du godkänna logikappen som ska ansluta till ditt OneDrive för företag-konto. Som tur är kan göra du det enkelt från i logikappen på Azure Portal. 

Här följer stegen för att auktorisera din logikapp för att ansluta till ditt OneDrive för företag-konto:

1. Om du vill skapa en anslutning till OneDrive för företag i Logic appdesigner väljer **visa Microsoft hanterade API: er** i nedrullningsbara listan anger *OneDrive för företag* i sökrutan. Välj utlösaren eller åtgärden som du kommer att tycka att använda:  
   ![](./media/connectors-create-api-onedriveforbusiness/onedriveforbusiness-1.png)
2. Om du inte skapat några anslutningar till OneDrive för företag innan du kan hämta uppmanas du att tillhandahålla ditt OneDrive för företag-autentiseringsuppgifter. Dessa autentiseringsuppgifter används för att auktorisera din logikapp för att ansluta till och få åtkomst till ditt OneDrive för företag-konto data:  
   ![](./media/connectors-create-api-onedriveforbusiness/onedriveforbusiness-2.png)
3. Ange ditt OneDrive för företag-användarnamn och lösenord för att godkänna din logikapp:  
   ![](./media/connectors-create-api-onedriveforbusiness/onedriveforbusiness-3.png)   
4. Lägg märke till anslutningen har skapats och du kan nu fortsätta med andra steg i logikappen:  
   ![](./media/connectors-create-api-onedriveforbusiness/onedriveforbusiness-4.png)
