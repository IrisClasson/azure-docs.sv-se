## <a name="authentication"></a>Authentication

Om du vill köra följande exempel behöver du ett Azure AD-program och tjänstens huvud namn som har åtkomst till resurser. Se [skapa ett Azure AD-program](/azure/active-directory/develop/howto-create-service-principal-portal) för att skapa ett kostnads fritt Azure AD-program och lägga till roll tilldelning på prenumerations nivå. Du behöver också `directory (tenant) ID`, `application ID` och `client secret`.
