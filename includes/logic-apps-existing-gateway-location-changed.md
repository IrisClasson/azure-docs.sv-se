**F**: Varför ändrades platsen för min befintliga gateway? <br/>
**S**: Gateway-resurser som skapades före den 3 maj 2017 flyttades till de ursprungliga platserna för Azure AD-klienten för arbets- eller skolkontot som skapade dessa gateways. Dessa ändringar bör dock inte påverka logikappar som körs för närvarande, vilket bör fortsätta att fungera som vanligt. Med gatewayens allmänna tillgänglighet i maj kan gateway-resursplatser skilja sig från logikappplatser.
