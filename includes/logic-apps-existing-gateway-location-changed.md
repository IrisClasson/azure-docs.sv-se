**Frågor och**: Varför har plats för mitt befintliga gateway ändringen? <br/>
**En**: Gateway-resurser som skapats innan den 3 maj 2017 har flyttats till de ursprungliga platserna för Azure AD-klient för arbets- eller skolkonto konto som har skapat dessa gateways. Dessa ändringar inte bör dock påverka pågående logic apps, som bör fortsätta arbeta som vanligt. Med gatewayens allmänt tillgängliga i maj, resursplatser för gateway kan skilja sig från logic app-platser.
