**F**: Varför ändrades platsen för min befintliga Gateway? <br/>
**A**: gateway-resurser som skapats före den 3 maj 2017 flyttades till den ursprungliga platsen för Azure AD-klienten för det arbets-eller skol konto som skapade dessa gatewayer. Dessa ändringar påverkar dock inte körning av logi Kap par, vilket bör fortsätta att fungera som vanligt. Med gatewayens allmänna tillgänglighet i maj kan gateway-resursens platser skilja sig från Logic app-platser.
