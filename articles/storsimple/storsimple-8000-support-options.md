# <a name="storsimple-solution-support"></a>Stöd för StorSimple-lösning

## <a name="storsimple-support"></a>Support för StorSimple

Microsoft erbjuder flexibla supportalternativ för StorSimple-företagslagringskunder. Vi är djupt engagerade i att leverera en supportupplevelse av hög kvalitet som gör att du kan maximera effekten av din investering i StorSimple-lösningen och Microsoft Azure. Som StorSimple-kund får du:

* 24x7-möjligheten att skicka supportbiljetter via Azure-portalen.
* Hjälpdesk åtkomst för allmänna supportfrågor och djup teknisk hjälp.
* Lokalt språkstöd där det finns tillgängligt.
* Aviseringsetablering och hantering för hälso- och prestandainsikter.
* Tillgång till programuppdateringar som täcker större, mindre och underhållskorrigeringar.
* Stöd för Lagringsmatriser i StorSimple 8000-serien och StorSimple-virtuella matriser i ett enda paket.

## <a name="local-language-support"></a>Stöd för lokalt språk
Förutom engelska, lokala språk stöd ges på följande språk under kontorstid: spanska, portugisiska, japanska, koreanska, taiwanesiska och traditionell kinesiska.

## <a name="support-scope"></a>Supportomfattning 
Support för fakturerings- och prenumerationshanteringsrelaterade problem finns på alla supportnivåer. För att få StorSimple-support måste kunden vara aktivt registrerad för antingen StorSimple Standard eller Premium supportplaner. StorSimples supportteam ansvarar för att lösa alla problem som påverkar StorSimple-lösningen. För att få support för Azure-relaterade problem som inte är direkt relaterade till StorSimple måste kunden registreras i en lämplig Azure-supportplan. Se [här](https://azure.microsoft.com/support/plans/) för mer information. Supportteamet refererar icke-StorSimple-supportärenden till Azure-teamet för uppföljning baserat på kundrättigheter för Azure-support.

## <a name="cancellation-policy"></a>Policy för avbokning
För att få StorSimple-support måste kunden köpa Standard- eller Premium-supportplaner under prenumerationsperioden. Avbokning resulterar inte i en proportionell återbetalning. StorSimples supportplaner är berättigade att minska på EA-årsdagen. Microsoft kan dock inte ge support till StorSimple-kunder utan giltiga supportavtal.

## <a name="renewal-policy"></a>Policy för förnyelse


Vid köpet av Lagringsmatriser i StorSimple 8000-serien ges stöd under nästa EA-årsdag. Kunden måste förnya StorSimple-supporten på EA-årsdagen. StorSimple support plan order är samtidiga. Kunder meddelas via e-post om förestående supportutgången för Lagringsmatriser i StorSimple 8000-serien och förväntas följa upp med Microsoft-konto/säljteamen eller deras Microsoft Licensing Solution Partner (LSP) för att förnya StorSimple-supporten.


Standard Azure-support täcker inte StorSimple-maskinvarusupport. Om du omfattas av Premier- eller Unified Microsoft-supporten måste du fortfarande köpa Standard StorSimple-supportförnyelse. StorSimple support förnyelse kan anpassas till EA årsdagen genom att förvärva den nödvändiga supporten SKU med licenskvantitet lika med antalet apparater och den enhetskvantitet som beställts är det återstående antalet månader av stöd som behövs fram till EA årsdagen om alla enheter har samma utgångsdatum för supportkontraktet. Om enheterna har olika utgångsdatum för supportkontrakt måste varje apparat täckas med en support-SKU med den enhetskvantitet som beställts som det återstående antalet månader av stöd som behövs fram till EA:s jubileumsdatum per varje apparat. 

Stöd för Lagringsmatriser i StorSimple 8000-serien tillhandahålls baserat på hur StorSimple-matrisen köptes.
