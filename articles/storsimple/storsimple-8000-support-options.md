# <a name="storsimple-solution-support"></a>Support för StorSimple-lösning

## <a name="storsimple-support"></a>StorSimple-support

Microsoft erbjuder flexibla supportalternativ för StorSimple enterprise storage-kunder. Vi är djupt arbetar för att leverera en högkvalitativ supportupplevelse som hjälper dig att maximera effekten av din investering i StorSimple-lösningen och Microsoft Azure. Som kund StorSimple får du:

* 24 x 7 möjlighet att skicka supportbegäranden via Azure portal.
* Hjälpa tillgång till supportavdelningen för stöd för allmänna frågor och djupgående teknisk hjälp.
* Stöd för lokala språk där det är tillgängligt.
* Avisera etablering och hantering för insights hälsa och prestanda.
* Åtkomst till som täcker högre eller lägre version, uppdateringar och korrigeringar för underhåll.
* Stöd för lagringsmatriser i StorSimple 8000-serien och StorSimple Virtual Array i ett enda paket.

## <a name="support-scope"></a>Supportens omfattning

StorSimple-supportteamet är ansvarig för att åtgärda alla problem som påverkar StorSimple-lösningen. Kunden måste vara registrerade i en lämplig Azure-supportplan för att få support för Azure-relaterade problem som inte är direkt relaterade till StorSimple. Se [här](https://azure.microsoft.com/support/plans/) mer information. Supportteamet refererar icke StorSimple supportärenden till Azure-teamet för uppföljning baserat på kundernas rättigheter för Azure-support.

## <a name="cancellation-policy"></a>Annulleringspolicy
För att få support för StorSimple, måste kunden köpa supportavtal för Standard eller Premium för hela prenumerationstiden. Annulleringen resulterar inte i en proportionell återbetalning. StorSimple-support är minskning berättigade vid EA anniversary. Microsoft kan dock inte kan ge stöd åt StorSimple kunder utan giltigt supportavtal.

## <a name="renewal-policy"></a>Förnyelsepolicy


Support ges via nästa EA årsdagen av inköp av lagringsmatriser i StorSimple 8000-serien. Kunden måste förnya EA anniversary StorSimple-support. StorSimple support plan order är coterminous. Kunderna meddelas via e-postmeddelanden om nära förestående stöd för förfallodatum för lagringsmatriser i StorSimple 8000-serien och som förväntas följa upp med Microsoft-konto/sales-team eller sin Microsoft-Partner LSP (Licensing Solution) att förnya StorSimple-support.


Standard Azure-support omfattar inte support för StorSimple-maskinvaran. Om du täcks under Premier eller Unified Microsoft support, måste du ändå köpa Supportförnyelse för storsimple. För storsimple kan justeras med EA-årsdagen genom att skaffa det obligatoriska stödet SKU med licensantal som är lika med antalet av utrustning och kvantiteten sorterade som det återstående antalet månaders stöd som de behöver tills EA årsdagen om alla enheter har samma utgångsdatum för support-kontrakt. Om enheterna som har stöd för olika kontraktets utgångsdatum kan varje installation måste förses med en stöd SKU med enheten kvantitet sorterade som det återstående antalet månaders stöd som de behöver tills EA-årsdagen varje varje produkt.
