# <a name="storsimple-for-cloud-solutions-provider-program-frequently-asked-questions"></a>StorSimple för Cloud Solutions Provider Program: Vanliga frågor och svar


## <a name="overview"></a>Översikt
Följande är frågor och svar som du kan ha som StorSimple-partner när du distribuerar eller hanterar en StorSimple Virtual Array i Azure-portalen.

Frågor och svar ordnas i följande kategorier:

 - [CSP-modell (StorSimple and Cloud Solutions Provider Program)](#storsimple-and-csp-model)
 - [Distribuera och hantera virtuell matris som partner](#deploy-and-manage-storsimple-as-a-partner)
 - Stöd för StorSimple-partners för CSP

## <a name="storsimple-and-csp-model"></a>StorSimple- och CSP-modell

### <a name="what-does-it-mean-to-have-storsimple-as-a-part-of-the-cloud-solutions-provider-csp-program"></a>Vad innebär det att ha StorSimple som en del av Cloud Solutions Provider (CSP) programmet?
 StorSimple för CSP gör det möjligt för våra CSP-partners att sälja StorSimple Virtual Array vidare. Den här modellen gör det möjligt för CSP-partnerna att äga kundens livscykel från sluten till med direkt etablering, fakturering och support för Microsofts molntjänster. 

Mer information finns [i StorSimple i CSP-programmet](https://www.microsoft.com/en-us/server-cloud/products/storsimple/). 

### <a name="is-storsimple-a-usage-based-or-based-on-seat-licenses-like-office-365-and-enterprise-mobility-suite-ems"></a>Är StorSimple en användningsbaserad eller baserad på platslicenser som Office 365 och Enterprise Mobility Suite (EMS)?
StorSimple är en användningsbaserad tjänst. Kunder som anskaffar StorSimple Virtual Array via CSP-vägen betalar för användningen av virtuell matris och Azure, faktureras som separata radobjekt i CSP-fakturan. 

### <a name="is-the-storsimple-business-model-similar-to-that-of-other-services-such-as-the-office-365-and-the-enterprise-mobility-suite"></a>Liknar StorSimple-affärsmodellen andra tjänster, till exempel Office 365 och Enterprise Mobility Suite? 
Ja. Modellen är en rabattmodell i grossistledet precis som andra användningsbaserade tjänster i Azure. Grossistrabatten för StorSimple liknar andra Azure-tjänster. Den här modellen ger våra CSP-partner en möjlighet att sälja mer och driva Azure-förbrukning via StorSimple Virtual Array.  

### <a name="which-storsimple-skus-are-available-at-launch"></a>Vilka StorSimple SKU:er finns tillgängliga vid lanseringen?
Samma SKU:er som är tillgängliga med de fristående StorSimple-tjänsterna. 

### <a name="is-there-any-difference-between-the-storsimple-directly-purchased-from-microsoft-under-ea-and-storsimple-as-a-part-of-csp"></a>Finns det någon skillnad mellan StorSimple direkt köpt från Microsoft under EA och StorSimple som en del av CSP?
Nej. Under CSP kan kunder också köpa andra tjänster som erbjuds av CSP-partnern tillsammans med StorSimple Virtual Array (SVA) under en faktura. Observera att för att upphandla Den fysiska storsimple 8000-seriens fysiska enhet måste kunderna fortfarande gå igenom EA-rutten. 

### <a name="in-which-markets-is-storsimple-available-for-csp-at-launch"></a>På vilka marknader, är StorSimple tillgängligt för CSP vid lanseringen?
StorSimple Virtual Array för CSP finns på alla marknader där StorSimple finns idag. Mer information finns i [listan över regioner där StorSimple är tillgängligt](https://azure.microsoft.com/regions/services/).

### <a name="what-kind-of-solutions-can-a-partner-deliver-with-storsimple-and-csp"></a>Vilken typ av lösningar kan en partner leverera med StorSimple och CSP? 
Det finns flera lösningar:

- Partnerna kan sälja StorSimple Virtual Array. 
- Partnerna kan leverera lösningar som är uppbyggda kring StorSimple och andra Azure-tjänster eller programvara från tredje part. Mer information finns i några möjliga [lösningar som finns runt StorSimple](https://aka.ms/storsimple-build-solutions).
- De kan också upptäcka nya affärsmodeller genom att leverera hanterade tjänster.

### <a name="what-are-the-incentives-available-for-storsimple-csp-partners-do-i-qualify-for-any-csp-program-incentives"></a>Vilka incitament är tillgängliga för StorSimple CSP-partner? Kvalificerar jag mig för några incitament för CSP-program?
Information om incitament finns i incitamenten för [CSP-program.](https://partner.microsoft.com/en-US/cloud-solution-provider/program-details)


## <a name="deploy-and-manage-storsimple-as-a-partner"></a>Distribuera och hantera StorSimple som partner

### <a name="how-can-i-administer-storsimple-virtual-array-in-csp"></a>Hur kan jag administrera StorSimple Virtual Array i CSP? 
Du kan lägga till StorSimple Virtual Array-prenumerationer på dina kunders konto via Partnercenter. Dessutom kan du använda Azure-portalen för att lägga till användare i prenumerationerna.

### <a name="is-the-azure-portal-approach-the-same-as-other-services-for-csp"></a>Är Azure-portalmetoden densamma som för andra tjänster för CSP? 
Ja. Med StorSimple Virtual Array för CSP bör det bästa sättet för partner att komma åt Azure-portalen vara via [Partner Center](https://partnercenter.microsoft.com/) där de kan hantera kunder och prenumerationer. För att hantera StorSimple-prenumerationer ska partnern eller kunden (beroende på beviljade behörigheter) logga in på Azure-portalen. 

### <a name="is-microsoft-shipping-a-new-portal-for-storsimple-for-csp"></a>Skickar Microsoft en ny portal för StorSimple för CSP? 
Nej. Du kan administrera StorSimple Virtual Array i CSP via Azure-portalen.

### <a name="i-have-provisioned-a-standalone-storsimple-subscription-for-my-customer-do-i-have-to-use-the-azure-portal-to-administer-my-device"></a>Jag har etablerat en fristående StorSimple-prenumeration för min kund. Måste jag använda Azure-portalen för att administrera min enhet? 
Ja. 

### <a name="is-the-microsoft-account-team-compensated-as-a-result-of-sales-of-storsimple-through-csp"></a>Kompenseras Microsoft-kontoteamet till följd av försäljningen av StorSimple via CSP?
Ja. Partnerförsäljningschefen och kundkontoteamet för slutkunden kommer att kompenseras för CSP-försäljning. Parterna förväntas ta ledningen i att sälja sina differentierade värde och erbjudanden.

## <a name="support-for-storsimple-and-csp"></a>Stöd för StorSimple och CSP

### <a name="are-there-any-forums-to-get-additional-support-for-csp-partners-to-deploy-and-manage-storsimple"></a>Finns det några forum för att få ytterligare stöd för CSP-partner för att distribuera och hantera StorSimple?
Du kan besöka StorSimple i CSP forum för att få svar på några vanliga frågor. Du måste gå med i [Azure Advisors Yammer-gruppen](https://www.yammer.com/azureadvisors) först. Nästa sökning efter och gå med i gruppen - **StorSimple Partner Advisors**.

### <a name="how-does-the-support-work-for-storsimple-for-csp"></a>Hur fungerar supporten för StorSimple för CSP? 
Supportmodellen för StorSimple i CSP är samma som för andra Azure-tjänster i CSP. Mer information finns i [Kundsupport för CSP](/partner-center/csp-overview).

Mer information om StorSimple för CSP finns på:

- [Microsoft Cloud Solution Provider Program](https://partner.microsoft.com/Membership)
- [Partnercenter](https://partnercenter.microsoft.com/)
