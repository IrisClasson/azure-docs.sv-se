# <a name="use-the-storsimple-device-manager-service-to-log-a-support-request-for-the-storsimple-virtual-array"></a>Använda Tjänsten StorSimple Device Manager för att logga en supportbegäran för StorSimple Virtual Array

## <a name="overview"></a>Översikt

StorSimple Device Manager ger möjlighet att **logga en ny supportbegäran** i servicesammanfattningsbladet. I den här artikeln beskrivs hur du kan logga en ny supportbegäran och hantera dess livscykel inifrån portalen.

## <a name="new-support-request"></a>Ny supportbegäran

Beroende på din [supportplan](https://azure.microsoft.com/support/plans/)kan du skapa supportärenden för ett problem på den virtuella storsimple-matrisen direkt från sammanfattningsbladet för StorSimple-enhetshanteraren.

#### <a name="to-log-a-new-request"></a>Så här loggar du en ny begäran

1. Gå till StorSimple Device Manager-tjänsten. Gå till avsnittet **SUPPORT + FELSÖKNING** i servicesammanfattningsbladsinställningarna och klicka sedan på Ny **supportbegäran**.
   
    ![Ny supportbegäran](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket1.png)

2. Gör **Basics** följande i basics-bladet:

    1. Välj **Issue type** **Teknisk**. 
    
    2. Den aktuella **prenumerations-,** **tjänsttypen** och tjänsten **Resurs** (StorSimple Enhetshanteraren) väljs automatiskt. 

    3. Ange en eller flera enheter som är registrerade på din tjänst och som har problem.

    4. Välj en lämplig **supportplan** om du har flera abonnemang kopplade till din prenumeration. Du behöver en betald supportplan för att aktivera teknisk support.

3. I **steg 2**väljer du **allvarlighetsgraden** och anger om problemet är relaterat till matrisen eller Tjänsten StorSimple Device Manager. Välj också en **kategori** för det här problemet och ge mer **information** om problemet.
   
    ![Ny supportbegäran](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket2.png)

4. Ange dina kontaktuppgifter i **steg 3.** Microsoft Support använder den här informationen för att kontakta dig för ytterligare information, diagnos och lösning.
   
    ![Ny supportbegäran](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket3.png)

## <a name="manage-a-support-request"></a>Hantera en supportbegäran

När du har skapat ett supportärende kan du hantera ärendets livscykel på portalen.

#### <a name="to-manage-your-support-requests"></a>Så här hanterar du dina supportförfrågningar

Om du vill komma till hjälp- och supportsidan navigerar du till **Bläddra > hjälp + support**.

![Hantera supportförfrågningar](./media/storsimple-virtual-array-log-support-ticket/manage-support-tickets.png)
