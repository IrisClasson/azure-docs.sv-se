# <a name="use-the-storsimple-device-manager-service-to-troubleshoot-the-storsimple-virtual-array"></a>Använda StorSimple Device Manager-tjänsten för att felsöka StorSimple Virtual Array
## <a name="overview"></a>Översikt

StorSimple Device Manager ger en **diagnostisera och lösa problem** inställningen inom sammanfattningsbladet för tjänsten, som visar några av de ofta förekommande problemen som kan uppstå med din virtuella matris och hur du löser dem. Den här självstudien introducerar självbetjäning felsökning funktioner i StorSimple Device Manager-tjänsten.

![Diagnostisera och lösa problem](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-main.png)

## <a name="diagnose-and-solve-issues"></a>Diagnostisera och lösa problem

Du kan visa några av de vanliga problem med StorSimple Virtual Array och granska lösningarna på de problem som är rätt från din sammanfattningsbladet för StorSimple Device Manager-tjänsten.

#### <a name="to-diagnose-an-issue-with-your-virtual-array"></a>Diagnostisera ett problem med din virtuella matris

1. Klicka på **diagnostisera och lösa problem** inställning i den vänstra rutan i din sammanfattningsbladet för StorSimple Device Manager-tjänsten för att visa en lista över vanliga problem.

2. **Expandera** på symtom på problemet uppstår om du vill granska **rekommenderade steg** att underlätta felsökningen. Om du vill kan granska du också detaljerad dokumentationslänkarna ytterligare referens.
   
    ![Felsöka ett problem](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-offline.png)

3. Om du inte att hitta en referens till problemet eller lösa det. Kontakta Microsoft Support för ytterligare hjälp.
