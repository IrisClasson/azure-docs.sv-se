# <a name="use-the-storsimple-device-manager-service-to-troubleshoot-the-storsimple-virtual-array"></a>Använda tjänsten StorSimple Device Manager för att felsöka StorSimple Virtual Array
## <a name="overview"></a>Översikt

StorSimple Enhetshanteraren ger en **diagnos och lösa probleminställning** i tjänsten sammanfattning blad, som belyser några av de vanligaste problemen som kan uppstå med din virtuella matris och hur du löser dem. Den här självstudien introducerar felsökningsfunktionen för självbetjäning som tillhandahålls i Tjänsten StorSimple Device Manager.

![Diagnostisera och lösa problem](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-main.png)

## <a name="diagnose-and-solve-issues"></a>Diagnostisera och lösa problem

Du kan visa några av de vanliga problemen med StorSimple Virtual Array och granska lösningarna på dessa problem direkt från sammanfattningsbladet för StorSimple-enhetshanteraren.

#### <a name="to-diagnose-an-issue-with-your-virtual-array"></a>Så här diagnostiserar du ett problem med den virtuella matrisen

1. Klicka på **Diagnostisera och lös inställningen för problem** i den vänstra rutan i sammanfattningsbladet för StorSimple-enhetshanteraren för att visa en lista över vanliga problem.

2. **Expandera** på symptomet på problemet som du stöter på för att granska **rekommenderade steg** som hjälper dig att felsöka problemet. Om du vill kan du också granska de detaljerade dokumentationslänkarna som finns för ytterligare referens.
   
    ![Felsöka ett problem](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-offline.png)

3. Om du inte kan hitta en referens till problemet eller lösa det kan du kontakta Microsoft Support för ytterligare hjälp.
