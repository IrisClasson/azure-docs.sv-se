# <a name="what-is-storsimple-for-cloud-solutions-providers-program"></a>Vad är StorSimple for Cloud Solutions Providers Program?


## <a name="overview"></a>Översikt

Microsoft Azure StorSimple är en unik metod för företagslagring med verkliga hybridmolnlagringsfunktioner. Det ger kunderna möjlighet att dra nytta av ekonomisk molnlagring för inaktiva data, samtidigt som deras verksamhetskritiska data är lokalt för högsta prestanda. 

StorSimple Virtual Array for Cloud Solutions Provider (CSP) gör det möjligt för partner att dra nytta av den här affärsmöjligheten. Partnerna äger kundens livscykel från slutna till sluten kund med direkt etablering, fakturering och support för Microsofts molntjänster. Kort sagt, partnerna kan nu handla StorSimple tillsammans med sina tjänster till kunder.

Mer information om StorSimple för CSP finns på översiktssidan för [Azure CSP.](https://docs.microsoft.com/azure/cloud-solution-provider/overview/azure-csp-overview)

Mer information om fakturering, prissättning, incitament och support i CSP finns i [StorSimple i CSP: Faq](storsimple-partner-csp-faq.md). 

## <a name="deploy-and-manage-storsimple-for-csp"></a>Distribuera och hantera StorSimple för CSP

StorSimple för CSP finns som en användningsbaserad tjänst på alla marknader där StorSimple finns idag. StorSimple för CSP använder Azure-portalen och StorSimple Device Manager-tjänsten. En CSP-partner kan skapa en StorSimple-enhetshanterare för att hantera StorSimple-virtuella matriser, resurser, volymer och säkerhetskopior. Du kan administrera alla virtuella matriser som är registrerade till tjänsten StorSimple Device Manager via Azure-portalen.

Mer information finns i [Distribuera och hantera den virtuella storsimple-matrisen för CSP](storsimple-partner-csp-deploy.md).
