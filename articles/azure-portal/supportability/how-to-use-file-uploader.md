# <a name="microsoft-azure-new-support-request-file-upload-utility-guidelines"></a>Microsoft Azure nya riktlinjer för filöverföring för supportbegäran

När du [skapar en supportbegäran](https://portal.azure.com/#create/Microsoft.Support)kan filöverföringsalternativet hjälpa dig att enkelt ladda upp diagnostikfiler eller andra filer som du tror är relevanta för incidenten.  

> [!NOTE]
> För att skydda din integritet ska du inte inkludera någon personlig information i din uppladdning.
>
>

Följ dessa riktlinjer när du använder filuppladdningsalternativet:

* Alla filer måste ha ett filnamnstillägg, till exempel *.docx* eller *.xlsx*. Se tabellen för de tillåtna filnamnstilläggen.
* Filnamnet får inte vara längre än 110 tecken.
* Du kan inte ladda upp mer än en fil.
* Filer kan inte vara större än 4 MB.
