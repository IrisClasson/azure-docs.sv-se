# <a name="unity-showcase-application"></a>Unity showcase ansökan

Unity showcase-programmet är ett Unity-baserat exempelprogram för att demonstrera Azure Remote Rendering-funktioner. Det kan köras både insida enhetredigeraren och på Hololens 2.

![Visa upp appen](./media/showcase-app.png "Visa upp appen")

## <a name="source-repository"></a>Källdatabas

Källkoden för det här programmet tillsammans med instruktioner för bygg-/distribution finns i [ARR-exempeldatabasen](https://github.com/Azure/azure-remote-rendering).
