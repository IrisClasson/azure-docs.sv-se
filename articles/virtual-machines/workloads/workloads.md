# <a name="workloads-on-azure-virtual-machines"></a>Arbets belastningar på Azure Virtual Machines

Virtuella datorer i Azure kan stödja många arbets belastningar. Varje arbets belastning kan ha särskild vägledning kring den optimala designen för distributioner på virtuella datorer i Azure. Här följer några rikt linjer som vi kan erbjuda för specialiserade arbets belastningar.

- [Data behandling med höga prestanda](./hpc/overview.md) i Azure använder CPU eller GPU-optimerade virtuella datorer för att lösa komplexa, beräknings intensiva problem.
- [SAP](./sap/get-started.md) i Azure kan köra dina verksamhets kritiska SAP-arbetsbelastningar och scenarier på en skalbar, kompatibel och företags beprövad plattform.
- [Oracle](./oracle/oracle-considerations.md) stöder körning av Oracle DB 12,1 standard-och Enterprise-versioner i Azure på avbildningar av virtuella datorer baserat på Oracle Linux. 
- [Omvärdering av stordator](./mainframe-rehosting/overview.md)
