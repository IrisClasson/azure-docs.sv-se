# <a name="workloads-on-azure-virtual-machines"></a>Arbetsbelastningar på Azure Virtual Machines

Virtuella datorer i Azure har stöd för många arbetsbelastningar. Varje arbetsbelastning kan ha specialiserade vägledning för den optimala designen för distribution på virtuella datorer i Azure. Här är några av de riktlinjer som vi kan erbjuda för särskilda arbetsbelastningar.


- [SAP](./sap/get-started.md) i Azure kan köra dina verksamhetskritiska SAP arbetsbelastningar och scenarier på en skalbar, kompatibel och företagsklass plattform.
- [Oracle](./oracle/oracle-considerations.md) stöder Oracle DB 12.1 Standard eller Enterprise Edition i Azure på avbildningar av virtuella datorer baserat på Oracle Linux. 
- [Stordatorprogram rehosting](./mainframe-rehosting/overview.md)
