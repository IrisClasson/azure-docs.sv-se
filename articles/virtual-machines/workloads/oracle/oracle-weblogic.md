# <a name="oracle-weblogic-server-azure-applications"></a>Oracle WebLogic Server Azure-program

## <a name="oracle-weblogic-server-is-a-scalable-enterprise-ready-java-ee-application-server"></a>Oracle WebLogic Server är en skalbar, företags klar Java-program Server.

Oracle WebLogic Server är världens första moln inbyggda, Enterprise Java Platform Application Server för utveckling och distribution av distribuerade företags program i flera nivåer. Med Azure WebLogic-Server kan du utnyttja molnbaserad data behandling genom att tillhandahålla större valmöjligheter och flexibilitet för WebLogic-migrering, inklusive lyft och Shift för dina Java-program i Azure med den minsta ansträngningen och största påverkan. Med erbjudandena kan du snabbt starta dina affärs program genom att automatiskt konfigurera virtuella nätverk, lagrings-och Linux-resurser, installera WebLogic-Server, Konfigurera säkerhet med en nätverks säkerhets grupp, belastnings utjämning med Azure App Gateway, autentisering med Azure Active Directory och över gång till databas anslutning.

Det finns fyra erbjudanden som är tillgängliga för att möta olika scenarier: en nod utan en administratörs Server, en nod med en administratörs Server, ett kluster och ett dynamiskt kluster.  Du bör bli kostnads fri att ge dem ett try-erbjudande som är tillgängliga utan kostnad.

_De här erbjudandena är en egen licens_. De förutsätter att du redan har uppfyllt rätt licenser med Oracle och att du har rätt licens för att köra erbjudanden i Microsoft Azure.

_Om du vill arbeta nära dina migrerings scenarier med teknik teamet som utvecklar dessa erbjudanden klickar du bara på knappen [kontakta mig](https://azuremarketplace.microsoft.com/en-us/marketplace/apps/oracle.oraclelinux-wls-cluster?tab=Overview) _ på [Azure Marketplace](https://azuremarketplace.microsoft.com/en-us/marketplace/apps/oracle.oraclelinux-wls-cluster?tab=Overview). Program chefer, arkitekter och tekniker kommer att gå tillbaka till dig inom kort och initiera samarbetet!

### <a name="oracle-weblogic-server-single-node"></a>Enskild nod för Oracle WebLogic-Server

Det här erbjudandet tillhandahåller en enskild virtuell dator och installerar Oracle WebLogic Server på den. Den skapar inte en domän eller startar administrations servern. Detta är användbart för scenarier med mycket anpassad domän konfiguration.

### <a name="oracle-weblogic-server-with-admin-server"></a>Oracle WebLogic-Server med administratörs Server

Det här erbjudandet tillhandahåller en enskild virtuell dator och installerar Oracle WebLogic Server på den. Den skapar en domän och startar administrations servern, vilket gör att du kan hantera domänen.

### <a name="oracle-weblogic-server-cluster"></a>Oracle WebLogic Server-kluster

Det här erbjudandet skapar ett kluster med hög tillgänglighet för virtuella Oracle WebLogic Server-datorer. Administrations servern och alla hanterade servrar startas som standard, vilket gör att du kan hantera domänen.

### <a name="oracle-weblogic-server-dynamic-cluster"></a>Dynamiskt kluster för Oracle WebLogic Server

Det här erbjudandet skapar ett hög tillgängligt och skalbart dynamiskt kluster av Oracle WebLogic Server Virtual Machines. Administrations servern och alla hanterade servrar startas som standard, vilket gör att du kan hantera domänen.
