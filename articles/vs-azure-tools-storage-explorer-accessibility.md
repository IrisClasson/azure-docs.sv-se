# <a name="storage-explorer-accessibility"></a>Hjälpmedel för Lagringsutforskaren

## <a name="screen-readers"></a>Skärmläsare

Storage Explorer stöder användningen av en skärmläsare på Windows och Mac. Följande skärmläsare rekommenderas för varje plattform:

Plattform | Skärmläsare
---------|--------------
Windows  | Nvda
Mac      | Berättarröst
Linux    | (skärmläsare stöds inte på Linux)

Om du stöter på ett hjälpmedelsproblem när du använder Storage Explorer öppnar du [ett problem på GitHub](https://github.com/Microsoft/AzureStorageExplorer/issues).

## <a name="zoom"></a>Zoom

Du kan göra texten i Storage Explorer större via inzoomning. Om du vill zooma in klickar du på **Zooma in** på Hjälp-menyn. Du kan också använda hjälpmenyn för att zooma ut och återställa zoomnivån till standardnivån.

![Zooma alternativ i hjälpmenyn][0]

Zoominställningen ökar storleken på de flesta gränssnittselement. Vi rekommenderar att du även aktiverar stora text- och zoominställningar för ditt operativsystem för att säkerställa att alla gränssnittselement skalas korrekt.

## <a name="high-contrast-themes"></a>Teman med hög kontrast

Storage Explorer har två hög kontrast teman, **Hög kontrast Ljus** och Hög kontrast **Dark**. Du kan ändra temat genom att välja in på menyn Hjälp > teman.

![Undermenyn Teman][1]

Temainställningen ändrar färgen på de flesta gränssnittselement. Det rekommenderas att också aktivera ditt OPERATIVSYSTEM matchande hög kontrast tema för att säkerställa att alla UI element är ordentligt färgade.
