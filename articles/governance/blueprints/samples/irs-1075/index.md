# <a name="overview-of-the-irs-1075-blueprint-sample"></a>Översikt över IRS 1075-skissexemplet

IRS 1075-skissexemplet ger styrningsriktlinjer med hjälp av [Azure Policy](../../../policy/overview.md) som hjälper dig att utvärdera specifika IRS 1075-kontroller. Den här skissen hjälper kunderna att distribuera en grundläggande uppsättning principer för all Azure-distribuerad arkitektur som måste implementera IRS 1075-kontroller.

## <a name="control-mapping"></a>Kontrollmappning

Avsnittet om kontrollmappning innehåller information om de principer som ingår i den här skissen och hur dessa principer täcker olika kontroller i IRS 1075. Resurser som tilldelats till en arkitektur utvärderas för icke-kompabilitet med tilldelade principer av Azure Policy. Mer information finns i [Azure Policy](../../../policy/overview.md).
