# <a name="azure-spatial-anchors-support-options"></a>Supportalternativ för Azure Spatial Anchors

## <a name="open-a-tech-support-ticket"></a>Öppna en teknisk supportbiljett

Så här öppnar du en teknisk supportbiljett i Azure Portal for Azure Spatial Anchors:

1. När [Azure-portalen](https://azure.microsoft.com/account/) är öppen väljer du hjälpikonen i den övre menyraden och väljer sedan **hjälp + supportknappen.**

![Hjälp och support för Azure Portal](./media/spatial-anchor-support.png)

1. När supportsidan för Hjälp + är öppen väljer du **+ Ny supportbegäran**.

![Ny supportbegäran för Azure portal](./media/spatial-anchor-support2.png)

1. När du fyller i supportbiljettfälten:

- Typ av problem: Teknisk
- Tjänst: Rumsliga ankare

![Biljettfält för Azure Portal-support](./media/spatial-anchor-support3.png)

## <a name="community-support"></a>Community-support

Support från användargruppen finns i [Azure Spatial Anchors on Stack Overflow](https://stackoverflow.com/questions/tagged/azure-spatial-anchors).

## <a name="provide-content-feedback"></a>Ge feedback om innehåll

Längst ned i varje innehållsartikel finns det en möjlighet att öppna ett GitHub-problem och ge feedback om Azure Spatial Anchor-dokumentationsinnehållet.

## <a name="provide-product-feedback"></a>Ge produktfeedback

Om du vill ge feedback delar du en idé eller ett förslag för Azure Spatial Anchors-tjänsten eller röstar på de idéer som andra har skickat in, på [Azure Spatial Anchors Feedback Forum](https://feedback.azure.com/forums/919252-azure-spatial-anchors).
