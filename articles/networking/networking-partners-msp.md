# <a name="about-azure-networking-msp-partners"></a>Om AZURE Networking MSP-partner

[Msp-partnerprogrammet (Azure Networking Managed Service Provider)](https://azure.microsoft.com/blog/enhancing-the-customer-experience-with-the-azure-networking-msp-partner-program/) gör det möjligt för nätverksfokuserade MSP:er, Telcos och Systems Integrators (SIs) att erbjuda moln- och hybridnätverkstjänster centrerade kring Azures portfölj av nätverksprodukter och -tjänster.

Azure Networking MSPs är en specialiserad uppsättning hanterade tjänsteleverantörer som tar itu med företagets molnnätverksbehov och utmaningar i alla aspekter av moln- och hybridnätverk. De hanterade nätverkstjänsterna omfattar en eller flera av följande tjänster: nätverksarkitektur, planering, distribution, drift, underhåll och optimering.

![Msp][0]

Kunder kan köpa MSP-hanterade tjänster erbjudanden i Azure Marketplace, eller direkt från MSP. MsPs kan distribuera och hantera Azure-nätverksresurser i kundernas prenumerationer och resursgrupper med hjälp av [Azure Lighthouse](https://azure.microsoft.com/services/azure-lighthouse/).
