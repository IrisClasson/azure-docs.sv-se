# <a name="quotas-and-limits-for-azure-container-instances"></a>Kvoter och begränsningar för Azure Container-instanser

Alla Azure-tjänster har vissa standardgränser och kvoter för resurser och funktioner. I den här artikeln beskrivs standardkvoter och begränsningar för Azure Container Instances.

Tillgängligheten för beräknings-, minnes- och lagringsresurser för Azure Container-instanser varierar beroende på region och operativsystem. Mer information finns i [Resurstillgänglighet för Azure Container Instances](container-instances-region-availability.md).
