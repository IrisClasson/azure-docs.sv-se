# <a name="manage-smart-groups"></a>Hantera smarta grupper
[Smart grupper](https://aka.ms/smart-groups) använder maskininlärningsalgoritmer grupp tillsammans aviseringar på grundval av delad förekomsten eller likheter, så att användaren kan nu hantera smarta grupper i stället för att hantera varje avisering enskilt. Den här artikeln visar dig hur du använder smart grupper i Azure Monitor.
1.  Om du vill se de smarta grupper som skapats för din aviseringsinstanser kan du antingen
     1. Klicka på **Smart grupper** från den **aviseringar sammanfattning** sidan    
    ![Övervakning](./media/alerts-managing-smart-groups/sg-alerts-summary.jpg)
     2. Klicka på aviseringar efter Smart grupper från sidan alla aviseringar   
     ![Övervakning](./media/alerts-managing-smart-groups/sg-all-alerts.jpg)
2.  Detta tar dig till listvyn för alla Smart grupper som skapats via din aviseringsinstanser. I stället för att leta efter flera aviseringar, kan du nu hantera smarta grupper i stället.   
![Övervakning](./media/alerts-managing-smart-groups/sg-list.jpg)
3.  När du klickar på någon Smart grupp öppnas sidan, där du kan se gruppering orsaken, tillsammans med aviseringarna som medlem. Den här aggregeringen låter dig arbeta med en enda smart grupp i stället för att leta efter flera aviseringar.   
![Övervakning](./media/alerts-managing-smart-groups/sg-details.jpg)
