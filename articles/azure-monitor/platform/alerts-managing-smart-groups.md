# <a name="manage-smart-groups"></a>Hantera smarta grupper

[Smarta grupper](https://aka.ms/smart-groups) använder maskininlärningsalgoritmer för att gruppera aviseringar på grundval av samförekomst eller likhet, så att användaren nu kan hantera smarta grupper i stället för att behöva hantera varje avisering individuellt. Den här artikeln kommer att gå igenom hur du kommer åt och använder smarta grupper i Azure Monitor.

1. Om du vill visa de smarta grupper som skapats för dina varningsinstanser kan du antingen

     1. Klicka på **Smarta grupper** från sidan Sammanfattning **av aviseringar**    
    ![Övervakning](./media/alerts-managing-smart-groups/sg-alerts-summary.jpg)
    
     1. Klicka på Aviseringar från smarta grupper från sidan Alla aviseringar   
     ![Övervakning](./media/alerts-managing-smart-groups/sg-all-alerts.jpg)

2. Detta tar dig till listvyn för alla smarta grupper som skapats via dina varningsinstanser. I stället för att gå igenom flera aviseringar kan du nu hantera de smarta grupperna i stället.   
![Övervakning](./media/alerts-managing-smart-groups/sg-list.jpg)

3. Om du klickar på en smart grupp öppnas informationssidan, där du kan se gruppningsorsaken, tillsammans med medlemsaviseringarna. Med den här aggregeringen kan du hantera en smart grupp i stället för att gå igenom flera aviseringar.   
![Övervakning](./media/alerts-managing-smart-groups/sg-details.jpg)
