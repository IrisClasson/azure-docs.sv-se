# <a name="how-to-pay-your-bill-for-microsoft-azure"></a>Så här betalar fakturan för Microsoft Azure

Den här artikeln gäller för kunder med ett Microsoft-avtal för kunder som registrerat sig för Azure via Azure-webbplatsen. [Kontrollera din åtkomst till en Microsoft-kundavtal](#check-access-to-a-microsoft-customer-agreement).

Standardbetalningsmetod för din faktureringsinformation profil kan vara ett kreditkort eller kontrollera/banköverföring överföring. Om du har Azure-krediter kan tillämpas de som automatiskt på fakturan varje faktureringsperiod. Om du vill bli berättigad att betala med kontroll/banköverföring, se [hur du kan betala med faktura](billing-how-to-pay-by-invoice.md).

Om din standardbetalningsmetod är Kontrollera/banköverföring, kan du skicka betalningen till den adress som anges längst ned på fakturan. Du kan också kan du betala online med ett kreditkort i Azure-portalen.

Om din standardbetalningsmetod är ett kreditkort, debiterar automatiskt vi ditt kreditkort varje faktureringsperiod när fakturan har skapats. Du kan också betala online i Azure portal om du vill använda ett annat kreditkort eller om automatisk avgiften misslyckas.

## <a name="pay-now-in-the-azure-portal"></a>Betala nu i Azure portal

Du måste vara fakturering profilens ägare, deltagare eller faktura manager för att betala för fakturor i Azure-portalen.

1. Logga in på [Azure-portalen](https://portal.azure.com).
1. Sök på **Cost Management + fakturering**.
1. I den **översikt** bladet klickar du på den *sammanfattning* fliken.

    ![Skärmbild som visar fliken Sammanfattning](./media/billing-how-to-pay-bill/billing-account-overview.png)

1. Om du har ett förfallodatum eller förfallodatum har passerat faktura, visas det totala beloppet.
1. Klicka på blå **betala nu** knappen.

    ![Skärmbild som visar betala nu](./media/billing-how-to-pay-bill/paynow-button-overview.png)

1. I det nya bladet klickar du på **Välj betalningsmetod** att välja ett befintligt kreditkort eller lägga till en ny.<!-- screenshot of select payment method link -->
1. När betalningsmetoden har valts klickar du på **betala nu**.

Status för fakturan kommer att uppdateras som betalas på bladet fakturor.
