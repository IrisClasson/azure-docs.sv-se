# <a name="change-your-azure-account-profile-information"></a>Ändra profilinformationen Azure-konto

Din profilinformation används för att kontakta dig om ditt Azure-konto. Du bör hålla det uppdaterat så att Microsoft kan meddela dig om viktiga meddelanden.

E-postadressen i administratörsprofil konto används av Microsoft för att skicka viktiga fakturering, tjänst och rekommendation-relaterade meddelanden om kontot. Vi rekommenderar att du anger en e-postadress som du kontrollerar regelbundet. Du kan uppdatera din profil för att ändra numret e adress och telefonnummer för kontot som kontoadministratör.

E-postadressen i en användares profil, konto-användare och inte är kontoadministratören, används av Microsoft för att skicka tjänstrelaterade meddelanden och rekommendationer. Alla fakturering meddelanden skickas till kontoadministratör. Du kan också uppdatera din profil om du vill ändra din e-postadress och telefonnummer, men den ändrar inte kontot e-postadress eller telefonnummer.

## <a name="change-your-profile-information"></a>Ändra profilinformationen

1. Logga in i [Azure-kontocentret](https://account.azure.com/Profile).
1. Välj knappen **Redigera information** och uppdatera information i **profilen**.

   ![Exemplet visar profilinformationen Azure](./media/billing-how-to-change-azure-account-profile/profile.png)

### <a name="send-notifications-to-an-additional-email-address"></a>Skicka meddelanden till en ytterligare e-postadress

Du kan ha tjänstmeddelanden och rekommendationer för skickas till en e-postadress förutom kontoadministratör e-postadressen som är associerade med ditt konto. Konto-administratörens e-postadress hämtar viktig information om alla prenumerationer i kontot, inklusive faktureringsrelaterade uppdateringar.

### <a name="change-the-account-administrator-email-address-in-your-profile"></a>Ändra kontoadministratör e-postadress i din profil

Kontoadministratören är den person som ställer in Azure-kontot och som tar emot viktiga e-postaviseringar om alla prenumerationer som ingår i kontot. Vi rekommenderar att du anger en e-postadress som kontoadministratören kontrollerar regelbundet.

### <a name="updating-your-profile-email-doesnt-update-your-sign-in-email-address"></a>Uppdatera din profil i e-post uppdateras inte din inloggning e-postadress

Uppdaterar profilen e-postadressen uppdateras inte din inloggning e-postadress. Om du vill ändra din inloggning e-postadress som du behöver överföra ägarskapet för kontot.

### <a name="updating-your-profile-address-doesnt-update-your-credit-card-billing-address"></a>Uppdatera din profil-adress uppdatera inte din faktureringsadress för kreditkort

Läs hur du uppdaterar din faktureringsinformation i [ändra det kreditkort som används för att betala för en Azure-prenumeration](billing-how-to-change-credit-card.md).

### <a name="you-cant-update-your-countryregion"></a>Du kan inte uppdatera land/region

Vi kan inte ändra land/region för ett befintligt konto på grund av tekniska begränsningar. Du kan dock skapa ett konto i den önskade land/regionen och sedan kontakta Azures support och migrera dina tjänster till det kontot.

## <a name="service-and-marketing-emails"></a>Tjänsten och marknadsföring e-postmeddelanden

Microsoft Azure-teamet skickar tjänstrelaterade e-postmeddelanden som informerar dig om viktiga aviseringar. Teamet skickar också konto rekommendation e-postmeddelanden, även kallat marknadsföringsmail, för att optimera och skala dina projekt.

Uppmanas du i den [Azure-portalen](https://portal.azure.com) att kontrollera eller uppdatera din e-postadress var 90: e dag. Meddelande syftet är att se till att Microsoft har uppdaterat kontaktinformation om de brådskande tjänstrelaterade meddelanden. Microsoft skickar e-postmeddelanden till din e-postadress för profilen med Azure-konto-relaterad information för:

- Tjänstmeddelanden
- Säkerhetsvarningar
- Fakturering syften
- Support
- Marknadsföring kommunikation
- Rekommendationer om bästa praxis, baserat på din Azure-användning

Ange den e-postadressen där du vill få information om ditt konto. Genom att ange en e-postadress, är du väljer i att få information från Microsoft.

![Exemplet visar uppdateringen din kontaktinformation box-enhet](./media/billing-how-to-change-azure-account-profile/update-contact-information.png)

### <a name="change-your-contact-email-address"></a>Ändra din e-postadress

Du kan ändra din e-postadress som på följande sätt:

**Alternativ 1**: Logga in på den [Azure Kontocenter](https://account.azure.com/Profile). Välj den **redigera information** och uppdatera sedan profilinformation för e-postadress. Uppdatera din profil e-postadress uppdatera inte din inloggning e-postadress.

**Alternativ 2**: Gå till den [kontaktinformation](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade) område i Azure-portalen och ange en e-postadress. Adressen som du anger får-tjänsten och marknadsföringsmaterial. E-postadressen i Azure-konto Center profil uppdateras för att matcha automatiskt. Uppdatera din profil e-postadress uppdatera inte din inloggning e-postadress.

**Alternativ 3**: I den [Azure-portalen](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade)väljer du ikonen med dina initialer eller bild. Välj snabbmenyn (**...** ). Välj sedan **Min kontaktinformation** från menyn och ange en e-postadress. Adressen som du anger får-tjänsten och marknadsföringsmaterial. E-postadressen i din Azure-konto Center profil uppdateras för att matcha automatiskt. Uppdatera din profil e-postadress uppdatera inte din inloggning e-postadress.

![Exempel som visar kontaktinformation uppdatering i Azure](./media/billing-how-to-change-azure-account-profile/azure-contact-information.png)

### <a name="opt-out-of-marketing-emails"></a>Avstår från marknadsföring e-postmeddelanden

Att välja bort marknadsföring e-postmeddelanden som tar emot:

1. Gå till den [rapportformuläret](https://account.microsoft.com/profile/permissions-link-request) att skicka en begäran med hjälp av din profil e-postadress. Du får en länk via e-post att uppdatera dina inställningar.
2. Den här länken visar typerna av marknadsföringsmaterial som e-postadressen är gjort det valet. Rensa alla ämnen som du vill välja bort och klicka sedan på **spara**.


![Exemplet visar hantera behörighetsalternativ för kommunikation](./media/billing-how-to-change-azure-account-profile/manage-communication-permissions.png)

### <a name="you-still-receive-service-notifications-if-you-opt-out-of-marketing-communications"></a>Du fortfarande ta emot tjänstmeddelanden om du avstår från marknadskommunikation

Om du har angett en e-postadress för kommunikation får tjänstmeddelanden, baserat på ditt konto. När du inte välja bort, om du endast välja bort marknadsföringsmaterial – du fortfarande ta emot tjänstmeddelanden.
