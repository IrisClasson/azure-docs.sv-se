# <a name="view-the-tax-documents-for-your-microsoft-customer-agreement"></a>Visa skattedokument för ditt Microsoft-kundavtal

Den här artikeln gäller för något faktureringskonto för en Microsoft-kundavtal. [Kontrollera om du har åtkomst till en Microsoft-kundavtal](#check-access-to-a-microsoft-customer-agreement).

## <a name="view-and-download-tax-documents"></a>Visa och hämta skattedokument

Du måste vara den ägare, deltagare, läsare eller faktura Manager av fakturering profilen att visa och ladda ned skattedokument. Läs mer om fakturering administratörsroller för Microsoft Customer avtal i [fakturering profil roller och uppgifter](billing-understand-mca-roles.md#billing-profile-roles-and-tasks).
<!-- TO DO: add link to manage access doc for MCA -->

1. Sök på **Cost Management + fakturering**.
2. Välj en profil för fakturering.
3. Välj **Fakturor**.
4. Hitta raden motsvarar skatte-dokumentet som du vill hämta fakturans i rutnätet för fakturan.
5. Klicka på ellipsknappen (`...`) i slutet av raden.
6. Välj **skatt dokumentet**. Beroende på land/region för din faktureringsinformation profil, kanske du ser flera skatt dokument per faktura.
