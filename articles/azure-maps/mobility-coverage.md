# <a name="azure-maps-mobility-coverage"></a>Azure Maps Mobility täckning

Azure Maps [mobilitetstjänsten](https://aka.ms/AzureMapsMobilityService) tillåter funktioner för offentliga överföring och bikeshare scooter resurs routning med multimodal utlösas alternativ. För offentliga tjänsten erbjuder söka efter i närheten stationer stoppas och linjer och innehåller information som schemalagda och i realtid offentliga överföra anslutningsbegäranden, lista över stoppas och dirigera guider, samt detaljerad och multimodala stegvisa resewebbplats och aviseringar. Mobilitetstjänsten returnerar också i realtid och information om statisk tillgänglighet till bil dela fordon och delade cyklar och skotrar.

Azure Maps har inte samma grad av information och Precision för alla städer och länder/regioner. Möjligheten att anropa data överförs beror på staden och data får inte innehålla alla alternativ för offentliga överföring och myndigheter som betjänar ort.

Följande tabell innehåller information om täckning för Azure Maps-mobilitetstjänsten.

| *      |Nästan fulla täckning för land/region.|
