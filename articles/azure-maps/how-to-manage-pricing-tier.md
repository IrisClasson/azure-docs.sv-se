# <a name="how-to-manage-pricing-tier-of-your-azure-maps-account"></a>Så här hanterar du prisnivån för ditt Azure Maps-konto

Du kan hantera prisnivån för ditt Azure Maps-konto via Azure portal. När du har skapat en [konto](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), du kan visa eller ändra prisnivå för ditt konto.

Mer information om hur du väljer rätt prisnivå, visas [välja rätt prisnivå](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier)

## <a name="view-pricing-tier"></a>Visa prisnivå

Om du vill visa vald prisnivån navigerar du till den **prisnivå** alternativ på inställningsmenyn.

![Visa valt prisnivå](./media/how-to-manage-pricing-tier/view-pricing-tier.png)

## <a name="change-pricing-tier"></a>Ändra prisnivå

När du har skapat ditt Azure Maps-konto kan du uppgradera eller nedgradera prisnivån för ditt Azure Maps-konto. Uppgradera eller nedgradera kan du gå till den **prisnivå** på inställningsmenyn och välj den prisnivå som inte är markerad. Klicka på knappen Spara för att spara vald prisalternativet nivå.

![Ändra prisnivå](./media/how-to-manage-pricing-tier/change-pricing-tier.png)

> [!Note]
> Du behöver inte skapa nya prenumerationsnycklar om du uppgraderar eller nedgraderar prisnivån för ditt Azure Maps-konto.
