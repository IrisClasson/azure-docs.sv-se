# <a name="manage-the-pricing-tier-of-your-azure-maps-account"></a>Hantera prisnivån för ditt Azure Maps-konto

Du kan hantera prisnivån för ditt Azure Maps-konto via Azure portal. När du har skapat en [konto](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), du kan visa eller ändra prisnivå för ditt konto.

Hämta mer information om [välja rätt prisnivå i Azure Maps](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier).

## <a name="view-your-pricing-tier"></a>Visa din prisnivå

Om du vill visa vald prisnivån navigerar du till den **prisnivå** alternativ på inställningsmenyn.

![Visa valt prisnivå](./media/how-to-manage-pricing-tier/view-pricing-tier.png)

## <a name="change-a-pricing-tier"></a>Ändra en prisnivå

När du har skapat ditt Azure Maps-konto kan du uppgradera eller nedgradera prisnivån för ditt Azure Maps-konto. Uppgradera eller nedgradera kan du gå till den **prisnivå** alternativ på inställningsmenyn. Välj den prisnivå som inte är markerad. Välj den **spara** för att spara vald prisalternativet nivå.

![Ändra en prisnivå](./media/how-to-manage-pricing-tier/change-pricing-tier.png)

> [!NOTE]
> Du behöver att generera nya prenumerationsnycklar om du uppgraderar eller nedgraderar prisnivån för ditt Azure Maps-konto.
