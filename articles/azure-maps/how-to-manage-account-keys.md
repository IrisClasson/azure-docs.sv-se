# <a name="manage-your-azure-maps-account"></a>Hantera ditt Azure Maps-konto

Du kan hantera ditt Azure Maps-konto via Azure-portalen. När du har ett konto kan du implementera API:erna på din webbplats eller i mobilappen.

Om du inte har en Azure-prenumeration kan du skapa ett [kostnadsfritt](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) konto innan du börjar.

## <a name="create-a-new-account"></a>Skapa ett nytt konto

1. Logga in på [Azure-portalen](https://portal.azure.com).

2. Välj **Skapa en resurs** i det övre vänstra hörnet i Azure-portalen.

3. Sök efter och välj **Kartor**. Välj sedan **Skapa**.

4. Ange informationen för ditt nya konto.

[![Ange kontoinformation för Azure Maps i Azure-portalen](./media/how-to-manage-account-keys/new-account-portal.png)](./media/how-to-manage-account-keys/new-account-portal.png#lightbox)

## <a name="delete-an-account"></a>Ta bort ett konto

Du kan ta bort ett konto från Azure-portalen. Navigera till sidan kontoöversikt och välj **Ta bort**.

[![Ta bort ditt Azure Maps-konto i Azure-portalen](./media/how-to-manage-account-keys/account-delete-portal.png)](./media/how-to-manage-account-keys/account-delete-portal.png#lightbox)

Du ser då en bekräftelsesida. Du kan bekräfta att ditt konto tas bort genom att skriva dess namn.
