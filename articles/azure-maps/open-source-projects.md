# <a name="azure-maps-community---open-source-projects"></a>Azure Maps-communityn – projekt med öppen källkod

Dessa initiativ med öppen källkod skapas och underhålls av Azure Maps-teamet. De är inte en del av standardprodukt- eller tjänsteerbjudanden.

Följande listar några av de mest populära Azure Maps-projekten och exemplen med öppen källkod.

| Projektnamn | Beskrivning |
|-|-|
| [Exempel på Azure Maps-kod](https://github.com/Azure-Samples/AzureMapsCodeSamples) | En samling kodexempel för att använda Azure Maps i webbaserade appar. |
| [Exempel på Azure Maps Gov-molnkod](https://github.com/Azure-Samples/AzureMapsCodeSamples) | En samling kodexempel för att använda Azure Maps via Azure Government Cloud. |
| [Azure Maps & Azure Active Directory-exempel](https://github.com/Azure-Samples/Azure-Maps-AzureAD-Samples) | En samling exempel som visar hur du använder Azure Active Directory med Azure Maps. | 
| [Implementera rumslig IoT-analys med Azure Maps](https://github.com/Azure-Samples/iothub-to-azure-maps-geofencing) | Att spåra och samla in relevanta händelser som inträffar i tid och rum är ett vanligt IoT-scenario. |
| [Exempel på bärbara Azure Maps-datorer](https://github.com/Azure-Samples/Azure-Maps-Jupyter-Notebook) | En samling pythonexempel med Hjälp av Azure Maps REST-tjänsterna. |
| [Bot Framework - Intressepunkt skicklighet](https://github.com/microsoft/botframework-solutions/tree/488093ac2fddf16096171f6a926315aa45e199e7/skills/csharp/pointofinterestskill) | Intressefärdigheten ger POI-relaterade funktioner till en virtuell assistent med Azure Maps med Azure Bot Service och Bot Framework. |

En längre lista som innehåller community-skapade projekt med öppen källkod finns [här](https://github.com/microsoft/Maps/blob/master/AzureMaps.md)

## <a name="supportability-of-open-source-projects"></a>Stöd för projekt med öppen källkod

Följande satser gäller i alla Azure Maps som skapats och underhålls projekt och exempel med öppen källkod:

- Azure Maps projekt och exempel med öppen källkod skapas av Microsoft och communityn.
- Azure Maps projekt och exempel med öppen källkod underhålls av Microsoft och communityn.
- Azure Maps projekt med öppen källkod och exempel använder tekniker som stöds och rekommenderas.
- Azure Maps projekt med öppen källkod och exempel är ett community-initiativ – personer som arbetar med initiativet till förmån för andra och har sitt vanliga vanliga jobb också.
- Azure Maps projekt och exempel med öppen källkod är INTE en produkt och stöds därför inte av Premier Support eller andra officiella supportkanaler.
- Azure Maps projekt med öppen källkod och exempel stöds på liknande sätt som andra projekt med öppen källkod som utförs av Microsoft med stöd från communityn av communityn.
