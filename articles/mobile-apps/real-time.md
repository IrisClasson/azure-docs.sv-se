# <a name="build-real-time-applications-with-azure-signalr-service"></a>Skapa realtidsprogram med Azure SignalR Service

Med [Azure SignalR Service](https://azure.microsoft.com/services/signalr-service/)kan du enkelt lägga till funktioner i realtid i program. Med den här realtidsfunktionen kan tjänsten skicka innehållsuppdateringar till anslutna mobila applikationer. Därför uppdateras appar utan att servern behöver avsökas eller skicka in nya HTTP-begäranden om uppdateringar.

## <a name="azure-signalr-service-features"></a>Azure SignalR-tjänstfunktioner
- Har brett klientstöd för ett brett spektrum av klienter, till exempel webb- och mobilwebbläsare, skrivbordsprogram, serverprocess, IoT-enheter och spelkonsoler.
- Tillåter flera instanser att arbeta tillsammans för att skala till miljontals anslutningar till mobila program. Den stöder också flera globala regioner för fragmentering, hög tillgänglighet eller katastrofåterställning.
- Erbjuder omfattande API:er för olika meddelandemönster för att skicka meddelanden till en viss anslutning, alla anslutningar eller en delmängd av anslutningar som tillhör en viss användare eller placerades i en godtycklig grupp.
