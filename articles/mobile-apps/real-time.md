# <a name="build-real-time-applications-with-azure-signalr-service"></a>Bygg real tids program med Azure SignalR-tjänsten

Med [Azure SignalR-tjänsten](https://azure.microsoft.com/services/signalr-service/)kan du enkelt lägga till real tids funktioner i program. Med den här funktionen i real tid kan tjänsten skicka innehålls uppdateringar till anslutna mobila program. Därför uppdateras appar utan att behöva avsöka servern eller skicka nya HTTP-begäranden om uppdateringar.

## <a name="azure-signalr-service-features"></a>Azure SignalR service-funktioner
- Har brett klient stöd för en mängd olika klienter, till exempel webb-och mobil webbläsare, skriv bords program, Server processer, IoT-enheter och spel konsoler.
- Tillåter att flera instanser arbetar tillsammans för att skala till miljon tals mobil program anslutningar. Det stöder också flera globala regioner för horisontell partitionering, hög tillgänglighet eller haveri beredskap.
- Innehåller omfattande API: er för olika meddelande mönster för att skicka meddelanden till en viss anslutning, alla anslutningar eller en delmängd av anslutningar som tillhör en viss användare eller placerats i en godtycklig grupp.
