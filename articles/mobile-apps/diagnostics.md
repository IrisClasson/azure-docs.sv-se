# <a name="monitor-failures-and-errors-in-real-time-for-your-mobile-application"></a>Övervaka fel och fel i realtid för din mobilapplikation
När ditt program används av tusentals användare, det är sannolikt att ha buggar. Användarna kan stöta på oväntade programfel. Ur användarperspektiv, allt de vill ha är ett fungerande och tillförlitligt program. Program som är buggy och misslyckas ger inte en bra användarupplevelse. Besvikna användare kommer att avinstallera appen, lämna en dålig recension, eller offentligt kritisera och klaga på sina erfarenheter.

Att förstå verkliga felbeteenden är viktigt för mobila applikationer. Jobbet som utvecklare är inte klart så fort du släpper till appbutiker. Det är viktigt att kontinuerligt övervaka och mäta prestanda för att säkerställa appstabilitet och upprätthålla användarnas tillfredsställelse. Vad du behöver är ett sätt att övervaka ditt program. Du måste förstå vad som hände när något går fel så att du kan prioritera och åtgärda problem som är avgörande för dina affärsbehov.

## <a name="importance-of-failure-monitoring"></a>Vikten av felövervakning
Som utvecklare måste du:
- Lär känna till information om programmets hälsa och hur det fungerar för användarna.
- Hitta grundorsaken och orsakerna till varför och var felet uppstod på en användares enhet.
- Var proaktiva vid triage problem och lösa fel att göra ditt program stabilt.
- Skapa bättre program och iterera snabbare för att ge en bra användarupplevelse så att användarna fortsätter att använda ditt program.

Dra nytta av följande tjänst för att övervaka ditt program för fel, diagnostisera problem och snabbt åtgärda dem för att ge en stabil och högkvalitativ mobil applikationsupplevelse.

## <a name="visual-studio-app-center"></a>Visual Studio App Center
[App Center Diagnostics-tjänsten](/appcenter/diagnostics/) hjälper dig att övervaka hälsotillståndet för ditt program. Det ger detaljerade insikter så att du kan förstå vad som hände och varför programmet misslyckades. App Center Diagnostics hjälper dig också att prioritera och hantera felrapporter.

**Huvudfunktioner**
- Automatiska fel och fel i realtid, smart grupperade, för att belysa de allvarligaste problemen och relevant information du behöver.
- Slutför stackspårningar och enhetsdata som ger kontext och klarhet som hjälper dig att identifiera grundorsaken.
- Kraftfull sökfunktion som hjälper dig att hitta de problem du bryr dig mest om.
- Analytics-integrering så att du kan förstå användarnas beteenden och se händelser som leder fram till ett fel.
- Anpassade egenskaper, användar-ID och bifogade filer så att du kan lägga till fler sammanhang för att diagnostisera dina problem.
- Aviseringar, meddelanden och integreringar av felspårare så att du kan hålla koll på nya och oväntade fel som påverkar användarna.
- Enkel SDK-integrering med en liner för att komma igång med den här tjänsten inom några minuter.
- Plattformsstöd för iOS, Android, macOS, tvOS, Xamarin, React Native, Unity, Cordova, WPF och WinForms.
