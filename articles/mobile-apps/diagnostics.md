# <a name="monitor-crashes-and-errors-in-real-time-for-your-mobile-application"></a>Övervaka krascher och fel i real tid för ditt mobil program
När ditt program är i jokertecken och används av tusentals användare, är det sannolikt att du har buggar och att användarna kommer att köra oväntade program krascher. Från användar perspektivet är alla de vill ha ett fungerande och tillförlitligt program. Program som är fel sökning och krasch ger inte en korrekt användar upplevelse. Besvikna användare kommer att avinstallera appen, lämna en felaktig granskning eller offentligt criticize och klaga på sin erfarenhet.

Därför är det viktigt att förstå det verkliga krasch beteendet för mobila program. Ditt jobb som utvecklare är inte slutfört så snart du släpper till App Store. Det är lika viktigt att kontinuerligt övervaka och mäta prestanda för att säkerställa stabiliteten i appen och upprätthålla användarnas tillfredsställelse. Det du verkligen behöver är ett sätt att övervaka programmet och förstå vad som gick fel så att du kan ägna din tid åt att prioritera och åtgärda problem som är viktiga för dina affärs behov.

## <a name="importance-of-crashes-monitoring"></a>Betydelsen av krascher som övervakar
Som utvecklare måste du:
- Läs mer om **programmets hälso tillstånd** och hur det fungerar **för slutanvändare**.
- **Hitta rotor saken och** orsaken till varför och var kraschen inträffade på en användares enhet.
- Var **förebyggande på prioritering problem och lös krascher** för att göra programmet stabilt.
- Skapa bättre program och **iterera snabbare** för att ge en bra användar upplevelse så att användarna kommer tillbaka till att använda programmet.

Dra nytta av tjänsten nedan för att övervaka ditt program för krascher, diagnostisera problem och snabbt åtgärda dem för att tillhandahålla en stabil och högkvalitativ mobil program upplevelse.

## <a name="visual-studio-app-center"></a>Visual Studio App Center
[App Center Diagnostics](/appcenter/diagnostics/) service hjälper dig att övervaka hälso tillståndet för ditt program, ger detaljerade insikter om vad som hände och varför programmet kraschar, och gör det möjligt att prioritera och hantera krasch rapporter.

**Viktiga funktioner**
- Real tid, automatiska **krascher och fel**, som är smart grupperade, för att markera de mest allvarliga problemen och relevant information som du behöver.
- **Slutför stack spårningar och enhets** data som innehåller kontext och klarhet för att hjälpa dig att identifiera rotor saken.
- **Kraftfull Sök funktion** som hjälper dig att hitta de problem du bryr dig om.
- Analys integrering så att du kan förstå dina användar beteenden och se händelser som leder till en krasch.
- Anpassade egenskaper, användar-ID och bifogade filer så att du kan lägga till ytterligare kontext som kan hjälpa dig att diagnostisera dina problem.
- **Aviseringar, aviseringar och fel spårs integreringar** så att du kan hålla koll på nya och oväntade krascher som påverkar din användare.
- **En enda liner SDK-integration** för att komma igång med den här tjänsten inom några minuter.
- **Plattforms stöd** – iOS, Android, MacOS, tvOS, Xamarin, reakta Native, Unity, Cordova, WPF, WinForms.
