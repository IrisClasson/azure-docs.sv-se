# <a name="content-delivery-network-on-azure-with-azure-content-delivery-network"></a>Nätverk för innehållsleverans på Azure med Azure Content Delivery Network
[Azure Content Delivery Network](https://azure.microsoft.com/services/cdn/) erbjuder utvecklare en global lösning för att snabbt leverera innehåll med hög bandbredd som videor, bilder eller statiska filer till användare genom att cachelagring deras innehåll till hundratals närvaropunkter över hela världen. Content Delivery Network kan också påskynda dynamiskt innehåll, som inte kan cachelagras, för att minska laddningstiderna, spara bandbredd och öka svarstiden. Du får dessa fördelar oavsett om du utvecklar eller hanterar webbplatser eller mobilappar eller kodar och distribuerar strömmande media, spelprogramvara, uppdateringar av inbyggd programvara eller IoT-slutpunkter.

## <a name="azure-content-delivery-network-features"></a>Funktioner i Azure Content Delivery Network
  - Arbeta sömlöst med Azure-tjänster som webbprogram, medietjänster, lagring och molntjänster.
  - Hantera enkelt trafikspikar och tunga laster.
  - Integrera med Akamai och Verizon för att bygga och skräddarsy innehållsleveransstrategi.
  - Använd dynamisk platsacceleration.
  - Kontrollera cachelagringsbeteende för innehållsleveransnätverk med cachelagringsregler.
  - Få stöd för en anpassad HTTPS-domän.
  - Visa Azure-diagnostikloggar för att se kärnanalys och exportera dem till Azure Blob-lagring, Azure Event Hubs eller Azure Monitor.
  - Förbättra prestanda genom att komprimera filer.
  - Använd geofiltrering för att begränsa innehållsleveransnätverkets innehåll efter land eller region.
