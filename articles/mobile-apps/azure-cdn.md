# <a name="content-delivery-network-on-azure-with-azure-cdn-service"></a>Content Delivery Network på Azure med Azure CDN-tjänst
[Azure CDN](https://azure.microsoft.com/services/cdn/) erbjuder utvecklare en global lösning för att snabbt leverera innehåll med hög bandbredd, till exempel videor, bilder eller statiska filer till användare genom att cachelagra sitt innehåll till hundratals punkter av närvaro (pop) över hela världen. Azure CDN kan också påskynda dynamiskt innehåll, som inte kan cachelagras och gör att du kan minska inläsnings tider, spara bandbredd och öka svars tider – oavsett om du utvecklar eller hanterar webbplatser eller mobila program, eller kodar och distribuerar strömmande Media, spel program, uppdateringar av inbyggd program vara eller IoT-slutpunkter.

## <a name="azure-cdn-features"></a>Funktioner i Azure CDN
  - Fungerar **sömlöst** med Azure-tjänster, inklusive webb program, Media Services, lagring och Cloud Services.
  - **Hantera enkelt trafik toppar och tung belastningar**.
  - **Integrering med Akamai och Verizon** för att bygga och skräddarsy strategin för innehålls leverans.
  - Acceleration av dynamisk webbplats.
  - Styra Azure CDN cachelagring med **regler för cachelagring**.
  - HTTPS- **stöd för anpassade domäner**.
  - **Azure Diagnostics-loggar** för att Visa Core Analytics och exportera dem till Blob Storage, Event Hubs eller Azure Monitor.
  - Förbättra prestanda genom att **Komprimera filer**.
  - **Geo-filtrering** för att begränsa Azure CDN innehåll efter land/region.
