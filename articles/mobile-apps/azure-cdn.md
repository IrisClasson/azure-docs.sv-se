# <a name="content-delivery-network-on-azure-with-azure-content-delivery-network"></a>Content Delivery Network på Azure med Azure Content Delivery Network
[Azure Content Delivery Network](https://azure.microsoft.com/services/cdn/) erbjuder utvecklare en global lösning för att snabbt leverera innehåll med hög bandbredd, till exempel videor, bilder eller statiska filer till användare genom att cachelagring av sitt innehåll till hundratals punkter av närvaro i hela världen. Content Delivery Network kan också påskynda dynamiskt innehåll, som inte kan cachelagras, för att minska inläsnings tider, spara bandbredd och öka svars tiderna. Du får de här fördelarna oavsett om du utvecklar eller hanterar webbplatser eller mobilappar eller kodar och distribuerar strömmande media, spelprogram, uppdateringar av inbyggd program vara eller IoT-slutpunkter.

## <a name="azure-content-delivery-network-features"></a>Funktioner i Azure Content Delivery Network
  - Arbeta sömlöst med Azure-tjänster som webb program, Media Services, lagring och moln tjänster.
  - Hantera enkelt trafik toppar och tung belastningar.
  - Integrera med Akamai och Verizon för att bygga och skräddarsy strategin för innehålls leverans.
  - Använd dynamisk webbplats acceleration.
  - Styra Content Delivery Network cachelagring med regler för cachelagring.
  - Få HTTPS-stöd för anpassade domäner.
  - Visa Azure Diagnostics-loggar för att se Core Analytics och exportera dem till Azure Blob Storage, Azure Event Hubs eller Azure Monitor.
  - Förbättra prestanda genom att komprimera filer.
  - Använd filter för att begränsa Content Delivery Network innehåll efter land eller region.
