# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>AI-driven moln tjänst med Azure Kognitiv sökning
[Azure kognitiv sökning](https://azure.microsoft.com/services/search/) är en moln lösning för Sök-som-en-tjänst som tillhandahåller utvecklares API: er och verktyg för att lägga till en omfattande Sök upplevelse över privat, heterogent innehåll i webb-, mobil-och företags program. Din anpassade kod anropar data inmatning (indexering) för att skapa och läsa in ett index. På den andra sidan utfärdar din program kod frågor om begär Anden och hanterar svar. Sök funktionen definieras i-klienten med hjälp av funktioner från Azure Kognitiv sökning, med frågekörningen över ett beständigt index som du skapar, äger och lagrar i din tjänst.

## <a name="azure-cognitive-search-features"></a>Funktioner i Azure Kognitiv sökning
- **Texts ökning i fri hands text**: ger fullständig texts ökning, enkel frågesyntax och Lucene-frågesyntax.
- **Enkla bedömnings profiler**: modell relevans som en funktion av värden i själva dokumenten.
- Geolokalisering: utforskar data baserat på ett Sök resultats närhet till en fysisk plats.
- **Funktioner för användar upplevelse**: Ange Autoavsluta, Sök förslag, sortering och växling.
- **Kognitiv sökning**: kan tillämpas på en indexerings pipeline för att extrahera text information från rå data för bild-och text analyser.
- **Kunskaps lager**: sparar de omfattande uppgifter som skapas vid indexering.
- **Data källor**: extrahera sökbart innehåll i primära data lager när du är ansluten till Azure SQL Database, Azure Cosmos DB och Azure Blob Storage.
- **Guiden Importera data**: konfigurerar indexerare. 
- **Index designern**: skapar ett index. 
- **Sök Utforskaren**: testar frågor och begränsar bedömnings profilerna.

Du hittar en detaljerad beskrivning av Azure Kognitiv sökning- [funktioner](/azure/search/search-what-is-azure-search#feature-descriptions).
