# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>AI-driven molntjänst med Azure Cognitive Search
[Azure Cognitive Search](https://azure.microsoft.com/services/search/) är en molnlösning för sökning som en tjänst som ger utvecklare API:er och verktyg för att lägga till en omfattande sökupplevelse över privat, heterogent innehåll i webb-, mobil- och företagsprogram. Den anpassade koden anropar datainmatning (indexering) för att skapa och läsa in ett index. Å andra sidan utfärdar programkoden frågebegäranden och hanterar svar. Sökupplevelsen definieras i klienten med hjälp av funktioner från Azure Cognitive Search, med frågekörning över ett beständigt index som du skapar, äger och lagrar i din tjänst.

## <a name="azure-cognitive-search-features"></a>Funktioner i Azure Cognitive Search
- **Freeform textsökning:** Ger fulltextsökning, enkel frågesyntax och Lucene-frågesyntax.
- **Enkla bedömningsprofiler**: Modellreleva som en funktion av värden i själva dokumenten.
- **Geosearch**: Utforskar data baserat på närheten till ett sökresultat till en fysisk plats.
- **Funktioner för användarupplevelser**: Ange komplettera automatiskt, sökförslag, sortering och växling.
- **Kognitiv sökning**: Kan användas på en indexeringspipeline för att extrahera textinformation från rått innehåll för bild- och textanalys.
- **Kunskapslager**: Sparar anrikningar som skapas under indexeringen.
- **Datakällor**: Extrahera sökbart innehåll i primära datalager när du är ansluten till Azure SQL Database, Azure Cosmos DB och Azure Blob-lagring.
- **Guiden Importera data**: Konfigurerar indexerare. 
- **Indexdesigner**: Står upp ett index. 
- **Sökutforskare:** Testar frågor och förfinar bedömningsprofiler.

Hitta en detaljerad beskrivning av Azure Cognitive [Search-funktioner](/azure/search/search-what-is-azure-search#feature-descriptions).
