# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>Geospatiala API: er för att lägga till kartor, lösningar för spatial analys och mobilitet med Azure Maps

[Azure Maps](https://azure.microsoft.com/services/azure-maps/) tillhandahåller plattforms plattform internt i Azure-molnet, erbjuder en portfölj med geospatiala tjänster som använder nya mappnings data för att tillhandahålla korrekt geografisk kontext till webb-och mobil program. Det gör det möjligt för utvecklare, företag och ISV att skapa plats medveten program, IoT, mobilitet, logistik och till gångs spårnings lösningar.

## <a name="azure-maps-features"></a>Azure Maps funktioner
- **Sök** innehåller information om intresse punkter i närheten (POI), återge platser på en karta och koder för att få motsvarande latitud-/longitud-koordinater.
- Förbättra res tiden och undvika gridlock med **trafik**.
- Integrera tydliga och lättlästa **kartor** i din lösning.
- **Routning** för att presentera de kortaste eller snabbaste vägarna som är tillgängliga – till flera mål i taget, eller för att tillhandahålla specialiserade vägar och fakta för förhands visning, cyklister och kommersiella fordon.
- Med **tids zoner** är det enkelt för användarna att se hur lång tid det är på plats i världen.
- **Spatiala åtgärder** för att förbättra plats information.
- **Geolokalisering** för att leta upp ett land i en IP-adress.
- **Data tjänst** för att ladda upp och lagra geospatiala data för användning med spatiala åtgärder eller bild sammansättning.
- **Mobility** tillhandahåller lokaliserings information i real tid på närliggande offentliga överförings tjänster, inklusive stopp, routningsinformation och res tids uppskattningar.
