# <a name="cognitive-services-helps-to-solve-business-problems-by-using-ai"></a>Cognitive Services hjälper till att lösa affärsproblem genom att använda AI
[Azure Cognitive Services](https://azure.microsoft.com/services/cognitive-services/) tillhandahåller kraftfulla tjänster för att använda dina appar med intelligenta algoritmer för att se, höra, tala, förstå och tolka dina användares behov för att omvandla din verksamhet med artificiell intelligens (AI). Du kan göra dina mobila applikationer smarta, engagerande och kontextuellt relevanta för användarna. Utvecklare kan använda API: er, SDK:er och tjänster för att lägga till AI i sina erbjudanden och hjälpa till att skapa intelligenta program utan att ha direkta AI- eller datavetenskapliga färdigheter.

## <a name="cognitive-services-capabilities"></a>Funktioner för Cognitive Services
- **Visuellt innehåll**
    - Bildmärkning, miniatyrer
    - OCR (handskriftsigenkänning)
    - Ansiktsspårning
    - Emotion upptäckt
    - Videoinsikter
    - Bild- och videomodering
    - Anpassad bildigenkänning

- **Tal**
    - Tal till text (taltranskription)
    - Anpassad talavskription (komplexa ord, bullriga miljöer)
    - Text till tal
    - Högtalar-ID och autentisering
    - Talöversättning i realtid

- **Språk**
    - Kontextuell språköverensning, anpassad avsiktsanalys
    - Sentimentanalys, nyckelfrasidentifiering
    - Textöversättning upp till 60 språk
    - Textmoderering
    - Stavningskontroll

- **Beslut** 
    - Innehållsmoderator
    - Leverans av personligt innehåll
    - Avvikelsedetektor

- **Sök**
    - Automatiska sökförslag 
    - Förstärkning av enhetsinformation
    - Omfattande nyheter, bild, video, webb och lokala företag enhet sökning
    - Skräddarsydda och anpassade sökupplevelser

- Stöd för behållare för Cognitive Services
