# <a name="mobile-application-development-in-microsoft"></a>Utveckling av mobilprogram i Microsoft
Beslutsfattare, utvecklingsgrupper och IT-avdelningar behöver implementera enhetliga strategier för utveckling av mobilprogram som:
- Fokuserar på de typer av program som passar företagets verksamhetsmål, till exempel B2E-program (företag till anställd), B2B-program (företag till företag) eller B2C-program (företag till konsument).
- Bestämmer om du ska investera i att skapa program för iOS, Android eller Windows med språk för en plattform, eller använda en plattformsoberoende metod.
- Levererar serverdelstjänster som kan skalas.
- Integrerar sömlöst med serverdelsdata, oavsett om de lagras online eller offline.
- Skapar säkra program med funktioner för autentisering och enhetshantering.
- Levererar kontinuerligt och snabbt.
- Ansluter till serverdelssystem, antingen i molnet eller lokalt.
- Integrerar med befintliga verktyg och processer.
- Utvecklar programmet med verksamheten.
- Gör publiceringsprocessen snabbare och låt utvecklarna fokusera på att skapa program av hög kvalitet.

Den här dokumentationsserien utforskar tjänster som tillhandahålls av Microsoft och som du kan använda för att skapa säkra, pålitliga, skalbara och molnbaserade mobilprogram.

**Målgrupp**

Den här dokumentationsserien är riktad till dem som är utvecklare som är nybörjare eller experter på att skapa mobilprogram och som vill lära sig mer om de molntjänster som kan hjälpa dem att bli framgångsrikare inom programutveckling. Du får en kortfattad översikt över alla Microsofts tjänster, verktyg och tekniker som du behöver känna till. Det finns inga kodexempel eller demonstrationer i den här dokumentationen. Syftet med dokumentationen är att ge dig en översikt över programutvecklingsprocessen och de associerade tjänster som används på en hög nivå. Varje artikel innehåller länkar till andra resurser för mer ingående information. Varje del av serien kan läsas separat, så att du kan välja enbart artiklar som du är intresserad av.
