# <a name="perform-automated-testing-on-thousands-of-mobile-devices-simultaneously"></a>Utföra automatiserad testning på tusentals mobila enheter samtidigt
Testning kan vara utmanande och allt mer komplex på grund av fragmentering av enhet, program komplexitet, snabba lanserings cykler, korta sessioner och höga användar förväntningar. Du måste regelbundet se till att du levererar noggrant testade program som ger en bra kund upplevelse. Testning är inte en enkel uppgift och är inte en bra användning av utvecklar tiden för att testa varje version manuellt.

Testning av utmaningar kan hanteras effektivt. Du kan testa mobila program över en mängd olika enheter genom att använda automatiserad testning. Du kan använda moln tjänster för att påskynda lanseringen av mobilappar och lösa testnings utmaningar som lägger till Flask halsar i utvecklingen och distributionen av program.

## <a name="importance-of-automated-testing"></a>Betydelsen av automatiserad testning
- Öka din test täckning genom att rikta in sig på olika form faktorer, anslutnings typer och OS-versioner.
- Spara tid, pengar och resurser genom att testa på olika enheter och OS-täckning.
- Förbättra noggrannheten genom att undvika upprepade manuella tester och få mer tid att skapa nya automatiserade program varu test och hantera avancerade funktioner.
- Testa kontinuerligt, analysera programmets prestanda och hitta och åtgärda problem innan användarna kan köra dem och frigöra snabbt.
- Leverera program med hög kvalitet genom att se till att dina kunder på ett brett utbud av enheter får bästa möjliga upplevelse.

## <a name="visual-studio-app-center"></a>Visual Studio App Center
[App Center Test](/appcenter/test-cloud/) är en molnbaserad tjänst som du kan använda för att köra automatiserade UI-tester på tusentals verkliga enheter i molnet över Android och iOS för interna och hybrid mobilappar. Med över 2 000 enheter kan du skriva test skript som kan köras automatiskt för att göra testningen enkel.

**Huvudfunktioner**
   - Tester som körs på verkliga enheter som inte är rotade eller som är jailbrokade för att ge bästa möjliga uppskattning av användarnas miljöer.
   - Skärm dum par samlas in och bifogas automatiskt till varje test steg. De kan visas sida vid sida för alla riktade enheter i en test körning.
   - Enhets loggar för alla mål enheter i en test körning och stack spår för misslyckade test steg.
   - Samtidiga test körningar på flera enheter samtidigt för att påskynda tiden till resultat.
   - Populära test Ramverks stöd som Appium (Java med JUnit) för Android och iOS, espresso (Java) för Android, Calabash för Android och iOS, Xamarin. UITest för Android och iOS och XCUITest för iOS.
   - Plattforms stöd för iOS-, Android-Xamarin och reagera internt.
