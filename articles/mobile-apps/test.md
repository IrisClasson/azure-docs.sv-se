# <a name="perform-automated-testing-on-thousands-of-mobile-devices-simultaneously"></a>Utföra automatiserade tester på tusentals mobila enheter samtidigt
Testning kan vara utmanande och alltmer komplexa på grund av enhetsfragmentering, programkomplexitet, snabba utgivningscykler, korta sessioner och höga användarförväntningar. Du måste regelbundet se till att du levererar noggrant testade program som ger en bra kundupplevelse. Testning är inte en lätt uppgift, och det är inte en bra användning av utvecklare tid att manuellt testa varje utgåva.

Testningsutmaningar kan hanteras effektivt. Du kan testa mobila program på en mängd olika enheter med hjälp av automatisk testning. Du kan använda molntjänster för att påskynda lanseringen av mobilappar och övervinna testning utmaningar som lägger flaskhalsar till utveckling och distribution av program.

## <a name="importance-of-automated-testing"></a>Vikten av automatiserad testning
- Öka testtäckningen genom att rikta in dig på olika formfaktorer, anslutningstyper och OS-versioner.
- Spara tid, pengar och resurser genom att testa på olika enheter och OS-täckning.
- Förbättra noggrannheten genom att undvika repetitiva manuella tester och få mer tid att skapa nya automatiserade programvarutester och hantera komplexa funktioner.
- Testa kontinuerligt, analysera programmets prestanda och hitta och åtgärda problem innan användarna stöter på dem och släpp snabbt.
- Leverera högkvalitativa program genom att se till att dina kunder på ett brett spektrum av enheter får den bästa upplevelsen.

## <a name="visual-studio-app-center"></a>Visual Studio App Center
[App Center Test](/appcenter/test-cloud/) är en molnbaserad tjänst som du kan använda för att köra automatiserade gränssnittstester på tusentals verkliga enheter i molnet i Android och iOS för inbyggda och hybridmobilappar. Med över 2 000 enheter kan du skriva testskript som kan köras automatiskt för att göra testningen enkel.

**Huvudfunktioner**
   - Tester som körs på riktiga enheter som inte är rotade eller jailbroken för att ge den bästa tillnärmningen av användarnas miljöer.
   - Skärmbilder fångas automatiskt och kopplas till varje teststeg. De kan visas sida vid sida för alla riktade enheter i en testkörning.
   - Enhetsloggar för alla målenheter i en testkörning och stapla spårningar för misslyckade teststeg.
   - Samtidiga testkörningar på flera enheter samtidigt för att påskynda tiden till resultaten.
   - Populära testramsstöd som Appium (Java med JUnit) för Android och iOS, Espresso (Java) för Android, Calabash för Android och iOS, Xamarin.UITest för Android och iOS och XCUITest för iOS.
   - Plattformsstöd för iOS, Android Xamarin och React Native.
