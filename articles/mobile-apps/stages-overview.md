# <a name="different-stages-in-mobile-application-development"></a>Olika steg i utvecklingen av mobil program
Att skapa ett mobilt program omfattar flera steg och utvecklare kan enkelt dra nytta av Microsofts tjänster, verktyg och tekniker för att skapa mobil program med hög klass.

## <a name="app-development"></a>Apputveckling
Utvecklare kan bygga interna klient program med en enda plattform med språk som mål-C och Java, plattforms oberoende appar med Xamarin, reagera inbyggd, Unity och hybrid med Cordova. De kan använda IDE: er och kod redigerare, till exempel Visual Studio, Visual Studio för Mac eller Visual Studio Code på PC-och Mac-datorer för att skapa klient program.

## <a name="plan-and-design-with-back-end-services"></a>Planera och utforma med backend-tjänster
 Microsoft tillhandahåller en mängd olika tjänster för att konfigurera en säker, skalbar och tillförlitlig Server del för ditt mobil program. Utvecklare kan skapa server lösa API: er och utlösare utan att behöva underhålla sina egna servrar. De kan utnyttja tjänster, t. ex. push-meddelanden, data synkronisering offline, autentisering och data, som tillsammans bildar en omfattande och mycket mobil Server del som en tjänst som har stöd för en mängd olika app-konstruktions scenarier. Att skapa en server dels tjänst har många krav:
   - Gräns för lagring av data på enheten.
   - Synkronisera data över flera enheter.
   - Se till att programmet körs utan nätverks anslutning.
   - Skicka meddelanden till användare.
   - Minska batteri förbrukningen.
   - Bygg skalbar, tillförlitlig och säker server del.
   - Automatisk Server Maintainence.

## <a name="devops-and-continuous-monitoring"></a>DevOps och kontinuerlig övervakning
Utvecklare som skapar mobila och molnbaserade program kan dra nytta av kontinuerlig integrering (CI), kontinuerlig leverans (CD) och kontinuerlig övervakning av tjänster för att skapa inbyggda versioner för alla plattformar kör UI automation-tester på tusentals fysiska enheter, automatisera sina versioner genom beta test kanaler eller distribuera direkt till app-butiker. Dessutom kan de samla in användnings analys, krasch-och undantags information för sina program.

## <a name="additional-services"></a>Ytterligare tjänster
Det finns många Azure-tjänster som kan förbrukas beroende på typ av och typ av program som skapas.
  - Azure Kognitiv sökning för att söka efter alla typer av interna data med snabb frågor.
  - Azure Cognitive Services att använda artificiell intelligens i ditt program.
  - Azure SignalR för att aktivera kommunikation i real tid i ditt program.
