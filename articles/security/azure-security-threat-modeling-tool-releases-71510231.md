# <a name="threat-modeling-tool-update-release-71510231---1112018"></a>Threat Modeling Tool uppdateringsutgåvan 7.1.51023.1 - 11/1/2018

Som det ursprungligen anges i den [GA viktig](./azure-security-threat-modeling-tool-releases-71509112.md), har vi släppt en uppdatering (7.1.51023.1) till Microsoft Threat Modeling Tool som användare av förhandsversionen (förhandsversion klienter med version < 7.1.50911.2) måste du uppgradera till den stödda GA-versionen. Den här versionen innehåller inte några nya funktioner och korrigeringar.

- Användare av förhandsversionen kommer automatiskt att hämta uppgraderingen när klienten öppnas. Om du väljer att inte installera den nya uppdateringen, stängs förhandsversionen av verktyget.
- Användare av GA-versionen av verktyget uppmanas att välja om de vill uppgradera.
- Användare av 2016-versionen av verktyget kommer att påverkas inte.
