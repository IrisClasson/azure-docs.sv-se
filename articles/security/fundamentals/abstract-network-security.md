# <a name="azure-network-security"></a>Azure-nätverkssäkerhet
## <a name="abstract"></a>Abstrakt
Azure nätverks tjänster maximerar flexibilitet, tillgänglighet, återhämtning, säkerhet och integritet efter design. Den här white paper innehåller information om nätverksfunktioner i Azure. Det beskriver också hur kunder kan använda inbyggda säkerhetsfunktioner i Azure för att skydda sina informations till gångar.
De avsedda mål grupperna för den här white paper är:
- Tekniska chefer, nätverks administratörer och utvecklare som söker efter säkerhetslösningar som är tillgängliga och som stöds i Azure.
- Små och medel stora företag eller affärs process chefer som vill ha en översikt över Azure-teknikerna och tjänsterna som är relaterade till nätverks säkerhet i det offentliga Azure-molnet.
