# <a name="azure-security-mvp-program"></a>Azure Security MVP-program

Microsoft Most Valuable profession ALS (MVP) är community-ledare som har visat ett förhöjtt engagemang för att hjälpa andra. MVP gör det möjligt för andra att få ut mesta möjliga av sina erfarenheter med Microsofts tekniker. De delar sina passion, verkliga kunskaper och teknisk expertis med communityn och med Microsoft.

Microsoft Azure känner nu igen community-experter med särskilda kunskaper om Azure-säkerhet. Microsoft MVP: er kan tilldelas MVP: er i Microsoft Azure i Azures säkerhets bidrags områden.

![Microsoft MVP](./media/mvp/azure-security-mvp-fig1.png)

Det finns inget riktmärke för att bli en MVP. Detta är delvis eftersom det varierar beroende på teknik och dess livs cykel. Några av villkoren omfattar:

- Effekten av en kandidat utvärderas bidrag till online forum som Microsoft Answers, TechNet och MSDN
- Wikis och onlineinnehåll
- Konferenser och användar grupper
- Poddsändningar, webbplatser, Bloggar och sociala medier
- Artiklar och böcker.
