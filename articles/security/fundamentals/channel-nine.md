[Hantera hemligheter för Azure-appar](https://channel9.msdn.com/events/Build/2016/P456)

Varje allvarlig app som du distribuerar på Azure har kritiska hemligheter – anslutnings strängar, certifikat och nycklar. Silly misstag vid hantering av dessa hemligheter leder till allvarliga konsekvenser – läckor, avbrott, överträdelser av efterlevnad. I takt med att flera av de senaste undersökningarna pekar ut kan Silly misstag orsaka fyra gånger mer data intrång än angripare. I den här sessionen går vi igenom några metod tips för att hantera viktiga program hemligheter. Dessa bästa metoder kan likna vad som är vanligt, men många utvecklare behöver inte göra det. Vi går också igenom hur du använder Azure Key Vault för att implementera dessa metod tips. Som en extra förmån kan du följa dessa metoder för att demonstrera kompatibilitet med standarder som SOC. De första 10 minuterna i sessionen är nivå 100 och de gäller för alla molnappar som du utvecklar på vilken plattform som helst. Resten är nivå 200-300 och fokuserar på appar som du bygger på Azure-plattformen.

[Skydda Azure-Virtual Network med hjälp av nätverks säkerhets grupper med Narayan Annamalai](https://channel9.msdn.com/Shows/Azure-Friday/Sucruing-your-Azure-Virtual-Network-using-Network-ACLs-with-Narayan-Annamalai)

Narayan Annamalai-chef lär dig att använda nätverks säkerhets grupper i en Azure-Virtual Network. Du kan skapa kontroll åtkomst till objekt i Azure via undernät och nätverk! Du lär dig hur du kontrollerar åtkomst och skapar grupper i Azure med hjälp av PowerShell.

[Azure AD Privileged Identity Management: Säkerhets guide, aviseringar, recensioner](https://channel9.msdn.com/Series/Azure-Active-Directory-Videos-Demos/Azure-AD-Privileged-Identity-Management-Security-Wizard-Alerts-Reviews)

* Hantera skydd för Office 365-arbets belastnings bara administrativa roller
* Konfigurera Azure Multi-Factor Authentication (MFA) för privilegierade roll aktiveringar
* Mäta och förbättra din position för klient säkerhet
* Övervakning och korrigering av säkerhets resultat
* Granska vem som behöver finnas i privilegierade roller för regelbundna certifierings arbets flöden
