# <a name="microsoft-trust-center"></a>Microsoft Säkerhetscenter
På Azure-webbplatsen för säkerhets information på Azure.com får du den information du behöver för att planera, utforma, distribuera, konfigurera och hantera dina moln lösningar på ett säkert sätt. Med Microsoft Trust Center har du också den information du behöver för att vara säker på att Azure-plattformen som du kör dina tjänster på är säker.

Vi vet att du har frågor till Azure när du ska anförtro dina program och data till Azure. Var finns det? Vem har åtkomst till den? Vad gör Microsoft för att skydda det? Hur kan du kontrol lera att Microsoft gör vad det står?

Och vi har svar. Eftersom det är dina data kan du bestämma vem som har åtkomst och du arbetar med oss för att bestämma var du befinner dig. För att skydda dina data använder vi den senaste säkerhets tekniken och kryptografi tekniken i världs klass. Vår efterlevnad granskas oberoende av varandra och vi är transparenta på många nivåer, från hur vi hanterar juridiska krav för dina kund data till säkerheten i vår kod.

Det här hittar du på Microsoft Trust Center:

* [Säkerhet](https://aka.ms/tcsecurity) – lär dig hur alla Microsoft Cloud-tjänsterna skyddas.
* [Sekretess](https://aka.ms/tcprivacy) – förstå hur Microsoft garanterar sekretess för dina data i Microsoft-molnet.
* [Efterlevnad](https://aka.ms/tccompliance) – Upptäck hur Microsoft hjälper organisationer att uppfylla nationella, regionala och branschspecifika krav som reglerar insamling och användning av enskildas data.
* [Transparens](https://aka.ms/tctransparency) – Visa hur Microsoft anser att du kan kontrol lera dina data i molnet och hur Microsoft hjälper dig att förstå hur data hanteras.
* [Produkter och tjänster](https://aka.ms/tcproductsservices) – se alla Microsoft Cloud produkter och tjänster på ett och samma ställe
* [Tjänst förtroende Portal](https://aka.ms/tcservicetrportal) – få kopior av oberoende gransknings rapporter för Microsofts moln tjänster, riskbedömningar, rekommenderade säkerhets metoder och relaterat material.
* [Nyheter](https://aka.ms/tcwhatsnew) – ta reda på vad som är nytt i Microsoft Cloud-förtroende
* [Resurser](https://aka.ms/tcresources) – Undersök fakta blad, videor och fallstudier på Microsoft Trusted Cloud

[Microsoft Trust Center](https://www.microsoft.com/trustcenter) har det du behöver för att förstå vad vi gör för att skydda Microsoft Cloud.
