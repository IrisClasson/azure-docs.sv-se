# <a name="application-architecture-on-azure"></a>Programarkitektur på Azure
För att skydda dina molnbaserade lösningar på Microsoft Azure är ett starkt arkitektur bas viktigt. Arkitekter, designers och implementerare drar nytta av en stark kunskap om program-och tjänst arkitektur. Den här grundläggande kunskapen hjälper dig att förstå alla komponenter i dina molnbaserade lösningar och gör det lättare att integrera säkerheten i alla aspekter av din design och implementering.

Vi har följande för att hjälpa dig med dina arkitektur undersökningar och design:

* Arkitektur ritningar
* Moln-och företags symbol uppsättning
* Visio-mall för 3D-skiss

## <a name="architectural-blueprints"></a>Arkitektur ritningar
Microsoft publicerar en uppsättning [arkitektur ritningar](https://aka.ms/azblueprints) på hög nivå som visar hur du skapar specifika typer av system med hjälp av Microsoft-produkter.
Varje skiss innehåller:

* Platt 2D Visio 2003-baserad fil som du kan hämta och ändra
* Färgad 3D-perspektiv PDF-fil för att presentera skissen på mindre tekniska mål grupper
* Video som går igenom 3D-versionen

## <a name="cloud-and-enterprise-symbol-set"></a>Moln-och företags symbol uppsättning
[Visa övnings videon för Visio och symboler](https://aka.ms/CnESymbolsVideo) och [Ladda ned molnet och företags symbolen](https://aka.ms/CnESymbols) för att skapa tekniskt material som beskriver Azure, Windows Server, SQL Server med mera. Du kan använda symbolerna i arkitektur diagram, utbildningsmaterial, presentationer, datablad, informationsgrafik, whitepapers och till och med tredje parts böcker om han eller hon vill använda Microsoft-produkter. De är dock inte avsedda att användas i användar gränssnitt.

## <a name="3d-blueprint-visio-template"></a>Visio-mall för 3D-skiss
3D-versionerna av [Microsoft Architecture-ritningar](https://aka.ms/azblueprints) skapades ursprungligen i ett verktyg som inte kommer från Microsoft. En ny Visio 2013-mall (och senare) har levererats den 5 augusti 2015 som en del av en [Microsoft Architecture Certificate-kurs som distribueras på EDX.org](https://docs.microsoft.com/azure/architecture/).

Mallen är också tillgänglig utanför kursen.

* [Visa övnings videon](https://aka.ms/3dBlueprintTemplateVideo) först så att du vet vad det kan göra
* Ladda ned [Visio-mallen Microsoft 3D skiss](https://aka.ms/3DBlueprintTemplate)
* Ladda ned [moln-och företags symboler](https://docs.microsoft.com/azure/architecture/) som ska användas med 3D-mallen
