# <a name="azure-security-courses-from-microsoft-virtual-academy"></a>Azures säkerhets kurser från Microsoft Virtual Academy
Microsoft Virtual Academy tillhandahåller kostnads fri utbildning online som hjälper utvecklare, IT-proffs och data personal och studenter att lära sig den senaste tekniken, bygga sina kunskaper och fortsätta sina karriärer.

På den här sidan hittar du en granskad samling av Azure Security-relaterade kurser. Besök [Microsoft Virtual Academy](https://mva.microsoft.com/) för att se alla kurser de har tillgängliga.

[Utveckling/testning i molnet](https://mva.microsoft.com/en-us/training-courses/devtest-in-the-cloud-16274?l=9aAijd7LC_2005190311)

Är du en utvecklare som behöver leverera snabbare och bättre program? Att flytta dina utvecklings-och test miljöer till molnet kan hjälpa dig att uppnå exakt det! Lär dig hur du får det gjort och ta reda på fördelarna med att flytta. Dessutom kan du se demonstrationer och presentationer som visar hur Microsoft Azure kan stödja dina utvecklings-och testnings behov. Här ingår en lektion om säkerhets utveckling och distributions metoder.

[Vanliga aktiviteter för Linux på Azure](https://mva.microsoft.com/en-us/training-courses/common-tasks-for-linux-on-azure-16191?l=J0Hvb7qJC_1204668937)

Om du har frågor om hur du använder Linux på Microsoft Azure plattform har den här detaljerade kursen svar på dig. Utforska några vanliga uppgifter med experterna från [Linux-Academy](https://linuxacademy.com/). Lär dig mer om att skapa en virtuell Linux-dator (VM) i Azure, komma åt den virtuella Linux-datorn med program vara för fjärr skrivbord och köra virtuella värdar. Många säkerhets tekniker och konfigurationer behandlas i den här kursen.

[Skydda molnet](https://mva.microsoft.com/en-us/training-courses/secure-the-cloud-14037?l=lQIkkst0B_5300115881)

I den här sessionen får du lära dig hur Microsoft kan hjälpa dig att uppfylla globala krav för efterlevnad, till exempel ISO 27001/27018, FedRAMP, PCI och HIPAA, med nya säkerhets kontroller. Dessa kontroller sträcker sig från data kryptering i vilo läge, nyckel hantering, VM-skydd, loggning och övervakning, till tjänster mot skadlig kod, identitets hantering, åtkomst kontroller och mycket annat.

[Utforma och implementera lösningar för Cloud Data Platform](https://mva.microsoft.com/en-us/training-courses/design-and-implement-cloud-data-platform-solutions-15711?l=jbCdW0j1B_3005244527)

Lär dig mer om funktionerna i Microsoft Cloud Data Platform-lösningar. Få en plattforms översikt och hör om säkerhetsfunktioner, alternativ för hög tillgänglighet, teknik för övervakning och hantering av moln data med mera. Dessutom får du vägledning om hur du kan identifiera kompromisser och fatta beslut om att utforma offentliga och hybrid moln lösningar med hjälp av Microsoft Cloud Data Platform-funktioner.

[Hantera och skydda identiteter i ett moln och i en mobil värld](https://mva.microsoft.com/en-us/training-courses/manage-and-secure-identities-in-a-cloud-and-mobile-world-14013?l=GIJ2GcvrB_405192797)

I den här sessionen får du lära dig hur Azure Active Directory och Microsoft Advanced Threat Analytics hjälper dig att skydda och hantera användar identitet, identifiera säkerhets överträdelser innan de orsakar skada och ge användarna en enda identitet för åtkomst till hela företaget resurser. Utforska de tekniker som används för att identifiera skugg-IT, hantera program åtkomst och övervaka misstänkt aktivitet via avancerad säkerhets rapportering, analys av användar beteende, granskning och avisering.

[Säkerhet i en moln aktive rad värld](https://mva.microsoft.com/en-us/training-courses/security-in-a-cloudenabled-world-12725?l=CfLHobAcB_3904300474)

Experter vägleder dig genom översikten över kund ansvar i [Microsoft Cloud Security för Enterprise Architects](https://www.microsoft.com/download/48121) -affisch. Experterna ger också rekommendationer för att ställa av en del av din säkerhets position, inklusive styrning, inne slutnings strategier, säkerhets åtgärder, högt mervärde till gångs skydd, informations skydd och användar-och enhets säkerhet, med en särskild tonvikt på skydd av administrativ kontroll. Lär dig från samma ramverk som Microsoft cybersäkerhet-teamet använder för att utvärdera kundernas moln säkerhet och för att bygga en säkerhets översikt.

[Microsoft Azure IaaS djup](https://mva.microsoft.com/en-us/training-courses/microsoft-azure-iaas-deep-dive-14339?l=PtppYVQgB_8300115888)

Lär dig hur du använder Microsoft Azure infrastruktur funktioner. Om du är IT-proffs behöver du inte ha tidigare erfarenhet av Azure. Den här kursen vägleder dig genom att skapa och konfigurera virtuella Azure-datorer, virtuella Azure-nätverk och anslutningar på olika platser för att komma igång med molnet. Säkerhets funktioner och överväganden ingår i hela kursen.

[Komma igång med Azure-säkerhet för IT-proffs](https://mva.microsoft.com/training-courses/getting-started-with-azure-security-for-the-it-professional-11165?l=HfHzCXSAB_7404300474)

I den här demon-kursen tar ett team med säkerhets experter och Azure-tekniker dig bortom de grundläggande certifieringarna och utforskar vad som är möjligt i Azure. Se hur du utformar och använder olika tekniker för att se till att du har den säkerhet och arkitektur du behöver för att kunna starta dina projekt i molnet. Gå in på Data Center åtgärder, konfiguration av virtuell dator, nätverks arkitektur och lagrings infrastruktur.

[Djupgående att gå in i Azure Resource Manager scenarier och mönster](https://mva.microsoft.com/en-us/training-courses/deep-dive-into-azure-resource-manager-scenarios-and-patterns-13793?l=i1m06ZJYB_7001937557)

Utforska Azure Resource Manager med ett expert team, som visar skript och verktyg som gör det enkelt att skapa eller sätta igång element i program infrastrukturen. Utforska användningen av rollbaserad åtkomst kontroll (RBAC) för att implementera säkerhet med Azure Resource Manager.

[Azure Rights Management Services Core-kunskaper](https://mva.microsoft.com/en-us/training-courses/azure-rights-management-services-core-skills-10500?l=QLoxMwuCB_1805094681)

Ta reda på varför informations skydd är "måste ha" krav i din organisation och hur Rights Management skyddar organisationens intellektuella egendom, oavsett var den reser över enheter och molnet. Få praktisk erfarenhet och teknisk kunskap – hur man gör från Microsoft-experter.
