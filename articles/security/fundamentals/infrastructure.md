# <a name="azure-infrastructure-security"></a>Azure-infrastruktur säkerhet
Microsoft Azure körs i datacenter som hanteras och drivs av Microsoft. Dessa geografiskt spridda datacenter uppfyller viktiga branschstandarder, till exempel ISO/IEC 27001:2013 och NIST SP 800-53, för säkerhet och tillförlitlighet. Datacenter hanteras, övervakas och administreras av Microsofts driftpersonal. Driftpersonalen har många års erfarenhet av att leverera världens största onlinetjänster med kontinuitet på 24 x 7.
