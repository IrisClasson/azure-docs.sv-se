# <a name="azure-security-overview"></a>Översikt över Azure-säkerhet

Den Microsoft Azure säkerhets informations webbplatsen på Azure.com är din plats för tekniska svar på dina Azures säkerhets frågor. Om du är intresse rad av Azure-kompatibilitet och hur Azure skyddar sin egen infrastruktur kan du gå till [Microsoft Trust Center](https://microsoft.com/en-us/trustcenter/default.aspx).

Vi vet att säkerheten är ett jobb i molnet och hur viktigt det är att du hittar korrekt och tidsödande information om Azure-säkerhet. Ett av de bästa orsakerna till att använda Azure för dina program och tjänster är att dra nytta av den breda uppsättningen säkerhets verktyg och funktioner som är tillgängliga. Med dessa verktyg och funktioner kan du skapa säkra lösningar på den säkra Azure-plattformen.

Den här sidan och webbplatsen kommer att utvecklas när nya Azure-tjänster är online och befintliga tjänster uppdateras. Våra säkerhets erbjudanden ökar ständigt och vi kommer att dela den tekniska informationen om dessa förbättringar här. Vi rekommenderar att du går tillbaka till den här webbplatsen regelbundet och ser vad som är nytt!

## <a name="using-the-azure-security-information-site"></a>Använda Azures webbplats för säkerhets information

När du tittar på vänster sida av sidan visas en samling kategorier av säkerhets ämnen. Klicka eller tryck på avsnittet för att se de artiklar vi har för den kategorin.

Varje kategori har en översikts artikel. Målet med översikts artikeln är att ge dig en översikt över vad Azure måste erbjuda när det gäller säkerhet för den kategorin. Översikts artiklarna hjälper dig att förstå de produkter, tjänster och funktioner som finns tillgängliga i Azure som du kan använda för att förbättra säkerheten för dina distributioner. Översikts artiklarna innehåller flera länkar till tekniskt innehåll så att du kan lära dig mer om ämnena och hur du implementerar dem.

Webbplatsen för Azure-säkerhetsinformation hanterar för närvarande följande kategorier:

* Resurser
* Nätverkssäkerhet
* Lagringssäkerhet
* Säkerhet för virtuella datorer
* Slutpunktsskydd
* Säkerhetshantering och övervakning
* Identitetshantering
* PaaS-tjänster
* Bransch Specific
* Arkitektur

> [!NOTE]
> Alla artiklar vi har på webbplatsen för Azure säkerhets information visas i navigeringen till vänster på sidan. Om du vill se alla artiklar klickar eller trycker du på **Visa mer** längst ned i listan Kategorier.
>
>

Vi vill ha din inaktuella information! Om det finns kategorier som du vill lägga till kan du berätta för oss! Om det finns artiklar som du vill lägga till i en befintlig kategori kan du berätta för oss! Du kan använda avsnittet kommentarer nedan eller skriva till oss på [azsecinfosite@microsoft.com](mailto:azsecinfosite@microsoft.com) så införlivar vi dina rekommendationer i våra planer.
