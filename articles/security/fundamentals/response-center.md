# <a name="microsoft-security-response-center"></a>Microsoft Security Response Center

Microsoft Security Response Center (MSRC) leds av några av världens mest erfarna säkerhets experter. De identifierar, övervakar, svarar och löser säkerhets incidenter och lokala och molnbaserade sårbarheter dygnet runt, varje dag på året.

Förutom det kontinuerliga arbetet har MSRC-teamet ett antal resurser som är tillgängliga för dig så att du kan förstå hur du skyddar dina Azure-tillgångar och-distributioner mer effektivt.

## <a name="the-msrc-blog"></a>MSRC-bloggen

[MSRC-bloggen](https://blogs.technet.microsoft.com/msrc/) är den plats där du kan hämta de senaste nyheterna om vad MSRC gör för att skydda dig mot moln risker.

## <a name="white-papers"></a>White paper-faktablad

MSRC har publicerat ett antal [fakta blad](https://technet.microsoft.com/library/bb969102.aspx) som hjälper dig att förstå vad de gör och hur de fungerar. Vissa ger insikter om hur vi skyddar Microsoft-molnet och innehåller användbar information om hur du kan använda samma säkerhetskonfigurationer.

## <a name="security-researcher-engagement-and-bounty-programs"></a>Säkerhets forskares engagemang och Bounty-program

MSRC stöder samarbete och relationer med säkerhets forskare globalt för att gå vidare till Microsofts produkt säkerhet.

Microsoft Bounty-program betala forskare för:

- Nya utnyttjande-tekniker
- Försvars idéer som minimerar nya utnyttjanden
- Identifiering av kritiska sårbarheter i Microsofts lokala och molnbaserade program.

## <a name="next-steps"></a>Nästa steg

Läs mer om dessa program på sidan [MSRC-fel Bounty](https://technet.microsoft.com/security/dn425036) och [MSRC-bloggen](https://blogs.technet.microsoft.com/msrc/).

Om du vill veta mer om MSRC går du till [MSRC](https://technet.microsoft.com/library/dn440717.aspx)-startsidan.
