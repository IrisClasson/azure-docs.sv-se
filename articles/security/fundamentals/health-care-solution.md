# <a name="a-practical-guide-to-designing-secure-health-care-solutions-in-azure"></a>En praktisk guide för att utforma säkra hälso vårds lösningar i Azure
Hälso branschens starter, system integrerare (SIs), oberoende program varu leverantörer (ISV) och sjukvårds organisationer som överväger att flytta till Azure, letar efter rikt linjer som hjälper dem att uppfylla sina krav på säkerhet.

[En praktisk guide för att utforma säkra hälso vårds lösningar i Microsoft Azure](https://aka.ms/azureindustrysecurity) hjälper dig att förstå hur du kan förbättra säkerheten för dina lösningar med hjälp av de Azure-tjänster och-funktioner som du kan konfigurera baserat på dina krav.
Innehållet är uppdelat i tre huvud avsnitt:

1. Vägledning om hur du använder moln teknik, inklusive riskhantering, delat ansvar, etablering av ett informations hanterings system, förståelse för bransch-och lokala bestämmelser och fast ställande av standard operativa procedurer.
2. Viktiga säkerhets principer som är både justerade mot en standard hanterings standard för informations säkerhet, till exempel ISO 27001 och standard utvecklings processer, till exempel Microsoft Security Development Lifecycle (SDL).
3. Tillämpa nyckel principerna för användnings fall genom att demonstrera justeringen från ett lösnings arkitekt perspektiv där kraven för lösningarna är justerade med standarden för informations säkerhets hantering.

Vi hoppas att du hittar [en praktisk guide för att utforma säkra hälso vårds lösningar](https://aka.ms/azureindustrysecurity) och om du har frågor eller förslag kan du meddela oss genom att lämna en kommentar nedan.
