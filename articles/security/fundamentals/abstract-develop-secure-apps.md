# <a name="develop-secure-cloud-applications-on-azure"></a>Utveckla säkra moln program på Azure
## <a name="abstract"></a>Abstrakt
Det här dokumentet är en allmän guide till de säkerhets frågor och kontroller som du bör tänka på i varje fas i livs cykeln för program varu utveckling när du utvecklar program för molnet. Implementering av dessa koncept innan du släpper produkten kan hjälpa dig att bygga säkrare program vara. Rekommendationerna som presenteras i det här dokumentet kommer från vår erfarenhet av Azure-säkerhet och våra kunders upplevelser.

Det här dokumentet är avsett att vara en resurs för programmerare, utvecklare och testare på alla nivåer som skapar och distribuerar säkra Azure-lösningar.
