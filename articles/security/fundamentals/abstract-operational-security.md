# <a name="azure-operational-security"></a>Drift säkerhet i Azure
## <a name="abstract"></a>Abstrakt
Microsoft Azure drift säkerhet avser tjänster, kontroller och funktioner som är tillgängliga för användare för att skydda data, program och andra till gångar i Azure. Azures drift säkerhet bygger på ett ramverk som införlivar den kunskap som du har fått via olika funktioner som är unika för Microsoft, inklusive Microsoft Security Development Lifecycle (SDL), Microsoft Security Response Center-programmet. och djupgående medvetenhet om cybersäkerhet Threat landskap. I den här white paper beskrivs hur du kan använda Azure för att nå drift säkerhet. Det omfattar flera Azure-tjänster, inklusive:
* Azure Monitor-loggar
* Azure Backup
* Azure Security Center
* Azure Monitor
* Azure Network Watcher
* Azure Storage-analys
* Azure Active Directory
