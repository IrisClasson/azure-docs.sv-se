# <a name="a-practical-guide-to-designing-secure-health-care-solutions-in-azure"></a>En praktisk guide till design av säkra hälsovård lösningar i Azure
Hälsotillstånd bransch nystartade företag, systemintegratörer (SI), oberoende programvaruleverantörer (ISV) och vårdorganisationer överväger en övergång till Azure söker vägledning som hjälper dem att införliva säkerhetskontroller för att uppfylla skyldigheterna efterlevnad.

[En praktisk Guide till utforma säkra sjukvård lösningar i Microsoft Azure](https://aka.ms/azureindustrysecurity) hjälper dig att förstå hur du kan förbättra säkerheten för dina lösningar med hjälp av Azure-tjänster och funktioner som du kan konfigurera baserat på dina krav.
Innehållet är uppdelad i tre huvuddelar:

1. Att tänka på hur du använder molntekniken, inklusive riskhantering, delade ansvar, upprätta ett hanteringssystem för informationssäkerhet, förståelse av branschbestämmelser och lokala föreskrifter och standarddriftsprocedurer.
2. Viktiga säkerhetsprinciper som är både justeras till en standard management standarden för informationssäkerhet, till exempel ISO 27001, och standard standardutvecklingsprocesser, till exempel Microsoft Security Development Lifecycle (SDL).
3. Tillämpa huvudprinciperna för att använda fall demonstrera justering ur en solution architect, där kraven för lösningarna som är anpassade till standard information security management.

Vi hoppas att du har hittat [en praktisk Guide till utforma säkra sjukvård lösningar](https://aka.ms/azureindustrysecurity) användbara och om du har några frågor eller förslag kan berätta för oss genom att lämna en kommentar här nedan.
