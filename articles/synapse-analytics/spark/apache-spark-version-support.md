# <a name="supported-language-and-runtime-versions-for-apache-spark-and-dependent-components"></a>Språk- och körningsversioner som stöds för Apache Spark och beroende komponenter  

Apache Spark i Azure Synapse Analytics-tjänsten stöder flera olika körningstider och tjänster som det här dokumentet listar versionerna.

## <a name="spark-version"></a>Spark-version

Spark 2.4 är den version som för närvarande stöds.

## <a name="operating-system-version"></a>Operativsystemversion

Apache Spark i Azure Synapse körs på Ubuntu version 16.04.

## <a name="java-version"></a>Java-version

Apache Spark i Azure Synapse innehåller Java version 1.8.0_222

## <a name="scala-version"></a>Scala-version

Apache Spark i Azure Synapse innehåller Scala 2.11.12

## <a name="net-core-version"></a>.NET Core-version

Apache Spark i Azure Synapse innehåller .NET Core 3.0

## <a name="net-support-for-apache-spark"></a>.NET-stöd för Apache Spark

Azure Synapse innehåller .NET-stöd för Apache Spark 0.5.0

## <a name="delta-lake"></a>Data Lake

Synapse Analytics innehåller stöd för Delta Lake 0.5.0

## <a name="python-version"></a>Python-version

Apache Spark i Azure Synapse innehåller en installation av Python version 3.6.1
