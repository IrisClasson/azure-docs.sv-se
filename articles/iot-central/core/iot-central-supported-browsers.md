# <a name="supported-browsers-for-azure-iot-central"></a>Webbläsare som stöds för Azure IoT Central

*Den här artikeln gäller för operatörer, kompilerare och administratörer.*

Azure IoT Central kan nås över de flesta moderna skriv bord, surfplattor och webbläsare. I följande artikel beskrivs en lista över vilka webbläsare som stöds och vilken anslutning som krävs.  

## <a name="supported-browsers"></a>Webbläsare som stöds
Vi rekommenderar att du använder den mest aktuella webbläsaren som är kompatibel med ditt operativ system. Följande webbläsare stöds:
- Microsoft Edge (senaste versionen)
- Safari (senaste versionen, endast Mac)
- Chrome (senaste versionen)
- Firefox (senaste versionen)

## <a name="required-protocols"></a>Protokoll som krävs
Azure IoT Central kräver att nätverket stöder både HTTPS-och WebSocket-protokollen för utgående anslutningar.
