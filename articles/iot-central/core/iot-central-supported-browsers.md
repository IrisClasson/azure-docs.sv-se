# <a name="supported-browsers-for-azure-iot-central"></a>Webbläsare som stöds för Azure IoT Central

*Den här artikeln gäller för operatörer, kompilerare och administratörer.*

Azure IoT Central kan nås på de flesta moderna skrivbord, surfplattor och webbläsare. I följande artikel beskrivs listan över webbläsare som stöds och nödvändig anslutning.  

## <a name="supported-browsers"></a>Webbläsare som stöds
Vi rekommenderar att du använder den senaste webbläsaren som är kompatibel med ditt operativsystem. Följande webbläsare stöds:
- Microsoft Edge (senaste versionen)
- Safari (senaste versionen, endast Mac)
- Chrome (senaste versionen)
- Firefox (senaste versionen)

## <a name="required-protocols"></a>Nödvändiga protokoll
Azure IoT Central kräver att nätverket stöder både HTTPS- och WebSocket-protokollen för utgående anslutning.
