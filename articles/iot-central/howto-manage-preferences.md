# <a name="manage-your-application-preferences"></a>Hantera din programinställningar

*Den här artikeln gäller för operatörer, kompilerare och administratörer.*

IoT Central ger dig flexibilitet att anpassa dina program så att de passar dina behov. Vi tillhandahåller även vissa flexibilitet på basis av per användare anpassa egna vyn. Den här artikeln beskrivs olika anpassningsalternativ som en användare kan använda för sina profiler.

## <a name="changing-language"></a>Ändra språk

IoT Central stöds i 17 språk. Du kan växla språk du föredrar med hjälp av den **språkväljare** på det övre navigeringsfältet. När du har ändrat ditt språk, vi kommer ihåg *din* val och tillämpa den på alla dina program. Observera att anpassning i programmet bilderna instrumentpanelen inte lokaliseras. 

![IoT Central Språkväljare](media/howto-manage-profile/language-picker.png)

## <a name="changing-theme"></a>Ändra tema

Vi har stöd för både mörkt tema och ljusa temat. Även om det mörka temat är tema, kan du ändra detta genom att klicka på namnet på din längst upp till höger och växla temat.

![Temaväljare IoT Central](media/howto-manage-profile/theme-picker.png)
