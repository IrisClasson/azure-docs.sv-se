# <a name="azure-sql-database-features"></a>Funktioner i Azure SQL-databas

Azure SQL Database-tjänsten delar en gemensam kodbas med den senaste stabila versionen av SQL Server. De flesta av standardfunktionerna för SQL-språk, frågebearbetning och databashantering är identiska i SQL Server och Azure SQL Database. De funktioner som är vanliga i SQL Server och alla smaker av Azure SQL Database är:
  - Språkfunktioner - [Kontroll av flödesspråksnyckelord,](https://docs.microsoft.com/sql/t-sql/language-elements/control-of-flow) [markörer](https://docs.microsoft.com/sql/t-sql/language-elements/cursors-transact-sql), [datatyper,](https://docs.microsoft.com/sql/t-sql/data-types/data-types-transact-sql) [DML-satser,](https://docs.microsoft.com/sql/t-sql/queries/queries) [predikat,](https://docs.microsoft.com/sql/t-sql/queries/predicates) [sekvensnummer,](https://docs.microsoft.com/sql/relational-databases/sequence-numbers/sequence-numbers) [lagrade procedurer](https://docs.microsoft.com/sql/relational-databases/stored-procedures/stored-procedures-database-engine)och [variabler](https://docs.microsoft.com/sql/t-sql/language-elements/variables-transact-sql).
  - Databasfunktioner - [Automatisk justering (plan forcing),](https://docs.microsoft.com/sql/relational-databases/automatic-tuning/automatic-tuning) [Ändra spårning](https://docs.microsoft.com/sql/relational-databases/track-changes/about-change-tracking-sql-server), [Databassortering](https://docs.microsoft.com/sql/relational-databases/collations/set-or-change-the-database-collation), [Inneslutna databaser](https://docs.microsoft.com/sql/relational-databases/databases/contained-databases), [Slutna användare](https://docs.microsoft.com/sql/relational-databases/security/contained-database-users-making-your-database-portable), [Datakomprimering](https://docs.microsoft.com/sql/relational-databases/data-compression/data-compression), [Databaskonfigurationsinställningar,](https://docs.microsoft.com/sql/t-sql/statements/alter-database-scoped-configuration-transact-sql) [Onlineindexåtgärder](https://docs.microsoft.com/sql/relational-databases/indexes/perform-index-operations-online), [Partitionering](https://docs.microsoft.com/sql/relational-databases/partitions/partitioned-tables-and-indexes)och [Temporaltabeller](https://docs.microsoft.com/sql/relational-databases/tables/temporal-tables) ([se komma igång-guide](sql-database-temporal-tables.md)).
  - Säkerhetsfunktioner - [Programroller](https://docs.microsoft.com/sql/relational-databases/security/authentication-access/application-roles), [Dynamisk datamaskering](https://docs.microsoft.com/sql/relational-databases/security/dynamic-data-masking) ([se komma igång-guide),](sql-database-dynamic-data-masking-get-started.md) [Säkerhet på radnivå](https://docs.microsoft.com/sql/relational-databases/security/row-level-security)och identifiering av hot - se komma igång guider för [enstaka databaser och elastiska pooler](sql-database-threat-detection.md) och [hanterad instans](sql-database-managed-instance-threat-detection.md).
  - Funktioner för flera modeller - [Grafbearbetning](https://docs.microsoft.com/sql/relational-databases/graphs/sql-graph-overview), [JSON-data](https://docs.microsoft.com/sql/relational-databases/json/json-data-sql-server) [(se komma igång-guide),](sql-database-json-features.md) [OPENXML,](https://docs.microsoft.com/sql/t-sql/functions/openxml-transact-sql) [Spatial,](https://docs.microsoft.com/sql/relational-databases/spatial/spatial-data-sql-server) [OPENJSON](https://docs.microsoft.com/sql/t-sql/functions/openjson-transact-sql)och [XML-index](https://docs.microsoft.com/sql/t-sql/statements/create-xml-index-transact-sql).

Azure SQL Database hanterar dina databaser och garanterar deras hög tillgänglighet. Vissa funktioner som kan påverka hög tillgänglighet eller inte kan användas i PaaS-världen har begränsade funktioner i Azure SQL Database. Dessutom beror vissa databasfunktioner på vilken typ av Azure SQL-databas som du skapar. Dessa funktioner beskrivs i tabellerna nedan. Med Azure SQL Database kan du skapa en databas som en del av en [hanterad instans](sql-database-managed-instance.md), som en enda databas eller som en del av en elastisk pool. Om du behöver mer information om skillnaderna kan du hitta dem på de separata sidorna för [endatabas och elastiska pooler](sql-database-transact-sql-information.md) eller [Hanterad instans](sql-database-managed-instance-transact-sql-information.md).
