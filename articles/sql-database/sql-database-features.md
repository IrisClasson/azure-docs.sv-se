# <a name="azure-sql-database-features"></a>Azure SQL Database funktioner

Azure SQL Database Service delar en gemensam kodbas med den senaste stabila versionen av SQL Server. De flesta standard-SQL-språk, fråge bearbetning och funktioner för databas hantering är identiska i SQL Server och Azure SQL Database. De funktioner som är vanliga i SQL Server och alla varianter av Azure SQL Database är:
  - Språk funktioner – [kontroll av nyckelord för Flow-språk](https://docs.microsoft.com/sql/t-sql/language-elements/control-of-flow), [markörer](https://docs.microsoft.com/sql/t-sql/language-elements/cursors-transact-sql), [data typer](https://docs.microsoft.com/sql/t-sql/data-types/data-types-transact-sql), [DML-instruktioner](https://docs.microsoft.com/sql/t-sql/queries/queries), [predikat](https://docs.microsoft.com/sql/t-sql/queries/predicates), [sekvensnummer](https://docs.microsoft.com/sql/relational-databases/sequence-numbers/sequence-numbers), [lagrade procedurer](https://docs.microsoft.com/sql/relational-databases/stored-procedures/stored-procedures-database-engine)och [variabler](https://docs.microsoft.com/sql/t-sql/language-elements/variables-transact-sql).
  - Databas funktioner – [Automatisk justering (schema framtvingande)](https://docs.microsoft.com/sql/relational-databases/automatic-tuning/automatic-tuning), [ändrings spårning](https://docs.microsoft.com/sql/relational-databases/track-changes/about-change-tracking-sql-server), [databas sortering](https://docs.microsoft.com/sql/relational-databases/collations/set-or-change-the-database-collation), [inneslutna databaser](https://docs.microsoft.com/sql/relational-databases/databases/contained-databases), [inneslutna användare](https://docs.microsoft.com/sql/relational-databases/security/contained-database-users-making-your-database-portable), [data komprimering](https://docs.microsoft.com/sql/relational-databases/data-compression/data-compression), [konfigurations inställningar för databasen](https://docs.microsoft.com/sql/t-sql/statements/alter-database-scoped-configuration-transact-sql) [Online index åtgärder](https://docs.microsoft.com/sql/relational-databases/indexes/perform-index-operations-online), [partitionering](https://docs.microsoft.com/sql/relational-databases/partitions/partitioned-tables-and-indexes)och [temporala tabeller](https://docs.microsoft.com/sql/relational-databases/tables/temporal-tables) ([se komma igång-guide](sql-database-temporal-tables.md)).
  - Säkerhetsfunktioner – [program roller](https://docs.microsoft.com/sql/relational-databases/security/authentication-access/application-roles), [dynamisk data maskning](https://docs.microsoft.com/sql/relational-databases/security/dynamic-data-masking) ([se komma igång-guide](sql-database-dynamic-data-masking-get-started.md)), [säkerhet på radnivå](https://docs.microsoft.com/sql/relational-databases/security/row-level-security)och hot identifiering – se komma igång guider för [enkel databas och elastiska pooler](sql-database-threat-detection.md) [ hanterad instans](sql-database-managed-instance-threat-detection.md).
  - Funktioner för flera modeller – [diagram bearbetning](https://docs.microsoft.com/sql/relational-databases/graphs/sql-graph-overview), [JSON-data](https://docs.microsoft.com/sql/relational-databases/json/json-data-sql-server) ([se komma igång-guide](sql-database-json-features.md)), [OpenXML](https://docs.microsoft.com/sql/t-sql/functions/openxml-transact-sql), [spatialdata](https://docs.microsoft.com/sql/relational-databases/spatial/spatial-data-sql-server), [openjson](https://docs.microsoft.com/sql/t-sql/functions/openjson-transact-sql)och [XML-index](https://docs.microsoft.com/sql/t-sql/statements/create-xml-index-transact-sql).

Azure SQL Database hanterar dina databaser och garanterar deras hög tillgänglighet. Vissa funktioner som kan påverka hög tillgänglighet eller som inte kan användas i PaaS World har begränsade funktioner i Azure SQL Database. Dessutom beror vissa databas funktioner på vilken typ av Azure SQL Database som du skapar. Dessa funktioner beskrivs i tabellerna nedan. Med Azure SQL Database kan du skapa en databas som en del av en [hanterad instans](sql-database-managed-instance.md), som en enskild databas eller som en del av en elastisk pool. Om du behöver mer information om skillnaderna kan du hitta dem på separata sidor för [en enkel databas och elastiska pooler](sql-database-transact-sql-information.md) eller [hanterade instanser](sql-database-managed-instance-transact-sql-information.md).
