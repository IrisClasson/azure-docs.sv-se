# <a name="what-is-azure-vmware-solution-by-cloudsimple"></a>Vad är Azure VMware-lösning från CloudSimple

**Azure VMware Solution by CloudSimple** är en fullständigt hanterad tjänst som ger dig möjlighet att köra VMware-plattformen i Azure. Den här lösningen innehåller verktyg för vSphere, vCenter, vSAN, NSX-T och motsvarande verktyg. Din VMware-miljö körs internt på Azure bare metal-infrastruktur i Azure-molnplatser. Tjänsten innehåller alla funktioner som krävs för att konsumera VMware-plattformarna effektivt och säkert.

![VMware-lösning på Azure by CloudSimple Översikt](media/azure-vmware-solution-by-cloudsimple.png)

## <a name="features"></a>Funktioner

* Tillhandahållande av VMware-molnmiljöer på begäran. Möjlighet att lägga till och ta bort kapacitet på begäran.
* VMware-plattformsdistribution, uppgradering, säkerhetskopiering av hanteringsplan, hälso-/kapacitetsövervakning, aviseringar, felsökning och reparation.
* Underlag nätverkstjänster som krävs för att aktivera VMware, inklusive L2/L3-tjänster och brandväggsregelhantering.
* Nätverkstjänster av edgetyp, inklusive VPN-, offentliga IP- och internetgateways. Dessa tjänster körs på Azure och bär säkerhets- och DDoS-skyddet för Azure.
* Kapacitetsreservation för lägre kostnader.
* Anslutning med låg latens med låg hastighet till Azure och lokalt.
* Lösningsarkitekturer för kunder att använda Azure-tjänster på ett integrerat sätt och dra nytta av det här unika "VMware-molnet i ett offentligt moln"-arkitektur. Azure-tjänsterna omfattar Azure AD, lagring, programgateways och andra.
* Infrastruktur som är helt tillägnad dig och är fysiskt isolerad från infrastruktur för andra kunder.
* Hantering har sådan aktivitetshantering, användning, fakturering/mätning och användarhantering.
* 24x7 kundsupport.

## <a name="benefits"></a>Fördelar

* **Operativ kontinuitet**. CloudSimple erbjuder inbygg åtkomst till VMware-plattformar. CloudSimple-arkitekturen är kompatibel med din befintliga:
    * Program
    * Åtgärder
    * Säkerhet
    * Säkerhetskopiering
    * Haveriberedskap
    * Granska
    * Verktyg för efterlevnad
    * Processer
* **Ingen omskolning.** VMware plattform kompatibilitet kan du använda befintliga färdigheter och kunskaper.
* **Infrastruktur agility**. Du behöver inte längre förutsäga alla dina kapacitetsbehov och sedan sluta med bortkastad kapacitet eller brist på infrastruktur. CloudSimple levereras som en molntjänst och du kan lägga till eller minska kapaciteten när som helst
* **Säkerhet**. Åtkomst till CloudSimple-miljön via Azure ger inbyggt DDoS-skydd och säkerhetsövervakning.
* **Lägre kostnad**. CloudSimple-plattformen är mycket konstruerad och ger hög automatisering, driftseffektivitet och stordriftsfördelar. Dessutom publicerar CloudSimple lösningsarkitekturer som utnyttjar närvaron av VMware i ett offentligt moln för att sänka kostnaderna. Exempel är Azure AD, säkerhetskopiering till Azure-lagring, programgateway, belastningsutjämnare och andra.
* **En ny hybridplattform**. Tjänsten möjliggör åtkomst med hög hastighet och låg latens till resten av Azure. Dessutom möjliggör CloudSimple-hantering enhetlig hantering av virtuella VMware-datorer och resten av Azure med samma användargränssnitt och API. Dina utvecklingsteam kan dra nytta av både offentliga och privata plattformar på ett integrerat och konsekvent sätt.
* **Övervakning, felsökning och support för infrastruktur**. CloudSimple driver din underliggande infrastruktur som en tjänst. Maskinvaran för fel misslyckas ersätts automatiskt. Du kan fokusera på förbrukning medan CloudSimple ser till att miljön fungerar smidigt.
* **Policykompatibilitet**. Behåll dina VMware-baserade verktyg, säkerhetsrutiner, granskningspraxis och efterlevnadscertifieringar.

## <a name="scenarios"></a>Scenarier

* **Datacenter pensionering eller migrering**. Få ytterligare kapacitet när du når gränserna för ditt befintliga datacenter eller uppdatera maskinvara. Det är enkelt för dig att lägga till nödvändig kapacitet i molnet och eliminera huvudvärken att hantera uppdateringar av maskinvara. Minska risken och kostnaden för molnmigreringar jämfört med tidskrävande konverteringar eller bakåtkrekturering. Använd välbekanta VMware-verktyg och kunskaper för att påskynda molnmigreringar. I molnet använder du Azure-tjänster för att modernisera dina program i din takt.
* **Expandera på efterfrågan**. Expandera till molnet för att möta oförutsedda behov, till exempel nya utvecklingsmiljöer eller säsongskapacitetsskurningar. Du kan enkelt skapa ny kapacitet på begäran och behålla den bara så länge du behöver den. Minska din förhandsinvestering, påskynda etableringshastigheten och minska komplexiteten med samma arkitektur och principer i både lokalt och molnet.
* **Haveriberedskap och virtuella skrivbord i Azure-molnet**. Upprätta fjärråtkomst till data, appar och skrivbord i Azure-molnet. Med anslutningar med hög bandbredd laddar du upp och hämtar data snabbt för att återställa från incidenter. Nätverk med låg latens ger dig snabba svarstider som användarna förväntar sig av en skrivbordsapp. Med CloudSimple är det enkelt att replikera alla dina principer och nätverk i molnet med CloudSimple-portalen och välbekanta VMware-verktyg. Den enkla återhämtningen och replikeringen minskar avsevärt ansträngningen och risken med att skapa och hantera DR- och VDI-implementeringar.
* **Högpresterande program och databaser**. CloudSimple tillhandahåller en hyperkonvergerad arkitektur som är utformad för att köra dina mest krävande VMware-arbetsbelastningar. Kör Oracle, Microsoft SQL-server, mellanprogram och högpresterande no-SQL-databaser. Upplev molnet som ditt eget datacenter med nätverksanslutningar med hög hastighet på 25 Gbit/s som gör att du kan köra hybridappar som sträcker sig över lokala, VMware på Azure och privata Azure-arbetsbelastningar utan att kompromissa med prestanda.
* **Äkta hybrid**. Förena DevOps över VMware och Azure. Optimera VMware-administration för Azure-tjänster och lösningar som kan tillämpas i alla dina arbetsbelastningar. Få tillgång till offentliga molntjänster utan att behöva utöka ditt datacenter eller bakåtkrekta dina program. Centralisera identiteter, åtkomstkontrollprinciper, loggning och övervakning för VMware-program på Azure.
