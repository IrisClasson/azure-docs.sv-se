# <a name="discover-jupyter-notebooks-in-the-azure-ai-gallery"></a>Identifiera Jupyter Notebooks i Azure AI-galleriet

**[Jupyter Notebooks](https://gallery.cortanaintelligence.com/notebooks)**  i [Azure AI-galleriet](http://gallery.cortanaintelligence.com) inkludera kod, datavisualiseringar och dokumentation i en enda, interaktiv arbetsyta.
Anteckningsböcker i galleriet ger självstudier och detaljerade förklaringar av avancerade machine learning-tekniker och lösningar.

## <a name="discover"></a>Utforska
  Om du vill bläddra efter anteckningsböcker i galleriet, öppna den [galleriet](http://gallery.cortanaintelligence.com) och klicka på **anteckningsböcker** överst på startsidan för galleriet.

 Den **[Jupyter Notebooks](https://gallery.cortanaintelligence.com/notebooks)** sidan visar en lista med de populäraste notebooks.
Klicka på **se alla** att visa alla anteckningsböcker.
Den här sidan kan du bläddra alla anteckningsböcker i galleriet. Du kan också söka genom att välja filtervillkoren till vänster på sidan och anger sökvillkor högst upp.

 Klicka på valfri anteckningsbok för att öppna informationssidan för denna notebook och läsa mer. På den här sidan kan du kommentera, ge feedback eller ställa frågor via avsnittet med kommentarer. Du kan också dela den med vänner och kollegor med resurs-funktionerna i LinkedIn eller Twitter. Du kan också skicka en länk till anteckningsboken för att bjuda in andra användare att visa sidan.

![Dela det här objektet med vänner](./media/gallery-how-to-use-contribute-publish/share-links.png)

![Lägga till egna kommentarer](./media/gallery-how-to-use-contribute-publish/comments.png)

## <a name="download"></a>Ladda ned
Du kan hämta en kopia av alla anteckningsboken från galleriet i din Machine Learning Studio-arbetsyta.

### <a name="download-from-the-gallery"></a>Ladda ned från galleriet
Hämta en kopia av en bärbar dator från galleriet:

1. Öppna den bärbara datorn detaljsida i galleriet
2. Klicka på **öppna i Studio**
3. Välj Region och arbetsytan som du vill använda
   
    ![Öppna Jupyter-anteckningsboken från galleriet](./media/gallery-jupyter-notebooks/open-notebook-from-gallery.png)

När du klickar på **öppna i Studio**, anteckningsboken läses in i Machine Learning Studio-arbetsyta och öppnas (om du inte redan är inloggad till Studio, uppmanas du att logga in med ditt Microsoft-konto innan anteckningsboken är kopierat till din arbetsyta). Du kan hitta den bärbara datorn igen senare genom att klicka på **anteckningsböcker** till vänster på sidan Studio.

### <a name="download-in-machine-learning-studio"></a>Ladda ned i Machine Learning Studio
Du kan också öppna någon av samma galleriet bärbara datorer när du arbetar i Machine Learning Studio:

1. I Machine Learning Studio, klickar du på **+ ny**
2. Välj **Notebook** – en lista över anteckningsböcker från galleriet visas att du kan välja bland eller du kan hitta en specifik notebook med sökrutan
3. Peka med musen på en bärbar dator och väljer **öppna anteckningsboken**
   
    ![Öppna galleriet anteckningsboken i Machine Learning Studio](./media/gallery-jupyter-notebooks/open-notebook-from-studio.png)

En kopia av den bärbara datorn hämtas och öppnas i din arbetsyta i avsnittet Jupyter Notebooks i Machine Learning Studio.
Anteckningsboken visas, tillsammans med dina anteckningsböcker på den **anteckningsböcker** sidan Studio (på den [Studio startsida](https://studio.azureml.net/), klickar du på **anteckningsböcker** till vänster).
