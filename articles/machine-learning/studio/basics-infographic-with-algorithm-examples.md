# <a name="downloadable-infographic-machine-learning-basics-with-algorithm-examples"></a>Nedladdningsbar infografik: Machine Learning-grunder med algoritmexempel

Ladda ned den här lättanvända informations grafiken-översikten över grunderna i Machine Learning och lär dig mer om populära algoritmer som används för att besvara vanliga Machine Learning-frågor. Algoritm exempel hjälper Machine Learning att förstå vilka algoritmer som ska användas och vad de använder.

## <a name="popular-algorithms-in-machine-learning-studio-classic"></a>Populära algoritmer i Machine Learning Studio (klassisk)

[Azure Machine Learning Studio (klassisk)](https://studio.azureml.net/) levereras med ett stort bibliotek med algoritmer för förutsägelse analys. Den här informations grafiken identifierar fyra populära familjer av algoritmer – regression, avvikelse identifiering, klustring och klassificering – och innehåller länkar till arbets exempel i [Azure AI Gallery](https://gallery.azure.ai/). Galleriet innehåller exempel på experiment och självstudier som visar hur dessa algoritmer kan användas i många verkliga lösningar.
