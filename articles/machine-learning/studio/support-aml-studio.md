Du kan **skicka in förfrågningar om** ytterligare utbildningsmaterial med hjälp av knappen Feedback om **innehåll** i slutet av varje artikel.

## <a name="get-service-support"></a>Få servicesupport

Kolla in dessa supportresurser:

+ **Teknisk support för Azure-kunder**: [Skicka och hantera supportförfrågningar](https://docs.microsoft.com/azure/azure-portal/supportability/how-to-create-azure-support-request) via Azure-portalen.

+ **Användarforum**: Ställ frågor, svara på frågor och få kontakt med andra användare i [Azure Machine Learning Studio (klassiskt) supportforum på MSDN](https://aka.ms/aml-forum-studio).

+ **Stack-spill:** Besök Azure Machine Learning-communityn på [StackOverflow](https://stackoverflow.com/questions/tagged/azure-machine-learning) som har taggats med "Azure-Machine-Learning".

+ **Dela produktförslag** och funktionsförfrågningar i vår [feedbackkanal](https://feedback.azure.com/forums/257792-machine-learning)för Azure Machine Learning . Om du vill dela din feedback väljer du knappen **Produktfeedback** i slutet av varje artikel.
