# <a name="team-data-science-process-for-developer-operations"></a>Team Data Science Process för utvecklare

Den här artikeln handlar Developer-åtgärder (DevOps)-funktioner som är specifika för en avancerad analys och Cognitive Services-lösningsimplementering. Dessa utbildningsmaterial är relaterade till Team Data Science Process (TDSP) och Microsoft och program med öppen källkod och verktyg, vilket är användbart för att visualisera, köra och leverera data science-lösningar. Den refererar till ämnen som täcker DevOps-Toolchain som är specifikt för datavetenskap och AI-produkter och lösningar.
