# <a name="team-data-science-process-for-data-scientists"></a>Tdsp för dataexperter

Den här artikeln ger vägledning till en uppsättning med mål som vanligtvis används för att implementera omfattande datavetenskapliga lösningar med Azure-tekniker. Vägleds du genom:

- Förstå en analytics-arbetsbelastning
- med hjälp av Team Data Science Process
- med hjälp av Azure Machine Learning
- Grunderna för överföring av replikeringsdata och lagring
- att tillhandahålla dokumentation om data-källa
- med hjälp av verktyg för bearbetning av trafikanalys

Dessa utbildningsmaterial är relaterade till Team Data Science Process (TDSP) och Microsoft och program med öppen källkod och verktyg, vilket är användbart för att visualisera, köra och leverera data science-lösningar.
