# <a name="explore-data-in-the-team-data-science-process"></a>Utforska data i TDSP (Team Data Science Process)

Att utforska data är ett steg i [teamdatavetenskapsprocessen](overview.md).

I följande artiklar beskrivs hur du utforskar data i tre olika lagringsmiljöer som vanligtvis används i datavetenskapsprocessen:

* Utforska [Azure blob-behållardata](explore-data-blob.md) med [Pandas](https://pandas.pydata.org/) Python-paketet.
* Utforska [SQL Server-data](explore-data-sql-server.md) med hjälp av SQL och med hjälp av ett programmeringsspråk som Python.
* Utforska [Hive-tabelldata](explore-data-hive-tables.md) med Hive-frågor.

Dessutom beskriver videon, [förbearbetningsdata i Azure Machine Learning Studio](https://azure.microsoft.com/documentation/videos/preprocessing-data-in-azure-ml-studio/), de vanliga modulerna för rengöring och omvandling av data.
