# <a name="additional-documentation"></a>Ytterligare dokumentation 

Vissa resurser utanför den här docs.microsoft.com webbplats kan hjälpa dig att förstå och arbeta med din Microsoft Azure FXT Edge Filer hybridcache.

## <a name="avere-vfxt-for-azure-cache-documentation"></a>Avere vFXT för Azure-cachedokumentation

[Avere vFXT för Azure](https://docs.microsoft.com/azure/avere-vfxt/) är en liknande klustercacheprodukt som använder virtuella noder i Microsoft Azure. Den skapas med Azure-portalen, men använder samma kontrollpanelkonfigurationsprogramvara som Azure FXT Edge Filer-maskinvarucachen. 

## <a name="legacy-hardware-documentation"></a>Äldre maskinvarudokumentation 

Denna teknik användes tidigare i produkter från Avere Systems. Användardokumentation för dessa äldre produkter finns på [webbplatsen för Avere Legacy Documentation](https://azure.github.io/Avere/). 

Dessa dokument kan ha användbar information:

* [Guiden för klusterkonfiguration](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html) – innehåller översiktsinformation om vanliga klusterkonfigurations- och hanteringsuppgifter och innehåller detaljerade förklaringar av klusterinställningssidorna 

* [Instrumentpanelsguide](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) - Förklarar funktionerna på **fliken Instrumentpanel på** kontrollpanelen

* [FXT Cluster Creation Guide](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html) - Guide för skapande av kluster från tidigare produkter
