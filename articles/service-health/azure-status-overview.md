# <a name="azure-status"></a>Azure-status
[Azure-status](https://azure.microsoft.com/status/) ger dig en global vy över hälsotillståndet för Azure-tjänster och regioner. Med Azure-status får du aktuell information på tjänstetillgänglighet. Status för Azure är tillgängligt för alla att visa alla tjänster som rapporterar sina tjänstens hälsotillstånd, samt incidenter med stor påverkan. Om du är en aktuella Azure-användare kan dock rekommenderar vi starkt att du kan använda den anpassade upplevelsen i [Azure Service Health](https://aka.ms/azureservicehealth). Azure Service Health innehåller alla avbrott, aktiviteter för planerat underhåll och rådgivning för tjänsten.

![Azure-statussidan](./media/azure-status-overview/azure-status.PNG)

## <a name="azure-status-updates"></a>Status för Azure-uppdateringar
Azure-statussidan uppdateras i realtid som hälsotillståndet för ändring av Azure-tjänster. Om du låter Azure-statussidan öppen kan du styra den hastighet med vilken sidan uppdateras med nya data. Du kan se den senaste gången sidan uppdaterades överst på sidan.

![Uppdatering av Azure-status](./media/azure-status-overview/update.PNG)

## <a name="azure-status-history"></a>Azure-statushistorik
Medan Azure-statussidan visas alltid den senaste informationen i hälsotillstånd, du kan visa äldre händelser med hjälp av den [sidan för Azure-status tidigare](https://azure.microsoft.com/status/history/).

Det kan du visa händelser för upp till 90 dagar tidigare, inklusive en preliminär rotorsaken, begränsning och nästa steg för Azure.

## <a name="rss-feed"></a>RSS-flöde
Status för Azure innehåller också [en RSS-feed](https://azure.microsoft.com/status/feed/) av ändringar av hälsotillståndet för Azure-tjänster som du kan prenumerera på.
