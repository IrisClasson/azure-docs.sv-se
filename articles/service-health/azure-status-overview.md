# <a name="azure-status-overview"></a>Översikt över Azure-status

[Azure-status](https://status.azure.com/status/) ger dig en global vy över hälso tillståndet för Azure-tjänster och-regioner. Med Azure-status kan du få information om tjänstens tillgänglighet. Azure-status är tillgänglig för alla som vill visa alla tjänster som rapporterar sin tjänste hälsa, samt incidenter med omfattande påverkan. Om du är en aktuell Azure-användare rekommenderar vi dock att du använder den anpassade upplevelsen i [Azure Service Health](https://aka.ms/azureservicehealth). Azure Service Health innehåller alla avbrott, kommande planerade underhålls aktiviteter och service rådgivare.

![Azure-statussida](./media/azure-status-overview/azure-status.PNG)

## <a name="azure-status-updates"></a>Azure status uppdateringar

Sidan Azure-status uppdateras i real tid när hälso tillståndet för Azure-tjänster ändras. Om du lämnar sidan Azure-status Öppen kan du styra den hastighet som sidan uppdateras med nya data. Längst upp kan du se den senaste gången Sidan uppdaterades.

![Uppdatering av Azure-status](./media/azure-status-overview/update.PNG)

## <a name="azure-status-history"></a>Status historik för Azure

Sidan Azure-status visar alltid den senaste hälso informationen, du kan visa äldre händelser på [sidan Azure status historik](https://status.azure.com/status/history/).

Från och med händelser som publicerades i november 2019 visar sidan status historik händelser upp till 365 dagar tidigare, inklusive en preliminär orsak, lösning och nästa steg för Azure.

## <a name="rss-feed"></a>RSS-flöde

Azure-status ger också [en RSS-feed](https://status.azure.com/status/feed/) med ändringar av hälso tillståndet för Azure-tjänster som du kan prenumerera på.
