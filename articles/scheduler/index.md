# <a name="scheduler-documentation"></a>Dokumentation om Scheduler

> [!IMPORTANT]
> [Azure Logic Apps](../logic-apps/logic-apps-overview.md) ersätter Azure Scheduler, som [dras tillbaka](../scheduler/migrate-from-scheduler-to-logic-apps.md#retire-date). Om du vill fortsätta att arbeta med de jobb du konfigurerat i Scheduler behöver du [migrera till Azure Logic Apps](../scheduler/migrate-from-scheduler-to-logic-apps.md) så snart som möjligt. 
>
> Scheduler är inte längre tillgänglig i Microsoft Azure-portalen, men [REST API](/rest/api/scheduler) och [Azure Scheduler PowerShell-cmdletar](scheduler-powershell-reference.md) är fortfarande tillgängliga så att du kan hantera dina jobb och jobbsamlingar.

Lär dig mer om att skapa, schemalägga, köra, hantera och uppdatera automatiserade återkommande jobb i molnet. Självstudier, API-referenser och annan dokumentationen som visar hur du schemalägger och övervakar jobb, som återkommande programåtgärder och rutinunderhåll.
