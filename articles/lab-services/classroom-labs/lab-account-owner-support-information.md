# <a name="set-up-support-information-lab-account-owner-in-azure-lab-services"></a>Konfigurera supportinformation (labbkontoägare i Azure Lab Services)
I den här artikeln beskrivs hur du (som labbkontoägare) kan ange supportinformation som labbskapare (lärare) och labbanvändare (deltagare) kan använda för att få hjälp om de stöter på tekniska problem med att använda labbkontot eller labbet. 

Supportinformationen omfattar:

- URL
- E-post
- Telefon
- Ytterligare instruktioner 

## <a name="specify-support-information"></a>Ange supportinformation
1. Logga in på [Azure Portal](https://portal.azure.com).
2. I sökfältet anger du **Lab Services**och väljer **Lab Services** i sökresultaten. 
3. Välj ditt labbkonto i listan över labbkonton. 
4. Byt till sidan **Internt stöd** gör du följande:
    1. Ange **support-URL.** 
     2. Ange **supportmeddelandet**. 
     3. Ange **supporttelefonen**.
     4. Ange detaljerade **supportinstruktioner** (valfritt). Labbägare och användare ser den här texten tillsammans med supportkontaktinformationen. Webbadresser omvandlas automatiskt till länkar. 
     5. Välj **Spara** i verktygsfältet.
