# <a name="azure-database-for-mysql-videos"></a>Azure Database för MySQL-videor

[Översikt: Azure Database för PostgreSQL och MySQL](https://channel9.msdn.com/Events/Connect/2017/T147)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T147/player]

Azure Database för PostgreSQL och Azure Database för MySQL sammanför community edition databasmotorer och funktioner i en helt hanterad tjänst – så att du kan fokusera på dina appar i stället för att hantera en databas. Kolla in för att få en snabb översikt över fördelarna med att använda tjänsten och se några av funktionerna i praktiken.

[Djupdykning funktioner för hanterade tjänster för MySQL och PostgreSQL](https://channel9.msdn.com/Events/Connect/2017/T148)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T148/player]

Azure Database för PostgreSQL och Azure Database för MySQL sammanför community edition databasmotorer och funktionerna i en helt hanterad tjänst. Kolla in för att få djupgående information om hur de här tjänsterna fungerar, hur vi garantera hög tillgänglighet och snabb skalning (inom några sekunder), så du kan uppfylla kundernas behov. Du får också lära dig om några av de underliggande investeringarna i säkerhet och tillgänglighet i hela världen.

[Hur du kommer igång med den nya Azure Database for MySQL-tjänsten](https://channel9.msdn.com/events/Build/2017/B8045)

>[!VIDEO https://channel9.msdn.com/Events/Build/2017/B8045/player]

I den här videon från maj 2017 Microsoft //Build konferens, Lär dig mer om Microsofts hanterad MySQL-tjänst i Azure. Videon beskriver Microsofts strategi för att stödja databassystem för öppen källkod i Azure. Videon beskriver vad det innebär att du som utvecklare att utveckla eller distribuera program som använder MySQL i Azure. Den här videon visar en översikt över arkitekturen i tjänsten och visar Azure Database for MySQL är integrerad med andra Azure-tjänster som Web Apps.
