# <a name="azure-database-for-mysql-videos"></a>Azure Database för MySQL-videor

[Översikt: Azure Database för PostgreSQL och MySQL](https://channel9.msdn.com/Events/Connect/2017/T147)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T147/player]

Azure Database for PostgreSQL och Azure Database for MySQL samlar databasmotorer och funktioner i Community Edition i en fullständigt hanterad tjänst – så att du kan fokusera på dina appar i stället för att behöva hantera en databas. Lyssna på dig för att få en snabb överblick över fördelarna med att använda tjänsten och se några av funktionerna i aktion.

[Djupdykning om hanterade servicefunktioner för MySQL och PostgreSQL](https://channel9.msdn.com/Events/Connect/2017/T148)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T148/player]

Azure Database for PostgreSQL och Azure Database for MySQL samlar databasmotorer för community edition och funktionerna i en fullständigt hanterad tjänst. Lyssna på hur dessa tjänster fungerar – hur vi säkerställer hög tillgänglighet och snabb skalning (inom några sekunder), så att du kan uppfylla dina kunders behov. Du får också lära dig om några av de underliggande investeringarna i säkerhet och tillgänglighet över hela världen.

[Så här kommer du igång med den nya Azure Database for MySQL-tjänsten](https://channel9.msdn.com/events/Build/2017/B8045)

>[!VIDEO https://channel9.msdn.com/Events/Build/2017/B8045/player]

I det här videoklippet från Microsoft //Build-konferensen i maj 2017 kan du läsa mer om Microsofts hanterade MySQL-erbjudande i Azure. Videon går igenom Microsofts strategi för att stödja databassystem med öppen källkod i Azure. Videon beskriver vad det innebär för dig som utvecklare att utveckla eller distribuera program som använder MySQL i Azure. Den här videon visar en översikt över tjänstens arkitektur och visar att Azure Database for MySQL är integrerat med andra Azure-tjänster som Webbappar.
