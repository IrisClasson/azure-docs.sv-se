# <a name="connect-trend-micro-deep-security"></a>Anslut trend mikro djup säkerhet

Med Trend Micro Deep Security-kopplingen kan du enkelt ansluta dina Deep Security-loggar med Azure Sentinel, för att visa instrumentpaneler, skapa anpassade aviseringar och förbättra undersökningen. Detta ger dig mer insikt i organisationens nätverk och system och förbättrar dina funktioner för säkerhetsdrift.


## <a name="prerequisites"></a>Krav

- Läs- och skrivbehörigheter till Arbetsytan Azure Sentinel.

## <a name="connect-azure-sentinel-to-trend-micro-deep-security"></a>Anslut Azure Sentinel till Trend Micro Deep Security

1. I Azure-portalen navigerar du till **Azure Sentinel** > **Data-kopplingar** och väljer sedan **trend micro deep security-anslutningsappen.**

2. Välj **Öppna kopplingssida**.

3. Följ instruktionerna och nästa steg.
