# <a name="connect-extrahop-revealx"></a>Anslut ExtraHop Visa (x)

Med ExtraHop Reveal (x) data Connector kan du enkelt ansluta ditt Reveal (x)-system med Azure Sentinel för att visa instrument paneler, skapa anpassade aviseringar och förbättra undersökningen. Den här integrationen ger dig möjlighet att få insikt i din organisations nätverk och förbättra dina funktioner för säkerhets åtgärder.


## <a name="prerequisites"></a>Förutsättningar

- Läs-och skriv behörigheter till Azure Sentinel-arbetsytan.

- ExtraHop-identifiering eller kommando apparat med version 7,8 eller senare av inbyggd program vara med ett användar konto som har obegränsade (administratör) privilegier.

## <a name="connect-azure-sentinel-to-extrahop-revealx"></a>Anslut Azure Sentinel till ExtraHop Reveal (x)

1. I Azure Portal går du till **Azure Sentinel** > **data Connectors** och väljer sedan **ExtraHop Reveal (x)-** anslutaren.

2. Välj **Öppna kopplings sida**.

3. Följ instruktionerna på sidan **ExtraHop Reveal (x)** .
