# <a name="connect-extrahop-revealx"></a>Ansluta ExtraHop Reveal(x)

Med ExtraHop Reveal(x)-dataanslutningen kan du enkelt ansluta ditt Reveal(x)-system med Azure Sentinel för att visa instrumentpaneler, skapa anpassade aviseringar och förbättra undersökningen. Den här integreringen ger dig möjlighet att få insikt i organisationens nätverk och förbättra dina funktioner för säkerhetsdrift.


## <a name="prerequisites"></a>Krav

- Läs- och skrivbehörigheter till Arbetsytan Azure Sentinel.

- ExtraHop Discover- eller Command-apparat med firmware version 7.8 eller senare med ett användarkonto som har obegränsad (administratör) behörighet.

## <a name="connect-azure-sentinel-to-extrahop-revealx"></a>Anslut Azure Sentinel till ExtraHop Reveal(x)

1. I Azure-portalen navigerar du till **Azure Sentinel** > **Data-kopplingar** och väljer sedan **ExtraHop Reveal(x)-anslutningen.**

2. Välj **Öppna kopplingssida**.

3. Följ instruktionerna på sidan **ExtraHop Reveal(x).**
