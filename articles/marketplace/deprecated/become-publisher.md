# <a name="become-a-cloud-marketplace-publisher"></a>Bli en Cloud Marketplace-utgivare

Den här artikeln beskriver hur du registrerar sig som en Cloud Marketplace-utgivare. Beroende på ditt valda publicerings alternativ och det aktuella medlemskapet Microsoft Partner Network några av följande steg kanske inte krävs. När du har registrerat dig kan du skapa Marketplace-erbjudanden för Azure Marketplace eller AppSource.


|  | Registrerings steg | Varaktighet | Information |  
|:--- |:--- |:--- |:--- |  
| 1 | Registrera dig i Microsoft Partner Network | 15 min | [Registrera dig i Microsoft Partner Network](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-account) |  
| 2 | Skapa ett Microsoft-konto <ul>(krävs för Azure Marketplace Transact-erbjudanden, rekommenderas för andra)</ul> | 15 min | [Skapa ett Microsoft-ID](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-account) |  
| 3 | Skapa ett partner Center-konto | 15 min | [Skapa ett Partnercenter-konto](https://partner.microsoft.com/dashboard/account/v3/enrollment/introduction/azureisv) |  
| 4 |  Logga in på Cloud Partner Portal | 1-3 dagar | [Logga in på Cloud Partner Portal](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-account#sign-in-with-a-work-account) |  
| 5 | Registrera dig i Microsoft Developer Center (dev Center) (för Azure Marketplace Transact-erbjudanden) | 5-10 dagar | [Registrera i Dev Center](/azure/marketplace/deprecated/register-dev-center) |  


## <a name="1-register-in-microsoft-partner-network"></a>1. registrera i Microsoft Partner Network  
| 15 min |  
Deltagande i Microsofts moln marknads plats kräver medlemskap i Microsoft Partner Network (MPN), som också ger viktiga fördelar, program och prestanda spårning av partner. Om din organisation ännu inte är medlem, kan du gå till Microsoft Partner Network (MPN) för att bli en officiell Microsoft-partner och få ytterligare förmåner och stöd för publicering i Marketplace. 

*   Om du vill registrera dig i Microsoft Partner Network går du till [sidan medlemskap](https://partner.microsoft.com/membership) och följer stegen för att starta din registrering.  

När du har registrerat dig registrerar du Microsoft Partner Network-ID för din organisation. Du behöver detta under steg 3 när du skickar in ditt registrerings formulär för Marketplace.
*   Lär dig mer om fördelarna med [Microsoft Partner Network](https://partner.microsoft.com/commercial)
*   Om din organisation har ett befintligt medlemskap, kommer du att ansluta till din organisation under registreringen. Om du använder [partner Center](https://partnercenter.microsoft.com/partner/home)kommer du att kunna se ID: t i ditt partner Center-konto. Om du använder en [partner medlemskaps Center (PMC)](https://partners.microsoft.com/partnerprogram/PartnerMembershipCenter.aspx)kommer du att kunna se det i ditt partner medlemskap Center-konto. 

Öppna ett ärende med Microsoft [partner support](https://partner.microsoft.com/support) för problem med MPN-registrering.

## <a name="2-create-a-microsoft-account"></a>2. skapa ett Microsoft-konto  
| 15 min |  
En Microsoft-konto, t. ex. *\@Outlook.com eller \@Live.com*, används för att få åtkomst till många Microsoft-enheter och-tjänster. Det krävs för Azure Marketplace Transact-erbjudanden och vi rekommenderar att de används för alla moln utgivare. Mer information om fördelarna och användningen av en Microsoft-konto [finns på den här sidan](https://account.microsoft.com/account/). Du kommer att använda en personlig Microsoft-konto för att få åtkomst till Cloud Partner Portal, den publicerings plattform som du ska använda för att skapa och hantera dina Marketplace-erbjudanden. 

[! VIKTIGT] Tänk på att en Microsoft-konto som är kopplad till ditt arbets-e-postmeddelande inte kommer att tillåta åtkomst till portalen.

> [!Note]
> Om du redan har en Microsoft-konto kan du läsa [rikt linjerna för att skapa ett Microsoft-ID för att hantera ett Marketplace-konto](https://docs.microsoft.com/azure/marketplace/partner-center-portal/manage-account#manage-tenants) för att avgöra om det kan användas för Marketplace-publicering. 
> 
> Användare med ett Azure Active Directory federerat företags konto bör läsa [den här artikeln](https://docs.microsoft.com/azure/marketplace/partner-center-portal/manage-account#manage-users) innan du försöker skapa eller använda ditt befintliga Microsoft-konto.

*   Om du vill ha hjälp med att skapa eller hantera din Microsoft-konto går du till [support sidorna för Microsoft-konto Hjälp](https://support.microsoft.com/products/microsoft-account?category=manage-account).

## <a name="4-sign-into-cloud-partner-portal"></a>4. Logga in på Cloud Partner Portal
| 1-3 dagar | <br>
Vårt onboarding-team kommer att validera Marketplace-registrerings information som anges i steg 3. När du har godkänt får du ett välkomst meddelande med autentiseringsuppgifter och vägledning om hur du loggar in på [Cloud Partner Portal](https://cloudpartner.azure.com).

Du kommer att använda [Cloud Partner Portal](https://cloudpartner.azure.com) för att skapa, publicera och hantera Marketplace-erbjudanden. När du får ett e-postmeddelande om godkännande kan du omedelbart logga in på [Cloud Partner Portal](https://cloudpartner.azure.com) med din Microsoft-konto för att få till gång till ytterligare teknisk information om publicering på Marketplace och skapa ditt erbjudande. 
*   Mer information om hur du använder Cloud Partner Portal [finns i avsnittet komma igång artikeln i avsnittet Läs mer](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cloud-partner-portal-getting-started-with-the-cloud-partner-portal). Du måste vara inloggad på Cloud Partner Portal för att få åtkomst till dessa uppgifter.
*   Om du har frågor eller vill schemalägga ett onboarding-telefonsamtal, granskar du andra artiklar i den här hand boken för utgivaren eller öppnar en supportbegäran under problem typ > Marketplace-onboarding inifrån Cloud Partner Portal.
*   Om du inte fick välkomst meddelandet kontrollerar du mappen spam för ett e-postmeddelande med ämnes raden "Välkommen till Microsoft Marketplace!." [Kontakta Microsoft-supporten](https://support.microsoft.com/getsupport?wf=0&tenant=classiccommercial&oaspworkflow=start_1.0.0.0&locale=&supportregion=&pesid=16230&forceorigin=esmc&ccsid=636595105151894820)om det inte finns något e-postmeddelande i skräppostmappen. På sidan support väljer du: problem typ > Marketplace-onboarding.  Microsoft Marketplace Publisher onboarding-teamet kommer att vara glad att hjälpa dig med dina frågor. 


## <a name="5-register-in-dev-center-for-transact-publishing-option-only"></a>5. registrera dig i dev Center (endast för Transact Publishing option) 
| 5-10 dagar |  
Ett [dev Center](https://developer.microsoft.com/store/register) -konto krävs för erbjudanden som använder alternativet för Transact-publicering: innehåller virtuella datorer, Azure-appar och SaaS-appar som kon figurer ATS för försäljning via Azure. Information om [dev Center](https://developer.microsoft.com/store/register) -kontot används av Microsoft för att verifiera juridisk information, skatte-och bank information för din organisation som ska användas för avbetalnings syfte när du säljer och fakturerar licens avgifter för program vara via Azure Marketplace. 

Du kan starta registrerings processen för [dev Center](https://developer.microsoft.com/store/register) innan du skapar ditt erbjudande i Cloud Partner Portal, men du måste slutföra innan du publicerar Marketplace-erbjudandet. Med den här stegets normala varaktighet på 5-10 dagen rekommenderar vi att du startar den här processen så snart som möjligt om du tänker använda alternativet för att använda Transact-publicering. 

Enskilda registreringar i [dev Center](https://developer.microsoft.com/store/register) måste vara en giltig representant för din organisation och måste tillhandahålla personlig information för att verifiera sin identitet. Registrera dig för Microsoft-konto, som du skapade i steg 2, och Använd samma Microsoft-konto som du använde för att få åtkomst till Cloud Partner Portal.

*   Registrera dig för ett [dev Center](https://developer.microsoft.com/store/register) -konto genom att [Starta registrerings processen här](https://developer.microsoft.com/store/register).
*   Mer information om hur du skapar en Microsoft-konto finns i avsnittet *skapa en Microsoft-konto* .
 
När du har slutfört registreringen av [dev Center](https://developer.microsoft.com/store/register) länkar du din moln partner profil med ditt [dev Center](https://developer.microsoft.com/store/register) -konto ([se detaljerade anvisningar här](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cloud-partner-portal-manage-publisher-profile)). Det här steget måste utföras innan du publicerar ditt Transact-erbjudande. 

>[!Note]
>För att frångå $99-registrerings avgiften för Developer Center, fyller du i [registrerings formuläret för Marketplace](https://partner.microsoft.com/dashboard/account/v3/enrollment/introduction/azureisv) så får du ett e-postmeddelande som innehåller din kampanj kod. Mer information finns i så här [skapar du ett partner Center-konto](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-account) .

>[!Important]
>Innan du skapar ett Microsoft dev Center-konto kontrollerar du att din organisation inte redan har ett dev Center-konto för att undvika dubbla konton. Mer information finns i avsnittet [Registrera dig i dev Center](https://docs.microsoft.com/azure/marketplace/register-dev-center) .
