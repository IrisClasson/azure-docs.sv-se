# <a name="become-a-cloud-marketplace-publisher"></a>Bli en Cloud Marketplace Publisher

Den här artikeln beskriver registrering som en molnmarknadsplatsutgivare. Beroende på vilket publiceringsalternativ du har valt och ditt nuvarande medlemskap i Microsoft Partner Network kanske det inte krävs några av följande steg. När du har registrerat dig kan du skapa marketplace-erbjudanden för Azure Marketplace eller AppSource.


|  | Registreringssteg | Varaktighet | Information |  
|:--- |:--- |:--- |:--- |  
| 1 | Registrera dig i Microsoft Partner Network | 15 min | [Registrera dig i Microsoft Partner Network](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-account) |  
| 2 | Skapa ett Microsoft-konto <ul>(krävs för Azure Marketplace transact erbjudanden, rekommenderas för andra)</ul> | 15 min | [Skapa ett Microsoft-ID](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-account) |  
| 3 | Skapa ett Partnercenter-konto | 15 min | [Skapa ett Partnercenter-konto](https://partner.microsoft.com/dashboard/account/v3/enrollment/introduction/azureisv) |  
| 4 |  Logga in på Cloud Partner Portal | 1-3 dagar | [Logga in på Cloud Partner Portal](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-account#sign-in-with-a-work-account) |  
| 5 | Registrera dig i Microsoft Developer Center (Dev Center) (för Azure Marketplace-transacterbjudanden) | 5-10 dagar | [Registrera i Dev Center](/azure/marketplace/deprecated/register-dev-center) |  


## <a name="1-register-in-microsoft-partner-network"></a>1. Registrera dig i Microsoft Partner Network  
| 15 min |  
Deltagande i Microsofts molnmarknadsplats kräver medlemskap i Microsoft Partner Network (MPN), vilket också ger viktiga fördelar, program och spårning av partnerprestanda. Om din organisation ännu inte är medlem går du med i Microsoft Partner Network (MPN) för att bli en officiell Microsoft-partner och får ytterligare fördelar och stöd för publicering på marketplace. 

*   Om du vill registrera dig i Microsoft Partner Network besöker du [sidan Medlemskap](https://partner.microsoft.com/membership) och följer stegen för att starta registreringen.  

När du har registrerat dig registrerar du Microsoft Partner Network ID för din organisation. Du behöver detta under steg 3 när du skickar in ditt registreringsformulär för marketplace.
*    Läs mer om fördelarna med [Microsoft Partner Network](https://partner.microsoft.com/commercial)
*    Om din organisation har ett befintligt medlemskap kommer du att gå med i din organisation under registreringen. Om du använder [Partner Center](https://partnercenter.microsoft.com/partner/home)kan du se ID:t i ditt Partner Center-konto. Om du använder [PMC (Partner Membership Center)](https://partners.microsoft.com/partnerprogram/PartnerMembershipCenter.aspx)kan du se det i ditt partnermedlemskapscenterkonto. 

Öppna en biljett med [Microsoft-partnersupport](https://partner.microsoft.com/support) för problem med MPN-registrering.

## <a name="2-create-a-microsoft-account"></a>2. Skapa ett Microsoft-konto  
| 15 min |  
Ett Microsoft-konto, * \@ \@t.live.com outlook.com*ex. Det krävs för Azure Marketplace transact erbjudanden, och vi rekommenderar dess användning till alla molnutgivare. Mer information om fördelarna med och användningen av ett Microsoft-konto [finns på den här sidan](https://account.microsoft.com/account/). Du kommer att använda ett personligt Microsoft-konto för att komma åt Cloud Partner Portal, den publiceringsplattform som du använder för att skapa och hantera dina marketplace-erbjudanden. 

[! VIKTIGT] Observera att ett Microsoft-konto som är kopplat till din e-postadress på jobbet inte tillåter dig att komma åt portalen.

> [!Note]
> Om du redan har ett Microsoft-konto läser du [riktlinjerna för att skapa ett Microsoft-ID för att hantera ett marketplace-konto](https://docs.microsoft.com/azure/marketplace/partner-center-portal/manage-account#manage-tenants) för att avgöra om det kan användas för marketplace-publicering. 
> 
> Användare med ett Azure Active Directory-federerat företagskonto bör läsa [den här artikeln](https://docs.microsoft.com/azure/marketplace/partner-center-portal/manage-account#manage-users) innan de försöker skapa eller använda ditt befintliga Microsoft-konto.

*    Om du vill ha hjälp med att skapa eller hantera ditt Microsoft-konto besöker du [supportsidorna för Microsoft-kontot](https://support.microsoft.com/products/microsoft-account?category=manage-account).

## <a name="4-sign-into-cloud-partner-portal"></a>4. Logga in på Cloud Partner Portal
| 1–3 dagar | <br>
Vårt Marketplace Onboarding Team validerar registreringsuppgifterna för marknadsplatsen som anges i steg 3. När du har godkänts får du ett välkomstmeddelande med kontouppgifter och vägledning om inloggning till [Cloud Partner Portal](https://cloudpartner.azure.com).

Du kommer att använda [Cloud Partner Portal](https://cloudpartner.azure.com) för att skapa, publicera och hantera marketplace-erbjudanden. När du har fått välkomstmeddelandet för godkännande kan du omedelbart logga in på [Cloud Partner Portal](https://cloudpartner.azure.com) med ditt Microsoft-konto för att få tillgång till ytterligare teknisk information om publicering på Marketplace och skapa ditt erbjudande. 
*    Mer information om hur du använder Cloud Partner Portal [finns i avsnittet Komma igång i Avsnittet Lär](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cloud-partner-portal-getting-started-with-the-cloud-partner-portal)dig . Du måste vara inloggad på Cloud Partner Portal för att få tillgång till dessa uppgifter.
*    Om du har frågor eller vill schemalägga ett introduktionsöversiktsanrop läser du andra artiklar i den här Publisher-guiden eller öppnar en supportbegäran under Problemtyp > Marketplace Onboarding från Cloud Partner Portal.
*    Om du inte fick välkomstmeddelandet kontrollerar du om det finns ett e-postmeddelande med ämnesraden Välkommen till Microsoft Marketplace!. Om det inte finns någon e-post i skräppostmappen kontaktar du [Microsofts support](https://support.microsoft.com/getsupport?wf=0&tenant=classiccommercial&oaspworkflow=start_1.0.0.0&locale=&supportregion=&pesid=16230&forceorigin=esmc&ccsid=636595105151894820). På supportsidan väljer du: Problemtyp > Marketplace Onboarding.  Vårt Onboarding-team för Marketplace Publisher hjälper gärna till med dina frågor. 


## <a name="5-register-in-dev-center-for-transact-publishing-option-only"></a>5. Registrera dig i Dev Center (endast för alternativ för transactpublicering) 
| 5–10 dagar |  
Ett [Dev Center-konto](https://developer.microsoft.com/store/register) krävs för erbjudanden som använder alternativet för transact-publicering: inkluderar virtuella datorer, Azure-appar och SaaS-appar som konfigurerats för att sälja via Azure. [Dev](https://developer.microsoft.com/store/register) Center-kontoinformationen används av Microsoft för att validera den juridiska, skattemässiga och bankinformation för din organisation som ska användas i utbetalningssyfte när du säljer och fakturerar dina licensavgifter för programvara via Azure Marketplace. 

[Registreringsprocessen](https://developer.microsoft.com/store/register) för Dev Center kan startas innan du skapar ditt erbjudande i Cloud Partner Portal, men måste slutföras innan du publicerar ditt marketplace-erbjudande. Med tanke på den typiska varaktigheten på 5-10 dagar rekommenderar vi att du startar den här processen så snart som möjligt om du tänker använda alternativet för transactpublicering. 

Den person som registrerar sig i [Dev Center](https://developer.microsoft.com/store/register) måste vara en giltig representant för din organisation och måste tillhandahålla personlig information för att verifiera sin identitet. Registrera dig med ditt Microsoft-konto, skapad i steg 2 och använd samma Microsoft-konto som du använde för att komma åt Cloud Partner Portal.

*    Om du vill registrera dig för ett [Dev Center-konto](https://developer.microsoft.com/store/register) [startar du registreringsprocessen här](https://developer.microsoft.com/store/register).
*    Mer information om hur du skapar ett Microsoft-konto finns i avsnittet *Skapa ett Microsoft-konto.*
 
När du har slutfört [Dev Center-registreringen](https://developer.microsoft.com/store/register) kan du länka din Cloud Partner-profil till ditt Dev [Center-konto](https://developer.microsoft.com/store/register) [(se detaljerade steg här).](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cloud-partner-portal-manage-publisher-profile) Det här steget måste slutföras innan du publicerar ditt transact-erbjudande. 

>[!Note]
>Om du vill avstå från registreringsavgiften för 99 USD i Developer Center fyller du i [registreringsformuläret för marknadsplatsen](https://partner.microsoft.com/dashboard/account/v3/enrollment/introduction/azureisv) så får du ett e-postmeddelande med din kampanjkod. Mer information finns i hur du [skapar ett Partner Center-konto.](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-account)

>[!Important]
>Innan du skapar ett Microsoft Dev Center-konto kontrollerar du att din organisation inte redan har ett Dev Center-konto för att undvika dubblettkonton. Mer information finns i avsnittet [Registrera dig i Dev Center.](https://docs.microsoft.com/azure/marketplace/register-dev-center)
