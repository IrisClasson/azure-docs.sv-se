# <a name="co-sell-partner-engagement"></a>Deltagande i partnersäljsamarbete

Microsofts program för co-försäljning gör det möjligt för partner att arbeta direkt med Microsoft på gemensamma Sälj möjligheter. Du kan komma åt en omfattande community med kunder och skapa nya försäljnings möjligheter som kan hjälpa till att öka företagets tillväxt.

## <a name="partner-selling-motions"></a>Partner Sälj initiativ

Det finns tre olika partner Sälj rörelserna:

* *Sälj partner*: du kan sälja Microsoft-teknik via egna säljare.
* *Tjänster Co-Sälj*: säljare och partners samarbetar för att sälja ditt projekt eller hanterade tjänster med Microsoft Technology Embedded.
* *IP-Co-Sälj*: säljare/Partners samarbetar för att sälja din upprepade, paketerade IP-lösning *.

\* några av de här lösningarna för IP-samförsäljning kan få ytterligare incitament när de har sålts och distribuerats med Microsofts sälj team.
