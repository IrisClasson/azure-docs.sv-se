# <a name="co-sell-partner-engagement"></a>Samförsäljnings partner engagemang

Microsofts program för co-försäljning gör det möjligt för partner att arbeta direkt med Microsoft på gemensamma Sälj möjligheter. Du kan komma åt en omfattande community med kunder och skapa nya försäljnings möjligheter som kan hjälpa till att öka företagets tillväxt.

## <a name="partner-selling-motions"></a>Partner Sälj initiativ

Det finns tre olika partner Sälj rörelserna:

* *Sälj partner*: Du kan sälja Microsoft-teknik via egna säljare.
* *Tjänster i Co-Sälj*: Säljare och partners samarbetar för att sälja ditt projekt eller hanterade tjänster med Microsoft-teknik Embedded.
* *IP-Co-Sälj*: Säljare/partner samarbetar för att sälja din upprepnings bara, paketerade IP-lösning *.

\*Några av de här IP-relaterade lösningarna kan få ytterligare incitament när de har sålts och distribuerats med Microsofts försäljnings team.
