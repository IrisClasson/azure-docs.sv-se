# <a name="co-sell-partner-engagement"></a>Deltagande i partnersäljsamarbete

Microsofts co-sell-program gör det möjligt för partner att arbeta direkt med Microsoft om gemensamma försäljningsmöjligheter. Du kan nå en stor gemenskap av kunder och skapa nya försäljningsmöjligheter som kan bidra till att påskynda företagens tillväxt.

## <a name="partner-selling-motions"></a>Partner säljer motioner

Det finns tre partner som säljer motioner:

* *Sälj via partner:* Du kan sälja Microsoft-teknik via dina egna säljare.
* *Tjänster medsäljersäljning:* Säljare och partner samarbetar för att sälja ditt projekt eller hanterade tjänster med Inbyggd Microsoft-teknik.
* *IP co-sell*: Säljare / partners samarbetar för att sälja din repeterbara, paketerade IP-lösning *.

\*Vissa av dessa IP-lösningar för samförsäljning kan få ytterligare incitament när de säljs och distribueras med Microsofts fältförsäljningsteam.
