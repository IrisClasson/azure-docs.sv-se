# <a name="cloud-solution-providers"></a>Leverantörer av molnlösningar

Programvara erbjudanden kan nu nå miljontals kvalificerade Microsoft-kunder som hanteras av Cloud Solution Provider (CSP) partner med minimal investering för marknadsföring och försäljning.  Utgivare kan nu få CSP-partner kanaler, förutom allmän tillgänglighet av erbjudanden via [Microsofts web butiker](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace).

Utgivare kan välja att delta i ett erbjudande för tillgänglighet i en kanal för CSP-partner, för ett nytt erbjudande eller en befintlig. Det här alternativet kan CSP-partner att sälja dina produkter och skapa tillsammans lösningar för kunder.

Utgivare har ansvaret för att tillhandahålla avbrottssupport till slutkunder och för att tillhandahålla en mekanism för CSP-partner och/eller kunder att kontakta dig för support. Det är bästa praxis för att ange CSP-partner med användardokumentation, utbildning, och tjänsten health/avbrottsmeddelanden (som tillämpligt) så att CSP-partner är lämpade att hantera nivå 1 supportärenden från våra kunder.

I följande tabell visar typer av erbjudanden och prismodellerna som kan anmäla sig till CSP-partner kanal:

| **Erbjudandetyp**    | **BYOL**  |  **Kostnadsfri** | **PAYG**   |
| :---------------- | :---------|:----------|:-----------|
| Virtuella datorer  | Ja | Ja | Ja |
| Containrar | Gäller inte | Ja | Gäller inte |
| Lösningsmallar | Gäller inte | Ja | Gäller inte |
| Managed Applications | Gäller inte | Ja | Ja |
| IT-lösning SaaS (per plats priser) | Gäller inte | Ja | Ja |
|   |   |   |
