# <a name="azure-appsource-and-marketplace-review-policies"></a>Azure AppSource och Marketplace granska principer

Klassificering och principer för Azure AppSource och Marketplace visas här.

* I AppSource och Azure Marketplace granskningar tolererar inte Microsoft följande beteende eller innehåll:
    * Bot-cyberattacker
    * Sekretess kompatibilitetsbrott
    * Skräppost innehåll
    * Bedrägeri innehåll
    * Stötande innehåll: obscent, olämpliga eller stötande språk eller gester
    * Ogiltigt innehåll
    * Missbruk, hateful eller hotfulla innehåll
    * Kränkande innehåll
    * Upprepad bidrag av samma eller liknande innehåll
    * Annonsering, inklusive befordran av andra appar och tjänster

* Microsoft förbehåller sig rätten att ta bort granska-relaterat innehåll som skickas av en användare av någon anledning.
* Microsoft förbehåller sig rätten att blockera en användare från att skicka granska-relaterat innehåll av någon anledning.
* Utgivare får inget meddelande på borttagning av granska relaterat innehåll som relaterar till sina program.
* Användare får inget meddelande på borttagning av granskning-relaterat innehåll som har lämnat in.
* Microsoft är inte skyldig att ge en förklaring om borttagning av granskning-relaterat innehåll.
* Microsoft följer inte förfrågningar om att återställa borttagna granska-relaterat innehåll.

## <a name="review-guidelines-for-customers"></a>Riktlinjer för granskning för kunder

* Inte skickar innehåll som kan vara strider mot den [AppSource och Azure Marketplace granska princip](./rating-review-policies.md).
* Omfattar inte personligt identifierbar information (PII), till exempel e-postadress, fysiska adress, telefonnummer och osv.
* Se till att granska är tydlig, läsbara och informativa.
* Kontrollera att åsikter som uttrycks i Granska är balanserade och neutral.
* Kom ihåg att du lägger upp i ett öppet forum och agera utifrån dessa.

## <a name="review-guidelines-for-publishers"></a>Riktlinjer för granskning för utgivare

* Inte publicera innehåll som kan vara strider mot den [AppSource och Azure Marketplace granska princip](./rating-review-policies.md).
* Inkludera inte några marknadsföringsinnehåll eller annonser i din kommentar.
* Uppgradera inte andra program eller tjänster i din kommentar.
* Ditt svar bör direkt relaterade till specifika appen och granskning.
* Använd inte kapslade svar och inte publicera kapslade kommentarer över flera granskningar.
* Ändra inte granskningar på något sätt, t.ex att erbjuda ersättning eller kompensation incitament till granskare för att ändra betyg eller granskningar.
* Alltid vara korrekt och Överväg att nöjda kunder är de bästa medel för din app.
