### <a name="industries"></a>Branscher

Branschval gäller endast för erbjudanden som publiceras på AppSource och Consulting Services som publiceras på Azure Marketplace. Välj branscher och/eller vertikaler om ditt erbjudande tillgodoser branschspecifika behov och ropar branschspecifika funktioner i erbjudandebeskrivningen. Du kan välja upp till två (2) branscher och två (2) vertikaler per vald bransch.

### <a name="applicable-products"></a>Tillämpliga produkter

Välj de produkter som appen arbetar med för att erbjudandet ska visas under utvalda produkter i AppSource.

### <a name="search-keywords"></a>Sök nyckelord

Sökord kan hjälpa kunderna att hitta ditt erbjudande när de söker. Identifiera de bästa sökorden för ditt erbjudande, införliva dem i ditt erbjudande sammanfattning och beskrivning samt i sökordet i avsnittet erbjudande lista detaljer.

## <a name="link-to-your-offer-page-from-your-website"></a>Länk till din erbjudandesida från din webbplats

När du länkar från AppSource- eller Azure Marketplace-märket på din webbplats till dina uppgifter på marketplace kan du stödja stark analys och rapportering genom att inkludera följande frågeparametrar i slutet av webbadressen:
* **src**: Inkludera källan från vilken trafiken dirigeras till AppSource (till exempel webbplats, LinkedIn eller Facebook).
* **mktcmpid**: Ditt marknadsföringskampanj-ID, som kan innehålla upp till 16 tecken i valfri kombination av bokstäver, siffror, understreck och bindestreck (till exempel *blogpost_12*).

Följande exempel-URL innehåller båda föregående frågeparametrar:`https://appsource.microsoft.com/product/dynamics-365/mscrm.04931187-431c-415d-8777-f7f482ba8095?src=website&mktcmpid=blogpost_12`

Genom att lägga till parametrarna i appsource-URL:en kan du granska hur effektiv kampanjen är i instrumentpanelen för analys i [Partner Center](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/marketplaceinsights).
