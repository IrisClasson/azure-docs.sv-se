<a name="managing-users-on-cloud-partner-portal"></a>Hantera användare på molnpartnerportalen
======================================

Cloud Partner Portal kan du lägga till användare med rollbaserad åtkomst till virtuella datorerbjudanden. Det hjälper till att hantera åtkomst och behörigheter medan flera personer arbetar med erbjudanden.

Så här lägger du **till användare och tilldelar roller** för erbjudandepublicering:

<a name="step-1-to-add-users-and-assign-roles-click-the-users-tab-on-the-left-side-navigation-pane"></a>Steg 1. Om du vill lägga till användare och tilldela roller klickar du på fliken Användare i navigeringsfönstret på vänster sida.
--------------------------------------------------------------------------------------------

![Fliken Användare](./media/cloud-partner-portal-how-to-manage-users/userstab.png)

<a name="step-2-click-add-user"></a>Steg 2. Klicka på Lägg till användare.
-----------------------

![Välj Lägg till användare](./media/cloud-partner-portal-how-to-manage-users/adduser.png)

<a name="step-3-type-email-address-and-select-a-role-assignment"></a>Steg 3. Skriv e-postadress och välj en rolltilldelning.
--------------------------------------------------------

Du kan lägga till den nya användaren som "ägare" eller "deltagare".

**E-post**: Lägg till e-postadresser till lagkamrater som kommer att arbeta med att publicera erbjudandet. Microsoft-konton (Outlook, Hotmail och Live) och Org-ID:er stöds.

o Lägg till ett e-postalias/säkerhetsgrupp för team/grupp om personen som arbetar med erbjudandet lämnar organisationen.

o Se till att e-post-ID:er som tillhandahålls på Cloud Partner Portal övervakas för all kommunikation från Microsoft.

**Roll**: Se tabellen nedan för att identifiera lämplig användarrolltyp.

![Tilldela](./media/cloud-partner-portal-how-to-manage-users/assignrole.png)
![rollrollnivå](./media/cloud-partner-portal-how-to-manage-users/roleaccesslevel.png)

För närvarande kan endast ägare komma åt flikarna Azure-utbetalningar och Azure-kund i Insights.

<a name="step-4-select-the-publisher-name-you-want-to-add-to-the-new-user-then-click-add-to-complete-user-addition"></a>Steg 4. Välj det Publisher-namn som du vill lägga till i den nya användaren och klicka sedan på Lägg till för att slutföra användartillägget.
----------------------------------------------------------------------------------------------------------------------------

> [!NOTE]
> De "utgivarnamn" som din e-post läggs till för som ägare eller deltagare för visas i listrutan för Publisher för val.

![Tilldela roll - välj utgivare](./media/cloud-partner-portal-how-to-manage-users/assignselectpublisher.png)

Om du behöver ta bort en användare som har lagts till behöver du bara söka efter deras e-post på fliken Användare och tryck sedan på delete-knappen till höger.
