<a name="managing-users-on-cloud-partner-portal"></a>Hantera användare på Cloud Partner Portal
======================================

Med Cloud Partner Portal kan du lägga till användare med rollbaserad åtkomst till erbjudanden för virtuella datorer. Det hjälper dig att hantera åtkomst och behörigheter samtidigt som flera personer arbetar med erbjudanden.

Följ stegen nedan om du vill **lägga till användare och tilldela roller** för erbjudande publicering:

<a name="step-1-to-add-users-and-assign-roles-click-the-users-tab-on-the-left-side-navigation-pane"></a>Steg 1. Om du vill lägga till användare och tilldela roller klickar du på fliken användare i navigerings fönstret till vänster.
--------------------------------------------------------------------------------------------

![Fliken användare](./media/cloud-partner-portal-how-to-manage-users/userstab.png)

<a name="step-2-click-add-user"></a>Steg 2. Klicka på Lägg till användare.
-----------------------

![Välj Lägg till användare](./media/cloud-partner-portal-how-to-manage-users/adduser.png)

<a name="step-3-type-email-address-and-select-a-role-assignment"></a>Steg 3. Skriv e-postadress och välj en roll tilldelning.
--------------------------------------------------------

Du kan lägga till den nya användaren som ägare eller deltagare.

**E-post**: Lägg till e-postadresserna till de grupp medlemmar som ska arbeta med publiceringen av erbjudandet. Microsoft-konton (Outlook, Hotmail och Live) och org ID: n stöds.

o Lägg till ett "grupp-/grupp-e-postalias/säkerhets grupp" om personen som arbetar på erbjudandet lämnar organisationen.

o se till att e-post-ID: n som anges på Cloud Partner Portal övervakas för kommunikation från Microsoft.

**Roll**: referera till tabellen nedan för att identifiera rätt typ av användar roll.

![tilldela roll](./media/cloud-partner-portal-how-to-manage-users/assignrole.png)
![Rolls nivå](./media/cloud-partner-portal-how-to-manage-users/roleaccesslevel.png)

För närvarande kan endast ägare få åtkomst till Azure-utbetalningar och Azure-kundflikar i insikter.

<a name="step-4-select-the-publisher-name-you-want-to-add-to-the-new-user-then-click-add-to-complete-user-addition"></a>Steg 4. Välj det utgivar namn som du vill lägga till den nya användaren och klicka sedan på Lägg till för att slutföra användar tillägget.
----------------------------------------------------------------------------------------------------------------------------

> [!NOTE]
> "Utgivar namn" som din e-post läggs till som ägare eller deltagare visas i list rutan för val av utgivare. "

![Tilldela roll-Välj Utgivare](./media/cloud-partner-portal-how-to-manage-users/assignselectpublisher.png)

Om du behöver ta bort en användare som har lagts till behöver du bara söka efter e-postmeddelandet på fliken användare och sedan trycka på knappen Ta bort på den högra sidan.
