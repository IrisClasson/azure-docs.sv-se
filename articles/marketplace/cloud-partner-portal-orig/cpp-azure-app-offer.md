# <a name="azure-application-offer"></a>Erbjudandet för Azure-program

<table> <tr> <td>Det här avsnittet beskrivs hur du publicerar ett nytt erbjudande för hanterade Azure-program till den <a href="https://azuremarketplace.microsoft.com/marketplace/">Microsoft Azure Marketplace</a>.  Varje hanterat program innehåller en Azure Resource Manager-mallen som definierar de tekniska resurser som används av programmet, vilket omfattar vanligtvis en eller flera virtuella datorer och andra Azure - eller webbaserade stödtjänster. </td> <td><img src="./media/cloud-partner-portal-publish-managed-app/azureapp-icon1.png"  alt="Azure application icon" /></td> </tr> </table>

## <a name="publishing-overview"></a>Översikt över publicering

På nedanstående video, [mallar för att skapa lösningar och hanterade program för Azure Marketplace](https://channel9.msdn.com/Events/Build/2018/BRK3603), är en översikt över hur du skapar en Azure Resource Manager-mall för att definiera en Azure lösning och sedan hur du därefter publicera apperbjudandet på Azure Marketplace.

## <a name="publishing-process"></a>Process för att publicera

I följande artiklar hjälper dig att processen för att skapa och publicera ett hanterat Azure-program:

1. [Registrera ett konto på Microsoft Developer Center](./cloud-partner-portal-dev-center-accounts-registration.md) att bli godkänd för att publicera på Azure Marketplace.
2. [Publicera ett nytt erbjudande för Azure-program](./cloud-partner-portal-managed-app-publish.md) på Azure Marketplace.
3. [Gör ditt erbjudande live](./cloud-partner-portal-managed-app-offer-go-live.md)! 


## <a name="next-steps"></a>Nästa steg

Därefter måste du kontrollera de krav som krävs för detta erbjudandetyp och påbörja publiceringsprocessen som beskrivs i [författare hanterat program](./cloud-partner-portal-managed-app-publish.md).
