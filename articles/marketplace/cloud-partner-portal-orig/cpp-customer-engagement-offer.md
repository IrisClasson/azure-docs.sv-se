## <a name="publishing-process"></a>Process för att publicera

I efterföljande artiklar i det här avsnittet vägleder dig genom processen att skapa och publicera en Dynamics 365 för kundengagemang lösning med stegen:

1. [Registrera](https://partners.microsoft.com/PartnerProgram/simplifiedenrollment.aspx) på Microsoft Partner Network (MPN), om du inte redan har gjort.
2. [Skapa en lösning för Dynamics 365](./dynamics-365-solution-preparation.md).
3. [Skapa ett paket med AppSource](./appsource-package-preparation.md) och tillhörande marknadsföringsinnehåll.
4.  Ladda upp ditt AppSource-paket till Azure storage och [Generera en SAS-nyckel](./store-your-appsource-package-to-azure-storage-and-generate-sas-key.md).
5. [Skapa associerade erbjudandet](./setup-your-application-in-the-publishing-portal.md).
6. [Publicera ditt erbjudande](./request-to-publish-the-app.md) publiceras.


## <a name="next-steps"></a>Nästa steg
