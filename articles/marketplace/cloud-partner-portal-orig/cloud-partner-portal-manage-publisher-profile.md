# <a name="manage-azure-marketplace-and-appsource-publisher-profile"></a>Hantera utgivarprofilen "Azure Marketplace" och "AppSource"

Det här dokumentet är en genomgång av visning och hantering av användare till din registrerade utgivarprofil.

Vid det här laget har du slutfört stegen för att bli en Azure Marketplace och AppSource-utgivare. Utgivarprofilen registreras på **[Cloud Partner Portal](https://cloudpartner.azure.com/)** efter godkännande av din partnerförfrågan. Din utgivarprofil gäller för alla erbjudanden och SKU:er som publiceras från kontot som användes vid partnerregistreringen.

Om du inte har registrerat ditt företag som molnpartner läser [du Komma igång med molnpartnerportalen](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cloud-partner-portal-getting-started-with-the-cloud-partner-portal).

**Utgivarprofil:** Din utgivarprofil skiljer ditt företag på Azure Marketplace och AppSource. Den består av ditt utgivar-ID, visningsnamn och ägar-e-post. En välhanterad profil ökar din synlighet och hjälper marketplace-användare att enkelt identifiera och välja ett lämpligt erbjudande.

> [!NOTE]
> Utgivarens ID och registrerade ägar-E-post kan inte redigeras när du publicerar ditt första erbjudande. Du kan dock visa din profil och redigera utgivarens visningsnamn från Cloud Partner Portal.

**Om du vill visa och hantera din utgivarprofil**klickar du på **fliken Publisher-profil**längst upp till höger .

![Publisher Profile Tab- utgivare profil](./media/cloud-partner-portal-how-to-manage-publisher-profile/publisherprofilenew.png)

![Information om Publisher-profil – information om utgivarens profil](./media/cloud-partner-portal-how-to-manage-publisher-profile/publisherprofiledetails.png)

**Länka ditt Dev Center-konto:** Du kan också länka ditt befintliga Dev Center-konto till din utgivarprofil på Cloud Partner Portal.
Logga först in på portalen med samma e-postadress som används för att registrera ditt Dev Center-konto. Sedan visar din utgivarprofilsida din Dev Center-kontostatus, dev center-kontoägare e-post och Dev Center-kontonamn.

>[!NOTE]
>Dev Center-kontoregistrering är obligatorisk för publicering av betalmarknadsplats SKU:er.

Om du inte har ett [Developer Center-konto](./cloud-partner-portal-dev-center-accounts-registration.md)kan du skapa ett konto. Lägg sedan till den registrerade e-postadressen som ägare på Cloud Partner Portal för ditt erbjudande. En ägare kan bara lägga till den registrerade e-postadressen i din publiceringsprofil från avsnittet **Användare.** Logga sedan in på portalen med samma e-postadress och länka ditt Dev Center-konto.
