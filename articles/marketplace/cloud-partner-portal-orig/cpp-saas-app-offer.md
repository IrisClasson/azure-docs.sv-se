# <a name="azure-saas-application-offer"></a>Azure SaaS-program-erbjudande

<table> <tr> <td>Det här avsnittet beskrivs hur du publicerar ett software-as-a-service (SaaS)-program på Microsoft <a href="https://azuremarketplace.microsoft.com/marketplace/">Azure Marketplace</a> eller <a href="https://appsource.microsoft.com">Microsoft AppSource</a>. Använd SaaS-program när din lösning ska distribueras i din Azure-prenumeration och kunder kommer att logga in via ett användargränssnitt som du utforma och hantera att testa programmet. <a href="https://azure.microsoft.com/services/active-directory">Azure Active Directory</a> (AAD) används för att utnyttja dina befintliga utvärderingsmiljö. </td> <td><img src="./media/cloud-partner-portal-publish-web-app/saasapp-icon1.png"  alt="Azure SaaS application icon" /></td> </tr> </table>


## <a name="publishing-benefits"></a>Publicera fördelar

Här är några av fördelarna med att visa en lista över dina program på en Microsoft marketplace:

- Nå 100 miljoner Azure Active Directory-användare i Office 365 och Dynamics 365.
- Utöka försäljningsteamet: nå företag över hela världen och få en försäljningskanal som engagerar användare, genererar leads och initierar samtal med nya kunder branscher.

- Hämta åtgärdsinriktade insikter: Vi kommer att dela insikter om hur appen fungerar på AppSource, vad som fungerar bra och hur att ytterligare förbättra din försäljning procedurer.


## <a name="publishing-process"></a>Process för att publicera

Om du vill skapa och publicera dina appar på Microsoft Azure Marketplace eller Microsoft AppSource, det här avsnittet vägleds du genom följande steg:

1. Bekräfta att du uppfyller förutsättningarna, inklusive partner registreringar: [Dev Center registrering](https://dev.windows.com/en-us/registration?accountProgram=Azure) *eller* [Microsoft Partner Network-registreringen](https://partner.microsoft.com/).
2.  Definiera dina marketplace-erbjudandet och planer.
3.  Ange marknadsföring innehåll och prissättning.
4.  Testa och certifiera ditt erbjudande i den [Cloud Partner Portal](https://cloudpartner.azure.com).
5.  Driftsätt med ditt erbjudande.
