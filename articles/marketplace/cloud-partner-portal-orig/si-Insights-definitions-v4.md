<a name="seller-insights-definitions"></a>Säljare insikter-definitioner
=======================

Följande tabell innehåller definitioner för många av de termer som används i säljar insikter.

|  **Löp**                 |  **Definition**                                                                                                                              |
|  --------------------------------------------    |  ---------------------------------------------------------------------------------------------------------------------------------           |
| **Typ av Azure-licens**                               | Den typ av licens avtal som kunder använder för att köpa Azure.  Kallas även för kanal.                                                  |
| **Azure-licens typ: Cloud Solution Provider**      | Slutanvändaren kan köpa Azure och ditt Marketplace-erbjudande via moln lösnings leverantören, som agerar som åter försäljare.                 |
| **Azure-licens typ: Enterprise**                   | Slutanvändaren kan köpa Azure och ditt Marketplace-erbjudande via en Enterprise-avtal, som loggas direkt med Microsoft.                  |
| **Azure-licens typ: Enterprise genom åter försäljare**  | Slutanvändaren säljer Azure och Marketplace-erbjudandet via en åter försäljare som fören klar sina Enterprise-avtal med Microsoft.     |
| **Azure-licens typ: betala per användning**                | Slutanvändaren debiterar Azure och Marketplace-erbjudandet genom ett "betala per användning"-avtal, undertecknat direkt med Microsoft.                |
| **Avgifts belopp (CC)**                              | Det belopp som debiteras kunden, i *kund valutan* (CC) för fakturering.                                                                 |
| **Avgifts belopp (PC)**                               | Det belopp som debiteras kunden i *utbetalnings valutan* (PC).                                                                      |
| **Avgifts datum**                                      | Datumet då kundens kostnad beräknades (vanligt vis direkt efter användnings perioden).                                             |
| **Moln instans namn**                              | Den Microsoft Cloud där en VM-distribution ägdes.                                   |
| **Moln instans namn: Azure Global**                | Det offentliga globala Microsoft-molnet.                           |
| **Moln instans namn: Azure Government**        | Myndighets Microsoft-moln för någon av följande myndigheter: Kina, Tyskland eller USA i Amerika.                           |
| **Samlings status**  | För en speciell avgift, den senaste statusen för fakturerings-och insamlingens livs cykel.  Status kan till exempel vara insamling pågår, samlas in eller åter betalning.                  |
| **Kärn storlek**  | Antalet virtuella kärnor som presenteras av den virtuella datorn som erbjudandet körs på.               |
| **Kund ort**  | Orts namnet som tillhandahålls av kunden. Detta kan skilja sig från staden i en kunds Azure-prenumeration.  |
| **Kundens kommunikations språk**        | Det språk som kunden föredrar för kommunikation.                                  |
| **Kund företags namn**                  | Företags namnet som tillhandahålls av kunden. Detta kan skilja sig från staden i en kunds Azure-prenumeration.                                  |
| **Kund land**                       | Landets/regionens namn som tillhandahålls av kunden.  Detta kan skilja sig från landet/regionen i en kunds Azure-prenumeration.                               |
| **Kund valuta (CC)**                 | Valutan som kunden föredrar för priser och fakturering.                            |
| **Kund-e-post**                         | E-postadressen som tillhandahålls av slut kunden.  Detta kan skilja sig från e-postadressen i en kunds Azure-prenumeration.                   |
| **Förnamn för kund**                              | Förnamnet som tillhandahålls av kunden.  Detta kan skilja sig från namnet som anges i en kunds Azure-prenumeration.                    |
| **Kundens efter namn**                               | Efter namnet som tillhandahålls av kunden.  Detta kan skilja sig från namnet som anges i en kunds Azure-prenumeration.                    |
| **Kund-ID**                               | Den unika identifierare som tilldelats till en kund.  En kund kan ha noll eller flera prenumerationer på Azure Marketplace.                    |
| **Kund betalnings typ**                            | Typ av betalnings instrument som används av kunden.  Detta kan till exempel vara kort, faktura eller elektronisk kontroll.                             |
| **Kund post nummer**                             | Post numret från kunden.  Detta kan skilja sig från post numret som anges i en kunds Azure-prenumeration.                                                                 |
| **Kund tillstånd**                                   | Den stat (adress) som tillhandahålls av kunden.  Detta kan skilja sig från det senaste namnet som angavs i en kunds Azure-prenumeration.                                                                 |
| **Inköps datum**                                    | Det första datumet då kunden köpte ett erbjudande som publicerats av dig.                 |
| **Datum förlorat**                                        | Det senaste datumet då kunden avbröt den sista av alla erbjudanden som tidigare har köpts.                                                         |
| **Uppskattad utökad avgift (CC)**                   | Den uppskattade utökade avgiften för antalet användnings enheter för en viss SKU (i kundens valuta). Det här värdet kan inte vara exakt på grund av avrundning eller trunkering-fel.                            |
| **Uppskattad utökad avgift (PC)**                   | Den uppskattade utökade avgiften för antalet användnings enheter för en viss SKU, baserat på sekundär Exchange-konvertering vid användning av datum, beräknas (i utgivar valutan). Det här värdet kan inte vara exakt på grund av avrundning eller trunkering-fel.                  |
| **Beräknad utbetalning (PC)**                            | Den uppskattade betalningen för antalet användnings enheter för en specifik SKU, baserat på sekundär Exchange-konvertering på det datum då användningen beräknas (i utgivar valutan). Det här värdet kan inte vara exakt på grund av avrundning eller trunkering-fel.                        |
| **Uppskattat pris (PC)**                             | Det beräknade priset för en användnings enhet för en specifik SKU, baserat på sekundär Exchange-konvertering vid användning av datum, beräknas (i utgivar valutan). Det här värdet kan inte vara exakt på grund av avrundning eller trunkering-fel.                                             |
| **Är ny kund**                                  | Värdet gör att du vet om det är en ny kund som förbrukar eller köper ett eller flera av dina Marketplace-erbjudanden för första gången (eller inte).  Värdet är "Ja" om det är samma kalender månad för "inköps datum".  Värdet är "nej" om kunden har köpt något av dina erbjudanden före den kalender månad som rapporter ATS.                                                       |
| **Är för hands version av SKU**                                  | Värdet meddelar dig om du har taggat SKU som "Preview". Värdet är "Ja" om SKU: n har taggats enligt detta och bara Azure-prenumerationer som har godkänts av du kan distribuera och använda den här avbildningen.  Värdet är "nej" om SKU: n inte har identifierats som "för hands version".                                          |
| **Är kampanj kontakt deltagande**                   | Värdet gör att du vet om kunden proaktivt har valt reklam kontakt från utgivare. För tillfället visar vi inte alternativet för kunder, så vi har angett "nej" på tavlan. När den här funktionen har distribuerats kommer vi att börja uppdatera enligt detta.                                          |
| **Marketplace-licens typ**                         | Fakturerings metoden för Marketplace-erbjudandet.                             |
| **Marketplace-licens typ: debiteras via Azure**   | Microsoft är din agent för det här Marketplace-erbjudandet och fakturerar kunder åt dig. (Antingen PAYG kredit kort eller företags faktura)       |
| **Marketplace-licens typ: ta med din egen licens** | Den virtuella datorn kräver en licens nyckel från kunden som ska distribueras. Microsoft fakturerar inte kunder för deras användning av det här Marketplace-erbjudandet. |
| **Marketplace-licens typ: kostnads fri**                   | Marketplace-erbjudandet är konfigurerat att vara kostnads fritt för alla användare. Microsoft fakturerar inte kunder för användningen av det här Marketplace-erbjudandet.    |
| **Marketplace-licens typ: Microsoft as åter försäljare**  | Microsoft är din åter försäljare för det här Marketplace-erbjudandet.                               |
| **Företags namn för Marketplace-prenumeration**  | Företags namnet som tillhandahålls av kunden i Azure-prenumerationen.                           |
| **Prenumerations-ID för Marketplace**            | Den unika identifierare som är kopplad till Azure-prenumerationen som kunden använde för att köpa Marketplace-erbjudandet.  Det var tidigare ett GUID för Azure-prenumerationen.  I vissa fall som en SAAS-beställning krävs ingen prenumeration.           |
| **Microsoft-avgift (CC)**                               | Microsofts avgift för transaktionen i kundens valuta.                              |
| **Erbjudandets namn**                                | Namnet på Marketplace-erbjudandet.                                                             |
| **Erbjudande typ**                                | Typ av Microsoft Marketplace erbjudande.  Det kan till exempel vara ett hanterat program, licenser för virtuella datorer eller behållare.                           |
| **Annullera datum för beställning**                         | Datumet då Marketplace-ordern annullerades.                                                       |
| **Order-ID**                                  | Den unika identifieraren för kund ordern för din Marketplace-tjänst.  Användnings erbjudanden för virtuella datorer är inte kopplade till någon order.                 |
| **Inköps datum för beställning**                       | Datumet då Marketplace-ordern skapades.                                                        |
| **Orderkvantitet**                       | Antalet beställningar för samma SKU som är associerad med en Azure-prenumeration.  För SAAS-beställningar representerar det antalet platser som är kopplade till den här instansen av en order.                                            |
| **Order status**                              | Status för en Marketplace-order vid den tidpunkt då data senast uppdaterades.  Ordningen kan till exempel vara aktiv eller annullerad.             |
| **Order status: aktiv**                           | Kunden har köpt en order och har inte annullerat sin beställning.   |
| **Order status: avbruten**                        | Kunden köpte tidigare en order och annullerade sedan sin beställning.        |
| **Utbetalnings belopp (PC)**                             | Det belopp som betalas ut till dig, i din önskade utbetalnings valuta (PC).                                       |
| **Utbetalnings valuta (PC)**                           | Den valuta som används för dina utbetalningar.                                                                |
| **Utbetalnings datum**                                    | Datumet då betalnings förfrågan skickades från Microsoft till din valda finansiella institution.                                                                         |
| **Utbetalnings status**                                  | Status för utbetalningen vid den tidpunkt då data senast uppdaterades.                                   |
| **Utbetalnings status: pågår**               | Transaktionen är inte klar för utbetalning. (Mer information finns i samlings status)                             |
| **Utbetalnings status: betald**                          | Transaktionen inkluderades i en beräkning av tidigare utbetalningar. Positiva värden betalas, och negativa värden debiteras mot totalt belopp som förfaller. |
| **Utbetalnings status: kommande utbetalning**                   | Transaktionen är klar för utbetalning och tas med i nästa tillgängliga beräknings beräkning.                      |
| **Pris (CC)**                                   | Priset för en användnings enhet för en specifik SKU (i kundens valuta).                                                                  |
| **Providerns e-postadress**                                 | E-postadressen till leverantören som är involverad i relationen mellan Microsoft och slut kunden. Om kunden är ett företag via åter försäljare är det åter försäljaren.  Om en moln lösnings leverantör (CSP) är involverad, kommer den att vara CSP.                                                                 |
| **Providernamn**                                 | Namnet på leverantören som är involverad i relationen mellan Microsoft och slut kunden. Om kunden är ett företag via åter försäljare är det åter försäljaren.  Om en moln lösnings leverantör (CSP) är involverad, kommer den att vara CSP.                                                                 |
| **Resurs-URI**                                  | Den sökväg där distributions artefakten distribuerades.                                            |
| **SKU**                                           | SKU-namn som definieras under publiceringen. Ett erbjudande kan ha många SKU: er, men en SKU kan bara associeras med ett enda erbjudande.              |
| **SKU-fakturerings typ**                                 | Fakturerings metoden för SKU: n.                                                                                                             |
| **SKU-fakturerings typ: ta med din egen licens**         | Den virtuella datorn eller tjänsten kräver en licens nyckel från kunden som ska distribueras. Microsoft fakturerar inte kunder för deras användning av dessa Marketplace-erbjudanden.   |
| **SKU-fakturerings typ: Microsoft as åter försäljare**          | Microsoft är din åter försäljare för denna SKU.   |
| **SKU-fakturerings typ: kostnads fri**                           | SKU: n är konfigurerad för att vara kostnads fri för alla användare. Microsoft fakturerar inte kunder för deras användning av denna SKU.                           |
| **SKU-fakturerings typ: betald**                           | Microsoft är din agent för den här SKU: n och fakturerar kunder för din räkning. (Antingen PAYG kredit kort eller företags faktura)                   |
| **SKU-fakturerings typ: utvärdering**                          | Kunden är i sin utvärderings period och kommer att konverteras till betald om de inte avbryts eller tas bort.                                         |
| **Skatte belopp (CC)**                                  | Det skatte belopp som tillämpas på kundens faktura i kund valutan (CC).                                                               |
| **Transaktions datum**                                 | Datum för den transaktion som registrerats i din utbetalnings rapport.                                                                           |
| **Transaktions typ**                               | Typ av transaktion som rapporteras.  Detta kan till exempel vara en avgift, åter betalning eller utbetalnings justering.                                                                                        |
| **Slutdatum för utvärdering**                                 | Det datum då utvärderings perioden för den här ordern upphör eller har avslut ATS.                                                                           |
| **Datum för användning**                                       | Datumet då kund användningen ägde rum.                                                                                                             |
| **Användnings antal**                                       | Den rapporterade kund användningen av SKU: n. För VM-avbildningar representerar användnings poster användningen för den rapporterade perioden för den virtuella datorns storlek och SKU.                                               |
| **Användnings referens**                                       | Identifieraren för en eller flera dagar av kund användning för en specifik SKU som är kopplad till en post i utbetalnings rapporten.                                               |
| **Användningstyp**                                       | En beskrivning av användningen som mäts. (Normaliserad användning eller rå användning)                                                              |
| **Användnings typ: normaliserad användning**                | Förbrukade timmar normaliserade för att redovisa antalet virtuella dator kärnor som ingår i användningen: [antal virtuella dator kärnor] x [timmar RAW Usage]. Virtuella datorer som anges som "SHAREDCORE" använder 1/6 (eller 0,1666) som [antal VM-kärnor] multiplikatorn.                                        |
| **Användnings typ: rå Usage**                        | Användnings timmar för ditt Marketplace-erbjudande.  Den presenteras i form av kalender timmar.  Det är antalet virtuella datorer som har körts.                           |
| **Användnings enheter**                                   | Mått enheten för den angivna användningen. Virtuella datorer mäts alltid med Tim mått enheter.                               |
| **VM-storlek**                                       | Representerar storleken på den virtuella maskin varan som är justerad mot Azure-erbjudandet. Exempel:,, och `Standard_G4` `Basic_A0` `Standard_A11` `Standard_D12`   |
|  |  |
