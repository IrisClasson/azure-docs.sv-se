# <a name="dynamics-365-for-operations-help-wiki"></a>Dynamics 365 for Operations Hjälp Wiki

Microsoft Dynamics 365 for Operations är Microsofts affärsprogram för företag. Det gör det möjligt för människor att snabbt fatta smartare beslut genom ett intelligent användargränssnitt som ger tillgång till insikter och data i realtid. Det gör det möjligt för företag att omvandla och förnya genom att leverera beprövad affärslogik som gör det möjligt för dem att omforma sina affärsprocesser snabbare. Det ger företagen flexibilitet att växa i sin egen takt och göra affärer nästan var som helst, när som helst, så att de kan skala sin verksamhet globalt för att möta företagens behov.

Den här användarhandboken kan hjälpa dig att lära dig hur du får Dynamics 365 for Operations att fungera för ditt företag.

Mer information finns i [Dynamics 365 for Operations Help Wiki.](https://docs.microsoft.com/dynamics365/operations/)
