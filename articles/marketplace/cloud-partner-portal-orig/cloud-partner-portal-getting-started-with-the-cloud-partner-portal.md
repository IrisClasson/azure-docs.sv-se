# <a name="getting-started-with-the-cloud-partner-portal"></a>Komma igång med Cloud Partner Portal

Den här artikeln innehåller en genom gång av Cloud Partner Portal. Lär dig vad du kan göra med Cloud Partner Portal för att arbeta med erbjudanden på [Azure Marketplace](https://azuremarketplace.microsoft.com/) och på [AppSource](https://appsource.microsoft.com/).

<a name="portal-tour"></a>Guidad visning av portalen
-----------

Cloud Partner Portal innehåller allt du behöver för att kunna erbjuda och köra en lyckad verksamhet i molnet. Här är en snabb översikt över de delar av den här webbplatsen som du förmodligen använder oftast.

### <a name="left-navigation-bar"></a>Vänster navigerings fält

När du först landar på portalen bör du se det komprimerbara navigerings fältet. Här kan du navigera bland meny alternativ.

![Partnerportalen i molnet](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/cloud-partner-portal-page.png)

Använd meny raden för att få mer information om vart och ett av följande objekt:

- **Nytt erbjudande** – starta ett nytt erbjudande här.
- **Alla erbjudanden** – kontrol lera status och status för alla dina aktuella erbjudanden.
- **Användare** - [Hantera och kontrol lera åtkomsten för ditt företag](./cloud-partner-portal-manage-users.md).
- **Lär dig** – kom igång med din utbildning och håll dig uppdaterad om vad som är nytt i Partner portalen.
- **Insights** – hitta insikter och användnings information om dina produkter och kunder genom att gå till insikts portalen.

### <a name="offer-page"></a>Erbjudande sida

När du har påbörjat ett nytt erbjudande, eller om du vill redigera ditt erbjudande, gör du mycket arbete på sidan erbjudande. Det är här du konfigurerar alla inställningar för ditt erbjudande och kontrollerar dess status.

![Erbjudande sida](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/offer-page.png)

På sidan erbjudande kan du:
- Lägg till innehåll eller kontrol lera ditt erbjudandes status genom att växla mellan två lägen på högsta nivån: **redigerare** och **status**.
- Använd **Åtgärds fältet** för att explicit hantera ditt erbjudandes ändringar genom att spara och ta bort dina inaktuella uppgifter.
- Navigera mellan de nödvändiga **formulären** för varje erbjudande när du ändrar inställningarna för ditt erbjudande.

### <a name="menu-bar"></a>Menyrad

Slutligen kan du alltid navigera till den översta meny raden längst upp på skärmen. Den här menyn innehåller interaktiva åtgärder för support och feedback. Du kan också komma åt all din utgivar konto information.

![Menyrad](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/menu-bar.png)

-   **Behöver du hjälp?** – Om du någonsin är förvirrad och behöver hjälp, skapar du en support förfrågan och hittar länkar till dokumentationen.
-   **Feedback** – har du något som du vill säga? Berätta för oss här. Microsoft vill göra Azure Marketplace, publicerings processen för AppSource och den övergripande moln partner portalen så enkelt och så intuitiv som möjligt. Vi prioritering och granskar all feedback och vi alltid respekterar din integritet. Vi kontaktar dig bara när du ber om mer information.
- **Utgivar konto** – första gången du besöker portalen ser du avtals avtalet för utgivare för att acceptera att bli en utgivare. Sedan kan du göra ändringar i din utgivar profil eller gå tillbaka till kontraktet. Det är också här du ansluter ditt dev Center-konto för att kunna sälja dina erbjudanden.
