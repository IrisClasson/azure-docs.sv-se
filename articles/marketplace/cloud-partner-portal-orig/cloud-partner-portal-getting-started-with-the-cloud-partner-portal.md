# <a name="getting-started-with-the-cloud-partner-portal"></a>Komma igång med Cloud Partner Portal

Den här artikeln innehåller en genomgång av Cloud Partner Portal. Läs om vad du kan göra med Cloud Partner Portal för att arbeta med erbjudanden på [Azure Marketplace](https://azuremarketplace.microsoft.com/) och på [AppSource](https://appsource.microsoft.com/).

<a name="portal-tour"></a>Portal turné
-----------

Cloud Partner Portal innehåller allt du behöver för att erbjuda och driva ett framgångsrikt företag i molnet. Här är en snabb översikt över de delar av den här webbplatsen som du troligen kommer att använda oftast.

### <a name="left-navigation-bar"></a>Vänster navigeringsfält

När du först landar på portalen bör du se det hopfällbara navigeringsfältet. Här kan du navigera bland menyalternativ.

![Partnerportalen i molnet](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/cloud-partner-portal-page.png)

Använd menyraden för att få mer information om vart och ett av följande objekt:

- **Nytt erbjudande** - Kick off ett nytt erbjudande här.
- **Alla erbjudanden** - Kontrollera tillståndet och statusen för alla dina aktuella erbjudanden.
- **Användare** - [Hantera och kontrollera åtkomsten för ditt företag](./cloud-partner-portal-manage-users.md).
- **Lär dig** - Starta din inlärning och håll dig uppdaterad om vad som är nytt i partnerportalen.
- **Insikter** – Hitta insikter och användningsinformation om dina produkter och kunder genom att besöka Insights Portal.

### <a name="offer-page"></a>Sidan Erbjudande

Fortsätter turnén, när du startar ett nytt erbjudande eller kommer tillbaka för att redigera ditt erbjudande, kommer du att göra en hel del arbete på erbjudandesidan. Här konfigurerar du alla inställningar för erbjudandet och kontrollerar dess status.

![Sidan Erbjudande](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/offer-page.png)

På sidan Erbjudande kan du:
- Lägg till innehåll eller kontrollera erbjudandets status genom att växla mellan två lägen på den översta nivån: **Redigerare** och **Status**.
- Använd **åtgärdsfältet** för att uttryckligen hantera erbjudandets ändringar genom att spara och ignorera dina indata.
- Navigera mellan de **formulär** som krävs för varje erbjudande när du ändrar inställningarna för erbjudandet.

### <a name="menu-bar"></a>Menyraden

Slutligen, på toppen av skärmen kommer du alltid att kunna navigera till den övre menyraden. Den här menyn innehåller interaktiva åtgärder för support och feedback. Du kan också komma åt all publisher-kontoinformation.

![Menyraden](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/menu-bar.png)

-   **Behöver du hjälp?** - Om du någonsin är förvirrad och behöver hjälp, skapa en supportbegäran och hitta länkar till dokumentationen.
-   **Feedback** - Har du något du vill säga? Låt oss veta här. Microsoft vill göra Azure Marketplace, AppSource-publiceringsprocessen och den övergripande molnpartnerportalupplevelsen så enkel och så intuitiv som möjligt. Vi triage och granska all feedback, och vi kommer alltid att respektera din integritet. Vi kontaktar dig endast när du ber om mer information.
- **Publisher-konto** – När du besöker portalen första gången ser du utgivarens avtal om att acceptera att bli utgivare. Från och med då kan du göra ändringar i din utgivarprofil eller gå tillbaka till kontraktet. Det är också här du ansluter ditt Dev Center-konto för att kunna sälja dina erbjudanden.
