<a name="cloud-partner-portal-gdpr-compliance"></a>GDPR-efterlevnad för Molnpartnerportalen
====================================

[General Data Protection Regulation (GDPR)](https://ec.europa.eu/info/law/law-topic/data-protection/reform_en/) är en EU-lagstiftning om dataskydd och integritet. GDPR inför regler för företag, myndigheter, ideella organisationer och andra organisationer som erbjuder varor och tjänster till människor i EU, eller som samlar in och analyserar data som är knutna till EU-invånare.

Cloud Partner Portal ger detaljerad information om dess behandling av kunddata och de säkerhetsåtgärder som används för att skydda dessa data. Denna information är tillgänglig via dess produktupplevelser.

<a name="discover"></a>Utforska
--------

Cloud Partner Portal ger tillgång till alla dina data på fliken Användare. Du kan identifiera dina personuppgifter där. Mer information om hur du lägger till/tar bort användare finns i vår [dokumentation](./cloud-partner-portal-manage-users.md).

<a name="manage"></a>Hantera
------

Du kan hantera dina personuppgifter genom att trycka på ta bort på ditt användarkonto på fliken Användare. Vi har roller som **ägare** och en **deltagare** som kan tilldelas varje specifik användare, och båda kan tilldelas/kopieras/uppdateras/tas bort.

<a name="consent"></a>Samtycke
-------

För att hantera medgivande i Cloud Partner Portal visas alltid ett utgivaravtal som du måste godkänna för att vara utgivare hos Microsoft. Om du inte samtycker till utgivarens avtal begränsar vi utgivaren att inte kunna sälja med Microsoft.
