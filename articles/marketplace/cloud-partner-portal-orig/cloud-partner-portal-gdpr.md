<a name="cloud-partner-portal-gdpr-compliance"></a>Cloud Partner Portal GDPR-kompatibilitet
====================================

[Allmän dataskyddsförordning (GDPR)](https://ec.europa.eu/info/law/law-topic/data-protection/reform_en/) är en europeisk union (EU) data skydds-och sekretess lagstiftning. GDPR tillämpar regler för företag, myndigheter, ideella organisationer och andra organisationer som tillhandahåller varor och tjänster till personer i EU eller som samlar in och analyserar data som är kopplade till EU-invånare.

Cloud Partner Portal innehåller detaljerad information om bearbetningen av kunddata och de säkerhets åtgärder som används för att skydda dessa data. Den här informationen kan nås via sin produkt upplevelse.

<a name="discover"></a>Utforska
--------

Cloud Partner Portal ger åtkomst till alla dina data på fliken användare. Du kan identifiera personliga data där. Information om hur du lägger till/tar bort användare finns i vår [dokumentation](./cloud-partner-portal-manage-users.md).

<a name="manage"></a>Hantera
------

Du kan hantera dina personliga data genom att trycka på ta bort på ditt användar konto på fliken användare. Vi har roller för en **ägare** och en **deltagare** som kan tilldelas till varje enskild användare, och båda kan tilldelas/kopieras/uppdateras/tas bort.

<a name="consent"></a>Givit
-------

För att hantera medgivande i Cloud Partner Portal, visas alltid ett avtal för utgivar avtal, som du måste samtycka till för att bli utgivare med Microsoft. Om du inte samtycker till avtals avtalet för utgivaren, så begränsar du att utgivaren inte kan sälja med Microsoft.
