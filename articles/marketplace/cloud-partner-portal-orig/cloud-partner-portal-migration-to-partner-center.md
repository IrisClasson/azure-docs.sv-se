# <a name="cloud-partner-portal-migration-to-partner-center"></a>Cloud Partner Portal-migrering till Partnercenter

För att effektivisera ditt konto och erbjuda hanterings erfarenhet, migrerar Microsoft erbjudanden från Cloud Partner Portal till Partner Center. Om du för närvarande hanterar samförsäljnings material i en OCP GTM-katalog (kommersiell partner go-to-Marketing) kan du uppdatera dina samförsäljnings material i Partner Center snart.

| **Erbjudande typ** | **Hantering av erbjudanden som migrerats från Cloud Partner Portal till Partner Center** | **Möjlighet att uppdatera samförsäljnings material i Partner Center** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| Program vara som en tjänst (SaaS) |     Tillgängligt nu       |   Allmän tillgänglighet runt feb. 2020      |
| Dynamics 365 för kund engagemang & PowerApps      |     Tillgängligt nu        |   Tillgängligt nu      |
| Dynamics 365 for Operations     |     Tillgängligt nu       |   Tillgängligt nu     |
| Dynamics 365 Business Central      |     Tillgängligt nu        |   Allmän tillgänglighet runt feb. 2020     |
| Azure Application     |     Allmän tillgänglighet dec. 2019       |   Allmän tillgänglighet runt feb. 2020     |
| Azure virtuell dator      |     Tillkännages senare       |   Tillkännages senare    |
| Azure-behållare     |     Tillkännages senare       |   Tillkännages senare     |
| Konsulttjänst      |     Tillkännages senare       |   Tillkännages senare     |
| IoT Edge modul      |     Tillkännages senare       |   Tillkännages senare     |
| Power BI tjänst program     |     Tillkännages senare       |   Tillkännages senare     |
