# <a name="office-365-dynamics-365-power-apps-and-power-bi-offer-publishing-guide"></a>Guide för Office 365, Dynamics 365, Powerapps och Power BI-erbjudandet publicering

AppSource-storefront innehåller erbjudanden som bygger på eller utöka Dynamics 365, Office 365, Power BI och Power Apps. Var och en av dessa produkter har specifika dokumentationen att hjälpa dig publiceringsprocessen. Nedan finns länkar till mer information för varje typ av erbjudande. 

AppSource har lista och testversioner med anrop till åtgärden ”kontakta mig” eller ”Prova nu”. Det finns för närvarande inga funktioner för transaktion i AppSource.

## <a name="product-specific-guides"></a>Produktspecifika guider

### <a name="office-365"></a>Office 365

Granska den [publicering processen och riktlinjer](https://docs.microsoft.com/office/dev/store/submit-to-the-office-store)

### <a name="power-apps"></a>Power-appar

Lär dig mer om hur kunder kan [provköra din app på AppSource.](https://powerapps.microsoft.com/blog/appsource-test-drive/)

### <a name="power-bi"></a>Power BI

Granska den [publicering processen och riktlinjer.](https://docs.microsoft.com/power-bi/developer/office-store)

### <a name="dynamics-365"></a>Dynamics 365

#### <a name="dynamics-365-for-finance-and-operations"></a>Dynamics 365 for Finance and Operations
När du skapar för Enterprise Edition, granska de [publicering processen och riktlinjer](https://docs.microsoft.com/dynamics365/unified-operations/dev-itpro/lcs-solutions/lcs-solutions-app-source)

Se [Dynamics 365 för Finance and Operations erbjudande](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cpp-dynamics-365-operations-offer) att publicera erbjudandet i partnerportalen i molnet.

### <a name="dynamics-365-for-customer-engagement"></a>Dynamics 365 för Customer Engagement
Granska den [publicering processen och riktlinjer](https://docs.microsoft.com/dynamics365/customer-engagement/developer/publish-app-appsource).

Se [Dynamics 365 för kundengagemang erbjudande](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cpp-customer-engagement-offer) att publicera erbjudandet i partnerportalen i molnet.

## <a name="next-steps"></a>Nästa steg

Om du inte redan gjort det, 

- [Registrera](https://azuremarketplace.microsoft.com/sell) i marketplace.

Om du är registrerad och arbeta i Cloud Partner Portal 

- [Logga in på partnerportalen i molnet](https://cloudpartner.azure.com) att skapa eller slutföra ditt erbjudande.
