# <a name="office-365-dynamics-365-power-apps-and-power-bi-offer-publishing-guide"></a>Office 365, Dynamics 365, Power Apps och Power BI erbjudande publicerings guide

AppSource-butik innehåller erbjudanden som bygger på eller utökar Dynamics 365, Office 365, Power BI och Power Apps. Var och en av dessa produkter har en detaljerad dokumentation som hjälper dig i publicerings processen. Se nedan för länkar till mer information för varje erbjudande typ. 

AppSource har betalat, lista och utvärderings erbjudanden, med anropet till åtgärden "Hämta nu", "kontakta mig" och "Prova nu". 

## <a name="product-specific-guides"></a>Produktspecifika guider

### <a name="office-365"></a>Office 365

Granska [publicerings processen och rikt linjerna](https://docs.microsoft.com/office/dev/store/submit-to-the-office-store)

### <a name="power-apps"></a>Power Apps

Lär dig mer om hur kunder kan [testa din app på AppSource.](https://powerapps.microsoft.com/blog/appsource-test-drive/)

### <a name="power-bi"></a>Power BI

Granska [publicerings processen och rikt linjerna.](https://docs.microsoft.com/power-bi/developer/office-store)

### <a name="dynamics-365"></a>Dynamics 365

#### <a name="dynamics-365-for-finance-and-operations"></a>Dynamics 365 for Finance and Operations
När du skapar för Enterprise Edition bör du läsa igenom [publicerings processen och rikt linjerna](https://docs.microsoft.com/dynamics365/unified-operations/dev-itpro/lcs-solutions/lcs-solutions-app-source)

Se [Dynamics 365 for finans and Operations-erbjudandet](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cpp-dynamics-365-operations-offer) för att publicera erbjudandet i Cloud Partner Portal.

### <a name="dynamics-365-for-customer-engagement"></a>Dynamics 365 för Customer Engagement
Granska [publicerings processen och rikt linjerna](https://docs.microsoft.com/dynamics365/customer-engagement/developer/publish-app-appsource).

Se [Dynamics 365 för kund engagemang](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cpp-customer-engagement-offer) för att publicera erbjudandet i Cloud Partner Portal.
