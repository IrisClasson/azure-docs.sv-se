# <a name="azure-application-support-tab"></a>Fliken Support för Azure-program

Använd fliken Support för att tillhandahålla teknisk support och kundsupportinformation.
 
## <a name="identify-contacts-and-websites"></a>Identifiera kontakter och webbplatser

Använd nästa skärmdump som en guide för att ge den information som krävs. Alla fältnamn som läggs till med en röd asterisk (*) måste fyllas i.

  ![Supportformulär för erbjudande](./media/azureapp-support-tab.png)

1. Ange information för följande fält under **Teknisk kontakt:**

   - **Namn\* ** - Ange namnet på den tekniska kontakten för din app. Den här kontakten tar emot teknisk kommunikation från Microsoft.
   - **E-post\* ** - Ange e-postadressen till den tekniska kontakten.
   - **Telefon\* ** - Ange telefonnumret till den tekniska kontakten.
  
2. Under **Supportkontakt**anger du information för följande fält:

   - **Namn\* ** - Ange namnet på supportkontakten för din app. Den här kontakten tar emot supportrelaterad kommunikation från Microsoft.
   - **E-post\* ** - Ange e-postadressen till supportkontakten för din app.
   - **Telefon\* ** - Ange telefonnumret till supportkontakten.
 
3. Under **Support-webbadresser**tillhandahåller du lämpliga supportadresser som Microsoft använder när dina kunder öppnar supportbiljetter. Dessa webbadresser kommer att vara för antingen Public Azure eller Azure Government Cloud.
4. Välj **Spara** när du är klar.
