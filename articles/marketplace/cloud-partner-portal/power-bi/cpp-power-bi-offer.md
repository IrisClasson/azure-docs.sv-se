# <a name="power-bi-app-offer"></a>Power BI app-erbjudande

Cloud Partner Portal har inte längre stöd för att skapa och hantera erbjudanden för Power BI-appar. Den här funktionen har flyttats till Microsoft [partner Center](https://partner.microsoft.com/pcv/). Mer information finns i [skapa en Power BI app för Microsoft AppSource](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-power-bi-app-offer).
