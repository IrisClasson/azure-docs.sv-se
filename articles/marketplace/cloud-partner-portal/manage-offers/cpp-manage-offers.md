# <a name="manage-azure-and-appsource-marketplace-offers"></a>Hantera Azure- och AppSource Marketplace-erbjudanden

I det här avsnittet beskrivs de allmänna åtgärder som används för att hantera erbjudanden för Azure- och AppSource Marketplaces.  Det förklarar till exempel hur du använder [Cloud Partner Portal](https://cloudpartner.azure.com/) för att skapa och publicera ett marketplace-erbjudande.  I det här avsnittet beskrivs också tekniska problem som är gemensamma för de flesta erbjudandetyper, till exempel textbaserade och bildtillgångar.


## <a name="standard-offer-operations"></a>Standarderbjudandeåtgärder

Cloud Partner Portal gör det möjligt för en utgivare att utföra följande grundläggande åtgärder på ett erbjudande.

|     Åtgärd      |  Beskrivning                                           |
|     ---------      |  -----------                                           |
| [Skapa erbjudande](./cpp-create-offer.md)   | Skapar en väntande erbjudandepost i den inbyggda databasen, men publicerar den inte automatiskt på en Microsoft-marknadsplats | 
| [Visa erbjudandestatus](./cpp-view-status-offer.md)   | Anger publiceringsstatus för ett erbjudande.  Det finns flera platser i portalen där statusinformation visas. |
| [Publicera erbjudande](./cpp-publish-offer.md) | Validerar och publicerar sedan en slutförd erbjudandepost på lämplig Microsoft-marknadsplats.  Erbjudandet sägs då vara *live.* |
| [Uppdatera erbjudande](./cpp-update-offer.md)   | Tillåter ändring av vissa egenskaper hos ett befintligt erbjudande.  Gäller både väntande och publicerade erbjudanden. |
| [Ta bort erbjudande](./cpp-delete-offer.md)   | Tar bort ett befintligt väntande eller levande erbjudande.  | 
|  |  |
