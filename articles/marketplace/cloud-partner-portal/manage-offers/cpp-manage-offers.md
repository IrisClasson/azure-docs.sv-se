# <a name="manage-azure-and-appsource-marketplace-offers"></a>Hantera Azure-och AppSource Marketplace-erbjudanden

I det här avsnittet beskrivs de allmänna åtgärder som används för att hantera erbjudanden för Azure-och AppSource-Marketplace.  Den förklarar till exempel hur du använder [Cloud Partner Portal](https://cloudpartner.azure.com/) för att skapa och publicera ett Marketplace-erbjudande.  Det här avsnittet beskriver också tekniska problem som är gemensamma för de flesta typer av erbjudanden, till exempel textbaserade och bild till gångar.


## <a name="standard-offer-operations"></a>Standard erbjudande åtgärder

Cloud Partner Portal gör det möjligt för en utgivare att utföra följande grundläggande åtgärder för ett erbjudande.

|     Åtgärd      |  Beskrivning                                           |
|     ---------      |  -----------                                           |
| [Skapa erbjudande](./cpp-create-offer.md)   | Skapar en väntande erbjudande post i onboarding-databasen, men publicerar den inte automatiskt på en Microsoft Marketplace | 
| [Visa erbjudandestatus](./cpp-view-status-offer.md)   | Tillhandahåller publicerings status för ett erbjudande.  Det finns flera platser i portalen där statusinformation visas. |
| [Publicera erbjudande](./cpp-publish-offer.md) | Validerar och publicerar sedan en slutförd erbjudande post till rätt Microsoft Marketplace.  Erbjudandet sägs sedan vara *Live*. |
| [Uppdatera erbjudande](./cpp-update-offer.md)   | Tillåter ändring av vissa egenskaper för ett befintligt erbjudande.  Gäller både för väntande och publicerade erbjudanden. |
| [Ta bort erbjudande](./cpp-delete-offer.md)   | Tar bort ett befintligt väntande eller Live-erbjudande.  | 
|  |  |
