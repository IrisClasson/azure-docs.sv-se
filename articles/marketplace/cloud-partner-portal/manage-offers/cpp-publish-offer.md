# <a name="publish-azure-marketplace-and-appsource-offers"></a>Publicera Azure Marketplace- och AppSource-erbjudanden

När du har fyllt i alla erbjudandedetaljer är det dags att publicera ditt erbjudande och ta det "live" på Azure Marketplace. Det finns några steg erbjudandet går igenom. Se till att både ditt marknadsföringsinnehåll och dina tekniska resurser uppfyller kvalitetskraven, att vara Azure-certifierad och gå live på webbplatsen.  Följande diagram illustrerar den allmänna publiceringsprocessen för erbjudanden.  Denna process skiljer sig något mellan de olika erbjudandetyperna.

![allmänt flöde för erbjudandepublicering](./media/publish-flow.png)


## <a name="publishing-process"></a>Publiceringsprocess

Följ följande steg för att publicera erbjudandet.

1. Klicka på **Publicera** under fliken **Redigerare** för att starta publiceringsprocessen.
2. Under fliken **Status** visas formuläret **Publiceringssteg** i flödesschema.  Det här formuläret anger också aktuell status för erbjudandet och rapporterade problem.
3. När du har kontrollerat att allt ser korrekt ut och fungerar som det ska i förhandsversionen är du redo att gå live. Klicka på **Gå live** under fliken **Status** så tar Microsoft ditt erbjudande i produktion och på lämplig marknadsplats. Vanligtvis går live tar flera timmar.

När som helst i publiceringsprocessen kan du också logga in och klicka på fliken **Alla erbjudanden** för att visa den senaste statusen för något av dina erbjudanden. Du kan klicka direkt på statusen för ditt erbjudande och se information om var ditt erbjudande finns i publiceringsprocessen.


## <a name="canceling-the-publishing-request"></a>Avbryta publiceringsbegäran

Du kan starta publiceringsprocessen och behöva avbryta din begäran. Du kan bara avbryta en publiceringsbegäran när publiceringsbegäran når ut *logga ut-steget i Publisher.* Om du vill avbryta klickar du på **Avbryt publicering**. Publiceringsstatusen återställs till steg 1 och om du vill publicera om måste du klicka på **Publicera** och följa stegen på fliken **Status.**


## <a name="next-steps"></a>Nästa steg

Under publiceringsprocessen är det användbart att övervaka [erbjudandestatus](./cpp-view-status-offer.md) tills erbjudandet publiceras.
