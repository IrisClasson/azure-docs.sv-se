# <a name="publish-azure-marketplace-and-appsource-offers"></a>Publicera Azure Marketplace-och AppSource-erbjudanden

När du har fyllt i all information om erbjudandet är det dags att publicera ditt erbjudande och ta det "Live" på Azure Marketplace. Det finns några steg som erbjudandet går igenom. Se till att både ditt marknadsförings innehåll och dina tekniska till gångar uppfyller kvalitets kraven, så att de blir Azure-certifierade och går live på webbplatsen.  Följande diagram illustrerar publicerings processen för allmänna erbjudanden.  Den här processen skiljer sig något mellan olika typer av erbjudanden.

![allmänt flöde för erbjudande publicering](./media/publish-flow.png)


## <a name="publishing-process"></a>Publicerings process

Använd följande steg för att publicera ditt erbjudande.

1. Klicka på **publicera** under fliken **redigerare** för att starta publicerings processen.
2. Under fliken **status** ser du **publicerings stegen** i formuläret flödes schema.  Det här formuläret anger också aktuell status för ditt erbjudande och rapporterade problem.
3. När du har kontrollerat att allting ser korrekt ut och fungerar korrekt i förhands granskning, är du redo att sätta igång. Klicka på **Go Live** på fliken **status** så tar Microsoft ditt erbjudande till produktion och på lämplig Marketplace. Normalt tar Live flera timmar.

När som helst i publicerings processen kan du också logga in och klicka på fliken **alla erbjudanden** för att visa den senaste statusen för något av dina erbjudanden. Du kan klicka direkt på statusen för ditt erbjudande och se informationen om var erbjudandet finns i publicerings processen.


## <a name="canceling-the-publishing-request"></a>Begäran om publicering avbryts

Du kan starta publicerings processen och behöva avbryta din begäran. Du kan bara avbryta en publicerings förfrågan när publicerings förfrågan når *utgivarens inloggnings* steg. Klicka på **Avbryt publicering**om du vill avbryta. Publicerings statusen återställs till steg 1, och om du vill publicera igen måste du klicka på **publicera** och följa stegen på fliken **status** .


## <a name="next-steps"></a>Nästa steg
