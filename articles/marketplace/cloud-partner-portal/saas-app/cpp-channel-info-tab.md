# <a name="saas-application-channel-info-tab"></a>Informationsfliken för SaaS-program kanal

Använda Channel informationsfliken för att ange kanalinformation om för appen, till exempel go-to-market (GTM) material och kontaktinformation.

![Kanalen info formulär](./media/saas-channel-info.png)

## <a name="to-configure-channel-information"></a>Konfigurera kanalinformation

1. Under **kanalinformation > GTM material**, anger du URL: en för den webbplats som är värd för ditt material för GTM.
2. Under **kanal Manager Kontakta**, ange en manager kontaktuppgifter som Microsoft kan kontakta för support och företag. Ange information för de här fälten: Namn, e-postadress och telefonnummer.
3. Välj **spara** att slutföra konfigurationen av kanalinformation.
