# <a name="container-marketplace-tab"></a>Fliken Marknadsplats för behållare

På fliken **Marketplace** på sidan **Nytt erbjudande** kan du förse dina potentiella kunder med marknadsföring, försäljning och juridisk information och avtal och hantera leads som genereras från marknadsplatsen. Lägg till ditt marknadsföringsspecifika innehåll i avsnittet **Översikt.**

## <a name="overview"></a>Översikt

I det här avsnittet anger du allmän information om ditt Azure Marketplace-erbjudande.  En asterisk (*) som läggs till i fältnamnet anger att det krävs.

![Information om Marketplace](./media/containers-marketplace-tab.png)

I följande tabell beskrivs syftet med och innehållet i dessa fält. Obligatoriska fält åtalas av en asterisk (*).

|    Field                  |       Beskrivning                                                            |
|  ---------                |     ---------------                                                          |
| **Titel\***                 | Erbjudandets titel. Det kommer att visas tydligt på marknaden.  Maximal längd är 50 tecken. |
| **Sammanfattning\***               | Kort sammanfattning av erbjudandet. Maximal längd är 100 tecken. |
| **Lång sammanfattning\***          | Längre sammanfattning av erbjudandet (även om det kan vara samma som **sammanfattningen**).  Maximal längd är 256 tecken. |
| **Beskrivning\***           | Beskrivning av erbjudandet.  Maximal längd är 3000 tecken, stöder enkel HTML-formatering.
| **Marknadsidentifierare**  | En unik URL som ska associeras med det här erbjudandet, som vanligtvis innehåller din organisation och ditt lösningsnamn. Maximal längd är 50 tecken.   |
| **Förhandsgranska prenumerations-ID:er\*** | Lägg till en till 100 prenumerationsidentifierare för förhandsgranskningsspelare. Dessa vitlistade prenumerationer har tillgång till erbjudandet när det har publicerats, innan det publiceras. |
|  |  |
