# <a name="container-support-tab"></a>Fliken stöd för behållare

Använd den **stöder** fliken den **nytt erbjudande** sidan för att ge teknisk hjälp och användaren Supportresurser för ditt erbjudande.  Följande indata områden tillhandahålls i formuläret Support: **Engineering Contact**, **kundsupport**, och **Support URL: er**. En asterisk (*) läggs till fältnamnet anger att det krävs.

## <a name="support-form-fields"></a>Stöd för formulärfält

Ange nödvändig information för fälten Engineering Contact och kundsupport.

### <a name="engineering-contact"></a>Engineering Contact

Kontakten tekniker är teknisk kontakt mellan din organisation och Microsoft. Följande information krävs.

- **Namn på** -namnet på den person eller grupp som fungerar som technical/tekniksupport.
- **E-post** -e-postadress för den här tekniska kontakt.
- **Phone** -telefonnummer för teknisk support.

### <a name="customer-support"></a>Kundsupport

Kontakta kundsupporten tar emot supportärenden som öppnas av kunder i Azure. Följande information krävs.

- **Namn på** -namnet på den person eller grupp som fungerar som kundsupport.
- **E-post** -e-postadress för den här supportkontakten.
- **Phone** -telefonnummer för kundsupport.

## <a name="next-steps"></a>Nästa steg

När du är klar med att tillhandahålla supportinformation är du redo att [publicera ditt erbjudande](./cpp-publish-offer.md).
