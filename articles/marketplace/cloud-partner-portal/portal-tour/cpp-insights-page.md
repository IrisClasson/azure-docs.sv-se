# <a name="insights-page"></a>Sidan med insikter

Sidan [**Insikter**](https://cloudpartner.azure.com/#insights) innehåller följande uppsättningar instrumentpaneler som hjälper dig att förstå och maximera försäljningen av dina marknadsplatserbjudanden.  Varje instrumentpanel nås via en flik med samma namn längst upp på sidan.


|  **Instrumentpanelen**    |  **Visat innehåll**               |
|  -------------    |  ---------------------               |
| **Sammanfattning**       | grafer, trender och värden för aggregerade data som sammanfattar marknadsplatsaktivitet för utgivarens erbjudanden |
| **Utbetalning**        | utbetalningar och relaterade transaktioner i grafiska och nedladdningsbara format |
| **Order & användning** | order och användningsinformation i grafiska och nedladdningsbara format |
| **Kund**      | kundinformation, inklusive deras inköpsprofil |
| **Distribution**    | information om lyckades och misslyckade driftsättning i format på både grafisk nivå och händelsenivå |
| **Nedladdningar**     | lista över nedladdningsbegäranden för de senaste 30 dagarna |
| **Analys**     | sammanfattning av webbanalys och analys av kampanjresultat |
|  |  |


## <a name="summary-dashboard"></a>Instrumentpanel för sammanfattningar
Instrumentpanelen Sammanfattning är den första fliken och är standardfliken som visas när du navigerar till sidan Statistik.  Den här avsiktliga designen återspeglar att den här instrumentpanelen ger utgivare den bredaste vyn av försäljningsaktiviteten för deras erbjudanden.  Det ger grafiska representationer av den normaliserade användningstrenden. månatlig användning. trendanvändning efter geografisk marknad, erbjudande och kund; och utbetalningstrenden.  Utdata kan anpassas genom att ange start- och slutdatum för analysen.  Följande bild visar en exempelsammanfattningsinstrumentpanel för Contoso för juni till augusti 2018.
