# <a name="insights-page"></a>Sidan med insikter

Sidan [**insikter**](https://cloudpartner.azure.com/#insights) innehåller följande uppsättningar av instrument paneler som hjälper dig att förstå och maximera försäljningen av dina Marketplace-erbjudanden.  Varje instrument panel nås via en flik med samma namn överst på sidan.


|  **Paneler**    |  **Visat innehåll**               |
|  -------------    |  ---------------------               |
| **Sammanfattning**       | diagram, trender och värden för sammanställda data som sammanfattar Marketplace-aktivitet för utgivarens erbjudanden |
| **Payout**        | utbetalningar och relaterade transaktioner i grafiskt och nedladdnings bara format |
| **Beställningar & användning** | beställningar och användnings information i grafiskt och nedladdnings Bart format |
| **Kund**      | kund information, inklusive inköps profil |
| **Distribution**    | information om lyckade och misslyckade distributioner i både grafik-och händelse nivå format |
| **Nedladdningar**     | lista över hämtnings begär Anden under de senaste 30 dagarna |
| **Analytics**     | Översikt över webb analys och analys av kampanj prestanda |
|  |  |


## <a name="summary-dashboard"></a>Instrumentpanel för sammanfattningar
Sammanfattnings instrument panelen är den första fliken och är standard fliken som visas när du navigerar till sidan insikter.  Denna avsiktliga design visar att den här instrument panelen ger utgivare den bredaste vyn av Sälj aktiviteten för sina erbjudanden.  Den ger grafiska representationer av den normaliserade användnings trenden. månatlig användning; tendens användning per geografisk marknad, erbjudande och kund. och utbetalnings trenden.  Utdata kan anpassas genom att ange start-och slutdatum för analysen.  Följande bild visar ett exempel på en översikts instrument panel för contoso i juni till och med augusti 2018.
