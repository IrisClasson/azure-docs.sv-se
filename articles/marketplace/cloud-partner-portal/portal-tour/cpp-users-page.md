# <a name="users-page"></a>Användarsidan

Sidan [**användare**](https://cloudpartner.azure.com/#users) visar en lista över alla registrerade användare av Cloud Partner Portal.  Kolumnerna innehåller varje användares e-postadress, Portal roll, erbjudande typ och publicerings organisation. Resultaten kan sorteras efter kolumn eller filtreras efter en angiven sträng.  

På den här sidan visas också knappen **Lägg till användare** för att lägga till användare i publicerings organisationen.  Mer information finns i [Hantera användare på Cloud Partner Portal](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-users.md).
