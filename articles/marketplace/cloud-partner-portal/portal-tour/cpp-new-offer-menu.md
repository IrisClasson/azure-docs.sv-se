# <a name="new-offer-menu"></a>Meny för nya erbjudanden

Med menyn **Nytt erbjudande** kan partner skapa instanser av erbjudandetyper som stöds av Cloud [Partner Portal](https://cloudpartner.azure.com).  Om du väljer menyalternativet **Nytt erbjudande,** i menyfältet för navigering på vänster sida, visas följande lista över erbjudandetyper.  Om du väljer en av dessa typer börjar processen för att skapa och publicera erbjudandet.

### <a name="next-steps"></a>Nästa steg

Om du skapar ett nytt erbjudande kan du använda föregående tabell för att navigera till erbjudandeavsnittet för att få vägledning om hur du skapar och publicerar erbjudandet.  Annars fortsätter portalen turnén med en undersökning av [sidan Alla erbjudanden](./cpp-all-offers-page.md).
