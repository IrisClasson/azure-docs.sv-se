# <a name="manage-your-azure-marketplace-and-appsource-publisher-profile"></a>Hantera din Azure Marketplace-och AppSource Publisher-profil

Den här artikeln förklarar hur du använder [Cloud Partner Portal](https://cloudpartner.azure.com/) för att visa och hantera din utgivar profil.  Du måste redan vara registrerad Microsoft Cloud partner för att få åtkomst till den här portalen.  Mer information finns i [Kom igång med Cloud Partner Portal](../../cloud-partner-portal-orig/cloud-partner-portal-getting-started-with-the-cloud-partner-portal.md).

Nu har du slutfört stegen för att bli en Azure Marketplace-och AppSource-utgivare. Utgivar profilen registreras på Cloud Partner Portal följande godkännande av din partner förfrågan. Din utgivar profil gäller för alla erbjudanden och SKU: er som publicerats från kontot som används vid partner registreringen.


## <a name="publisher-profile"></a>Utgivar profil

Din utgivar profil särskiljer ditt företag på Azure Marketplace och AppSource. Det består av ditt utgivar-ID, visnings namn och ägarens e-post (er). En väl hanterad profil ökar synligheten och hjälper Marketplace-användare att enkelt identifiera och välja ett lämpligt erbjudande.

> [!NOTE]
> Ditt utgivar-ID och din registrerade ägares e-postadress går inte att redigera när du publicerar ditt första erbjudande. Du kan dock visa din profil och redigera utgivarens visnings namn från Cloud Partner Portal.  <br/>
> Lägg till användare (deltagare och ägare) till din publicerings profil från avsnittet användare i det vänstra navigerings fönstret från avsnittet användare i Cloud Partner Portal.


### <a name="view-and-manage-your-publisher-profile"></a>Visa och hantera din utgivar profil

Visa och hantera din utgivar profil genom att klicka på fliken **utgivare profil** i den övre högra meny raden.

![Fliken utgivarens profil – utgivar profil](./media/publisherprofilenew.png)

Din **profil** dialog ruta visas.

![Profil information för utgivare – profil information för utgivare](./media/publisherprofiledetails.png)


### <a name="link-your-dev-center-account"></a>Länka ditt dev Center-konto

Du kan också länka ditt befintliga dev Center-konto till din utgivar profil på Cloud Partner Portal.  Logga först in på portalen med samma e-postadress som används för att registrera ditt dev Center-konto. Sedan visar sidan Publisher-profil din status för ditt dev Center-konto, ditt konto för dev Center-ägare och ett dev Center-konto.

> [!NOTE]
> Konto registrering för Developer Center är obligatoriskt för publicering av betalda marknads plats-SKU: er.

Om du inte har ett konto för utvecklar Center kan du skapa ett konto som beskrivs i [Registrera i dev Center](https://docs.microsoft.com/azure/marketplace/register-dev-center).  Lägg sedan till den registrerade e-postadressen som en ägare på Cloud Partner Portal för ditt erbjudande. En ägare kan bara lägga till den registrerade e-postadressen i din publicerings profil från avsnittet **användare** . Logga sedan in på portalen med samma e-postadress och länka ditt dev Center-konto.

![Utgivar profil information – Beskrivning av utgivar profil](./media/publisherprofiledescription.png)
