# <a name="manage-your-azure-marketplace-and-appsource-publisher-profile"></a>Hantera dina Azure Marketplace och AppSource utgivarprofil

Den här artikeln förklarar hur du använder den [Cloud Partner Portal](https://cloudpartner.azure.com/) att visa och hantera din publisher-profil.  Du måste redan vara registrerad Microsoft cloud partner till den här portalen.  Mer information finns i [Kom igång med partnerportalen i molnet](../../cloud-partner-portal-orig/cloud-partner-portal-getting-started-with-the-cloud-partner-portal.md).

Du har slutfört stegen för att bli en Azure Marketplace och AppSource utgivare vid den tidpunkten. Publisher-profil har registrerats på den partnerportalen i molnet efter godkännande av ditt partner-begäran. Din utgivarprofil gäller för alla erbjudanden och SKU: er som publiceras från det konto som används under registreringen av partner.


## <a name="publisher-profile"></a>Utgivarprofil

Din utgivarprofil särskiljer ditt företag på Azure Marketplace och AppSource. Den består av Publicerings-ID, namn och ägare email(s). En väl hanterad profilen kommer öka översyn och hjälper användarna att marketplace lätt kan identifiera och välja en lämplig erbjudandet.

> [!NOTE]
> Publisher ID och registrerad ägare för din e-post är inte redigerbara när du publicerar ditt första erbjudande. Men du kan visa din profil och redigera publisher visningsnamn från partnerportalen i molnet.  <br/>
> Lägg till användare (deltagare och ägare) till din publiceringsprofil från avsnittet för användare i det vänstra navigeringsfönstret från avsnittet för användare av partnerportalen i molnet.


### <a name="view-and-manage-your-publisher-profile"></a>Visa och hantera dina utgivarprofil

Om du vill visa och hantera dina utgivarprofil från det övre högra menyfältet, klickar du på den **utgivarprofil** fliken.

![Utgivarprofil profil fliken - utgivare](./media/publisherprofilenew.png)

Din **profil** dialogrutan visas.

![Profilinformation Publisher - profilinformation för utgivare](./media/publisherprofiledetails.png)


### <a name="link-your-dev-center-account"></a>Länka ditt Dev Center-konto

Du kan också länka ditt befintliga Dev Center-konto med din utgivarprofil på partnerportalen i molnet.  Först logga in på portalen med samma e-postadressen används för att registrera ditt Dev Center-konto. Din profilsida för utgivaren visar sedan ditt Dev Center kontostatus, e-postadress för ägare för Dev Center-konto och Dev Center-kontonamn.

> [!NOTE]
> Developer Center Kontoregistrering är obligatoriskt för publicering betald marknadsplatsen SKU: er.

Om du inte har ett Developer Center-konto kan du skapa ett konto som beskrivs i [registrera i Dev Center](../../register-dev-center.md).  Lägg sedan till den registrerade e-postadressen som ägare på partnerportalen i molnet för ditt erbjudande. Ägare kan bara lägga till den registrerade e-postadressen till din publiceringsprofil från den **användaren** avsnittet. Sedan logga in på portalen med samma e-postadress och länka ditt Dev Center-konto.

![Profilinformation Publisher - utgivare Profilbeskrivning](./media/publisherprofiledescription.png)
