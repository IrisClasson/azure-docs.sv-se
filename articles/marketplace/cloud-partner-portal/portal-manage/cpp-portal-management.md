# <a name="cloud-partner-portal-management"></a>Partnerportalen molnhantering

Det här avsnittet beskrivs vilka typer av åtgärder som är tillgängliga för att skapa och hantera konton som krävs för att använda den [Cloud Partner Portal](https://cloudpartner.azure.com).  Dessa konton som krävs för den [Azure Marketplace](https://azuremarketplace.microsoft.com) och [AppSource](https://appsource.microsoft.com).  


## <a name="background-information"></a>Grundläggande information

Innan du använder Cloud Partner Portal, bör du bekanta dig med följande artiklar:

- [Azure Marketplace-principer och villkor](../../policies-terms.md) visas de olika juridiska och tekniska krav och regler som gäller för användare av och utgivare kan den [Azure Marketplace](https://azuremarketplace.microsoft.com) och [AppSource](https://appsource.microsoft.com).
- [Bli en Cloud Marketplace Publisher](../../become-publisher.md) innehåller en översikt över alla steg som krävs för att bli en officiella Microsoft cloud utgivare.


## <a name="account-creation-and-management"></a>Konto skapande och hantering

Det här avsnittet innehåller följande artiklar som beskriver hur du hanterar marketplace publisher-konton.

- [Förutsättningar för att hantera konton](./cpp-manage-prerequisites.md) förklarar hur du blir utgivare registrerat moln, krav för att använda partnerportalen i molnet.
- [Hantera Azure Marketplace och AppSource utgivarprofil](./cpp-manage-publisher-profile.md) förklarar hur du använder Cloud Partner Portal för att visa och hantera din profil för Microsoft marketplace publisher. 
- [Hantera användare på Cloud Partner Portal](./cpp-manage-users.md) lägga till användare i din organisations moln publicera konto och hur du hanterar deras åtkomst, så att flera personer kan arbeta med samma erbjudanden.


## <a name="next-steps"></a>Nästa steg

Du måste uppfylla de [kraven för att hantera konton](./cpp-manage-prerequisites.md) innan du fortsätter.  Därefter kan du bör också dra den [Cloud Partner Portal Tour](../portal-tour/cpp-portal-tour.md) att bekanta dig med portalens användargränssnitt.
