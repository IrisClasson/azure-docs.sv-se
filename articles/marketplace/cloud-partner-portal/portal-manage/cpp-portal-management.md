# <a name="cloud-partner-portal-management"></a>Cloud Partner Portal hantering

I det här avsnittet beskrivs de olika typerna av åtgärder som du kan använda för att skapa och hantera konton som krävs för att använda [Cloud Partner Portal](https://cloudpartner.azure.com).  Dessa konton krävs för [Azure Marketplace](https://azuremarketplace.microsoft.com) och [AppSource](https://appsource.microsoft.com).  


## <a name="background-information"></a>Bakgrunds information

Innan du använder Cloud Partner Portal bör du bekanta dig med följande artiklar:

- [Principer och villkor för Azure Marketplace](../../policies-terms.md) listar de olika juridiska och tekniska krav och regler som gäller för användare av och utgivare till [Azure Marketplace](https://azuremarketplace.microsoft.com) och [AppSource](https://appsource.microsoft.com).
- [Bli en Cloud Marketplace-utgivare](https://docs.microsoft.com/azure/marketplace/become-publisher) ger en översikt över alla steg som krävs för att bli en officiell Microsoft-molnbaserad utgivare.


## <a name="account-creation-and-management"></a>Skapa och hantera konton

Det här avsnittet innehåller följande artiklar som förklarar hur du hanterar utgivar konton i Marketplace.

- Krav [för att hantera konton](./cpp-manage-prerequisites.md) förklarar hur du blir en registrerad moln utgivare, kravet på att använda moln partner portalen.
- [Hantera Azure Marketplace och AppSource Publisher-profilen](./cpp-manage-publisher-profile.md) förklarar hur du använder Cloud Partner Portal för att visa och hantera din Microsoft Marketplace Publisher-profil. 
- [Hantera användare på Cloud Partner Portal](./cpp-manage-users.md) lägga till användare i organisationens moln publicerings konto och hantera deras åtkomst för att tillåta att flera personer arbetar med samma erbjudanden.


## <a name="next-steps"></a>Nästa steg

Du måste uppfylla [kraven för att hantera konton](./cpp-manage-prerequisites.md) innan du fortsätter.  Därefter bör du också ta [Cloud partner Portals visningen](../portal-tour/cpp-portal-tour.md) för att bekanta dig med portalens användar gränssnitt.
