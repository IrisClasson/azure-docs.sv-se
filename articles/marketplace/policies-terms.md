# <a name="azure-marketplace-policies-and-terms"></a>Policyer och villkor för Azure Marketplace

>[!Note]
>”Marketplace” eller ”Azure Marketplace”: Microsoft ägs eller separata plattform, men med namnet genom vilka erbjudanden kan visas eller upptagna av kunder. Om inget annat anges, innehåller Marketplace Azure Marketplace och AppSource Marketplace för Azure-portalen.

Tack för visat intresse för publicera erbjudanden på Azure Marketplace. Vi arbetar hårt partnerskap med dig för att skapa en omfattande källa för molnlösningar och en rad för företag-erbjudanden som nöjda kunder över hela världen och hjälper dig att skapa din verksamhet.

Erbjudanden på Marketplace måste följa våra villkor och principer. Vi kan uppdatera dessa principer och villkoren då och då för att se till att en bra kundupplevelse och ge partner möjlighet att lyckas. Om du har feedback på våra principer eller villkor, berätta genom kommentarer i den [forum för Azure Marketplace och AppSource](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/bd-p/2222).
