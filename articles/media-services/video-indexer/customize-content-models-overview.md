# <a name="customizing-content-models-in-video-indexer"></a>Anpassa innehållsmodeller i Video Indexer

Video Indexer kan du anpassa några av sina modeller som ska anpassas till din specifika användning fall. Dessa modeller inkluderar [varumärken,](customize-brands-model-overview.md) [språk](customize-language-model-overview.md)och [person](customize-person-model-overview.md). Du kan enkelt anpassa dessa modeller med hjälp av videoindexerarens webbplats eller API.

Den här artikeln innehåller länkar till artiklar som förklarar fördelarna med varje typ av anpassning. Artikeln länkar också till programvägleder som visar hur du kan implementera anpassningen av varje modell.
