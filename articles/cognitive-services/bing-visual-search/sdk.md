# <a name="bing-visual-search-sdk"></a>Bing Visual Search SDK

Bing Visual Search innehåller följande klient SDK: er som du kan använda för att arbeta med API: T för Visual Search.

* [.NET-exempel](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [NuGet-paket](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.VisualSearch/1.2.0)
    * [.NET-biblioteken](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingVisualSearch) för definitioner och beroenden
* [Node.js-exempel](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * [Node.js-bibliotek](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) för definitioner och beroenden
* [Java-exempel](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * [Java-bibliotek](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples/tree/master/Search) för definitioner och beroenden
* [Python-exempel](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * [Python-biblioteken](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) för definitioner och beroenden

SDK-prov för varje språk som innehåller en Viktigt-fil med krav för att använda SDK tillsammans med anvisningar för att installera och köra exemplen.

För de här exemplen behöver du starta en prenumeration på S9-prisnivån enligt [Cognitive Services priser – API för Bing Search](https://azure.microsoft.com/en-us/pricing/details/cognitive-services/search-api/). 

Starta en prenumeration i Azure Portal:
1. Ange ”BingSearchV7” i textrutan längst upp i Azure-portalen där det står `Search resources, services, and docs`.  
2. Under Marketplace i den nedrullningsbara listan, väljer `Bing Search v7`.
3. Ange `Name` för den nya resursen.
4. Välj `Pay-As-You-Go` prenumeration.
5. Välj `S9` prisnivå.
6. Klicka på `Enable` att starta prenumerationen.

### <a name="sdk-tools"></a>SDK-verktyg
SDK-verktyg för att utforska funktionerna i Bing Visual Search kan hittas i Git-lagringsplatsen: [Bing Visual Search-verktyg](https://github.com/Azure-Samples/bing-visualsearch-tools)
