# <a name="how-to-sign-up-for-the-translator-speech-api"></a>Hur du registrerar dig för Translator Speech API

Följ dessa steg för att registrera dig för Translator Speech API.

[!INCLUDE [Deprecation note](../../../includes/cognitive-services-translator-speech-deprecation-note.md)]

## <a name="1-sign-into-azure"></a>1: Logga in på Azure

- Har du inte något konto? [Registrera dig för en Microsoft Azure-konto](http://azure.com/)
- Har du redan ett konto? [Logga in](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2: prenumerera på Microsoft Translator

- När du loggar in på Azure, går du till den [Azure Cognitive Services](https://portal.azure.com/#create/Microsoft.CognitiveServices) avsnittet
- Under ”API-typ”, väljer du Translator Speech API
- I avsnittet prisnivå väljer du prisnivån som bäst passar dina behov
    - Varje prenumeration har en kostnadsfri nivå. Den kostnadsfria nivån har samma funktioner och funktionalitet som de betalda prenumerationerna och har inte ett utgångsdatum.
    - Du kan ha endast en kostnadsfri prenumeration på ditt konto
- Fyll i resten av formuläret och välj knappen Skapa
- Prenumerationen är klar för direkt användning

## <a name="3-authentication-key"></a>3: autentiseringsnyckel

- Hämta din autentiseringsnyckeln genom att gå till menyn alla resurser > klickar du på din prenumeration
- Nyckel-värde används för autentisering, nyckel-värde som konfidentiell. Du behöver nyckelvärdet när du utvecklar din app
