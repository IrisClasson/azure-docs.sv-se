# <a name="featured-user-generated-content-for-the-anomaly-detector-api"></a>Aktuellt användargenererat innehåll för API:et för avvikelsedetektor

Använd den här artikeln för att upptäcka hur andra kunder tänker på och använder API:et för avvikelsedetektor. Följande resurser skapades av gruppen av Anomaly Detector-användare. De omfattar projekt med öppen källkod och andra bidrag som skapats av både Microsoft- och tredjepartsanvändare. Vissa av följande länkar finns på webbplatser som är externa för Microsoft och Microsoft ansvarar inte för innehållet där. Läs diskretion när du refererar till dessa resurser.
