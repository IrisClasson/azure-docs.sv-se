# <a name="bing-speech-client-samples"></a>Taligenkänning i Bing klient exempel

[!INCLUDE [Deprecation note](../../../includes/cognitive-services-bing-speech-api-deprecation-note.md)]

Taligenkänning i Bing tillhandahåller exempel från slut punkt till slut punkt för att demonstrera olika användnings fall, till exempel kommando igenkänning, kontinuerlig igenkänning och avsikts identifiering. Alla exempel är tillgängliga på GitHub och kan hämtas med följande länkar: README.md i varje databas och sidan [klient bibliotek](GetStarted/GetStartedClientLibraries.md) innehåller information om hur du skapar och kör exemplen.

- [REST API exempel](https://github.com/Azure-Samples/SpeechToText-REST)

- [JavaScript-exempel](https://github.com/Azure-Samples/SpeechToText-WebSockets-Javascript)

- [C#Skriv bords klient exempel](https://github.com/Azure-Samples/Cognitive-Speech-STT-Windows)

- [C#tjänst exempel](https://github.com/Azure-Samples/Cognitive-Speech-STT-ServiceLibrary)

- [Java på Android-exempel](https://github.com/Azure-Samples/Cognitive-Speech-STT-Android)

- [ObjectiveC på iOS-exempel](https://github.com/Azure-Samples/Cognitive-Speech-STT-iOS)

Alla SDK:er i Microsoft Cognitive Services och exempel är licensierade med MIT-licensen. Mer information finns i [License](https://github.com/Azure-Samples/SpeechToText-REST/blob/master/LICENSE).
