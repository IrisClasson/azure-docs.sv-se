# <a name="get-started-with-bing-speech-service-client-libraries"></a>Kom igång med klient bibliotek för Taligenkänning i Bings tjänsten

[!INCLUDE [Deprecation note](../../../../includes/cognitive-services-bing-speech-api-deprecation-note.md)]

Förutom att göra direkta HTTP-begäranden via en REST API ger Taligenkänning i Bing-tjänsten utvecklare med tal klient bibliotek på olika språk. Tal klient biblioteken:

- Stöd för mer avancerade funktioner i tal igenkänning, till exempel mellanliggande resultat i real tid, lång ljud ström (upp till 10 minuter) och kontinuerlig igenkänning.
- Ange ett enkelt och idiomatiskt-API på det språk som du föredrar.
- Dölj kommunikations information på låg nivå.

För närvarande finns följande Taligenkänning i Bing klient bibliotek:

- [C#Skriv bords bibliotek](GetStartedCSharpDesktop.md)
- [C#tjänst bibliotek](GetStartedCSharpServiceLibrary.md)
- [JavaScript-bibliotek](GetStartedJSWebsockets.md)
- [Java-bibliotek för Android](GetStartedJavaAndroid.md)
- [Mål-C-bibliotek för iOS](Get-Started-ObjectiveC-iOS.md)

## <a name="additional-resources"></a>Ytterligare resurser

- Sidan [exempel](../samples.md) innehåller fullständiga exempel för att använda tal klient bibliotek.
- Om du behöver ett klient bibliotek som inte stöds ännu kan du skapa ett eget SDK. Implementera ett [tal-WebSocket-protokoll](../API-Reference-REST/websocketprotocol.md) på plattformen och Använd valfritt språk.
