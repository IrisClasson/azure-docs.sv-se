# <a name="quickstart-use-the-computer-vision-client-library"></a>Snabbstart: Använda klientbiblioteket För datorseende

Kom igång med klientbiblioteket För datorseende. Följ dessa steg för att installera paketet och prova exempelkoden för grundläggande uppgifter. Computer Vision ger dig tillgång till avancerade algoritmer för bearbetning av bilder och returnerande information

Använd klientbiblioteket För visuellt innehåll för att:

* Analysera en bild för taggar, textbeskrivning, ansikten, barnförbjudet innehåll med mera.
* Identifiera utskriven och handskriven text med batchläsnings-API:et.
