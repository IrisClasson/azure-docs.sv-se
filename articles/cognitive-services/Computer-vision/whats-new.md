## <a name="january-2020"></a>Januari 2020

### <a name="read-api-30-preview"></a>Läs förhandsversionen av API 3.0

Du har nu möjlighet att använda version 3.0 av Läs-API:et för att extrahera utskriven eller handskriven text från bilder. Jämfört med tidigare versioner innehåller 3.0:
* förbättrad noggrannhet
* nytt utdataformat
* konfidenspoäng för varje extraherat ord
* stöd för både spanska och engelska språk med den ytterligare språkparametern

Följ en [snabbstart för extrahera text](https://docs.microsoft.com/azure/cognitive-services/computer-vision/quickstarts/csharp-hand-text?tabs=version-3) för att börja använda 3.0 API:et.
