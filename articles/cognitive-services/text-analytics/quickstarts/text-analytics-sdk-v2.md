# <a name="quickstart-use-the-text-analytics-client-library-v2"></a>Snabb start: Använd Textanalys klient bibliotek v2

Kom igång med Textanalys klient bibliotek v2. Följ de här stegen för att installera paketet och prova exempel koden för grundläggande uppgifter.

Använd Textanalys klient bibliotek v2 för att utföra:

* Känsloanalys
* Språkspårning
* Enhetsidentifiering
* Extraktion av nyckelfraser
