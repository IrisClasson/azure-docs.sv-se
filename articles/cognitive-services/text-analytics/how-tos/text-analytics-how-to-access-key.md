# <a name="how-to-find-endpoints-and-access-keys-for-the-text-analytics-cognitive-service"></a>Så här hittar du slutpunkter och åtkomst till nycklar för Text Analytics Cognitive Service

När du registrerar dig för textanalys kan hämta du en åtkomstnyckel för anpassade unika till din prenumeration. Den här nyckeln krävs för varje anrop till API för textanalys. Om du inte gjort [registrerad](text-analytics-how-to-signup.md), göra det nu för att få din nyckel. 

Om du behöver hjälp att hitta nyckeln eller för att avgöra om din prenumeration redan har textanalys, Använd följande instruktioner för att hämta nödvändig information. 

## <a name="find-your-service-endpoint-and-access-key"></a>Hitta din service-slutpunkt och åtkomstnyckel

1. Logga in på [Azure-portalen](https://portal.azure.com).

2. I det vänstra navigeringsfönstret väljer **alla tjänster**.

3. Skriv i filtret, *Cognitive Services*. Om din prenumeration har textanalys, visas den i listan med en API-typ av **API för textanalys**.

4. Klicka på länken för att öppna bladet för tjänsten. Nu kan du få en nyckel från **resurser > nycklar** eller genom att klicka på **visa åtkomstnycklar** i rutan information. Slutpunkten är också visas i rutan information.
