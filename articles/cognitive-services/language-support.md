# <a name="natural-language-support-for-azure-cognitive-services"></a>Stöd för naturligt språk för Azure Cognitive Services

Med Azure Cognitive Services kan du bygga program som kan se, höra, tala med och förstå dina användare. Mellan dessa tjänster stöds fler än tre dussin språk, så att användarna kan kommunicera med ditt program på naturliga sätt.

Den här artikeln är uppdelad i två delar. De första listar kärn språken som ofta stöds i Azure Cognitive Services. Det andra avsnittet fokuserar på ytterligare språk stöd för varje tjänst. Språkets tillgänglighet kan variera beroende på land/region och marknad.

## <a name="core-languages"></a>Kärn språk

De här kärn språken stöds i Azure Cognitive Services:

* Kinesiska
* Svenska
* Franska
* Tyska
* Italienska
* Japanska
* Koreanska ¹
* Portugisiska
* Spanska

> [!NOTE]
> ¹ LUIS, Video Indexer, Textanalys och tal till text stöds inte.

## <a name="additional-language-availability-by-service"></a>Ytterligare språk tillgänglighet per tjänst

Tabellerna fokuserar på språk tillgänglighet per tjänste kategori; undantagna kärn språk. Använd länkarna nedan om du vill veta mer om ytterligare språk stöd och land/region och marknads tillgänglighet.
