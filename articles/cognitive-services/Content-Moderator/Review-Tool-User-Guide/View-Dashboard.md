# <a name="view-dashboard"></a>Visa instrumentpanelen

Klicka på den **instrumentpanelen** FLIKEN för att se viktiga mått för alla innehållsgranskning klar i verktyget. Du kan se Totalt antal slutförda och pågående granskningar för bilder och text. Du kan se mått för standardteam och din undergrupper.
