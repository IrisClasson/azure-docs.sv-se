# <a name="upload-images"></a>Ladda upp bilder

Om du vill utforska användargränssnittet, överföra upp till fem bilder i taget, varje avbildning högst 2 MB i storlek. Glöm inte att kontrollera din filstorlekar innan du laddar upp.

![Ladda upp bilder](images/1-try-image-1.png)

## <a name="submit-for-moderation"></a>Skicka in för Moderering
Om du måste ladda upp fler än fem bilder i taget, ta bort oönskade bilder med hjälp av du istället Papperskorgen. Skicka bilder för automatiserad moderering.

![Skicka bilder](images/1-try-image-2.png)
