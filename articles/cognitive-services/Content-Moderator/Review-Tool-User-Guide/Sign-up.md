# <a name="sign-up-for-content-moderator"></a>Registrera dig för Content Moderator

Registrera dig på Content Moderator [granskningsverktyget](https://contentmoderator.cognitive.microsoft.com) webbplats. Du kan logga med ditt Microsoft-konto, eller så kan du skapa ett konto på webbplatsen.

![Registrera dig](images/0-signup-1.PNG)

## <a name="name-your-team"></a>Namnge ditt team

Ange ett namn för ditt granskningsteam. Alternativt kan du bjuda in dina kollegor för att ansluta till ditt team.

![Skapa Team](images/1-Team-Name.PNG)
