# <a name="whats-new-in-qna-maker"></a>Nyheter i QnA Maker

Läs om vad som är nytt i tjänsten. Dessa objekt kan släppa anteckningar, videor, blogginlägg och andra typer av information. Bokmärk den här sidan för att hålla dig uppdaterad med tjänsten.

## <a name="release-notes"></a>Viktig information

Läs om vad som är nytt med QnA Maker.

### <a name="march-2020"></a>Mars 2020

* TLS 1.2 tillämpas nu för alla HTTP-begäranden till den här tjänsten. Mer information finns i [Azure Cognitive Services-säkerhet](../cognitive-services-security.md).

### <a name="february-2020"></a>Februari 2020

* [NPM-paket](https://www.npmjs.com/package/@azure/cognitiveservices-qnamaker) med GenerateAnswer API

### <a name="november-2019"></a>November 2019

* [Usa:s regerings molnstöd](https://docs.microsoft.com/azure/azure-government/documentation-government-services-aiandcognitiveservices#qna-maker) för QnA Maker
* [Multi-turn-funktion](https://docs.microsoft.com/azure/cognitive-services/qnamaker/how-to/multiturn-conversation) i GA
* [Chit-chat-support](https://docs.microsoft.com/azure/cognitive-services/qnamaker/how-to/chit-chat-knowledge-base#language-support) tillgänglig på nivå 1-språk

### <a name="october-2019"></a>Oktober 2019

* [Ange uttryckligen språk](./how-to/language-knowledge-base.md#select-language-when-creating-first-knowledge-base) för alla kunskapsbaser i QnA Maker-tjänsten.

### <a name="september-2019"></a>September 2019

* Importera och exportera med [XLS-filformat](concepts/content-types.md)

### <a name="june-2019"></a>Juni 2019

* Förbättrad [rankermodell](concepts/query-knowledge-base.md#ranker-process) för franska, italienska, tyska, spanska, portugisiska

### <a name="april-2019"></a>April 2019

* Stöd för extrahering av webbplatsinnehåll
* [Stöd för SharePoint-dokument](how-to/add-sharepoint-datasources.md) från autentiserat åtkomst

### <a name="march-2019"></a>Mars 2019

* [Aktivt lärande](how-to/improve-knowledge-base.md) ger förslag på nya frågealternativ baserade på verkliga användarfrågor
* Förbättrad modell för språkbearbetning [ranker](concepts/query-knowledge-base.md#ranker-process) (NLP) för engelska
