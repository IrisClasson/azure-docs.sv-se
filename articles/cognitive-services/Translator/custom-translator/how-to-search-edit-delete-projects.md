# <a name="search-edit-and-delete-projects"></a>Sök, redigera och ta bort projekt

Anpassade Translator finns flera sätt att hantera dina projekt på effektivt sätt. Du kan skapa flera projekt, Sök baserat på dina villkor och redigera dina projekt. Det går också att ta bort ett projekt i anpassade Translator.  

## <a name="search-and-filter-projects"></a>Sök och filtrera projekt

Verktyget filter kan du söka projekt av olika filtervillkor. Filtreras som projektnamn, status, källa och målspråk och kategori för projektet.

1.  Klicka på filtreringsknappen.

    ![Sök efter projekt](media/how-to/how-to-search-project.png)

1.  Du kan filtrera efter några (eller alla) i följande fält: namn, status, källspråk, målspråk och kategori.

2.  Klicka på tillämpa.

    ![Filteralternativ för Search-projekt](media/how-to/how-to-search-project-filters.png)

3.  Ta bort filtret om du vill visa alla projekt genom att trycka på ”Rensa”.


## <a name="edit-a-project"></a>Redigera ett projekt

Anpassade Translator ger dig möjligheten att redigera namnet och beskrivningen för ett projekt. Andra projekt metadata som kategori, källspråk och målspråk inte är tillgängliga för redigera. Stegen nedan beskriver hur du redigerar ett projekt.

1.  Klicka på pennikonen som visas när du hovrar över ett projekt.

    ![Redigera projekt](media/how-to/how-to-edit-project.png)

2.  I dialogrutan du kan ändra på projektets namn eller beskrivning av projektet, men det går inte att ändra etikett, kategori eller språk par för projektet.

    ![Redigera dialogrutan projekt](media/how-to/how-to-edit-project-dialog.png)

3.  Klicka på filtreringsknappen.

## <a name="delete-a-project"></a>Ta bort ett projekt

Du kan ta bort ett projekt när du inte längre behöver den. Stegen nedan Beskriv hur du tar bort ett projekt.

1. Hovra över en projektpost, och klicka på Papperskorgen.

   ![Ta bort projekt](media/how-to/how-to-delete-project.png)

2. Bekräfta borttagning. Tar bort ett projekt raderas alla modeller som skapats i det aktuella projektet. Ta bort projektet påverkar inte dina dokument.

   ![Bekräfta vid borttagning](media/how-to/how-to-delete-project-confirm.png)
