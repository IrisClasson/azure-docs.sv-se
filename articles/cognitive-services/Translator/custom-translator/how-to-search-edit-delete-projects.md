# <a name="search-edit-and-delete-projects"></a>Söka, redigera och ta bort projekt

Custom Translator erbjuder flera sätt att hantera dina projekt på ett effektivt sätt. Du kan skapa många projekt, söka baserat på dina kriterier och redigera dina projekt. Det är också möjligt att ta bort ett projekt i Custom Translator.  

## <a name="search-and-filter-projects"></a>Sök och filtrera projekt

Med filterverktyget kan du söka efter projekt efter olika filterförhållanden. Den filtrerar som projektnamn, status, käll- och målspråk och projektkategori.

1.  Klicka på filterknappen.

    ![Sök projekt](media/how-to/how-to-search-project.png)

1.  Du kan filtrera efter alla (eller alla) av följande fält: projektnamn, status, källspråk, målspråk och kategori.

2.  Klicka på Ansök.

    ![Sök efter projektfilteralternativ](media/how-to/how-to-search-project-filters.png)

3.  Rensa filtret för att visa alla dina projekt genom att trycka på "Rensa".


## <a name="edit-a-project"></a>Redigera ett projekt

Custom Translator ger dig möjlighet att redigera namn och beskrivning av ett projekt. Andra projektmetadata som kategori, källspråk och målspråk är inte tillgängliga för redigering. Stegen nedan beskriver hur du redigerar ett projekt.

1.  Klicka på pennikonen som visas när du hovrar över ett projekt.

    ![Redigera projekt](media/how-to/how-to-edit-project.png)

2.  I dialogrutan kan du ändra antingen projektnamnet eller beskrivningen av projektet, men kan inte ändra projektetiketten, kategorin eller språkparet.

    ![Dialogrutan Redigera projekt](media/how-to/how-to-edit-project-dialog.png)

3.  Klicka på filterknappen.

## <a name="delete-a-project"></a>Ta bort ett projekt

Du kan ta bort ett projekt när du inte längre behöver det. Nedan beskrivs hur du tar bort ett projekt.

1. Hovra på alla projekt post och klicka på papperskorgen ikonen.

   ![Ta bort projekt](media/how-to/how-to-delete-project.png)

2. Bekräfta borttagning. Om du tar bort ett projekt tas alla modeller som har skapats i projektet bort. Om du tar bort projektet påverkas inte dina dokument.

   ![Dialogrutan Ta bort bekräftelse](media/how-to/how-to-delete-project-confirm.png)
