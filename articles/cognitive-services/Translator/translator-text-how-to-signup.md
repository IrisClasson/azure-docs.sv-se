# <a name="how-to-sign-up-for-the-translator-text-api"></a>Så här registrerar du dig för Translator Text API

## <a name="sign-in-to-the-azure-portal"></a>Logga in på Azure Portal

- Har du inte något konto? Du kan skapa ett [kostnadsfritt konto](https://azure.microsoft.com/free/) och experimentera utan kostnad.
- Har du redan ett konto? [Logga in](https://ms.portal.azure.com/)

## <a name="create-a-subscription-to-the-translator-text-api"></a>Skapa en prenumeration på Translator Text API

När du har loggat in på portalen kan du skapa en prenumeration på Translator Text API enligt följande:

1. Välj **+ Skapa en resurs**.
1. I sökrutan **Sök på Marketplace** anger du **Översättartext** och väljer den sedan från resultaten.
1. Välj **Skapa** om du vill definiera information för prenumerationen.
1. Välj den prisnivå som bäst passar dina behov i listan **Prisnivå.**
    1. Varje prenumeration har en kostnadsfri nivå. Den kostnadsfria nivån har samma funktioner som de betalda abonnemangen och upphör inte att gälla.
    1. Du kan bara ha en gratis prenumeration för ditt konto.
1. Välj **Skapa** för att slutföra skapandet av prenumerationen.

## <a name="authentication-key"></a>Autentiseringsnyckel

När du registrerar dig för Översättartext får du en anpassad åtkomstnyckel som är unik för din prenumeration. Den här nyckeln krävs för varje anrop till Translator Text API.

1. Hämta autentiseringsnyckeln genom att först välja lämplig prenumeration.
1. Välj **Nycklar** i avsnittet **Resurshantering** i prenumerationens information.
1. Kopiera någon av nycklarna som anges för din prenumeration.
