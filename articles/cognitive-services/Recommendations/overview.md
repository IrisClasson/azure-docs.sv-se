# <a name="recommendations-api-discontinued"></a>Förhandsversion av API för rekommendationer (inaktuell)

Förhandsversionen av API för rekommendationer utvecklas inte längre aktivt och togs officiellt ur bruk den 15 februari. 

Ett alternativ är den nya [mallen för rekommendationslösningar](https://aka.ms/recopcs) med öppen källkod, som gör att du kan hantera en rekommendationsmotor i din egen Azure-prenumeration. Den stöder huvudscenarierna i API för rekommendationer, och du får även friheten att distribuera till valfri Azure-region, mer kontroll över de dataresurser som används för att träna modeller samt rekommendationer. 

Du hittar mer information om mallen för rekommendationslösningar i [dokumentationen](https://github.com/Microsoft/Product-Recommendations).
