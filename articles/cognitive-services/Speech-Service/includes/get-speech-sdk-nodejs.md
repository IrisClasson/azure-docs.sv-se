### <a name="nodejs-package-manager-npm"></a>Node. js-paket hanteraren (NPM)

Installera Java Script Speech SDK genom att köra följande `npm install` kommando nedan.

```nodejs
npm install microsoft-cognitiveservices-speech-sdk
```

Mer information finns i avsnittet <a href="https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/javascript/node" target="_blank">Node. js tal SDK snabb start <span class="docon docon-navigate-external x-hidden-focus"> </span> </a>.
