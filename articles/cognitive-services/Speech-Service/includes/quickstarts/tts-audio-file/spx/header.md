I den här snabb starten använder du verktyget SPX från kommando raden för att konvertera text till tal som lagras i en ljudfil. Tjänsten för text till tal innehåller många alternativ för syntetiserade röster under [stöd för text till tal-språk](../../../../language-support.md#text-to-speech). Efter en konfiguration i taget kan du använda SPX-verktyget för att syntetisera tal från text med hjälp av kommandon från kommando raden.
