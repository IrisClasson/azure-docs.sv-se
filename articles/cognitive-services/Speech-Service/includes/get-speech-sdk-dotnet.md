#### <a name="c-nuget-package"></a>C# NuGet-paketet

.NET Speech SDK kan installeras från **.NET** Core `dotnet add` CLI med följande kommando.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

.NET Speech SDK kan installeras från **Pakethanteraren** med följande `Install-Package` kommando.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
