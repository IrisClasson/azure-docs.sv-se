|**Beskrivning**|Massutringsbiblioteket gör det möjligt för klientprogram att utföra massåtgärder i Azure Cosmos DB-konton. Massutnämningsbiblioteket tillhandahåller MassImport och MassUpdate-namnområden. BulkImport-modulen kan massinta dokument på ett optimerat sätt så att dataflödet som etablerats för en samling förbrukas i dess maximala omfattning. BulkUpdate-modulen kan massuppdatering av befintliga data i Azure Cosmos-behållare som korrigeringsfiler.|
