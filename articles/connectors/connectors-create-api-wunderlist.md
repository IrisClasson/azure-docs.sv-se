# <a name="monitor-and-manage-wunderlist-by-using-azure-logic-apps"></a>Övervaka och hantera Wunderlist med hjälp av Azure Logic Apps

Från och med den 6 maj 2020 [stänger Wunderlist-tjänsten ned](https://www.wunderlist.com/) och efterfrågade kunder för att [migrera sina data och börja använda Microsoft för att göra det i stället](https://www.wunderlist.com/blog/goodbye-from-wunderlist/). Du kan använda [Microsoft för att göra-anslutningen](https://docs.microsoft.com/connectors/todo/) för att ersätta den föråldrade [Wunderlist-anslutningen](https://docs.microsoft.com/connectors/wunderlist/).
