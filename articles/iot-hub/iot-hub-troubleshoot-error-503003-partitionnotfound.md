# <a name="503003-partitionnotfound"></a>503003 PartitionNotFound

I den här artikeln beskrivs orsakerna och lösningarna för **503003 PartitionNotFound-fel.**

## <a name="symptoms"></a>Symtom

Begäranden till IoT Hub misslyckas med felet **503003 PartitionNotFound**.

## <a name="cause"></a>Orsak

Det här felet är internt för IoT Hub och är sannolikt övergående. Se [IoT Hub interna serverfel orsak](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause).

## <a name="solution"></a>Lösning

Se [lösningar på interna serverfel i IoT Hub](iot-hub-troubleshoot-error-500xxx-internal-errors.md#solution).
