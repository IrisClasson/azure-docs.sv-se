# <a name="azure-notification-hubs-sdks"></a>Azure-meddelandehubbar SDK:er

Microsoft och tredje part publicerar SDK:er för Azure Notification Hubs. Microsoft-databaserna för Android, iOS och .NET innehåller också exempelappar för varje målplattform.

| SDK            | Utgivare      |
| -------------- | -------------- |
| [.NET-server](https://github.com/Azure/azure-notificationhubs-dotnet) | Microsoft |
| [Android](https://github.com/Azure/azure-notificationhubs-android)  |Microsoft |
| [Apache Cordova](https://github.com/derek82511/cordova-azure-notification-hubs) | Tredje part |
| [iOS](https://github.com/Azure/azure-notificationhubs-ios) |Microsoft |
| [Java-server](https://github.com/Azure/azure-notificationhubs-java-backend) |Microsoft |
| [Nodejs](https://github.com/Azure/azure-sdk-for-node) | Microsoft |
| [Reagera inbyggt](https://github.com/CatalystCode/react-native-azurenotificationhub) | Tredje part |

> [!Note]
> Microsoft ansvarar inte för kvaliteten, användbarheten eller supporten för lösningar från tredje part.

Vänligen meddela oss om det saknas.
