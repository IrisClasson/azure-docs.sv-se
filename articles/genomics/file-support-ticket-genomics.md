# <a name="how-to-contact-microsoft-genomics-for-support"></a>Så här kontaktar du Microsoft Genomics för support
I den här översikten beskrivs hur du lämnar in en supportbegäran om att kontakta Microsoft Genomics. Detta kan vara användbart om du inte kan felsöka problemet med hjälp av [felsökningsguiden](troubleshooting-guide-genomics.md) eller [vanliga frågor och svar](frequently-asked-questions-genomics.md). 


## <a name="file-a-support-ticket-through-the-azure-portal"></a>Arkivera en supportbiljett via Azure-portalen
Logga först in på ditt Azure-konto i Azure-portalen. Klicka på frågetecknet längst upp till `Help + support`höger och välj .

![Kontakta support på Azure Portal](./media/file-support-ticket/genomics-contact-support.png "Kontakta support på Azure Portal") 



Välj `New support request`sedan . 

![Ny supportbegäran](./media/file-support-ticket/new-support-request.png "Ny supportbegäran") 

Fyll i `Basics`, `Issue type` inklusive `Subscription`och din . Välj den `Issue type` mest relevanta för ditt problem. som ett exempel i `Technical` bilden nedan, har valts. Var noga `Microsoft Genomics` med att välja som relevant tjänst.  `Microsoft Genomics`kommer att listas `Intelligence and Analytics` under kategorin.   När du `Microsoft Genomics`har valt blir du `Support plan` också ombedd att ange din typ.

![Grunderna i supportbegäran](./media/file-support-ticket/support-request-basics.png "Grunderna i supportbegäran")


Lägg sedan till information om ditt problem. Verktyget för begäran kommer också att länka till befintlig dokumentation som kan vara till hjälp för dig. I det här exemplet är `Genomics keys`problemet relaterat till , men flera andra alternativ är också tillgängliga.

![Problem med supportbegäran](./media/file-support-ticket/support-request-problem.png "Problem med supportbegäran")

Lägg slutligen till din `Create` kontaktinformation och välj längst ned på skärmen för att skicka din supportbegäran.

![Kontakt med supportbegäran](./media/file-support-ticket/support-request-contact.png "Kontakt med supportbegäran")
