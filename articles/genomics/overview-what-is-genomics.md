# <a name="what-is-microsoft-genomics"></a>Vad är Microsoft Genomics?
Microsoft Genomics erbjuder en molnimplementering av Burrows-Wheeler Aligner (BWA) och Genome Analysis Toolkit (GATK) för sekundär analys. Tjänsten är ISO-certifierad och kompatibel med HIPAA-reglerna och erbjuder prisförutsägbarhet för dina genomsekvenseringsbehov. Lär dig hur du använder Microsoft Genomics-tjänsten och integrerar med vårt API genom att läsa våra snabbstarter, självstudier och dokumentation.

## <a name="support-your-most-demanding-sequencing-needs"></a>Stöd dina mest krävande sekvenseringsbehov
I stället för att hantera dina egna datacenter kan du dra nytta av omfattningen och erfarenheten av Microsoft när du kör arbetsbelastningar i exabyteskala. Vår molnimplementering av BWA-GATK är mycket överensstämmelse med Broad Institutes pipeline för bästa praxis.


## <a name="keep-your-business-running"></a>Håll verksamheten igång
Microsoft Genomics erbjuder ett serviceavtal på 99,99 % tillgänglighetsnivå (SLA) för att ta emot arbetsflödesförfrågningar. Läs mer om Microsoft Genomics [SLA](https://azure.microsoft.com/support/legal/sla/genomics/v1_0/).


## <a name="secure-your-data"></a>Skydda dina data
Microsoft Genomics-tjänsten är ISO 27001-, ISO 27018- och ISO 9001-certifierad och kompatibel med HIPAA-reglerna. Besök [Azure Säkerhetscenter](https://www.microsoft.com/trustcenter/security) för information om Azures plattformssäkerhet.
