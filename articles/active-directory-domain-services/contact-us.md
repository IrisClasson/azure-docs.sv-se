# <a name="azure-ad-domain-services---contact-us"></a>Azure AD-domäntjänster – kontakta oss

## <a name="feedback"></a>Feedback
Om du har feedback på våra tjänster eller icke-tekniska frågor kan du dela din begäran med produktgruppen Azure AD Domain Service genom att skicka ett e-postmeddelande till oss på: [Azure AD Domain Services Feedback](mailto:aaddsfb@microsoft.com). E-postmeddelanden till den här adressen granskas. Medlemmar i produktgruppen **kan** svara på begäran om ytterligare information. Förfrågningar om teknisk support till den här e-postadressen besvaras inte.

## <a name="technical-assistance"></a>Tekniskt bistånd
Om du har problem med den hanterade domänen kontrollerar du om stegen i [felsökningsguiden](troubleshoot.md) löser problemet. Om du fortfarande har problem loggar du in på Azure-portalen. Öppna startsidan för **Azure AD Domain Services** och välj Ny **supportbegäran** i åtgärdsfönstret under **Support + felsökning**.

![ny supportbegäran](./media/contact-us/supportRequest.png) 

Fyll i varje avsnitt i supportbegäran så noggrant som möjligt. När du väljer en tjänst kontrollerar du att du väljer **Azure Active Directory Domain Services (VM - Domain Controllers)** för att dirigera din begäran till rätt supporttekniker.

> [!IMPORTANT]
> Välj **Azure Active Directory-kataloger, domäner och objekt** i tjänstlistorna för hjälp med domänverifiering med Azure Active Directory.
>
> 

## <a name="feature-requests"></a>Funktionsbegäranden
Har du en uppfattning om hur vi kan förbättra Azure Active Directory Domain Services? Vi skulle vilja höra om det. Gå till Azure Feedback Forum, navigera till kategorin [Domain Services](https://feedback.azure.com/forums/169401-azure-active-directory?category_id=160593) och dela din idé. Medlemmar i produktgruppen granskar dessa begäranden.
