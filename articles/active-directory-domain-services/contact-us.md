# <a name="azure-ad-domain-services---contact-us"></a>Azure AD Domain Services – Kontakta oss

## <a name="feedback"></a>Feedback
Om du har feedback för vår tjänst eller icke-tekniska frågor kan dela du din begäran med Azure AD Domain Service produktgruppen av e-posta oss på: [Azure AD Domain Services Feedback](mailto:aaddsfb@microsoft.com). E-postmeddelanden till den här adressen granskas. Medlemmar i produktgruppen **kan** svara för att begära ytterligare information. Kommer inte att svara på begäranden för teknisk support till den här e-postadress.

## <a name="technical-assistance"></a>Teknisk support
Om du har problem med din hanterade domän kontrollerar du om stegen som beskrivs i den [felsökningsguiden](troubleshoot.md) lösa problemet. Om du fortfarande har problem med, loggar du in på Azure Portal. Öppna den **Azure AD Domain Services** startsida och välj **ny supportbegäran** från åtgärdsrutan under **Support och felsökning**.

![Ny supportförfrågan](./media/contact-us/supportRequest.png) 

Fyll i varje avsnitt på supportbegäran så noggrant som möjligt. När du väljer en tjänst, se till att du väljer **Azure Active Directory Domain Services (VM - domänkontrollanter)** att dirigera din begäran till rätt supporttekniker.

> [!IMPORTANT]
> Välj **Azure Active Directory-kataloger, domäner och objekt** i listorna tjänster för att få hjälp med att verifiera domänen med Azure Active Directory.
>
> 

## <a name="feature-requests"></a>Funktionsbegäranden
Har du en idé på hur vi kan förbättra Azure Active Directory Domain Services? Vi vill få information om den. Gå till Azure Feedback-forum, navigera till den [Domain Services](https://feedback.azure.com/forums/169401-azure-active-directory?category_id=160593) kategori och dela din idé. Medlemmar i produktgruppen granska dessa begäranden.
