# <a name="third-party-software-compatible-with-azure-ad-domain-services"></a>Programvara från tredje part som är kompatibel med Azure AD Domain Services

## <a name="one-identity-authentication-services"></a>En identitetsautentiseringstjänster

![En identitet](./media/compatible-software-icons/OneIdentity.png)

Med One Identity Authentication Services kan Unix-, Linux- och Mac OS X-system använda åtkomst, autentisering och auktorisering av en organisations befintliga Active Directory-infrastruktur (AD). Authentication Services stöder nu Azure Active Directory Domain Services som gör det möjligt för icke-Windows-resurser att använda samma nästa generations plattform som dina befintliga SaaS-lösningar redan använder.

[En identitetsautentiseringstjänster](https://www.quest.com/products/authentication-services/)

[KB-artikel – använda en identitetsautentiseringstjänster med Azure AD-domäntjänster](https://support.quest.com/authentication-services/kb/208427)


## <a name="red-hat-enterprise-linux"></a>Red Hat Enterprise Linux

![Red Hat](./media/compatible-software-icons/RedHat.png)

Red Hat Enterprise Linux är världens ledande Linux-plattform för företag som är byggd för att möta behoven hos dagens moderna företag. Red Hat Enterprise Linux är det självklara valet för arbetsbelastningar på virtuella datorer för virtuella datorer på Microsoft Azure. Red Hat Enterprise Linux är en öppen, tillförlitlig och säker plattform som utformats för kunder som vill ha distributionsflexibilitet för sina affärskritiska arbetsbelastningar – från datacenter till Azure-molnet – med hjälp av tätt integrerat support i företagsklass från Red Hat och Microsoft.

[Gå med i Red Hat Enterprise Linux 7.2 VM till Azure AD Domain Services](active-directory-ds-admin-guide-join-rhel-linux-vm.md)
