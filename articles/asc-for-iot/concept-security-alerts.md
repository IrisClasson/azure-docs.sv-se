# <a name="azure-security-center-for-iot-security-alerts"></a>Azure Security Center for IoT-säkerhetsaviseringar

Azure Security Center for IoT analyserar kontinuerligt din IoT-lösning med hjälp av avancerad analys och hotinformation för att varna dig för skadlig aktivitet.
Dessutom kan du skapa anpassade aviseringar baserat på dina kunskaper om förväntat enhetsbeteende.
En varning fungerar som en indikator på eventuella kompromisser och bör undersökas och åtgärdas.

I den här artikeln hittar du en lista över inbyggda aviseringar som kan utlösas på din IoT Hub och/eller IoT-enheter.
Förutom inbyggda aviseringar kan du med Azure Security Center for IoT definiera anpassade aviseringar baserat på förväntat IoT Hub- och/eller enhetsbeteende.
Mer information finns i [anpassningsbara aviseringar](concept-customizable-security-alerts.md).
