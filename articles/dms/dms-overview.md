# <a name="what-is-azure-database-migration-service"></a>Vad är Azure Database Migration Service?

Azure Database Migration Service är en fullständigt hanterad tjänst som utformats för att möjliggöra sömlösa migreringar från flera databaskällor till Azure-dataplattformar med minimal driftstopp (onlinemigreringar).

## <a name="migrate-databases-to-azure-with-familiar-tools"></a>Migrera databaser till Azure med välbekanta verktyg

Azure Database Migration Service integrerar några av funktionerna i våra befintliga verktyg och tjänster. Det ger kunderna en omfattande, högtillgänglig lösning. Tjänsten använder [datamigreringsassistenten](https://aka.ms/dma) för att generera utvärderingsrapporter som ger rekommendationer som hjälper dig att följa de ändringar som krävs innan en migrering utförs. Det är upp till dig att utföra alla åtgärder som krävs. När du är redo att påbörja migreringsprocessen utför Azure Database Migration Service alla nödvändiga steg. Du kan tryggt starta och sedan glömma dina migreringsprojekt i vetskap om att processen sker enligt Microsofts bästa praxis.

> [!NOTE]
> För att använda Azure Database Migration Service för att utföra en onlinemigrering måste du skapa en instans baserat på premiumprisnivån.
