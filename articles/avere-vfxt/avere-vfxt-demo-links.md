# <a name="avere-vfxt-demo-projects"></a>Avere vFXT demoprojekt

Exempel på självstudier finns på [GitHub](https://github.com/Azure/Avere). Dessa små projekt demonstrerar viktiga funktioner och användningsfall för Avere vFXT för Azure.

## <a name="video-rendering"></a>Videoåtergivning

* [Rendering med Azure Batch och Avere vFXT](https://github.com/Azure/Avere/blob/master/docs/maya_azure_batch_avere_vfxt_demo.md) - Ett 60-minutersprojekt som visar hur du använder Autodesk Maya med Azure Batch och ett Avere vFXT-kluster för att generera en animerad film

* [Varför använda Avere vFXT för rendering?](https://github.com/Azure/Avere/blob/master/docs/why_avere_for_rendering.md) - En demonstration som jämför renderingstider från nätverksansluten lagring med och utan ett Avere vFXT-kluster

## <a name="high-performance-computing"></a>Högpresterande datoranvändning

* [Metodtips för att förbättra starttiden för Azure Virtual Machine (VM)](https://github.com/Azure/Avere/blob/master/docs/azure_vm_provision_best_practices.md) – Ett reproducerbart test som använder Avere vFXT för Azure för att förbättra starttider när tusentals beräkningsklienter snabbt snurras

## <a name="vfxt-performance"></a>vFXT-prestanda

* [Mät vFXT-prestanda med vdbench](https://github.com/Azure/Avere/blob/master/docs/vdbench.md) - En grundläggande testinställning för att generera små och medelstora arbetsbelastningar för att testa vFXT-minne och diskundersystem

## <a name="client-setup"></a>Klientinstallation

* [Windows 10 arbetsstation för Avere vFXT](https://github.com/Azure/Avere/blob/master/docs/windows_10_avere_vfxt_mounted_workstation.md) - Visar hur du ställer in en Windows-arbetsstation och montera den på ett Avere vFXT-kluster
