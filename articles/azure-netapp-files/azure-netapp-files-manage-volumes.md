# <a name="manage-volumes-by-using-azure-netapp-files"></a>Hantera volymer med hjälp av Azure NetApp Files
Du kan hantera en volym genom att visa, uppdatera eller ta bort volymen. 

Om du vill ta bort en volym måste du först ta bort alla under objekt till en volym. 

1.  Klicka på bladet **Volymer** via bladet Hantera kapacitetspooler. 
2.  Använd vyn **Bläddra bland volymer** för att visa, uppdatera eller ta bort befintliga volymer.
