# <a name="azure-netapp-files-videos"></a>Videor för Azure NetApp-filer
Den här artikeln innehåller referenser till videor som innehåller djupgående diskussioner om hur du använder Azure NetApp-filer. 

Det finns flera videor som hjälper dig att lära dig mer om Azure NetApp-filer:  

* [Microsoft Ignite 2019: Kör dina mest krävande arbetsbelastningar för företagsfiler med Azure NetApp Files](https://myignite.techcommunity.microsoft.com/sessions/82938?source=sessions) ger en kort introduktion till Azure NetApp-filer, inklusive användningsfall och demo, och går sedan djupare på funktionerna och färdplanen.
* [Azure NetApp Files talks av Kirk Ryan](https://www.youtube.com/channel/UCq1jZkyVXqMsMSIvScBE2qg/playlists) är en serie videor, självstudier och demonstrationer som är dedikerade till Azure NetApp-filer.
