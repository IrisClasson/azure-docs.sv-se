# <a name="secure-hybrid-access-secure-legacy-apps-with-app-delivery-controllers-and-networks"></a>Säker hybridåtkomst: Säkra äldre appar med appleveransstyrenheter och nätverk

Du kan nu skydda dina lokala och molnåldriga autentiseringsprogram genom att ansluta dem till Azure AD med din befintliga programleveransstyrenhet eller nätverk. På så sätt kan du överbrygga klyftan och stärka din säkerhetsposition för alla program med Azure AD-funktioner som Azure AD Conditional Access och Azure AD Identity Protection.

Med hjälp av din befintliga nätverks- och leveransstyrenhet kan du enkelt skydda äldre program som fortfarande är viktiga för dina affärsprocesser men som du inte kunde skydda tidigare med Azure AD. Det är troligt att du redan har allt du behöver för att börja skydda dessa program.

* [Citrix Application Delivery Controller (ADC)](../saas-apps/citrix-netscaler-tutorial.md)
