# <a name="secure-hybrid-access-secure-legacy-apps-with-app-delivery-controllers-and-networks"></a>Säker hybrid åtkomst: säkra äldre appar med program leverans styrenheter och nätverk

Nu kan du skydda dina lokala och molnbaserade program för äldre autentisering genom att ansluta dem till Azure AD med din befintliga program leverans styrenhet eller ditt nätverk. På så sätt kan du överbrygga luckan och förstärka din säkerhets position över alla program med Azure AD-funktioner som Azure AD villkorlig åtkomst och Azure AD Identity Protection.

Med hjälp av din befintliga nätverks-och leverans kontroll kan du enkelt skydda äldre program som fortfarande är kritiska för dina affärs processer, men som du inte kunde skydda innan Azure AD. Det är troligt att du redan har allt du behöver för att börja skydda programmen.

![Bild som visar säker hybrid åtkomst](media/secure-hybrid-access/secure-hybrid-access.png)

* [Citrix Application Delivery Controller (ADC)](../saas-apps/citrix-netscaler-tutorial.md)
* [F5 Big-IP APM](https://aka.ms/f5-hybridaccessguide)
