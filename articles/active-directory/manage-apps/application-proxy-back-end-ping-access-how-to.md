# <a name="how-to-configure-an-application-proxy-application-to-use-pingaccess"></a>Konfigurera ett programproxyprogram för att använda PingAccess

Vårt samarbete med PingAccess gör det nu möjligt att utöka fördelarna med Application Proxy till program med hjälp av rubrikbaserad autentisering. Om dina program inte använder rubriker läser du vår dokumentation för enkel [inloggning](application-proxy-configure-single-sign-on-with-kcd.md) för mer information om andra alternativ.

## <a name="overview-of-steps-and-recommended-documents"></a>Översikt över steg och rekommenderade dokument

Om du vill konfigurera ett program med PingAccess finns det fyra steg:

1.  Konfigurera proxyanslutningar för program

2.  Skapa ett Azure AD-programproxyprogram

3.  Ladda ned & Konfigurera PingAccess

4.  Konfigurera program i PingAccess

Mer information om vart och ett av dessa steg finns i [dokumentationen för enkel inloggning med rubriker](application-proxy-configure-single-sign-on-with-ping-access.md).
