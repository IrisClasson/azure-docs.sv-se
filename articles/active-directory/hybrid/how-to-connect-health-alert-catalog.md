# <a name="azure-active-directory-connect-health-alert-catalog"></a>Hälsovarningskatalog för Azure Active Directory Connect 

Azure AD Connect Health-tjänst skicka aviseringar visar att din identitetsinfrastruktur inte är felfri. Den här artikeln innehåller aviseringar titlar, beskrivningar och reparationssteg för varje avisering. <br />
Fel, Varning och Förvarning är tre steg med aviseringar som genereras från Connect Health-tjänsten. Vi rekommenderar starkt att du vidtar omedelbara åtgärder vid utlösta aviseringar. <br />
Azure AD Connect Health-aviseringar löses på ett lyckat villkor. Azure AD Connect Health Agents identifiera och rapportera framgångsvillkoren till tjänsten med jämna mellanrum. För några få aviseringar är dämpningen tidsbaserad. Med andra ord, om samma feltillstånd inte observeras inom 72 timmar från aviseringsgenereringen, löses aviseringen automatiskt.

De senaste data som behandlas av hälso- och sjukvården är äldre än 2 timmar. | Kontrollera att hälsoagenterna har utgående anslutning till de nödvändiga tjänstslutpunkterna.
