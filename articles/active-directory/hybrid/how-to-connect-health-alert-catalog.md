# <a name="azure-active-directory-connect-health-alert-catalog"></a>Azure Active Directory Connect Health-Åtgärdskatalog 

Azure AD Connect Health-tjänsten skicka varningar indikera att din infrastruktur för Identitetshantering inte är felfri. Den här artikeln innehåller aviseringar rubriker, beskrivningar och steg för varje avisering. <br />
Fel, varning och Prewarning finns tre steg av aviseringar som genereras från Connect Health-tjänsten. Vi rekommenderar starkt att du vidta direkta åtgärder på utlösta aviseringar. <br />
Azure AD Connect Health-aviseringar supportärendena går att på ett lyckat tillstånd. Azure AD Connect Health-agenterna identifiera och rapportera lyckades villkor till tjänsten med jämna mellanrum. För några aviseringar är Undertryckning tidsbaserade. Om samma fel inte observeras inom 72 timmar från genereringen, matchas med andra ord automatiskt aviseringen.
