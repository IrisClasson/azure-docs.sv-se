# <a name="sync-service-manager-metaverse-search"></a>Synkroniseringshanteraren för tjänsten Metaversumsökning

![Sync Service Manager](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

Fliken Sök metaversum är användbart för felsökning av problem i data som är relaterade. Du kan skapa en fråga som baseras på en kombination av attribut i den övre delen. När du är nöjd med din fråga klickar du på **Search**. Resultatet visas i rutnätet längst ned. Du kan välja vilka kolumner som ska visas med **kolumninställningar**.

I sökresultaten väljer du ett objekt och **egenskaper** att se den [metaversumobjektegenskaperna](tshoot-connect-object-not-syncing.md#metaverse-object-properties).
