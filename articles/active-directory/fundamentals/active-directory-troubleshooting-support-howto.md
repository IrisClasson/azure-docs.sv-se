# <a name="find-help-and-open-a-support-ticket-for-azure-active-directory"></a>Hitta hjälp och öppna en supportbiljett för Azure Active Directory
Microsoft tillhandahåller global teknisk support, pre-sales, billing och prenumeration för Azure Active Directory (Azure AD). Support är tillgänglig både online och per telefon för Microsoft Azure-betalda prenumerationer och utvärderingsprenumerationer. Telefonsupport och faktureringssupport online är tillgänglig på fler språk. 

## <a name="find-help-without-opening-a-support-ticket"></a>Få hjälp utan att skapa någon supportbegäran

Innan du skapar en supportbegäran kan du gå igenom följande resurser och leta efter svar och information. 

* Innehåll som information eller kodexempel för IT-proffs och utvecklare finns i den [tekniska dokumentationen på docs.microsoft.com](https://docs.microsoft.com/azure/active-directory/).

* [Microsoft Technical Community](https://techcommunity.microsoft.com/) är platsen för våra IT-proffspartner och kunder att samarbeta, dela och lära. [Microsoft Technical Community Info Center](https://techcommunity.microsoft.com/t5/Community-Info-Center/ct-p/Community-Info-Center) används för meddelanden, blogginlägg, ask-me-anything (AMA) interaktioner med experter och mycket mer. Du kan också [gå med i gruppen för att skicka in dina idéer](https://techcommunity.microsoft.com/t5/Communities/ct-p/communities).


## <a name="open-a-support-ticket"></a>Öppna en supportbegäran

Om du inte kan hitta svar med hjälp av självhjälpsresurser kan du öppna en supportbiljett online. Du bör öppna varje supportbiljett för endast ett enda problem, så att vi kan ansluta dig till supporttekniker som är ämnesexperter för ditt problem. Azure Active Directory-teknikteam prioriterar också sitt arbete baserat på incidenter som genereras, så du bidrar ofta till serviceförbättringar.

### <a name="how-to-open-a-support-ticket-for-azure-ad-in-the-azure-portal"></a>Så här öppnar du en supportbiljett för Azure AD i Azure-portalen

> [!NOTE]
> För fakturerings- eller prenumerationsproblem måste du använda [administrationscentret för Microsoft 365](https://admin.microsoft.com).
> 

1. Logga in [på Azure-portalen](https://portal.azure.com) och öppna **Azure Active Directory**.
   
2. Bläddra ned till **Felsökning + Support** och välj Ny **supportbegäran**.
   
3. Välj **Teknisk**för **ärendetyp**på bladet **Grunderna.**

4. Välj din **prenumeration**.

5. För **Tjänst**väljer du **Azure Active Directory**.

6. Skapa en **sammanfattning** för begäran. Sammanfattningen måste vara under 140 tecken.
  
7. Välj en **problemtyp**och välj sedan en kategori för den typen. Nu erbjuds du också självhjälpsinformation för din problemkategori.
  
7. Lägg till resten av probleminformationen och klicka på **Nästa**. 

8. Nu erbjuds du självhjälpslösningar och dokumentation i **lösningsbladet.** Om ingen av lösningarna där löser problemet klickar du på **Nästa**. 

9. Fyll i de uppgifter som krävs och välj en [allvarlighetsgrad](https://azure.microsoft.com/support/plans/response/)på bladet **Detaljer** . 
 
![image](https://user-images.githubusercontent.com/13383753/76565580-1c284900-6468-11ea-8c0f-85af98097b6f.png)
 
10. Ange dina kontaktuppgifter och välj **Nästa**. 


8. Ange dina kontaktuppgifter och välj **Skapa**.
  ![Problemkategori självhjälp skärmdump](./media/active-directory-troubleshooting-support-howto/open-support-ticket.png)

### <a name="how-to-open-a-support-ticket-for-azure-ad-in-the-microsoft-365-admin-center"></a>Så här öppnar du en supportbiljett för Azure AD i administrationscentret för Microsoft 365

> [!NOTE]
> Stöd för Azure AD i [Microsoft 365-administrationscentret](https://admin.microsoft.com) erbjuds endast för administratörer. 

1. Logga in på [Microsoft 365-administrationscentret](https://admin.microsoft.com) med ett konto som har en EMS-licens (Enterprise Mobility + Security).

2. Välj **Ny tjänstbegäran** **på** supportpanelen:

3. På sidan **Supportöversikt** väljer du **Identitetshantering** eller **Användar- och domänhantering:**

4. För **Funktion**väljer du den Azure AD-funktion som du vill ha support för.

5. För **Symptom**väljer du ett lämpligt symptom, sammanfattar problemet och anger relevant information och väljer sedan **Nästa**.

6. Välj en av de erbjudna självhjälpsresurserna eller välj **Ja, fortsätt** eller **Nej, avbryt begäran**.

7. Om du fortsätter blir du tillfrågad om mer information. Du kan bifoga alla filer som du har som representerar problemet och sedan välja **Nästa**.

8. Ange dina kontaktuppgifter och välj **Skicka begäran**.
