# <a name="my-profile-preview-portal-overview"></a>Översikt över portalen Min profil (förhandsversion)

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

Du kan använda portalen **Min profil** (förhandsversion) till att hantera ditt arbets- eller skolkonto konto för att:

- Konfigurera och hantera din säkerhetsinformation för tvåstegsverifiering och återställning av lösenord.

- Hantera dina anslutna organisationer och enheter.

- Visa hur din organisation använder dina data.

>[!Important]
>Innehållet är avsett för användare. Om du är administratör hittar du mer information om hur du konfigurerar och hanterar en Azure Active Directory (Azure AD)-miljö i [dokumentationen om Azure Active Directory](https://docs.microsoft.com/azure/active-directory).

## <a name="access-the-my-profile-portal"></a>Öppna portalen Min profil
Du öppnar portalen **Min profil** genom att gå till https://myprofile.microsoft.com från den aktuella versionen av någon av följande webbläsare:

- Chrome
- Microsoft Edge
- Safari
- Firefox
- Internet Explorer 11

    ![Översiktssida för portalen Min profil](media/myprofile-portal/myprofile-portal-overview.png)
