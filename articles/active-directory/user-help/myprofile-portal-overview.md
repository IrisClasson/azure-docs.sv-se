# <a name="my-profile-preview-portal-overview"></a>Översikt över portalen Min profil (förhandsversion)

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

Den **min profil** portal (förhandsversion) hjälper dig att hantera ditt arbets- eller skolkonto konto genom att konfigurera och hantera din säkerhetsinformation, hantera anslutna organisationer och enheter och visa hur din organisation använder dina data.

>[!Important]
>Innehållet är avsett för användare. Om du är administratör hittar du mer information om hur du konfigurerar och hanterar en Azure Active Directory (Azure AD)-miljö i [dokumentationen om Azure Active Directory](https://docs.microsoft.com/azure/active-directory).

![Översiktssida för portalen Min profil](media/myprofile-portal/myprofile-portal-overview.png)

## <a name="go-to-the-my-profile-portal"></a>Gå till portalen för min profil

Du får den **min profil** portal (https://myprofile.microsoft.com), från den aktuella versionen av någon av följande webbläsare:

- Chrome
- Microsoft Edge
- Safari
- Firefox
- Internet Explorer 11

## <a name="my-profile-portal-articles"></a>Min profil portal artiklar

Detaljerad information om och vad som är tillgängligt från den **min profil** -portalen finns i följande artiklar:

|Artikel |Beskrivning |
|------|------------|
|[Lägg till säkerhetsinformation](user-help-security-info-overview.md)| Flera artiklar med instruktioner om hur du konfigurerar Microsoft Authenticator-appen, SMS eller telefonsamtal som din verifieringsmetod. Innehåller också artiklar om hur du lägger till en e-postadress eller fördefinierade säkerhetsfrågor så att du kan återställa ditt eget glömda lösenord om det behövs.|
|[Visa eller lämna din anslutna organisationer](myprofile-portal-organizations-page.md)| Instruktioner om hur du visar och lämna organisationer ansluten till arbets-eller skolkonto.|
|[Visa eller inaktivera dina anslutna enheter](myprofile-portal-devices-page.md)| Instruktioner om hur du visar eller inaktivera alla enheter som du har anslutit till arbets-eller skolkonto.|
|[Visa din sekretessrelaterade information](myprofile-portal-privacy-page.md)| Instruktioner för hur att visa vilka onlinetjänster är anslutna till ditt arbets- eller skolkonto, tillsammans med din organisations användningsvillkor använda instruktionen.|
