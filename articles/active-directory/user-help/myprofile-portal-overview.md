# <a name="my-profile-preview-portal-overview"></a>Översikt över portalen Min profil (förhandsversion)

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

Portalen **min profil** (för hands version) hjälper dig att hantera ditt arbets-eller skol konto genom att konfigurera och hantera din säkerhets information, hantera dina anslutna organisationer och enheter och se hur din organisation använder dina data.

>[!Important]
>Innehållet är avsett för användare. Om du är administratör hittar du mer information om hur du konfigurerar och hanterar en Azure Active Directory (Azure AD)-miljö i [dokumentationen om Azure Active Directory](https://docs.microsoft.com/azure/active-directory).

![Översiktssida för portalen Min profil](media/myprofile-portal/myprofile-portal-overview.png)

## <a name="go-to-the-my-profile-portal"></a>Gå till min profil Portal

Du kan gå till **min profil** portal (https://myprofile.microsoft.com) från den aktuella versionen av någon av följande webbläsare:

- Chrome
- Microsoft Edge
- Safari
- Firefox
- Internet Explorer 11

## <a name="my-profile-portal-articles"></a>Mina profil portal artiklar

Detaljerad information och anvisningar om vad som är tillgängligt i **min profil** Portal finns i följande artiklar:

|Artikel |Beskrivning |
|------|------------|
| [Lägg till din säkerhets information](user-help-security-info-overview.md) | Flera artiklar med instruktioner om hur du ställer in Microsoft Authenticator appen, SMS-meddelanden eller telefonsamtal som din verifierings metod. Innehåller även artiklar om hur du lägger till en e-postadress eller fördefinierade säkerhets frågor så att du kan återställa ditt eget lösen ord vid behov.|
| [Visa eller lämna dina anslutna organisationer](myprofile-portal-organizations-page.md) | Anvisningar om hur du visar och lämnar organisationer som är anslutna till ditt arbets-eller skol konto.|
| [Visa eller inaktivera dina anslutna enheter](myprofile-portal-devices-page.md) | Anvisningar om hur du visar eller inaktiverar enheter som du har anslutit till ditt arbets-eller skol konto.|
| [Visa din sekretessrelaterade information](myprofile-portal-privacy-page.md) | Anvisningar om hur du visar vilka onlinetjänster som är anslutna till ditt arbets-eller skol konto, tillsammans med din organisations användnings villkor.|
