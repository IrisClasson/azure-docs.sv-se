# <a name="change-your-work-or-school-account-password-from-the-change-password-page"></a>Ändra lösenordet för ditt arbets- eller skolkonto från sidan Ändra lösenord

Sidan **Ändra lösenord** på portalen Mitt **konto** hjälper dig att uppdatera ett befintligt lösenord för ditt arbets- eller skolkonto, förutsatt att du kommer ihåg lösenordet och att du inte är utelåst från ditt konto. Om du inte kommer ihåg ditt lösenord, om du är utelåst från ditt konto eller om du aldrig fick ett lösenord från din organisation kan du använda din säkerhetsinformation och din mobila enhet för att återställa lösenordet.

>[!Important]
>Den här artikeln är avsedd för användare som försöker uppdatera ett känt lösenord för ett befintligt arbets- eller skolkonto. Om du är en användare som försöker komma in på ett personligt konto, till exempel för Xbox, Hotmail eller Outlook.com, kan du prova förslagen i artikeln [När du inte kan logga in på ditt Microsoft-konto.](https://support.microsoft.com/help/12429/microsoft-account-sign-in-cant) Om du är administratör som försöker hitta mer information om hur du testar återställning av lösenord med självbetjäning för anställda eller andra användare läser du [Återställning av lösenord för självbetjäning](https://docs.microsoft.com/azure/active-directory/authentication/quickstart-sspr).

## <a name="update-a-password-from-the-change-password-page"></a>Uppdatera ett lösenord från sidan Ändra lösenord

1. Logga in på ditt arbets- eller skolkonto med ditt befintligahttps://myprofile.microsoft.com/) lösenord och gå sedan till sidan Mitt **konto** (

2. Välj **Lösenord** i det vänstra navigeringsfönstret eller välj länken **Ändra lösenord** i blocket **Lösenord.**

    ![Sidan Mitt konto, som visar den markerade länken Ändra lösenord](media/my-account-portal/my-account-portal-change-password.png)

3. Skriv ditt gamla lösenord och skapa och bekräfta sedan ditt nya lösenord.

    ![Ändra lösenordssida, visa lösenordsfält](media/my-account-portal/my-account-portal-change-password-page.png)

4. Välj **Skicka**.

    Ditt lösenord ändras och du blir ombedd att logga in på ditt arbets- eller skolkonto igen.
