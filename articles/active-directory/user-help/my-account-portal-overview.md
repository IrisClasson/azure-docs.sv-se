# <a name="my-account-portal-overview"></a>Översikt över min konto Portal

Portalen **mitt konto** hjälper dig att hantera ditt arbets-eller skol konto genom att konfigurera och hantera din säkerhets information, hantera dina anslutna organisationer och enheter och se hur din organisation använder dina data.

>[!Important]
>Innehållet är avsett för användare. Om du är administratör hittar du mer information om hur du konfigurerar och hanterar en Azure Active Directory (Azure AD)-miljö i [dokumentationen om Azure Active Directory](https://docs.microsoft.com/azure/active-directory).

![Min konto Portal, översikts sida](media/my-account-portal/my-account-portal-overview.png)

## <a name="go-to-the-my-account-portal"></a>Gå till mitt konto Portal

Du kan gå till **mitt konto** portal (https://myprofile.microsoft.com)från den aktuella versionen av någon av följande webbläsare:

- Chrome
- Microsoft Edge
- Safari
- Firefox
- Internet Explorer 11
