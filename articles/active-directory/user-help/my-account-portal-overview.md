# <a name="what-is-the-my-account-portal"></a>Vad är portalen Mitt konto?

Portalen **Mitt konto** hjälper dig att hantera ditt arbets- eller skolkonto genom att konfigurera och hantera din säkerhetsinformation, hantera dina anslutna organisationer och enheter och visa hur din organisation använder dina data.

Du kan komma till portalen **Mitt konto** (https://myprofile.microsoft.com), från den aktuella versionen av någon av följande webbläsare:

- Chrome
- Microsoft Edge
- Safari
- Firefox
- Internet Explorer 11

![Min kontoportal, översiktssida](media/my-account-portal/my-account-portal-overview.png)

>[!Important]
>Den här artikeln är avsedd för användare som försöker komma till portalen Mitt konto för att uppdatera sin säkerhetsinformation, enhetsinformation, lösenord, anslutna organisationer, sekretess eller tidigare inloggningsinformation. Om du är administratör och letar efter information om hur du aktiverar autentisering och andra Azure Active Directory-funktioner (Azure AD) för dina anställda och andra användningsområden läser du [Azure AD-dokumentationen för administratörer](https://docs.microsoft.com/azure/active-directory/).
