# <a name="update-your-profile-and-account-information-on-the-my-apps-portal"></a>Uppdatera din profil- och kontoinformation på portalen Mina appar

Du kan använda ditt arbets- eller skolkonto med den webbaserade **Portalen Mina appar** för att 

- Visa och starta många av organisationens molnbaserade appar
- Uppdatera en del av din profil och kontoinformation
- Se information **om dina grupper**
- Gör **åtkomstgranskningar** för dina appar och grupper. 

Om du inte har tillgång till Portalen **Mina appar** måste du kontakta din helpdesk för att få behörighet.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-portal.md)]

> [!Important]
> Det här innehållet är avsett för **Användare av Mina appar.** Om du är administratör kan du hitta mer information om hur du konfigurerar och hanterar dina molnbaserade appar i [dokumentationen för programhantering](https://docs.microsoft.com/azure/active-directory/manage-apps).

## <a name="view-your-organization-related-profile-information"></a>Visa din organisationsrelaterade profilinformation

Beroende på vad din organisation har valt att inkludera på din **profilsida** på portalen **Mina appar** kan du se dina specifika arbetsrelaterade uppgifter, tillsammans med dina enheter och aktiviteter, och eventuella ytterligare organisationer som du tillhör.

### <a name="to-view-your-profile-information"></a>Så här visar du din profilinformation

1. Logga in på ditt arbets- eller skolkonto, [gå till portalen **Mina appar** ](my-apps-portal-end-user-access.md).

2. På sidan **Appar** väljer du din profilbild i det övre högra hörnet, där det står ditt namn och din organisation och väljer sedan **Profil**.

    **Profilsidan** visas som visar din profilinformation.

    ![Profilsida som visar all tillgänglig information](media/my-apps-portal/my-apps-portal-profile-page.png)

3. På **profilsidan** kan du:

    - **Granska din organisationsrelaterade information.** Visa foto, namn, titel, tillhörande e-postadresser och kontorsinformation. Den här informationen hanteras av din organisation och du kan inte ändra den. Om du ser ett fel kontaktar du helpdesk.

    - **Granska dina enheter & aktivitet**. Kontrollera att varje enhet är välbekant och korrekt ansluten till din organisation. Om du inte känner igen en enhet väljer du **Inaktivera enhet** för att ta bort kopplingen till ditt konto. När du har inaktiverat en enhet tas den bort från den här sidan.

    - **Granska dina organisationer.** Se till att du fortfarande arbetar med var och en av de angivna organisationerna. Om du inte längre arbetar med en organisation rekommenderar vi starkt att du väljer **Logga in för att lämna organisationen**. När du har lämnat organisationen tas den bort från den här sidan.

## <a name="manage-your-work-or-school-account-information"></a>Hantera information om ditt arbets- eller skolkonto

Uppdatera och hantera din information om arbets- eller skolkonto från **profilsidan** i portalen **Mina appar.** Den här sidan hjälper dig att:

- Ändra lösenordet för ditt arbets- eller skolkonto.

- Aktivera återställning av lösenord (om administratören har aktiverat den här funktionen).

- Ange ytterligare säkerhetsverifieringsinformation.

- Granska organisationens användarvillkor.

- Logga ut överallt.

## <a name="change-your-password"></a>Ändra ditt lösenord

Om du vill ändra lösenordet för ditt arbets- eller skolkonto kan du välja **Ändra lösenord** i området **Hantera konto** på sidan **Profil.**

### <a name="to-change-your-password"></a>Så här ändrar du ditt lösenord

1. Välj **Ändra lösenord**i området **Hantera konto** på sidan **Profil** .

2. På sidan **Ändra lösenord** kontrollerar du att ditt användar-ID är korrekt och skriver sedan in ditt gamla lösenord och ditt nya lösenord i rutorna.

    ![Ändra lösenordssida](media/my-apps-portal/my-apps-portal-change-password-page.png)

3. Välj **Skicka**.

    Ditt lösenord har ändrats. Du måste logga in på alla appar som du tidigare har loggat in på med ditt arbets- eller skolkonto.

## <a name="set-up-and-use-password-reset"></a>Konfigurera och använda återställning av lösenord

Om du har glömt ditt lösenord, aldrig fått något från företagets support eller har blivit utelåst från ditt konto kan du återställa ditt eget lösenord.

>[!Important]
>Administratören måste aktivera den här funktionen och du måste registrera dig för att delta. Mer information om hur du registrerar dig och hur du återställer lösenordet finns i [Registrera dig för återställning av lösenord för självbetjäning](active-directory-passwords-reset-register.md) och Återställ ditt [arbets- eller skollösenord](active-directory-passwords-update-your-own-password.md).

## <a name="change-your-security-verification-information"></a>Ändra din säkerhetsverifieringsinformation

Om din organisation kräver att du använder tvåfaktorsverifiering kan du lägga till, uppdatera och ta bort tillhörande säkerhetsinformation från sidan **Ytterligare säkerhetsverifiering.**

Tvåfaktorsverifiering kräver att du använder två verifieringsinformation, till exempel ett lösenord och en PIN-kod, innan du kan komma åt ditt konto eller till organisationens information. Mer information om tvåfaktorsverifiering finns i [Konfigurera mitt konto för tvåstegsverifiering](multi-factor-authentication-end-user-first-time.md).

### <a name="to-change-your-security-information"></a>Så här ändrar du din säkerhetsinformation

1. Välj **Ytterligare säkerhetsverifiering**i området **Hantera konto** på sidan **Profil** .

    ![Ytterligare sida för säkerhetsverifiering](media/my-apps-portal/my-apps-portal-additional-verification-page.png)

2. På sidan **Ytterligare säkerhetsverifiering** kan du lägga till, ändra eller ta bort följande information:

    - **Standardverifieringsalternativ.** Välj den sekundära standardmetod som ska användas för tvåfaktorsverifiering. Den här metoden används automatiskt när som helst tvåfaktorsverifiering krävs när du anger ditt användarnamn och lösenord.

    - **Lägg till, uppdatera eller ta bort verifieringsmetoder.** Du kan lägga till ny, uppdatera befintlig eller ta bort gammal information som inte längre gäller.

    - **Konfigurera Microsoft Authenticator-appen.** Du kan välja att konfigurera Microsoft Authenticator-appen så att den fungerar som verifieringsmetod. Mer information om Microsoft Authenticator-appen finns i [Vad är Microsoft Authenticator-appen?](user-help-auth-app-overview.md)

3. Välj **Spara** för att spara ändringarna.

## <a name="review-your-organizations-terms-of-use-statement"></a>Granska organisationens användningsvillkor

Du kan välja att granska organisationens användarvillkor, om den är tillgänglig.

1. Välj **Granska användningsvillkor**i området **Hantera konto** på sidan **Profil.**

2. Granska organisationens användarvillkor och välj **Acceptera** för att bekräfta att du har läst och förstått användarvillkoren för din organisation.

    ![Sidan Användningsvillkor från portalen Mina appar](media/my-apps-portal/my-apps-portal-tou-page.png)

    Om din organisation inte har några användningsvillkor kan du välja **Klar** för att gå tillbaka till **profilsidan.**

## <a name="sign-out-of-everywhere"></a>Logga ut överallt

Du kan välja att logga ut från överallt där du är inloggad på ditt arbets- eller skolkonto. Detta inkluderar alla appar och enheter.

### <a name="to-sign-out-of-everywhere"></a>Så här loggar du ut överallt

1. Välj **Logga ut överallt**i området **Hantera konto** på sidan **Profil.**

2. I bekräftelserutan **Logga ut överallt** väljer du **Ja** för att bekräfta att du vill logga ut från alla dina sessioner och enheter. Välj **Nej** om du ändrar dig.
