# <a name="view-your-recent-sign-in-activity-on-the-my-profile-preview-portal"></a>Visa dina senaste aktiviteter logga in på portalen min profil (förhandsversion)
Du kan visa alla dina senaste inloggning aktiviteter med ditt arbets- eller skolkonto konto från den **senaste aktivitet** för den **min profil** portal.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

## <a name="view-your-recent-sign-in-activity"></a>Visa dina senaste inloggningsaktivitet
1. Logga in på ditt arbets- eller skolkonto konto och gå sedan till din https://myprofile.microsoft.com/ sidan.

2. Välj **senaste aktivitet** från den vänstra navigeringsfönstret eller Välj den **granskningsaktivitet** länka från den **senaste aktivitet** block.

    ![Min profilsida som visar markerade länkar för senaste aktivitet](media/myprofile-portal/myprofile-portal-recent-activity-links.png)

3. Expandera och granska inloggning objekten, se till att du känner igen var och en. Om du hittar något logga in som ser inte bekant ut vi rekommenderar starkt att du ändrar ditt lösenord för att skydda ditt konto om det har komprometterats.

    ![Sida med senaste aktiviteter med utökade inloggningsinformation](media/myprofile-portal/myprofile-portal-recent-activity-page.png)

    >[!Note]
    >Om du tror att ditt konto har komprometterats bör se [hur du kommer åt ett komprometterat microsoftkonto](https://support.microsoft.com/help/10494/microsoft-account-how-to-access-a-compromised-account) för mer hjälp med att återställa lösenordet eller avblockera ditt konto.

## <a name="next-steps"></a>Nästa steg

- När du visar dina senaste aktiviteter inloggning kan du välja att visa eller hantera din [säkerhetsinformation](user-help-security-info-overview.md), anslutna [enheter](myprofile-portal-devices-page.md) eller [organisationer](myprofile-portal-organizations-page.md), eller visa hur din organisation [använder dina data](myprofile-portal-privacy-page.md).
