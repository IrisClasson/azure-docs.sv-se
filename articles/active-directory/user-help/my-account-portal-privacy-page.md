# <a name="view-how-your-organization-uses-your-privacy-related-data"></a>Visa hur din organisation använder dina sekretessrelaterade data

Du kan visa hur din organisation använder dina data från **sekretesssidan** på portalen **Mitt konto.**

## <a name="view-your-privacy-related-info"></a>Visa din sekretessrelaterade information

1. Logga in på ditt arbets- eller https://myprofile.microsoft.com/ skolkonto och gå sedan till din sida.

2. Välj **Sekretess** i det vänstra navigeringsfönstret eller välj **sekretesslänken Granska** i **sekretessblocket.**

    ![Sidan Mitt konto, som visar markerade sekretesslänkar](media/my-account-portal/my-account-portal-privacy.png)

3. Granska informationen, inklusive:

    - **Tjänster.** En lista över onlinetjänster som du är ansluten till med ditt arbets- eller skolkonto.

    - **Användarvillkor**. Organisationens användarvillkor.

## <a name="next-steps"></a>Nästa steg

När du har tittat på hur din organisation använder dina data kan du:

- Visa eller hantera [din säkerhetsinformation](user-help-security-info-overview.md).

- Visa eller hantera dina anslutna [enheter](my-account-portal-devices-page.md).

- Visa eller hantera dina [organisationer](my-account-portal-organizations-page.md).

- Visa din [inloggningsaktivitet](my-account-portal-sign-ins-page.md).

- Du kan också visa dina Kontaktinställningar och sekretessinformation för Microsoft Office från [office-portalen, sekretesssidan säkerhet &](https://portal.office.com/account/#security).
