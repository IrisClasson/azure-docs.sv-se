# <a name="overview-about-changing-your-security-info-preview-methods-and-information"></a>Översikt över hur du ändrar metoder och information för din säkerhetsinformation (förhandsversion)
Du kan lägga till, redigera, ta bort och ändra din standardmetod för säkerhetsinformation från sidorna **Säkerhetsinformation**. Mer information om varje metod och hur du ändrar informationen finns i respektive metodbaserad artikel.

## <a name="two-factor-verification-versus-password-reset-authentication"></a>Tvåfaktorsverifiering jämfört med autentisering med återställning av lösenord
Metoder för säkerhetsinformation används för både tvåfaktorsverifiering och för återställning av lösenord. Dock kan inte alla metoder användas för båda.

|Metod|Används för|
|------|--------|
|Autentiseringsapp|Tvåfaktorsverifiering och autentisering med återställning av lösenord.|
|SMS|Tvåfaktorsverifiering och autentisering med återställning av lösenord.|
|Telefonsamtal|Tvåfaktorsverifiering och autentisering med återställning av lösenord.|
|E-postkonto|Endast autentisering med återställning av lösenord. Du måste välja en annan metod för tvåfaktorsverifiering.|
|Säkerhetsfrågor|Endast autentisering med återställning av lösenord. Du måste välja en annan metod för tvåfaktorsverifiering.|
