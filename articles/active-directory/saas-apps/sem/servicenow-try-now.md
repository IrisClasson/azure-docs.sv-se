# <a name="servicenow-and-azure-active-directory"></a>ServiceNow och Azure Active Directory

Påskynda införandet av ServiceNow i din organisation och öka användar användningen med vår färdiga integrering. Effektivisera din Hantering av IT-tjänster (ITSM) genom att ansluta ServiceNow till Azure Active Directory (Azure AD).

![Visar ServiceNow app-logotyp](./media/servicenow.png)

> [!div class="nextstepaction"]
> [Prova ServiceNow med Azure AD >](https://aka.ms/ServiceNowTryItNow)

## <a name="what-can-you-do-with-servicenow-and-azure-ad"></a>Vad kan du göra med ServiceNow och Azure AD?

- **Enkel inloggning** – eliminera flera användar namn och lösen ord och förenkla åtkomst genom att tillåta att användare loggar in på ServiceNow med sina Azure AD-konton.
- **Automatisera användar etablering** – automatisera skapandet, underhåll och borttagning av användar konton i ServiceNow när användare ansluter, flyttar eller lämnar din organisation.
- **Skydda, styr och övervaka åtkomst** säker åtkomst till ServiceNow genom att tillämpa principer för villkorlig åtkomst, styra åtkomsten till granskningar och övervaka åtkomst med inloggnings loggar.

## <a name="get-started"></a>Kom igång

Konfigurera [användar etablering](https://aka.ms/ServiceNowProvisioning) och [enkel inloggning](https://aka.ms/ServiceNowSSO) för ServiceNow med våra enkla konfigurations guider.
