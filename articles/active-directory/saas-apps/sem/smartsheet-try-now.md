# <a name="smartsheet-and-azure-active-directory"></a>Smartsheet och Azure Active Directory

Påskynda införandet av Smartsheet i din organisation och öka användar användningen med vår färdiga integrering. Gör det möjligt för användare att samar beta snabbt genom att ansluta Smartsheet till Azure Active Directory (Azure AD).

![Visar Smartsheet app-logotyp](./media/smartsheet.png)

> [!div class="nextstepaction"]
> [Prova Smartsheet med Azure AD >](https://aka.ms/SmartsheetTryItNow)

## <a name="what-can-you-do-with-smartsheet-and-azure-ad"></a>Vad kan du göra med Smartsheet och Azure AD?

- **Enkel inloggning** – eliminera flera användar namn och lösen ord och förenkla åtkomst genom att tillåta att användare loggar in på Smartsheet med sina Azure AD-konton.
- **Automatisera användar etablering** – automatisera skapandet, underhåll och borttagning av användar konton i Smartsheet när användare ansluter, flyttar eller lämnar din organisation.
- **Skydda, styr och övervaka åtkomst** säker åtkomst till Smartsheet genom att tillämpa principer för villkorlig åtkomst, styra åtkomsten till granskningar och övervaka åtkomst med inloggnings loggar.

## <a name="get-started"></a>Kom igång

Konfigurera [användar etablering](https://aka.ms/SmartsheetProvisioningDoc) och [enkel inloggning](https://aka.ms/SmartsheetSSODoc) för Smartsheet med våra enkla konfigurations guider.
