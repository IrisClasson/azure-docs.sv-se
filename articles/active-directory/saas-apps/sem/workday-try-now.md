# <a name="workday-and-azure-active-directory"></a>Workday och Azure Active Directory

Anslut Workday med Azure Active Directory (Azure AD) med hjälp av vår fördefinierade integrering och etablera eller avetablera användar konton automatiskt när anställda ansluter, flyttas eller lämnar.

![Visar logo typen för Workday-appen](./media/workday.png)

> [!div class="nextstepaction"]
> [Prova Workday med Azure AD >](https://aka.ms/WorkdayTryItNow)

## <a name="what-can-you-do-with-workday-and-azure-ad"></a>Vad kan du göra med Workday och Azure AD?

- **Automatisera användar etablering** – tilldela automatiskt program till medarbetare baserat på deras grupp, roll eller andra attribut i arbets dagen.
- **Synkronisera personal profiler** – se till att medarbetar data hålls uppdaterade mellan Azure AD och Workday.
- **Förenkla onboarding-och offboarding** – etablera användar konton automatiskt för nya anställningar så att de kan vara produktiva på dag ett och minska säkerhets riskerna genom att automatiskt etablera användar konton när de anställda lämnar.
- **Enkel inloggning** Ta bort användar namn och lösen ord och Tillåt användare att logga in med sina organisations konton som finns i Azure AD.

## <a name="get-started"></a>Kom igång

Konfigurera [användar etablering](https://aka.ms/WorkdayProvisioning) och [enkel inloggning](https://aka.ms/WorkdaySSODoc) för arbets dagar med våra enkla konfigurations guider.
