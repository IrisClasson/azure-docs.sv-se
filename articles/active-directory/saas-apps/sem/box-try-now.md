# <a name="box-and-azure-active-directory"></a>Box och Azure Active Directory

Påskynda införandet av box i organisationen och öka användar användningen med vår färdiga integrering. Gör det möjligt för användare att samar beta snabbt genom att ansluta rutan till Azure Active Directory (Azure AD).

![Visar Box app-logotypen](./media/box.png)

> [!div class="nextstepaction"]
> [Testa box med Azure AD >](https://aka.ms/BoxTryItNow)

## <a name="what-can-you-do-with-box-and-azure-ad"></a>Vad kan du göra med box och Azure AD?

- **Enkel inloggning** – eliminera flera användar namn och lösen ord och förenkla åtkomst genom att tillåta att användare loggar in på box med sina Azure AD-konton.
- **Automatisera användar etablering** – automatisera genereringen, underhållet och borttagningen av användar konton i rutan när användare ansluter, flyttar eller lämnar din organisation.
- **Skydda, styr och övervaka åtkomst** säker åtkomst till rutan genom att tillämpa principer för villkorlig åtkomst, styra åtkomsten till granskningar och övervaka åtkomst med inloggnings loggar.

## <a name="get-started"></a>Kom igång

Konfigurera [användar etablering](https://aka.ms/BoxSSODoc) och [enkel inloggning](https://aka.ms/BoxProvisioningDoc) för box med våra enkla konfigurations guider.
