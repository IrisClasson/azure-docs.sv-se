# <a name="grant-permissions-to-users-from-partner-organizations-in-your-azure-active-directory-tenant"></a>Bevilja behörigheter till användare från partnerorganisationer i din Azure Active Directory-klient

Azure Active Directory (Azure AD) B2B-samarbetsanvändare läggs till som gästanvändare i katalogen och gästbehörigheter i katalogen begränsas som standard. Ditt företag kan behöva vissa gästanvändare för att fylla roller med högre behörighet i organisationen. För att stödja definitionen av roller med högre behörighet kan gästanvändare läggas till i alla roller du önskar, baserat på organisationens behov.
