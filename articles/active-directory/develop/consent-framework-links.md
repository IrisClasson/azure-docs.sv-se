# <a name="how-application-consent-works"></a>Så här fungerar samtycke till program

Den här artikeln är att hjälpa dig att lära dig mer om hur Azure AD-medgivande ramverket fungerar så att du kan utveckla program mer effektivt.

## <a name="recommended-documents"></a>Rekommenderade dokument

- Få en allmän förståelse för [hur samtycke tillåter en resursägare att styra ett programs åtkomst till resurser](https://docs.microsoft.com/azure/active-directory/develop/active-directory-dev-glossary#consent).
- Få en stegvis översikt över [hur Azure AD-medgivanderamverket implementerar samtycke](https://docs.microsoft.com/azure/active-directory/develop/active-directory-integrating-applications).
- Mer information finns i [hur ett program med flera innehavare kan använda medgivanderamverket](https://docs.microsoft.com/azure/active-directory/develop/active-directory-devhowto-multi-tenant-overview) för att implementera "användar- och administratörstillstånd", vilket stöder mer avancerade programmönster på flera nivåer.
- Mer djup kan du läsa [om hur medgivande stöds på OAuth 2.0-protokolllagret under tilldelningsflödet för auktoriseringskod.](https://docs.microsoft.com/azure/active-directory/develop/active-directory-protocols-oauth-code#request-an-authorization-code)
