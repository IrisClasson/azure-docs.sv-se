# <a name="discover-and-use-sample-notebooks-in-azure-notebooks-preview"></a>Upptäck och använda exempeldatorböcker i förhandsversionen av Azure Notebooks

Internet har många samlingar av intressanta och användbara Jupyter-anteckningsböcker som du enkelt kan ta med i Azure Notebooks. Den här artikeln innehåller en lista över anmärkningsvärda exempel.

[!INCLUDE [notebooks-status](../../includes/notebooks-status.md)]

När du har identifierat en anteckningsbok som du vill utforska kan du förstå några detaljer som du vill förstå:

- Om en anteckningsbok redan finns i Azure Notebooks använder du knappen **Klona** på projektinstrumentpanelen för att göra en kopia av projektet i ditt eget konto.
- Vissa anteckningsböcker kräver specifika paket för att köras. Även om `!pip install` du kan `install.packages` använda (Python) eller (R) för att installera paket från en bärbar dator själv, kan du också [konfigurera ett Azure Notebooks-projekt med rätt miljö](configure-manage-azure-notebooks-projects.md).

## <a name="great-sample-notebooks"></a>Bra exempel anteckningsböcker

- [Startuppsättningen för Azure Notebooks](https://notebooks.azure.com/#sample-redirect) innehåller introduktioner till Python, R och F#, tillsammans med anteckningsböcker som visar dataåtkomst, Azure Machine Learning och några datavetenskapsövningar.
- [Introduktion till Jupyter anteckningsböcker](https://nbviewer.jupyter.org/github/jupyter/notebook/blob/master/docs/source/examples/Notebook/Notebook%20Basics.ipynb) (jupyter.org)
- [Introduktion till datavetenskap](https://github.com/jakevdp/PythonDataScienceHandbook/tree/master/notebooks) av Jake Vanderplas.
- [Galleri av intressanta anteckningsböcker](https://github.com/ipython/ipython/wiki/A-gallery-of-interesting-IPython-Notebooks) (Jupyter projekt på GitHub)
- [nbviewer.jupyter.org](https://nbviewer.jupyter.org)
- Fysikinstruktion från [fysikavdelningen vid Imperial College of London](https://notebooks.azure.com/cvanbreu/libraries)
- [Azure Machine Learning komma igång-exempel](https://notebooks.azure.com/azureml/projects/azureml-getting-started)
