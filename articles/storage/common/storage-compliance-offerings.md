# <a name="azure-storage-compliance-offerings"></a>Azure Storage-efterlevnadserbjudanden

För att hjälpa organisationer att uppfylla nationella, regionala och branschspecifika krav som styr insamling och användning av enskilda personers data, erbjuder Microsoft Azure & Azure Storage den mest omfattande uppsättningen certifieringar och intyg för alla molntjänstleverantörer.

Nedan följer du efterlevnadserbjudanden på Azure Storage för att säkerställa att din tjänst regleras med hjälp av Azure Storage-tjänsten. De gäller för följande Azure Storage-erbjudanden: Blobbar, Filer, Köer, Tabeller, Diskar, Cool Storage och Premium Storage.
