# <a name="azure-active-directory-b2c-partners"></a>Azure Active Directory B2C partner

Vårt partner nätverk utökar våra lösnings funktioner som hjälper dig att bygga sömlösa slut användar upplevelser. Med Azure AD B2C kan du integrera med partners för att aktivera Multi-Factor Authentication-metoder, Aktivera säker kundautentisering (SCA), utföra rollbaserad Access Control och bekämpa bedrägerier genom identitets verifiering och korrektur. Använd våra detaljerade genom gångar för att lära dig hur du integrerar appar med de partner som anges nedan.

>[!NOTE]
>[Azure Active Directory B2C Community-webbplatsen på GitHub](https://azure-ad-b2c.github.io/azureadb2ccommunity.io/) innehåller också exempel på anpassade principer från communityn.
