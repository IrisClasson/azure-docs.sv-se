# <a name="solutions-and-training-for-azure-active-directory-b2c"></a>Lösningar och utbildning för Azure Active Directory B2C

Azure Active Directory B2C (Azure AD B2C) gör det möjligt för organisationer att implementera affärslösningar som hjälper dem att få kontakt med sina kunder. Följande lösningsguider och utbildning är nedladdningsbara dokument som kommer att gå igenom dessa lösningar.

| [Fördelar med att använda Azure AD B2C](https://aka.ms/b2coverview) | Förstå fördelarna och de vanliga scenarierna med Azure AD B2C och hur dina program kan utnyttja den här CIAM-tjänsten. |
| [Få expertis inom Azure AD B2C: En kurs för utvecklare](https://aka.ms/learnAADB2C) | Den här end-to-end-kursen tar utvecklare genom en fullständig resa på att utveckla program med Azure AD B2C som autentiseringsmekanism. Tio djupgående moduler med labb täcker allt från att konfigurera en Azure-prenumeration till att skapa anpassade principer som definierar de resor som engagerar dina kunder. |
| [Göra det möjligt för partner, leverantörer och kunder att komma åt program med Azure active Directory](https://aka.ms/aadexternalidentities) | Varje organisations framgång, oavsett storlek, bransch eller efterlevnad och säkerhetsposition, bygger på organisatorisk förmåga att samarbeta med andra organisationer och få kontakt med kunder.<br><br>Den här guiden samlar Azure AD, Azure AD B2C och Azure AD B2B Collaboration och beskriver affärsvärdet och mekaniken för att skapa ett program eller en webbupplevelse som ger en konsoliderad autentiseringsupplevelse som är anpassad till kontexterna för din anställda, affärspartners och leverantörer samt kunder. |
| [Migrera programautentisering till Azure AD B2C i en hybridmiljö](https://aka.ms/MigratetoAADB2C) | I dagens moderna organisationer sker digital omvandling och flytt till molnet i etapper, vilket kräver att de flesta organisationer åtminstone tillfälligt arbetar i en hybrididentitetsmiljö.<br><br>Den här guiden fokuserar på att skapa migreringsplanen för att flytta ditt första program till Azure AD B2C och täcker övervägandena för att göra det i en hybrididentitetsmiljö. |
