# <a name="service-bus-messaging-samples"></a>Meddelandetjänsten Service Bus-exempel

Meddelandetjänsten Service Bus-exempel demonstrera viktiga funktioner i [Service Bus-meddelanden](https://azure.microsoft.com/services/service-bus/). För närvarande kan hittar du exemplen på två platser:

- [.NET och Java](https://github.com/Azure/azure-service-bus/tree/master/samples) . Se den [.NET readme](https://github.com/Azure/azure-service-bus/blob/master/samples/DotNet/Microsoft.ServiceBus.Messaging/README.md) i lagringsplatsen för beskrivningar av .NET-exempel och [Java readme](https://github.com/Azure/azure-service-bus/tree/master/samples/Java) beskrivningar av Java-exempel.
- [Typescript](https://github.com/Azure/azure-service-bus-node/tree/master/examples)

## <a name="service-bus-explorer"></a>Service Bus Explorer

Dessutom kan den [Service Bus Explorer](https://github.com/paolosalvatori/ServiceBusExplorer) är ett exempel på GitHub där du kan ansluta till ett Service Bus-namnområde för tjänsten och enkelt hantera meddelandeentiteter. Verktyget ger avancerade funktioner, till exempel import/export- och möjlighet att testa meddelandeentiteter och tjänster för vidarebefordran. Du hittar den fullständiga Service Bus Explorer käll- och dokumentation på [GitHub](https://github.com/paolosalvatori/ServiceBusExplorer).
