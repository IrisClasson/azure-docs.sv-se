# <a name="security-recommendations---a-reference-guide"></a>Säkerhetsrekommendationer - en referensguide

I den här artikeln visas de rekommendationer som kan visas i Azure Security Center. Rekommendationerna som visas i din miljö beror på vilka resurser du skyddar och din anpassade konfiguration.

Mer information om hur du svarar på dessa rekommendationer finns [i Åtgärda rekommendationer i Azure Security Center](security-center-remediate-recommendations.md).

Ditt säkra resultat baseras på antalet rekommendationer för Säkerhetscenter som du har slutfört. Om du vill bestämma vilka rekommendationer som ska matchas först kan du titta på allvarlighetsgraden för var och en och dess potentiella inverkan på ditt säkra resultat.

>[!TIP]
> Om en rekommendationsbeskrivning säger "Ingen relaterad princip" beror det oftast på att rekommendationen är beroende av en annan rekommendation och *dess* policy. Rekommendationen "Hälsofel för slutpunktsskydd bör till exempel åtgärdas...", bygger på rekommendationen att en slutpunktsskyddslösning ens *är installerad* ("Slutpunktsskyddslösning bör installeras..."). Den underliggande *rekommendationen* har en policy. Om du begränsar policyerna till att endast den grundläggande rekommendationen förenklas policyhanteringen.
