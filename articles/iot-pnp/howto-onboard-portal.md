# <a name="onboard-to-the-azure-certified-for-iot-portal"></a>Registrera i Azure Certified för IoT-portalen

Du kan använda [Azure-certifierad för IoT-portalen](https://aka.ms/ACFI) för att:

- Slutför certifierings processen för din IoT-enhet.
- Hitta IoT Plug and Play för hands versions modeller av enhets funktioner. Du kan använda dessa modeller för att snabbt bygga IoT Ready-enheter och integrera dem med lösningar.
- Publicera en enhets kapacitets modell på den offentliga lagrings platsen.

## <a name="sign-in"></a>Logga in

Använd ditt Microsoft _Work-konto_ för att logga in på [Azure-certifierad för IoT Portal](https://aka.ms/ACFI) -portalen. Portalen kontrollerar ditt medlemskap med Microsoft Partner Center när du loggar in:

- Om din organisation redan är medlem i [Microsoft Partner Center](https://docs.microsoft.com/partner-center/)beviljas du åtkomst till alla funktioner i portalen.
- Om din organisation inte är medlem i [Microsoft Partner Center](https://docs.microsoft.com/partner-center/)kan du köra certifierings test och bläddra i databaserna för enhets kapacitets modellen. Du kan inte publicera din enhet i den IoT Certified Device-katalogen.
- Om du ser ett meddelande om att ditt Microsoft Partner Network-ID saknas bör du [skapa ett partner Center-konto](https://docs.microsoft.com/partner-center/mpn-create-a-partner-center-account). När ditt konto har godkänts kan du gå tillbaka till portalen och publicera enheten. Mer information finns i [vanliga frågor och svar om Partner Center](https://support.microsoft.com/help/4340639/partner-center-account-faqs).

Innan du publicerar enheten i IoT Certified Device-katalogen läser du och godkänner program avtalet. Du hittar avtalet på sidan Inställningar i Azure-certifierad för IoT-portalen.
