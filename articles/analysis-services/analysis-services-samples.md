# <a name="azure-analysis-services-samples"></a>Exempel på Azure Analysis Services

Använd följande exempelresurser för att lära dig mer om och testa Analysis Services för din miljö.

## <a name="code-samples"></a>Kodexempel

[Analysis Services-databasen](https://github.com/Microsoft/Analysis-Services) på GitHub innehåller exempel på öppen källkod och communityprojekt. 

## <a name="tabular-model-project-and-database-samples"></a>Projekt- och databasexempel för tabellmodell

[Adventure Works for Analysis Services](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks-analysis-services) på GitHub är det vanligaste exempeltabellmodellprojektet. Du kan hämta ett VS-projekt eller en slutförd exempeltabellmodelldatabas.

## <a name="sample-database-on-azure"></a>Exempel på databas på Azure

Azure Synapse Analytics (SQL Data Warehouse) innehåller ett exempel på AdventureWorksDW-databas som kan inkluderas i en etablerad resurs. Mer information finns i [Snabbstart: Skapa och fråga en Azure Synapse Analytics SQL-pool](/azure/sql-data-warehouse/create-data-warehouse-portal).

## <a name="sample-databases-on-github"></a>Exempel på databaser på GitHub

Dessa exempeldatabaser på GitHub kan användas för att skapa och testa dina egna modeller. 

[Exempeldatabaser för Adventure Works](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks2014)

[Wide World Importers exempel databaser](https://github.com/Microsoft/sql-server-samples/releases/tag/wide-world-importers-v1.0)

## <a name="adventure-works-internet-sales-sample-model-in-azure-portal"></a>Exempelmodell för Adventure Works Internet Sales i Azure Portal

Om du har en Analysis Services-server i Azure-portalen kan du snabbt och enkelt skapa en exempelmodell. På serverns översiktssida i Azure-portalen klickar du på **Ny modell**och väljer sedan **Exempeldata**i **Välj en datakälla**.
