# <a name="azure-cdn-coverage-by-metro"></a>Azure CDN-täckning från Metro 
> [!div class="op_single_selector"]
> * [POP-platser efter region](cdn-pop-locations.md)
> * [POP-platser efter förkortning](cdn-pop-abbreviations.md)
> 


I den här artikeln visas aktuella metros som innehåller POP-platser (Point-of-presence), sorterade efter region, för CDN-produkter (Azure Content Delivery Network). Varje metro kan innehålla mer än en POP. Azure CDN från Microsoft har till exempel 130 POP-adresser i 82 tunnelbanor. 

> [!IMPORTANT]
> POP-ortplatser för **Azure CDN från Akamai** avslöjas inte individuellt.  
> 
> Eftersom varje Azure CDN-produkt har ett tydligt sätt att bygga sina CDN-infrastrukturer rekommenderar Microsoft att inte använda POP-platser för att bestämma vilken Azure CDN-produkt som ska användas. Tänk i stället på dess funktioner och slutanvändarens prestanda. Testa prestanda med varje Azure CDN-produkt för att välja rätt produkt för dina användare.
