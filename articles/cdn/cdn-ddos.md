# <a name="azure-cdn-ddos-protection"></a>Azure CDN DDoS-skydd

Ett innehållsleveransnätverk ger DDoS-skydd avsiktligt. Förutom den globala kapaciteten att absorbera volymetriska attacker har Azure CDN ytterligare DDoS-skydd enligt beskrivningen nedan, utan extra kostnad.

## <a name="azure-cdn-from-microsoft"></a>Azure CDN från Microsoft

Azure CDN från Microsoft skyddas av [Azure Basic DDoS](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview). Det är integrerat i Azure CDN från Microsoft-plattformen som standard och utan extra kostnad. Den fullständiga omfattningen och kapaciteten hos Azure CDN från Microsofts globalt distribuerade nätverk ger skydd mot vanliga nätverkslagerattacker genom trafikövervakning alltid och begränsning i realtid. Grundläggande DDoS-skydd skyddar också mot de vanligaste, ofta förekommande Layer 7 DNS Query Floods och Layer 3 och 4 volymetriska attacker som riktar sig till CDN-slutpunkter. Den här tjänsten har också en dokumenterad erfarenhet av att skydda Microsofts företagstjänster och konsumenttjänster från storskaliga attacker.

## <a name="azure-cdn-from-verizon"></a>Azure CDN från Verizon

Azure CDN från Verizon skyddas av Verzions egenutvecklade DDoS-begränsningsplattform. Det är integrerat i Azure CDN från Verizon som standard och utan extra kostnad. Det ger grundläggande skydd mot de vanligaste, ofta förekommande Layer 7 DNS Query Floods och Layer 3 och 4 volymetriska attacker som riktar sig till CDN-slutpunkter.

## <a name="azure-cdn-from-akamai"></a>Azure CDN från Akamai

Azure CDN från Akamai skyddas av Akamais egenutvecklade DDoS-begränsningsplattform. Det är integrerat i Azure CDN från Akamai som standard och utan extra kostnad. Det ger grundläggande skydd mot de vanligaste, ofta förekommande Layer 7 DNS Query Floods och Layer 3 och 4 volymetriska attacker som riktar sig till CDN-slutpunkter.
