# <a name="azure-cdn-ddos-protection"></a>Azure CDN DDoS Protection

Content delivery network ger DDoS protection avsiktligt. Utöver den globala kapaciteten att absorbera överför attacker, Azure CDN har ytterligare DDoS-skydd som beskrivs nedan, för utan extra kostnad.

## <a name="azure-cdn-from-microsoft"></a>Azure CDN från Microsoft

Azure CDN från Microsoft är skyddat av [Azure grundläggande DDoS](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview). Det är integrerad i Azure CDN från Microsoft-plattformen som standard och utan extra kostnad. Fullständig skalning och kapaciteten för Azure CDN från Microsofts globalt distribuerade nätverk ger skydd mot common layer nätverksattacker via ständig övervakning och i realtid problemlösning. Grundläggande DDoS-skydd även skyddar mot de vanligaste och mest ofta uppstår Layer 7 DNS-fråga översvämningar och nivå 3 och 4 överför attacker som mål CDN-slutpunkter. Den här tjänsten har också bevisade framgångar i skydda Microsoft enterprise och tjänster från storskaliga attacker.

## <a name="azure-cdn-from-verizon"></a>Azure CDN från Verizon

Azure CDN från Verizon skyddas av Verzions upphovsrättsskyddad DDoS minskning plattform. Det är integrerad i Azure CDN från Verizon som standard och utan extra kostnad. Det ger grundläggande skydd mot de vanligaste och mest ofta vill inträffar Layer 7 DNS-fråga översvämningar och nivå 3 och 4 överför attacker som mål CDN-slutpunkter.

## <a name="azure-cdn-from-akamai"></a>Azure CDN från Akamai

Azure CDN från Akamai skyddas av Akamais upphovsrättsskyddad DDoS minskning plattform. Det är integrerad i Azure CDN från Akamai som standard och utan extra kostnad. Det ger grundläggande skydd mot de vanligaste och mest ofta vill inträffar Layer 7 DNS-fråga översvämningar och nivå 3 och 4 överför attacker som mål CDN-slutpunkter.
