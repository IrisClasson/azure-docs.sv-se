# <a name="azure-data-box-edge-and-azure-data-box-gateway-1905-release-notes"></a>Azure Data Box Edge och Azure Data Box Gateway 1905 viktig information

## <a name="overview"></a>Översikt

Följande viktiga information identifiera kritiska öppna ärenden och löst problemen för 1905 övergång till Azure Data Box Edge och Azure Data Box-Gateway.

Den viktiga informationen uppdateras kontinuerligt och allteftersom allvarliga problem som kräver en lösning upptäcks, läggs de. Innan du distribuerar din Data Box Edge/Data Box-Gateway måste du noga igenom uppgifterna i viktig information.

Den här versionen motsvarar programvaruversionerna:

- **Data Box Gateway 1905 (1.6.887.626)**
- **Data Box Edge 1905 (1.6.887.626)**

> [!NOTE]
> Uppdatera 1905 kan tillämpas endast på Data Box Edge-enheter som kör GA-versionen av programvaran.

## <a name="whats-new"></a>Nyheter

- **Förbättringar för fältet Programmable Gate Array FPGA ()-loggning** – i den här versionen har vi gjort loggning och förbättringar av aviseringar som rör FPGA. Detta är en obligatorisk uppdatering för Data Box Edge om du använder funktionen Edge beräkning med FPGA. Läs mer om hur du [Transformera data med Edge-beräkning på din Data Box-Edge](data-box-edge-deploy-configure-compute-advanced.md).

## <a name="known-issues-in-ga-release"></a>Kända problem i GA-version

Inga nya problem är versionen som anges för den här versionen. Versionen anges problem har överförts från tidigare versioner. Om du vill se en lista över kända problem kan du gå till [kända problem i GA-version](data-box-gateway-release-notes.md#known-issues-in-ga-release).
