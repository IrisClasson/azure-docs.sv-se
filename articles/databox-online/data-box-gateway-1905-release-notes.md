# <a name="azure-data-box-edge-and-azure-data-box-gateway-1905-release-notes"></a>Versionsinformation om Azure Data Box Edge och Azure Data Box Gateway 1905

## <a name="overview"></a>Översikt

Följande versionsfakturor identifierar de kritiska öppna problemen och de lösta problemen för 1905-versionen för Azure Data Box Edge och Azure Data Box Gateway.

Viktig information uppdateras kontinuerligt och när kritiska problem som kräver en lösning upptäcks läggs de till. Innan du distribuerar databoxens edge/databoxgateway bör du noggrant granska informationen i viktig information.

Den här versionen motsvarar programvaruversionerna:

- **Data Box Gateway 1905 (1.6.887.626)**
- **Data Box Edge 1905 (1.6.887.626)**

> [!NOTE]
> Uppdatering 1905 kan endast tillämpas på Data Box Edge-enheter som kör GA-versionen av programvaran.

## <a name="whats-new"></a>Nyheter

- **Fpga-loggningsförbättringar (Field Programmemable Gate Array)** - I den här versionen har vi gjort loggnings- och varningsförbättringar relaterade till FPGA. Detta är en nödvändig uppdatering för Data Box Edge om du använder Edge-beräkningsfunktionen med FPGA. Mer information finns i hur du [omvandlar data med Edge-beräkning på din Data Box Edge](data-box-edge-deploy-configure-compute-advanced.md).

## <a name="known-issues-in-ga-release"></a>Kända problem i GA-utgåvan

Inga nya problem är kända för den här versionen. Alla release noterade frågor har fört över från tidigare utgåvor. Om du vill se en lista över kända problem går du till [Kända problem i GA-versionen](data-box-gateway-release-notes.md#known-issues-in-ga-release).
