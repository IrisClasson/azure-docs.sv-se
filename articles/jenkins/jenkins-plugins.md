# <a name="jenkins-plugins-for-azure"></a>Jenkins-plugin-program för Azure

Följande Jenkins plugin-program stöder olika funktioner för användning med Azure.
  
| Plugin-programmet Jenkins | Beskrivning                                   |
|------------------------------------------------------------------------------|
| [Azure App Service-pluginprogram](https://plugins.jenkins.io/azure-app-service)     | Jenkins-plugin-programmet att distribuera en Azure App Service (stöder för närvarande endast Web App). | 
| [Azure AD-plugin-programmet](https://plugins.jenkins.io/azure-ad)                       | Jenkins-plugin-programmet som har stöd för autentisering och auktorisering via Azure Active Directory. | 
| [Azure-artefakt Manager-pluginprogram](https://plugins.jenkins.io/azure-artifact-manager) | Plugin-programmet med Azure artefakt Manager är en artefakt-hanteraren kan du lagra dina artefakter till Azure Blob Storage. Plugin-programmet med Azure artefakt Manager fungerar transparent för Jenkins och dina jobb, det är som standard artefakt Manager. | 
| [Plugin-program för Azure Container Agents](https://plugins.jenkins.io/azure-container-agents) | Azure Container Agents-pluginprogram kan hjälpa dig att köra en behållare som en agent i Jenkins | 
| [Plugin-programmet med Azure Container Registry uppgift](https://plugins.jenkins.io/azure-container-registry-tasks)       | Jenkins-plugin-programmet att skicka en begäran om docker-build att [Azure Container Registry](/azure/container-registry/container-registry-tasks-overview). |
| [Plugin-program för Azure Container Service](https://plugins.jenkins.io/azure-acs)       | Plugin-programmet Jenkins distribuerar konfigurationer till Azure Container Service (AKS). | 
| [Plugin-program för Azure Credential](https://plugins.jenkins.io/azure-credentials)      | Jenkins-plugin-programmet att hantera Azure-autentiseringsuppgifter. | 
| [Plugin-program för Azure Function](https://plugins.jenkins.io/azure-function)           | Jenkins-plugin-programmet att distribuera en Azure-funktion. | 
| [Azure IoT Edge plugin](https://plugins.jenkins.io/azure-iot-edge)           | Plugin-programmet med Azure IoT Edge gör det enkelt att konfigurera en CI/CD-pipeline för att utveckla en IoT Edge-lösning på Jenkins. | 
| [Azure Service Fabric plugin](https://plugins.jenkins.io/service-fabric)     | Plugin-programmet Jenkins för Linux Azure Service Fabric-projekt. |
| [Plugin-program för Azure Storage](https://plugins.jenkins.io/windows-azure-storage)     | plugin-programmet för att ladda upp skapa artefakter som eller ladda ned skapa beroenden från Microsoft Azure Blob storage. | 
| [Azure VM agents-pluginprogram](https://plugins.jenkins.io/azure-vm-agents)         | Jenkins plugin-programmet för att skapa Jenkins-agenter på virtuella Azure-datorer (via Azure Resource Manager-mall). | 
| [Plugin-programmet för Azure VM-skalningsuppsättningen](https://plugins.jenkins.io/azure-vmss)           | Jenkins-plugin-programmet att distribuera avbildningar av Virtuella datorer till Azure-datorer anger. |
