# <a name="jenkins-plugins-for-azure"></a>Jenkins plugins för Azure

Följande Jenkins-plugins stöder olika funktioner för användning med Azure.
  
| Jenkins plugin | Beskrivning                                   |
|------------------------------------------------------------------------------|
| [Plugin-program för Azure App Service](https://plugins.jenkins.io/azure-app-service)     | Jenkins plugin för att distribuera en Azure App Service (för närvarande stöder endast Web App). | 
| [Plugin-program för Azure AD](https://plugins.jenkins.io/azure-ad)                       | Jenkins plugin som stöder autentisering & auktorisering via Azure Active Directory. | 
| [Plugin-program för Azure Artifact Manager](https://plugins.jenkins.io/azure-artifact-manager) | Azure Artifact Manager plugin är en Artifact Manager som låter dig lagra dina artefakter i Azure Blob Storage. Azure Artifact Manager plugin fungerar transparent till Jenkins och dina jobb, är det som standard Artifact Manager. | 
| [Plugin-program för Azure Container Agents](https://plugins.jenkins.io/azure-container-agents) | Azure Container Agents plugin kan hjälpa dig att köra en behållare som en agent i Jenkins | 
| [Plugin plugin-program för Azure-behållarregisteruppgift](https://plugins.jenkins.io/azure-container-registry-tasks)       | Jenkins plugin för att skicka en begäran om docker-build till [Azure Container Registry](/azure/container-registry/container-registry-tasks-overview). |
| [Azure Container Service-pluginprogram](https://plugins.jenkins.io/azure-acs)       | Jenkins plugin för att distribuera konfigurationer till Azure Container Service (AKS). | 
| [Plugin-program för Azure Credential](https://plugins.jenkins.io/azure-credentials)      | Jenkins plugin för att hantera Azure-autentiseringsuppgifter. | 
| [Plugin-program för Azure Function](https://plugins.jenkins.io/azure-function)           | Jenkins plugin för att distribuera en Azure-funktion. | 
| [Azure IoT Edge-plugin](https://plugins.jenkins.io/azure-iot-edge)           | Azure IoT Edge plugin gör det enkelt att konfigurera en CI / CD pipeline för att utveckla en IoT Edge-lösning på Jenkins. | 
| [Plugin-program för Azure Service Fabric](https://plugins.jenkins.io/service-fabric)     | Jenkins plugin för Linux Azure Service Fabric-projekt. |
| [Plugin-program för Azure Storage](https://plugins.jenkins.io/windows-azure-storage)     | plugin för att ladda upp byggartefakter till eller hämta byggberoenden från Microsoft Azure Blob-lagring. | 
| [Plugin-program för Azure VM-agenter](https://plugins.jenkins.io/azure-vm-agents)         | Jenkins plugin för att skapa Jenkins-agenter i virtuella Azure-datorer (via Azure Resource Manager-mall). | 
| [Plugin-insticksprogrammet för Azure-skalningsuppsättning för virtuella datorer](https://plugins.jenkins.io/azure-vmss)           | Jenkins plugin för att distribuera VM-avbildningar till Azure virtuella dator skalningsuppsättningar. |
