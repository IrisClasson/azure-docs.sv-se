# <a name="jenkins-plugins-for-azure"></a>Jenkins-plugin-program för Azure

Följande Jenkins-plugin-program stöder olika funktioner för användning med Azure.
  
| Jenkins-plugin | Beskrivning                                   |
|------------------------------------------------------------------------------|
| [Azure App Service-plugin](https://plugins.jenkins.io/azure-app-service)     | Jenkins-plugin-programmet för att distribuera en Azure App Service (stöds för närvarande endast webbappar). | 
| [Azure AD-plugin](https://plugins.jenkins.io/azure-ad)                       | Jenkins-plugin-programmet som stöder autentisering & auktorisering via Azure Active Directory. | 
| [Plugin-program för Azure artefakt Manager](https://plugins.jenkins.io/azure-artifact-manager) | Plugin-programmet Azure artefakt Manager är en artefakt hanterare som gör att du kan lagra artefakter i Azure Blob Storage. Plugin-programmet Azure artefakt Manager fungerar transparent för Jenkins och dina jobb, men är som standard artefakt hanterare. | 
| [Plugin-program för Azure Container Agents](https://plugins.jenkins.io/azure-container-agents) | Plugin-programmet Azure Container agents kan hjälpa dig att köra en behållare som en agent i Jenkins | 
| [Azure Container Registry aktivitets-plugin](https://plugins.jenkins.io/azure-container-registry-tasks)       | Jenkins-plugin-programmet för att skicka en Docker-build-begäran till [Azure Container Registry](/azure/container-registry/container-registry-tasks-overview). |
| [Plugin-program för Azure Container Service](https://plugins.jenkins.io/azure-acs)       | Jenkins-plugin-programmet för att distribuera konfigurationer till Azure Container Service (AKS). | 
| [Plugin-program för Azure Credential](https://plugins.jenkins.io/azure-credentials)      | Jenkins-plugin-programmet för att hantera Azure-autentiseringsuppgifter. | 
| [Plugin-program för Azure Function](https://plugins.jenkins.io/azure-function)           | Jenkins-plugin-programmet för att distribuera en Azure-funktion. | 
| [Azure IoT Edge-plugin](https://plugins.jenkins.io/azure-iot-edge)           | Med Azure IoT Edge plugin-programmet kan du enkelt konfigurera en CI/CD-pipeline för att utveckla en IoT Edge-lösning på Jenkins. | 
| [Azure Service Fabric-plugin-program](https://plugins.jenkins.io/service-fabric)     | Jenkins-plugin-program för Linux Azure Service Fabric-projekt. |
| [Plugin-program för Azure Storage](https://plugins.jenkins.io/windows-azure-storage)     | plugin-program för att ladda upp Bygg artefakter till eller hämta build-beroenden från Microsoft Azure Blob Storage. | 
| [Plugin-program för Azure VM-agenter](https://plugins.jenkins.io/azure-vm-agents)         | Jenkins-plugin-programmet för att skapa Jenkins-agenter i Azure Virtual Machines (via Azure Resource Manager-mall). | 
| [Plugin-program för skalnings uppsättning för virtuella Azure-datorer](https://plugins.jenkins.io/azure-vmss)           | Jenkins-plugin för att distribuera VM-avbildningar till skalnings uppsättningar för virtuella Azure-datorer. |
