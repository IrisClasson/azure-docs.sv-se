# <a name="jenkins-plugins-for-azure"></a>Jenkins-plugin-program för Azure

Följande Jenkins plugin-program stöder olika funktioner för användning med Azure.
  
| Plugin-programmet Jenkins | Beskrivning                                   |
|------------------------------------------------------------------------------|
| [Azure App Service-pluginprogram](https://plugins.jenkins.io/azure-app-service)     | Ett plugin-program med Jenkins för att distribuera en Azure App Service (stöder för närvarande endast Web App). | 
| [Azure AD-plugin-programmet](https://plugins.jenkins.io/azure-ad)                       | En Jenkins Plugin som har stöd för autentisering och auktorisering via Azure Active Directory. | 
| [Plugin-program för Azure Container Agents](https://plugins.jenkins.io/azure-container-agents) | Azure Container Agents-pluginprogram kan hjälpa dig att köra en behållare som en agent i Jenkins | 
| [Plugin-program för Azure Container Service](https://plugins.jenkins.io/azure-acs)       | En Jenkins-Plugin för att distribuera konfigurationer till Azure Container Service (AKS). | 
| [Plugin-program för Azure Credential](https://plugins.jenkins.io/azure-credentials)      | Jenkins-plugin-programmet att hantera Azure-autentiseringsuppgifter. | 
| [Plugin-program för Azure Function](https://plugins.jenkins.io/azure-function)           | Om du vill använda det här plugin-programmet för att distribuera till Azure-funktion, måste du först har en Azure-tjänstens huvudnamn i Jenkins-instansen. | 
| [Azure Service Fabric plugin](https://plugins.jenkins.io/service-fabric)     | En Jenkins Plugin för Linux Azure Service Fabric-projekt. |
| [Plugin-program för Azure Storage](https://plugins.jenkins.io/windows-azure-storage)     | Ett plugin-program för att ladda upp skapa artefakter som eller ladda ned skapa beroenden från Microsoft Azure Blob storage. | 
| [Azure VM agents-pluginprogram](https://plugins.jenkins.io/azure-vm-agents)         | En Jenkins-Plugin för att skapa Jenkins-agenter i Azure virtual machines (via Azure Resource Manager-mall). | 
| [Plugin-programmet för Azure VM-skalningsuppsättningen](https://plugins.jenkins.io/azure-vmss)           | Anger en plugin-programmet med Jenkins för att distribuera avbildningar av Virtuella datorer till Azure-datorer. |
