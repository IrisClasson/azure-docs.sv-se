# <a name="azure-data-catalog-developer-samples"></a>Exempel på Utvecklare av Azure Data Catalog

Kom igång med att utveckla Azure Data Catalog-appar med rest-API:et för datakatalogen. REST API för datakatalog är ett REST-baserat API som ger programmatisk åtkomst till datakatalogresurser för att registrera, kommentera och söka datatillgångar programmässigt.

## <a name="samples-available-on-githubcom"></a>Prover som finns på GitHub.com

* [Kom igång med Azure Data Catalog](https://github.com/Azure-Samples/data-catalog-dotnet-get-started/)
  
   Med exemplet kom igång visas hur du autentiserar med Azure AD för att registrera, söka och ta bort en datatillgång med rest-API:et för datakatalog.
   
* [Komma igång med Azure Data Catalog med tjänsthuvudnamn](https://github.com/Azure-Samples/data-catalog-dotnet-service-principal-get-started/)

   Det här exemplet visar hur du registrerar, söker efter och tar bort en datatillgång med hjälp av REST-API:et för datakatalogen. I det här exemplet används autentiseringen av tjänsthuvudnamn.

* [Import/exportverktyg för Azure Data Catalog](https://github.com/Azure-Samples/data-catalog-dotnet-import-export/)

   Det här exemplet som visar hur du använder REST-API:et för datakatalog för att hämta resurser från Azure Data Catalog och serialisera dem till en fil. Det visar också hur man tar en uppsättning tillgångar serialiserade som JSON och driva dem in i katalogen. Den stöder export av en delmängd av katalogen med hjälp av en sökfråga.

* [Massregister och kommentera i Azure Data Catalog](https://github.com/Azure-Samples/data-catalog-dotnet-excel-register-data-assets/)
  
   Det här exemplet som visar hur du massregisterar datatillgångar från en Excel-arbetsbok med REST-API för datakatalog och Open XML.
  
* [Ordlistetermer för massimport till Azure Data Catalog](https://github.com/Azure-Samples/data-catalog-bulk-import-glossary/)

   Det här exemplet visar hur du importerar ordlistor från CSV-filer till ADC-ordlista.

* [Massimportrelationer till Azure Data Catalog](https://github.com/Azure-Samples/data-catalog-bulk-import-relationship/)

   Det här exemplet visar hur du programmässigt importerar relationsinformation från en CSV-fil till en datakatalog.

* [Publicera relationer i Azure Data Catalog](https://github.com/Azure-Samples/data-catalog-dotnet-publish-relationships/)

   Det här exemplet visar hur du programmässigt kan publicera relationsinformation i en datakatalog.
